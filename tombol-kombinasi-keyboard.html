14. mengaktifkan menu print dengan mengunakan tombol kombinasi keyboard adalah dengan menekan tombol


Mengaktifkan menu print dengan menggunakan tombol kombinasi keyboard adalah dengan menekan tombol CTRL dan P secara bersamaan. Langkah ini disebut menggunakan "shortcut" di samping menggunakan menu biasa yang ada pada layar.

Pembahasan

Untuk melakukan hal tersebut cukup dengan menggunakan shortcut CTRL dan P atau tombol pintasan di Word. Saat kita menekan Ctrl+F2 untuk menampilkan pengaturan Cetak, kita kemudian dapat menekan Alt+P untuk menampilkan beberapa bantuan di layar lalu Alt+V, yang mengaktifkan area Print Preview (Pratinjau Cetak). Pengaturan lainnya adalah:

1. Di Word 2010 dan versi yang lebih baru, tampilkan tab File di pita, lalu klik Opsi.

2. Pilih opsi Sesuaikan (Word 2007) atau opsi Bilah Alat Akses Cepat (Word 2010 dan versi yang lebih baru) di sisi kiri kotak dialog.

3. Menggunakan Pilih Perintah Dari daftar drop-down, pilih File Tab. Cara ini menampilkan semua perintah yang ada di tab File di ribbon (pita).

4. Gulir daftar perintah yang tersedia dan pilih Mode Edit Pratinjau Cetak.

5. Klik tombol Tambah. Perintah dipindahkan ke daftar di sebelah kanan kotak dialog.

6. Klik Oke. Perintah baru sekarang muncul di Quick Access Toolbar (Bilah Alat Akses Cepat).
