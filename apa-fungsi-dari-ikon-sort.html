1. Fungsi ikon sort ascending dan sort descending


apa itu sort ? sort merupakan metode dimana data yang di simpan dalam array di urutkan bedasarkan kebutuhan tertentu

apa ascending ? ascending merupakan behavior yang ada pada metode looping pada bahasa pemrograman tertentu (++)

apa desceding ? sama seperti ascending, descending merupakan kebalikan dari ascending (--)

jadi kesimpulannya sort ascending adalah mengurutkan data yang diurutkan kedepan, dan sort descending adalah mengurutkan data yang diurutkan kebelakang
ascending :
mengurutkan data dari
smallest to largest
descending: mengurutkan data dr
largest to smalle

36. ikon yang berfungsi untuk mempercepat akses terhadap suatu file aplikasi disebut....a. pop-up menub. shortcutc. sort byd. file manager​


Jawaban:

Ikonn yang berfungsi untuk mempercepat akses terhadap suatu file aplikasi disebut "shortcut". Shortcut adalah ikon yang disediakan oleh sistem operasi atau alat pengurus file untuk mempercepat akses terhadap suatu file atau aplikasi. Ikon ini biasanya ditempatkan di desktop atau menu start pada sistem operasi Windows.

Dalam sistem operasi Mac, ikon untuk akses cepat ke aplikasi atau file disebut " Launchpad". Sedangkan dalam Linux, ikon untuk akses cepat ke aplikasi atau file disebut "Application Launcher" atau "Start Menu".

Pop-up menu adalah menu yang muncul secara otomatis ketika pengguna mengklik suatu ikon atau tombol tertentu, sedangkan "sort by" adalah fitur untuk menyortir file atau aplikasi dengan berdasarkan beberapa kriteria seperti nama, tanggal, ukuran, atau tipe file. File manager adalah aplikasi yang secara spesifik menyediakan pengelolaan file atau aplikasi di dalam suatu system operasi, seperti Windows Explorer dalam sistem operasi Windows atau Finder dalam sistem operasi Mac.
