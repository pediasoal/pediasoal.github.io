6. jenis media audio yang dapat ditambahkan pada electronic publication (epub) untuk buku elektronik adalah ....


Jenis media audio yang dapat ditambahkan pada electronic publication (epub) untuk buku elektronik adalah semua media yang bisa difungsikan untuk mendengarkan format file audio Mp3, Wav, dan Mpg seperti Earphone dan Headphone.

Pembahasan:

Epub adalah buku dengan format file standar untuk buku digital. Format epub sendiri awalnya dikembangkan oleh International Digital Publishing Forum (IDPF).

Epub merupakan salah satu format eBook yang dapat dibuka di perangkat elektronik seperti komputer, tablet, smartphone bahkan ponsel. Buku digital format epub (*.epub) dibuat menggunakan bahasa HTML dan xHTML terbuka dan dapat digunakan di perangkat apa pun.

Faktanya, e-book dalam format epub hampir sama dengan pdf, keduanya merupakan format e-book ringan yang mendukung audio dan video.

Fungsi Menu dan Icon Microsoft Publisher

Fungsi Menu dan Icon Microsoft Publisher

1. CONTENTS… Standard Toolbar Object Toolbar Menu Bar Formatting Toolbar

2. Menu bar: barisan menu yang menampilkan perintah- perintah yang digunakan untuk mengatur dan mengolah objek. Terdiri dari File, Edit, View, Insert, Format, Tools, Arrange, Window, Help, dan Adobe PDF

3.  File: berisi perintah-perintah dasar, seperti New, Open, Close, Import New Document, Save, Save As, Pubish to Web, Page Review, Pack and Go, Convert, Page Setup, Print Setup, Print Preview, Print, Exit.  Edit: berisi perintah-perintah untuk mengedit web page, gambar, dan objek lainnya. Seperti: Undo, Redo, Cut, Copy, Paste, Find, Replace, Office Clipboard, Delete Page, Delete Object, Delete Text, Select All, Go to Page, Links, Edit Story in Microsoft Word, Objects, dan Bussiness Information.  View: berisi perintah-perintah untuk melihat hasil web page atau hasil edit. View berisi submenu yaitu: Master Page, Ignore Master Page, Taskpane, Toolbars, Rulers, Special Characters, Boundaries and Guidelines, Zoom and Picture.

4.  Page: membuat page baru  Duplicate Page: membuat halaman baru yang sama persis  Symbol: Memasukkan Symbol  Design Gallery Object: Memasukkan Objek dari Design Gallery  Add to Content Library: Memasukkan item ke Content Library  Item from Content Library: Menunjukkan item-item yang tersimpan di Content Library  Navigation Bar: Memasukkan Navigation Bar  Form Control: Memasukkan Textbox, text area, checkbox, option button, list box, submit.  HTML Code Fragment: Memasukkan kode HTML ke dokumen  Business Information: Memasukkan informasi organisasi pembuat dokumen  Picture: memasukkan gambar ke dokumen  Text box: membuat text box  Text file: Memasukkan teks dari file  Object: Memasukkan Object ke dokumen  Hyperlink: Memasukkan Hyperlink ke dokumen  Bookmark: memasukkan bookmark/ penanda Insert

5.  Font: Mengatur ukuran, tipe, warna, efek dari font.  Character Spacing: Mengatur spasi di antara 2 karaktter  Paragraph: Mengatur paragraf dalam dokumen  Bullets and numbering: Memasukkan Bullets & Numbering  Horizontal Rules: Pengatur penggaris horizontal  Tabs: membuat tabs  Drop Cap; menentukan style huruf besar di awal paragraph  AutoFit Text: Mengatur Text agar pas dan rapi  Background: Memasukkan background baik dari file maupun yang ditawarkan Publisher  Format Publication: Memformat page, color schemes, font schemes, web site options  Page options: Memformat page  Color Schemes: memformat color schemes  Font Schemes: memformat font schemes  Styles: memilih tipe teks  Apply master Page: Memasukkan Master Page  Object: memformat objek Format
