17. cara membuat template pada microsoft word?​


Jawaban Dengan Langkah²

Dokumen perusahaan tidak melulu berisikan angka-angka dan tulisan. Ada kalanya gambar atau graphis pun perlu ada untuk menggambarkan besaran penjualan misalnya. Nah untuk memudahkan biasanya bagian administrasi sudah memiliki template sendiri bila ingin menyertakan gambar ke dalam laporan perusahaan.

Template Microsoft Word adalah dokumen Word yang menyertakan pengaturan khusus seperti teks boilerplate, macros, header dan footer, kamus khusus, toolbar, dan entri AutoText. Elemen-elemen yang sama akan selalu ada setiap kali Anda membuka template dan tidak dapat diubah. Dengan adanya template Anda juga sudah memotong durasi pengaturan file menjadi lebih cepat karena sudah ada template standar yang hanya perlu update tanggal maupun angka-angka nominal di dalamnya.

Merencanakan Template Word

Sebelum Anda membuat template Word Anda, buat daftar fitur yang ingin Anda sertakan di dalamnya seperti contoh di bawah ini:

Teks yang akan selalu disertakan dalam dokumen

Memformat kolom, margin, tab berhenti, catatan akhir, dan catatan kaki

Makro untuk mengotomatisasi tugas.

Bidang tanggal yang diperbarui secara otomatis setiap kali templat dibuka.

Alamat dan informasi kontak

Fields atau AutoText untuk informasi yang berubah seperti nomor halaman, judul dokumen, atau path file di header dan footer.

Teks placeholder dengan pemformatan tertentu seperti ukuran jenis huruf.

Setelah Anda menyusun semua elemen yang akan disertakan ke dalam template, buatlah prototipe dalam dokumen Word kosong dan lakukan penyesuaian bila dibutuhkan, dan jangan lupa simpan prototipenya.

Membuat template di Word 2019, 2016, 2013, DAN 2010

1. Buka File

2. Pilih Save As

3. Di kotak dialog Save As, di kotak teks File Name, ketik nama template

4. Kotak dialog Microsoft Word Save As menyimpan templat kustom

5. Jalur file berubah ke lokasi templat default. Templat di folder ini muncul di kotak dialog saat Anda membuat dokumen baru dari templat. Namun, Anda dapat memilih folder lain jika Anda mau

6. Pilih Save.

Membuat template di Word 2007

Anda bisa membuat template dengan cara berikut:

1. Pilih tombol Microsoft Office

2. Pilih Save As > Word Template

3. Di kotak dialog Save As, untuk menyimpan templat di folder templat default, di bawah Favorite Links pilih Template. Ini adalah folder yang disarankan karena templat di folder ini muncul di kotak dialog Template ketika Anda membuat dokumen baru berdasarkan templat. Namun, Anda juga dapat menavigasinya ke folder lain jika Anda mau.

4. Ketikkan nama deskriptif untuk template, lalu Save.

Membuat template di Word 2003

1. Pilih File > Save As

2. Ketikkan nama yang dapat dikenali untuk templat lalu Save as type dan pilih Document Template

3. Pilih Save.

Dokumen Anda sekarang disimpan sebagai templat dengan ekstensi file .dot atau .dotx dan dapat digunakan untuk membuat dokumen baru. Praktis bukan? Kini Anda tinggal mengganti konten secara reguler sesuai template yang sudah ada. Sama praktisnya ketika saat Anda membutuhkan dokumen tersebut dicetak kemudian Anda menggunakan jasa PrinterQoe yang merupakan solusi cetak dokumen online di seluruh wilayah Indonesia.

28. Buatlah rangkuman/resume tentang cara membuat grafik di microsoft word​


Penjelasan:

Microsoft office word adalah sebuah program yang dirancang untuk membantu kerja manusia agar lebih efektif. Pada saat bekerja kita pasti pernah mempunyai tugas untuk membuat sebuah grafik dari sebuah penelitian tertentu, misalkan grafik penjualan, grafik jumlah penduduk maupun grafik yang lain.

Grafik atau chart pada ms word berfungsi untuk menampilkan diagram dari sebuah data, sehingga kita bisa membaca data dengan melihat grafik tersebut. Untuk membuat sebuah grafik pada microsoft word, ikuti langkah-langkah berikut:

1.Buka lembar kerja ms word anda

2.Klik insert pada menubar,kemudian pada group illustrations pilih chart.

3.Selanjutnya akan muncul jendela insert chart. Pada jendela ini kita bisa memilih char atau grafik sesuai dengan tampilan atau bentuk yang kita inginkan. Misalkan kita ingin chart berupa garis, lingkaran, kolom dan lain-lain.

4.Selanjutnya pilih chart sesuai dengan keinginan anda. Anda bisa memilih template chart pda sidebar sebelah kiri, kemudian pilih model chart pada kolom bagian kanan.

5.Selanjutnya klik OK.

6.Setelah kita klik OK maka akan muncul chart pada lembar kerja microsoft word kita, bersamaan dengan itu juga muncul lembar kerja baru yaitu icrosoft excel.

7.untuk melakukan perubahan data, edit data pada lembar kerja microsoft excel.

Untuk memasukkan judul dan label lakukanlah langkah berikut.

Dari judul toolbar klik chart, pilih chart option, maka akan muncul beberapa pilihan menu

Klik chart title untuk memasukkan judul grafik

Klik category (X) axisi yaitu untuk nama di sumbu X

Klik category (Y) axis yaitu untuk nama di sumbu Y

Kemudian tekan esc, maka grafik yang kita inginkan sudah selesai dibuat.

Untuk mengganti type grafik ikuti langkah berikut:

Klik dua kali pada area grafik

Berarti grafik dalam kondisi aktif dan siap untuk diadakan perubahan

Klik chart pada toolbar, maka akan muncul pilihan jenis atau type grafik

Pilihlah grafik yang diinginkan

Tekan OK dan lihat hasilnya.

1. Fungsi Utama Microsoft Office Word:

Microsoft Word adalah program pengolah kata yang digunakan untuk membuat, mengedit, dan memformat dokumen teks. Fungsi utamanya adalah untuk membuat dokumen seperti surat, laporan, esai, dan banyak jenis dokumen lainnya.

2. Cara Membuat Teks Menjadi Tebal:

Anda dapat membuat teks menjadi tebal dengan cara:

Memilih teks yang ingin diubah.

Kemudian, di toolbar di atas, Anda akan menemukan ikon "Tebal" (T) yang bisa Anda klik atau menggunakan pintasan keyboard Ctrl + B.

3. Perbedaan antara Cut, Copy, dan Paste dalam Word:

Cut: Memindahkan teks atau objek dari tempat asalnya dan menempatkannya di tempat lain. Menghapus teks/objek dari sumber aslinya.

Copy: Menggandakan teks atau objek ke clipboard sehingga Anda dapat menempatkannya di tempat lain tanpa menghapus sumber aslinya.

Paste: Menempatkan teks atau objek yang telah Anda cut atau copy ke lokasi tertentu dalam dokumen.

4. Cara Menyisipkan Gambar ke dalam Dokumen Word:

Klik tempat di dokumen Anda di mana Anda ingin menyisipkan gambar.

Pilih "Insert" di toolbar, lalu pilih "Picture" untuk memilih dan menyisipkan gambar dari komputer Anda.

5. Pengaturan Margin dalam Word:

Margin adalah jarak antara teks dan batas dokumen. Anda dapat mengubahnya dengan masuk ke tab "Page Layout" dan memilih "Margins". Anda dapat memilih margin yang ada atau membuat margin kustom.

6. Cara Membuat Daftar Numerik:

Klik ikon "Numbering" di toolbar untuk mengaktifkan daftar numerik, atau Anda bisa menggunakan pintasan Ctrl + Shift + L.

7. Langkah-langkah Menyimpan Dokumen Baru:

Klik "File" > "Save As".

Pilih lokasi penyimpanan dan beri nama dokumen.

Pilih format file yang diinginkan (misalnya, .docx) dan klik "Save".

8. Membuat Halaman dengan Orientasi Potret dan Lanskap dalam Satu Dokumen:

Anda bisa memasukkan section breaks untuk mengubah orientasi halaman. Pergi ke "Page Layout" > "Breaks" > pilih "Next Page" di bawah "Section Breaks". Kemudian, pada halaman berikutnya, Anda dapat mengubah orientasi halaman dengan "Page Layout" > "Orientation".

9. Cara Membuka File Dokumen Baru:

Klik "File" > "New" atau gunakan pintasan Ctrl + N.

10. Cara Menyimpan Dokumen Word dalam Format PDF:

Klik "File" > "Save As".

Pilih lokasi dan nama file.

Pilih format "PDF (*.pdf)" dalam opsi "Save as type".

Klik "Save".

32. cara membuat kolom koran di MS Word​


1. Buka aplikasi Ms. Word dan bukalah dokumen yang akan dibuat menjadi kolom koran, atau ketiklah teks yang diinginkan.

2. Kemudian seleksi atau blok teks yang sudah diketik, yang akan dibuat kolom koran.

3. Selanjutnya klik “Menu Page Layout” dan pilih “Colums” (di sini kita dapat memilih jumlah kolomnya, baik itu 1 kolom, 2 kolom, 3 kolom, atau 2 kolom dominan kanan atau kiri).

4. Kita juga bisa memilih “More Columns” untuk mengatur sendiri jumlah kolom yang diinginkan.

5. Jika kita memilih “More Columns” maka akan muncul kotak dialog kolom. Pada bagian Presets (untuk menentukan model kolom yang diinginkan), Number of Columns (untuk menentukan jumlah kolom yang diinginkan), Width and Spacing (untuk mengatur lebar dan jarak kolom).

6. Terakhir klik “OK”, maka akan jadi seperti pada gambar. Demikianlah langkah-langkah membuat kolom koran pada Ms. Word.
