1. prosedur backup dan restore itu apa ?


backup = aktifitas untuk membuat salinan berkas utama agar suatu waktu dibutuhkan jika berkas utama tidak ada atau sedang dipakai maka bisa menggunakan salinan identik.

restore = mengembalikan file backup kedalam bentuk dan format file aslinya baik melalui proses decompress maupun tidak sama sekali.

2. Jelaskan apa fungsi backup, restore, dan recovery dalam pengaplikasiannya terhadap sistem operasi


Fungsi backup, restore, dan recovery dalam pengaplikasiannya terhadap sistem operasi adalah sebagai berikut yaitu:

Backup = adalah melakukan penyalinan data untuk mengamankan dari kehilangan data melalui penyimpanan fisik.Restore = artinya mengembalikan data yang telah hilang.Recovery = artinya adalah memulihkan sistem yang telah rusak pada sistem operasi.Penjelasan

Recovery itu hampir sama dengan backup dan restore. Ketiga kegiatan ini saling berkaitan, namun sebenarnya memiliki pengertian yang berbeda. Pencadangan data adalah tindakan pencegahan jika terjadi kerusakan atau kehilangan data.

6. 1) Apa itu Control Panel Hosting beserta fungsinya? 2) Bagaimana cara kerja Control Panel Hosting? 3) Apa itu EHCP? Mohon bantuanya Min :))


Hosting atau yang biasa disebut juga dengan web hosting adalah penyewaan server yang digunakan sebagai tempat penyimpanan data - data yang  sehingga data tersebut dapat diakses melalui internet. Data tersebut dapat berupa teks, gambar, video atau bahkan program yang berisi ke semua hal tersebut yang bisa kita sebut dengan website.

Pembahasan


Control Panel Hosting

adalah alat yang digunakan untuk mengatur segala macam pengaturan yang ada pada hosting. Beberapa fungsi control panel hosting antara lain

mengeloal email akun (menambah, menghapus, ataupun mengubah)mengupload file ke server yang berada pada hostinginstalasi software pada server memasang password pada sebuah directory yang ada di servermelihat statistik web (jika hosting digunakan untuk keperluan website) dan masih banyak lagi.

Control Panel Hosting sekarang sudah memilki tampilan user interface yang memudahlkan pengguna untuk mengubah pengaturan server dengan beberapa klik saja. Berbeda dengan sebelumya semua pengaturan akan memerlukan waktu yang banyak. Dengan adanya graphical user inferface di control panel hosting, pengguna cukup melakukan klik pada menu yang diinginkan dan mengganti setting sesuai dengan kebutuhan.

EHCP (Easy Hosting Control Panel)

adalah salah satu contoh control panel hosting yang open source dan dibuat menggunakan bahasa pemrograman PHP. EHCP sendiri tersedia secara gratis, berbeda dengan beberapa control panel hosting terkenal lain seperti cPanel dan Plesk yang berbayar. Oleh karena itu untuk menggunakan cPanel atau Plesk kita perlu mecari penyedia layanan hosting yang sudah menyediakan cPanel, sehingga penyedia layananlah yang akan membayar untuk cPanel. EHCP sendiri bisa digunakan pada server yang menggunakan linux sebagai sistem operasinya.

17. apa fungsi Local Security Policy,services,pada menu control panel?


Apa fungsi Local Security Policy ,Services pada menu control panel ?Jawaban Pendahuluan

Fungsi dari Local Security Policy adalah untuk memberikan aturan atau batasan-batasan untuk pengguna yang berada di grup yang telah dibuat.

Untuk services sendiri pada control panel adalah untuk mengatur services-services yang berjalan pada sistem operasi windows.

Pada microsoft windows, kita bisa melakukan berbagai macam pengaturan untuk sistem kita. Untuk melakukan pengaturan tersebut kita bisa melalui menu control panel. Sedangkan pada contoh pengaturan services adalah menonaktifkan services firewall dan masih banyak services lainnya.

28. Sebutkan contoh software utilitas untuk backup dan restore


Contoh software utilitas untuk backup dan restore antara lain:

DFINC Backup homeParagon Backup and RecoveryCobian BackupAOMEIEASUS Todo Backup

Pembahasan

Software utilitas untuk backup dan restore selalu digunakan sebagai software yang berhubungan dengan keperluan primer suatu komputer untuk mencadangkan data-data didalamnya dan mengembalikan seperti semula jika terjadi kondisi tertentu.

Beberapa software utilitas memberikan layana  gratis pada softwarenya seperti EASUS Todo, Paragon Backup and Recovery,  dan DFINC Backup home. Meskipun gratis, software-software ini sangat mumpuni untuk melakukan backup dan recovery secara profesional untuk komputer rumahan. Seringkali jika digunakan untuk keperluan kantor dan lebih besar maka diperlukan software yang dapat mempermudah prosedur backup and recovery  serta memiliki lisensi seperti Cobian Backup dan AOMEI. Adanya fitur-fitur tambahan dari software utilitas ini seperti adanya scheduler, kemampuan multithread, one-click dan cenderung lebih dinamis.  Sehingga penggunaan software utlitas untuk backup dan restore, lebih disesuaikan dengan kebutuhan penggunanya karena masing-masing software memiliki kelebihan dan kelemahannya. Umumnya pada sistem operasi seperti windows, mereka akan menyediakan software utilitas untuk backup dan restore versi built-in.
