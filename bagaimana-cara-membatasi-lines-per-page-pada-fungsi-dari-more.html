19. tolong jawab secepatnya- Kegunaan Margins - Cara costum margins - cara print dokumen - fungsi ikon font size - fungsi ikon color - fungsi dari ikon page border - cara membuat page border - fungsi ikon insert picture - Perbedaan insert foto dari explorer dan internet - fungsi dari ikon coloms di page layout - kegunaan explorer - cara membuat folder.


Jawaban:

Kegunaan Margins= Fungsi margin pada microsoft word adalah untuk mengatur jarak antara tepi pengetikan dengan tepi halaman kertas.

Cara costum margins= 1. Pada tab Page Layout, grup Page Setup, klik Margins dan kemudian pilih Custom Margins.

2. Pada kotak dialog Page Setup, tab Margins, isi margin untuk kiri (Left), kanan (Right), atas (Top), dan bawah (Bottom).

3. Klik OK untuk menutup kotak dialog.

Cara print dokumen= 1. Di komputer, buka Chrome.

2. Buka halaman, gambar, atau file yang ingin dicetak.

3. Klik File. Cetak. Atau, gunakan pintasan keyboard: Windows & Linux: Ctrl + p. Mac: ⌘ + p.

4. Di jendela yang muncul, pilih tujuan dan ubah setelan cetak pilihan Anda.

Klik Cetak.

Fungsi ikon font size= Font Size, berfungsi untuk menggantu ukuran font sesuai keperluan pengguna. Ukuran font tersedia dalam rentang skala 8 - 72. Font Color, berfungsi untuk mengganti warna font sesuai keperluan pengguna.

Fungsi ikon color= Pen color berfungsi Untuk memberikan warna pada gambar.Atau memberikan warna pada karakter.

Fungsi dari ikon page border= Page Borders digunakan apabila isi dari suatu halaman ingin diberi bingkai, corak atau warna latar belakang tertentu agar tampilannya lebih indah dan menarik.

Cara membuat page border=

Langkah-langkah:

Pada tab Page Layout (Word 2007 – 2010) atau tab Design (Word 2013 ke atas), Page Background grup, klik Page Borders.

Page Border - Page Background Group MS Word 2010

Akan muncul Page Border tab pada kotak dialog Borders and Shading.

Page Border - Borders And Shading MS Word 2010

Di bagian Setting, pilih bentuk bingkai yang diinginkan, misalnya:

Box: untuk membuat bingkai pada seluruh sisi halaman.

Shadow: untuk membuat bingkai dengan efek bayangan.

3-D: untuk membuat bingkai dengan efek 3-D. Catatan: efek 3-D hanya bisa digunakan pada gaya garis tertentu saja.

Custom: untuk membuat bingkai pada sisi tertentu halaman. Caranya: di bagian Preview, klik tombol atas, bawah, kanan, atau kiri.

Buat pengaturan pada:

Style: untuk menentukan gaya garis bingkai.

Color: untuk menentukan warna garis bingkai.

Width: untuk menentukan ketebalan garis bingkai.

Art: untuk memilih bingkai yang artistik, misalnya gambar bintang, pohon, dan lain-lain.

Fungsi ikon pictures=Untuk menyisip kan gambar dari media lain!

Perbedaan insert foto dari explorer dan internet= Insert foto dari explorer artinya kita menginput sebuah foto/gambar dari internal/dalam yaitu dokumen sendiri atau yang telah disediakan. Sedangkan dari internet artinya kita menginput sebuah foto/gambar dari eksternal/luar sebagai contoh dari Google Images

Fungsi dari ikon coloms di page layout= Columns: digunakan untuk mengatur jumlah kolom teks. Breaks: digunakan untuk mengatur kontinuitas halaman maupun kolom teks. Line Numbers: digunakan untuk mengatur kontinuitas nomor baris teks.

Kegunaan explorer= Dengan Windows Explorer, pengguna dapat melakukan hal-hal berikut: Membuat, memindahkan, menyalin, membuka, menyunting, atau menghapus berkas dan folder di dalam komputer. Mencari berkas-berkas dengan menggunakan query yang kompleks.

Cara membuat folder= 1.Buka File Explorer menggunakan salah satu cara berikut: ...

2.Navigasikan ke tempat yang Anda inginkan untuk membuat folder baru, lalu klik Folder Baru.

3.Ketikkan nama folder Anda, lalu tekan Enter.

40. apa fungsi page borders, watermark, page color, indent, spacing


Page Borders : digunakan untuk mengatur garis tepi halaman

Watermark : digunakan untuk memberikan efek tanda air di belakang teks.
