2. buatlah 10 formula/fungsi beserta penulisan formulanya dan jelaskan kegunaan dari formula tersebut


1. FUNGSI VALUE
digunakan untuk merubah nilai value menjadi nilai text. penulisannya :
=VALUE(text)
2. FUNGSI FIND
digunakan untuk menghasilkan posisi subtring dari sebuah string atau nomor yang divari, penulisannya :
=FIND(cari textt,pada text,mulai nomor)
3.FUNGSI MID
digunkan unntuk mengambil karakter tertentu dar sederet karakter, penulisannya:
=MID(text,posisi awal,jumlah karakter)
4. FUNGSI MAX
digunakan untuk mencari nila tertinggi dari sekumpulan data, penulisannya :
=MAX(number1,number2,...)
5. FUNGSI STEDEV
digunakan untuk menetukan standar devisiasi dari suatu range, penulisannya :
=STEDEV(number1,number2,..)
6. FUNGSI VAR
digunakan untuk menentukan nilai varience dari suatu range, penulisannya:
=VAR(number1,number2,..)
7. FUNGSI FACT
digunakan untuk menyatakan factorial dalam suatu bilangan, penulisanya:
=FACT(...)
8. FUNGSI GCD
digunakan untuk menyatakan pembagi umum terbesar, penulisannya:
=GCD(number1,number2,...)
9. FUNGSI LN
digunakan untuk menyatakan logaritma natural dari suatu bilangan, penulisannya:
=LN(number)
10. FUNGSI SQRT
digunakan untuk menghitung akar dari bilangan yang positif tidak boleh negatif, penulisannya:
=SQRT(...)

Jawaban:

Untuk membuat fungsi sendiri di Microsoft Excel, kamu dapat menggunakan fitur VBA (Visual Basic for Applications).

Berikut adalah langkah-langkahnya:

Buka Microsoft Excel dan tekan tombol "Alt + F11" untuk membuka jendela Visual Basic Editor.Pada jendela Visual Basic Editor, pilih "Insert" dan kemudian pilih "Module" untuk membuat modul baru.Pada modul baru tersebut, ketikkan kode-kode VBA yang akan menjadi fungsi kamu.

Jawaban:

Pembuatan Fungsi/Formula Anda Sendiri dengan MS. Excel:

1. Pertama, buka lembar kerja MS. Excel dan pilih sel yang akan menampung hasil dari fungsi/formula yang dibuat.

2. Selanjutnya klik ikon Fx pada Menu Ribbon Microsoft Excel untuk membuka jendela Insert Function.

3. Untuk membuat fungsi/formula sendiri, Anda harus merujuk ke nama User Defined Functions (UDF) di bagian atas jendela Insert Function.

4. Setelah itu, Anda bisa membuat sintaks/kode program yang berisi perintah-perintah baru yang dapat Anda gunakan untuk melakukan berbagai tugas dan operasi tertentu.

5. Jika sudah selesai, simpan file tersebut sebagai macro-enabled workbook (.xlsm).

6. Setelah itu, Anda bisa menggunakan fungsi/formula yang telah Anda buat kemudian memasukkannya kedalam sel yang tepat pada lembar kerja Excel.

13. what is formula to make desriptive text ?


apa rumus untuk membuat teks deskriptif..Struktur Descriptive Text (generic structure) adalah :
1. Identification (identifikasi) adalah pendahuluan , berupa gambaran umum tentang suatu topik.
2.Description (deskripsi) adalah berisi ciri-ciri khusus yang dimiliki benda, tempat, atau orang yang dideskripsikan.

- Menggunakan simple present tense

32. apa fungsi dari toolbar formula​


Jawaban:

Formula bar adalah kotak persegi panjang yang berfungsi untuk menampilkan dan mengedit isi dari sebuah sel yang sedang aktif. ... Selain mengetik langsung pada sebuah sel aktif kita bisa melakukan input data atau menuliskan rumus-rumus excel pada bagian formula bar ini.

Fungsi dari toolbar formula sendiri adalah untuk menampilkan dan mengedit isi dari sebuah sel yang sedang aktif. Selain itu sebagai tempat memasukkan rumus serta untuk mengedit atau memperbaiki rumus excel yang telah dibuat. Formula bar letaknya diantara ribbon dan worksheet area atau workspace.

Pembahasan

Pada bagian sebelah kiri formula bar terdapat Name Box yang menunjukkan alamat sel atau nama dari sel atau range yang sedang aktif. Di bagian name box ada icon segitiga kecil yang jika dipilih akan menunjukkan daftar nama-nama sel atau range atau nama lain yang sedang digunakan pada workboook file yang sedang dibuka. Jika file membuka fileyang baru, nama-nama tersebut tentunya masih kosong.

Diantara namebox dan formula bar ada 3 buah icon yang masing-masing mempunyai fungsi yakni :

Ikon Cancel (x) : Tombol ikon ini mempunyai fungsi yang sama dengan tombol Esc pada keyboard, yaitu untuk membatalkan perubahan atau editing data yang dilakukan pada formula bar atau sel aktif. Ikon Enter (✓) : Tombol ikon centang ini mempunyai fungsi seperti tombol enter pada keyboard, yaitu digunakan untuk menerima input data ataua perubahan pada sel aktif atau formula bar. Ikon Insert Function (fx) : Jika melakukan klik pada icon ini maka sebuah kotak jendela insert function akan terbuka. Ketika jendela muncul maka dapat kita temukan daftar fungsi excel yang bisa digunakan untuk menyusun sebuah rumus exel sesuai kebutuhan. Pelajari lebih lanjut

33. beberapa fungsi formula engineering yaitu​


Jawaban:

Misalkan di program Ms. Excel:

BIN2DEC

BIN2OCT

BIN2HEX

Penjelasan:

Ms. Excel termasuk salah 1 program pengolah angka. Dengan adanya formula Engineering, terdapat fitur-fitur untuk mengkonversi angka berbasis 2 (biner) ke angka berbasis lainnya (contohnya desimal).

Amatilah lampiran, Formula Engineering ditunjukkan tanda panah merah. Beberapa contoh formula Engineering, antara lain:

BIN2DEC = berfungsi untuk mengubah bilangan biner ke bilangan desimal

BIN2OCT = berfungsi untuk mengubah bilangan biner ke bilangan oktal

BIN2HEX = berfungsi untuk mengubah bilangan biner ke bilangan hksadesimal
