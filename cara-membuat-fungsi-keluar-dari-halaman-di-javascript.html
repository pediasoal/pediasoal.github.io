1. cara menempatkan fungsi pada javascript gimana?


Untuk dapat menggunakan JavaScript secara benar membutuhkan pengertian bagaimana Netscape Navigator memproses suatu dokumen HTML. Ketika suatu dokumen HTML di muat, browser akan mulai melakukan proses terhadap baris pertama dari dokumen. Browser membuat layout dari layar berdasarkan perintah HTML yang terdapat dalam dokumen tersebut. Setelah layar selesai digambar, layar tidak dapat digambar ulang tanpa melakukan proses terhadap dokumen baru. Bagaimana hal ini mempengaruhi frame dan window adalah topik pada bagian berikutnya. Sesuai dengan hal ini, maka instance dari suatu objek ada setelah proses dilakukan terhadap baris HTML yang menghasilkan instance tersebut. Dengan kata lain JavaScript tidak dapat mengacu pada suatu objek HTML seperti form, atau menjalankan suatu fungsi sampai browser memproses perintah tersebut. Sebagai contoh, JavaScript tidak dapat mengacu pada suatu objek form sampai browser memproses HTML untuk form tersebut. Demikian juga dengan proses perubahan  properti, setelah browser menggunakan properti dalam menghasilkan layout window, maka perubahan properti setelah itu tidak memiliki efek terhadap tampilannya. Walaupun nampaknya keterbatasan tersebut berat, namun dengan pengertian yang baik terhadap HTML proses dan JavaScript akan mengurangi frustasi. Prinsip utama yang perlu diingat bahwa suatu dokumen HTML diproses secara berurut dan JavaScript adalah bagian dari proses secara berurut tersebut. Objek Window Objek window adalah orang tua dari semua Objek. Hal tersebut termasuk semua window dan frame yang berada dalam suatu. Ketika browser mulai memuat suatu dokumen HTML, hal tersebut dimulai dengan suatu instance dari objek window. Jika dokumen HTML membuat suatu frame, informasi dari frame tersebut tersimpan dalam suatu objek frame array. Dengan kata lain, membuka suatu window akan menghasilkan suatu anak dari objek window. Kehebatan dari JavaScript adalah terletak pada kemampuan untuk melakukan utilisasi properti dan metode dari objek window. Bagian ini akan membahas properti dari objek  window berikut : objek Location objek History objek Document Diantara ketiga objek tersebut, document adalah objek yang paling penting dalam hirarki tersebut. Objek document itu sendirinya mengacu kepada objek lain. Hal lain yang penting dari hal ini adalah sebagai berikut: objek form objek anchor objek link.

JavaScript adalah bahasa pemrograman yang digunakan secara luas untuk mengembangkan aplikasi web. JavaScript digunakan untuk menambahkan interaktivitas pada halaman web, seperti membuat animasi, memvalidasi input pengguna, mengambil data dari server tanpa perlu me-refresh halaman, dan banyak lagi.

Fungsi JavaScript adalah untuk membuat halaman web lebih interaktif dan dinamis. Beberapa contoh penggunaan JavaScript diantaranya:

Validasi form: JavaScript dapat digunakan untuk memvalidasi input pengguna pada form, sehingga informasi yang diisi oleh pengguna tidak salah atau tidak valid.

Animasi: JavaScript dapat digunakan untuk membuat animasi pada halaman web, seperti menampilkan pop-up, memperbesar gambar, dan banyak lagi.

Menyediakan akses ke API: JavaScript dapat digunakan untuk mengakses API dari server, seperti API Google Maps, Twitter, Facebook, dan lain-lain.

Membuat game: JavaScript dapat digunakan untuk membuat game sederhana pada halaman web.

18. 1. Jelaskan fungsi JavaScript dalam pemrograman web !2. Apa yang dimaksud dengan ECMAScript Engine (JavaScript Engine) ?3. Sebutkan cara memasukkan kode JavaScipt ke dalam HTML !


1. Fungsi JavaScript Dalam Pemograman Web

JavaScript pada awal perkembangannya berfungsi untuk membuat interaksi antara user dengan situs web menjadi lebih cepat tanpa harus menunggu pemrosesan di web server. Sebelumjavascript, setiap interaksi dari user harus diproses oleh web server.

Bayangkan ketika kita mengisi form registrasi untuk pendaftaran sebuah situs web, lalu men-klik tombol submit, menunggu sekitar 20 detik untuk website memproses isian form tersebut, dan mendapati halaman yang menyatakan bahwa terdapat kolom form yang masih belum diisi.

2. JavaScript Engine adalah mekanisme internal yang dimiliki oleh web browser.JavaScript Engine dapat diumpamakan dengan compiler dalam bahasa pemograman lain, yakni algoritma yang digunakan untuk menjalankan JavaScript. Semakin cepat sebuah web browser menjalankan JavaScript akan semakin baik.

Biasanya disetiap rilis baru web browser seperti Google Chrome, Internet Explorer, maupun Mozilla Firefox, juga diikuti rilis terbaru JavaScript Engine yang menawarkan kecepatan lebih baik.

36. Bagaimana membuat function pada javascript?


Jawaban:

Pakai keyword function lalu tuliskan nama function yang mau dibuat. Seperti berikut:

function namaFunc(argumentJikaAda) {

berikan statement disini

}

Bisa juga menggunakan arrow function (ES6) dengan cara berikut:

const namaFunc = (argumentJikaAda) => {

berikan statement disini

}

const merupakan arti dari konstanta yang tidak bisa diubah, jadi kalau ingin membuat function dan gabisa diubah orang asing bisa pakai const, jadi cuma bisa diubah di declarationnya atau pas dideklarasi. Tapi kalau mau biar bisa diubah bisa pakai var atau let. Kalau pakai arrow function ini juga bisa dikasih async di dekat argument. Kalau mau cari tau tentang async bisa check aja di Internet mengenai asynchronous.

Penjelasan:

Aturan penggunaan nama function sama dengan membuat variable. Yaitu:

• Tidak boleh ada spasi

• Tidak boleh memakai symbol yang berhubungan dengan operasi atau disebut operator seperti contohnya +,-,*,/,%

• Bisa memakai underscore

• Tidak boleh diawali nomor seperti 123iniVariable tetapi bisa jika nomornya tidak di awal nama variable seperti variable123

• Biasanya nama variable atau function menggunakan camelCase dibanding underscore jika ada spasi contohnya cari_angka_terbesar diganti menjadi cariAngkaTerbesar.

40. Sebutkan 3 fungsi yang digunakan untuk menampilkan isi variabel pada javascript!​


Jawaban:

Pada JavaScript, terdapat beberapa fungsi yang dapat digunakan untuk menampilkan isi variabel, diantaranya:

1.) console.log(): Fungsi ini digunakan untuk menampilkan output di konsol browser. Contohnya: console.log(namaVariabel)

2.( document.write(): Fungsi ini digunakan untuk menampilkan output di halaman web. Contohnya: document.write(namaVariabel)

3.) window.alert(): Fungsi ini digunakan untuk menampilkan pop-up dengan pesan yang ditentukan. Contohnya: window.alert(namaVariabel)

Semua fungsi di atas dapat digunakan untuk menampilkan isi dari berbagai tipe data, seperti string, angka, boolean, dan lainnya. Namun, biasanya console.log() lebih sering digunakan untuk debugging kode dan document.write() lebih sering digunakan untuk menampilkan output secara dinamis di halaman web.

Jawaban:

1. Berikut adalah 3 fungsi yang dapat digunakan untuk menampilkan isi variabel pada JavaScript:

console.log()

Fungsi ini digunakan untuk menampilkan isi variabel di console browser. Contohnya

let nama = "John Doe";

console.log(nama);

Hasil yang akan ditampilkan di console browser adalah "John Doe".

2. document.write()

Fungsi ini digunakan untuk menampilkan isi variabel di halaman web. Contohnya:

let nama = "John Doe";

document.write(nama);

Hasil yang akan ditampilkan di halaman web adalah "John Doe".

3. alert()

Fungsi ini digunakan untuk menampilkan isi variabel dalam bentuk dialog pop-up. Contohnya:

let nama = "John Doe";

alert(nama);

Hasil yang akan ditampilkan adalah dialog pop-up dengan isi "John Doe".
