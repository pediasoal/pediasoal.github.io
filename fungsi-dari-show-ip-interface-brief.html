2. Apa fungsi dari IP ,router, switch, interface, access list​


IP address bisa digunakan sebagai alat atau media untuk mengidentifikasi host atau fitur antarmuka pada sebuah jaringan komputer.

Router memiliki fungsi untuk mengirimkan paket data atau suatu informasi melalui internet atau jaringan dari lokasi tertentu ke jaringan lainnya.

Switch mempunyai fungsi umum sebagai suatu concentrator yang dapat menerima serta membagikan data antara sejumlah perangkat komputer.

User Interface berperan sebagai penghubung yang memudahkan penggunanya agar dapat menjalankan suatu program, aplikasi, maupun mengunjungi website.

Access List merupakan sebuah tool yang digunakan di sisi AP (Access Point) untuk melakukan filtering koneksi dari client.

5. 2. Bagaimanakah perintah agar PC client dapat melakukan koneksi ke internet dan mengubah IP privat client ke IP publik yang ada di interface publik?​


Jawaban:

Untuk dapat melakukan koneksi ke internet dan mengubah IP privat client menjadi IP publik, terdapat beberapa langkah yang harus dilakukan.langkah-langkah umumnya:

1. Pastikan bahwa PC client terhubung dengan interface publik yang sudah tersambung ke internet.

2. Pada PC client, buka pengaturan jaringan dan cari opsi untuk mengatur IP address.

3. Ubah pengaturan IP address dari mode privat ke mode publik. Untuk pengaturan ini, cari opsi "Obtain an IP address automatically" atau "Use a public IP address" pada pengaturan jaringan.

4. Jika PC client terhubung ke jaringan dengan router atau firewall, pastikan bahwa perangkat tersebut telah dikonfigurasi untuk memberikan akses internet ke interface publik.

5. Coba untuk melakukan koneksi ke internet dengan membuka aplikasi web atau melakukan ping ke alamat web tertentu.

6. Jika koneksi ke internet berhasil, maka PC client sudah dapat menggunakan IP publik pada interface publik.

9. Sebutkan macam-macam port interface dan tuliskan fungsinya


Jawaban:

Berikut ini adalah beberapa macam port interface dan fungsinya:

Port USB (Universal Serial Bus) - Digunakan untuk menghubungkan perangkat-perangkat elektronik seperti mouse, keyboard, printer, dan lain-lain.

Port HDMI (High Definition Multimedia Interface) - Digunakan untuk menghubungkan perangkat-perangkat elektronik seperti komputer, TV, proyektor, dan lain-lain yang memiliki resolusi gambar tinggi.

Port VGA (Video Graphics Array) - Digunakan untuk menghubungkan perangkat-perangkat elektronik seperti komputer, TV, proyektor, dan lain-lain yang memiliki resolusi gambar sedang.

Port LAN (Local Area Network) - Digunakan untuk menghubungkan perangkat-perangkat elektronik ke jaringan lokal (misalnya jaringan kantor atau sekolah).

Port Audio - Digunakan untuk menghubungkan perangkat-perangkat audio seperti speaker, headphone, atau microphone.

Port Serial - Digunakan untuk menghubungkan perangkat-perangkat elektronik yang membutuhkan komunikasi data satu per satu (misalnya modul GPS atau RFID).

Port Parallel - Digunakan untuk menghubungkan perangkat-perangkat elektronik yang membutuhkan komunikasi data banyak sekaligus (misalnya printer atau scanner).

Port Firewire (IEEE 1394) - Digunakan untuk menghubungkan perangkat-perangkat elektronik yang membutuhkan kecepatan tinggi dalam mentransfer data (misalnya kamera video atau hard disk eksternal).

16. jelaskan fungsi perintah do show vlan brief pada switch​


Jawaban:

Perintah "do show vlan brief" pada switch memiliki fungsi untuk menampilkan ringkasan atau daftar VLAN yang ada dalam switch.

VLAN (Virtual Local Area Network) adalah metode untuk membagi jaringan fisik menjadi beberapa jaringan logis yang terisolasi satu sama lain. Dengan menggunakan VLAN, administrator jaringan dapat memisahkan lalu lintas jaringan berdasarkan kriteria seperti departemen, fungsi, atau kelompok pengguna.

Ketika Anda menjalankan perintah "do show vlan brief" pada switch, hasilnya akan menampilkan daftar VLAN yang ada beserta informasi penting, seperti ID VLAN, nama VLAN, dan status VLAN. Informasi ini berguna untuk melihat konfigurasi dan status VLAN pada switch.

Contoh output dari perintah "do show vlan brief" dapat berupa tabel yang mencantumkan ID VLAN, nama VLAN, dan status VLAN. ID VLAN adalah angka yang digunakan untuk mengidentifikasi VLAN, sedangkan nama VLAN adalah nama yang diberikan oleh administrator jaringan untuk menggambarkan VLAN tersebut. Status VLAN menunjukkan apakah VLAN tersebut sedang aktif atau tidak.

Perintah "do show vlan brief" sangat berguna ketika Anda perlu memeriksa konfigurasi dan status VLAN dalam switch, terutama ketika Anda ingin memverifikasi apakah VLAN yang diinginkan sudah dibuat atau mengidentifikasi konflik VLAN. Informasi yang diperoleh dari perintah ini dapat membantu dalam pemecahan masalah dan pengaturan jaringan yang lebih efisien.

18. Apa yang dimaksud sistem operasi berfungsi sebagai interface ​

Penjelasan:

user interface

Antarmuka pengguna (bahasa Inggris: user interface) merupakan bentuk tampilan grafis yang berhubungan langsung dengan pengguna (user). Antarmuka pengguna berfungsi untuk menghubungkan antara pengguna dengan sistem operasi, sehingga komputer tersebut bisa digunakan.

Pengertian antarmuka ( interface) adalah salah satu layanan yang disediakan sistem operasi sebagai sarana interaksi antara pengguna dengan sistem operasi. Antarmuka adalah komponen sistem operasi yang bersentuhan langsung dengan pengguna. Terdapat dua jenis antarmuka, yaitu Command Line Interface(CLI) dan Graphical User Interface(GUI).

User interface jika diartikan dalam bahasa Inggris adalah tampilan antar muka pengguna. User interface berfungsi sebagai mediator antara user dengan komputer.

User interface sangat berperan penting dalam dunia komputer karena dengan adanya user interface maka kemudahan user dalam mengoperasikan suatu perangkat komputer menjadi lebih mudah.

User interface dibagi menjadi 2 yaitu :

1. command line interface (CLI)

Merupakan user interface berupa text ataupun perintah-perintah yang diketikan langsung oleh user untuk melakukan atau mengeksekusi sesuatu. Contoh dari text interface : Command Line Interface (Unix, Linux) dan juga Command Prompt (Windows).

2. Graphic User Interface (GUI)

GUI merupakan user interface berupa tampilan grafis ataupun gambar yang dipergunakan untuk melakukan komunikasi dengan komputer. Contoh Graphic User Interface : Button, Toolbox, Textarea, dan masih banyak lagi.

20. Jika wlan dan ether2 di-bridge, maka interface manakah yang disetting IP dan setting DHCP server?.


Jawaban:

Settingan IP dan DHCP Server berada pada Bridge Connection

Penjelasan:

Pada kasus permasalahan, diminta WLAN dan Port Ether2 di bridge, konsep bridge sendiri yaitu dapat menghuungkan jaringan/port yang berbeda dan menjadikannya dalam satu jalur koneksi, dalam hal ini WLAN dan Ether2 menjadi 1 koneksi. Maka dari itu, router akan membuat interface baru dibawah bridge menu, contohnya pada mikrotik, ketika kita ingin menjadikannya bridge, maka kita akan membuat interface bridge terlebih dahulu, kemudian mengisikan port yang ingin disatukan. Agar lebih mudahnya, berikut ini merupakan langkah-langkahnya (penulsi menggunakan router Mikrotik RouterOS.

1. Masuk pada menu Bridge, lalu klik tanda (+), kemudian isikan nama interface bridge, setelah itu klik ok

2. Masuk ke tab Ports, lalu isikan interface WLAN dan Ether2

3. Kemudian masuk ke menu IP > Address List, kemudian klik tanda (+), tambahkan IP yang sesuai dan pilih interface bridge (nama bridge yang dibuat tadi).

4. Kemudian, masuk ke menu IP > DHCP Server > klik tanda (+) > isikan name sesuai dengan keinginan, kemudian pilih interface bridge (nama bridge yang telah dibuat)

5. Maka dalam hal ini, bridge yang telah dibuat telah berisikan IP dan sebagai DHCP server

Kesimpulan,

Dari pertanyaan yang diberikan, maka WLAN dan Ether2 di bridge, maka interface yang diberikan IP dan setting DHCP Server adalah interface Bridge.

23. sebutkan macam macam port interface dan tuliskan fungsinya


1. port Serial
Port yang berfungsi untuk melakukan transfer data dengan kecepatan lebih kecil dengan jenis paralel. sehingga kini port ini kurang populer dipergunakan.

2. port PS/2
Port serial 2, seperti halnya port serial: melakukan transfer data, namun relatif lebih cepat dari port serial sebelumnya. kini masih dipakai untuk menghubungkan komputer dengan mouse atau keyboard jenis PS/2

3. port Paralel
adalah port yang berfungsi mengirim dan menerima beberapa bit data pada satu saat melalui satu set kabel. dipergunakan untuk menghubungkan komputer dengan modem eksternal yang memiliki kabel jenis paralel.

4. port USB
adalah port pengembangan dari port serial dan kini tengah populer digunakan. Bisa digungsikan untuk menghubungakn komputer dengan flashdisk, scanner atau printer yang mendukung kabel USB.

5. port VGA
adalah port yang berfungsi menghubungkan monitor dengan CPU.
