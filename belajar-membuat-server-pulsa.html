16. Bagaimana cara membuat Master Server?


Edit file konfigurasi mysql
$sudo nano /etc/mysql/mysql.conf.d/mysqld.cnf atau untuk versi yang lama
$sudo nano /etc/mysql/my.cnf
Edit/tambahkan baris berikut di file konfirgurasi
bind-address = 192.168.1.1 atau * (address server master)
server-id = 1
log_bin = /var/log/mysql/mysql-bin.log
binlog_do_db = database1 (database yg mau direplikasi)

Kalau mau lebih dari satu database tinggal tambahkan baris lagi
binlog_do_db = database2
binlog_do_db = database3 dst
binlog_ignore_db = database4 (database yg tdk ikut direplikasi)
Refresh database mysql $sudo service mysql restart
Login ke Mysql
#mysql -u root -p
mysql>GRANT REPLICATION SLAVE ON *.* TO 'slave_user'@'%'
IDENTIFIED BY 'password'; (buat user replikasi)
mysql>FLUSH PRIVILEGES;
mysql>USE database1;
mysql>FLUSH TABLES WITH READ LOCK;
mysql>SHOW MASTER STATUS;
Akan keluar tampilan seperti berikut
mysql> SHOW MASTER STATUS;
+------------------+----------+--------------+------------------+
| File | Position | Binlog_Do_DB | Binlog_Ignore_DB |
+------------------+----------+--------------+------------------+
| mysql-bin.000001 | 107 | database1 | |
+------------------+----------+--------------+------------------+
1 row in set (0.00 sec)

Catat File dan Position yang akan digunakan di SERVER SLAVE
(contoh disini mysql-bin.000001 dan 107)

Backup database yang akan direplikasi untuk diinstal di SERVER SLAVE
#mysqldump -u root -p --opt database1 > database1.sql

Unlock Tabel
mysql>UNLOCK TABLES;
mysql>QUIT;
Langkah di SERVER MASTER sudah selesai, lanjut ke SERVER SLAVE


SERVER SLAVE

Buat database yang akan direplikasi di MySQL
#mysql -uroot -p
mysql>CREATE DATABASE database1;
Import struktur database yang sudah dilakukan di server master sebelumnya
#mysql -u root -p database1 < /path/to/database1.sql
Edit file konfigurasi MySQL
$sudo nano /etc/mysql/mysql.conf.d/mysqld.cnf
Edit/tambah file konfigurasi mysql
server-id = 2 (nomor harus beda dari Server Master)
relay-log = /var/log/mysql/mysql-relay-bin.log
log_bin = /var/log/mysql/mysql-bin.log
binlog_do_db = database1 (list database yg direplikasi)
Restart MySQL
$sudo service mysql restart
Buka MySQL
#mysql -u root -p
mysql>CHANGE MASTER TO MASTER_HOST='12.34.56.789',
MASTER_USER='slave_user',
MASTER_PASSWORD='password',
MASTER_LOG_FILE='mysql-bin.000001',
MASTER_LOG_POS= 107; (master log & master pos disesuaikan)

Start Slave
mysql>START SLAVE;
mysql>SHOW SLAVE STATUS\G
Jika ada masalah saat koneksi bisa ditambah perintah berikut
mysql>SET GLOBAL SQL_SLAVE_SKIP_COUNTER = 1;
mysql>SLAVE START;

27. pulsa dan kuota terbuat dari apa​


Pulsa dan kuota merupakan data yang dapat berbentuk sinyal listrik, sinyal optik, ataupun gelombang radio, tergantung dari metode transmisi yang digunakan.

Penjelasan:

Kuota dalam dunia telekomunikasi Indonesia merujuk pada sejumlah data yang diberi harga tertentu (disebut juga "paket data"). Inti dari telekomunikasi adalah bertukar data, karenanya data-lah yang diperjual belikan. Operator menjual kuota data dengan harga tertentu (misalkan 5 GB dibanderol dengan harga 50.000, dan sebagainya).

Lalu apa sebenarnya data ini? Data adalah informasi yang dibuat dalam bentuk biner. Satu satuan terkecilnya adalah binary digit atau disingkat bit. Biner adalah sistem bilangan 1 dan 0. Misalkan, huruf "A" dalam bentuk biner adalah 01000001, "B" adalah 01000010, dan seterusnya. Jadi saat kita mengirim data, data tersebut akan diubah dalam bentuk biner sebelum dikirimkan.

Mengapa harus diubah ke biner? Karena dengan mengubah data ke bentuk biner, kita dapat merepresentasikannya dalam bentuk lain. Misalkan, jika kita ingin merepresentasikan "A" dengan sinyal listrik, kita dapat menyatakan 0 sebagai off "tidak ada arus" dan 1 sebagai on "ada arus". Intinya, mengubah data ke bentuk biner membuat data tersebut bisa dikirimkan dalam bentuk sinyal.

Bagaimana bentuk sinyal ini? bentuk sinyalnya beragam, tergantung metode pengirimannya. Jika sinyal dikirimkan melalui kabel fiber optik, bisa jadi bentuknya adalah sinyal cahaya, sinyal dikirim secara wireless, sinyalnya berbentuk gelombang radio.

29. terbuat dari apakah pulsa?


Jawaban:

Pulsa berasal dari berbagai macam bentuk dari sinyal, seperti sinyal listrik, sinyal optik, hingga sampai gelombang radio, relatif sesuai tiang sinyal listrik, dan operator tranmisinya yang akan digunakan. Pulsa juga bisa dianggap sebagai bagian dari gelombang sinusoidal, yang bergerak atau merambat melewati sebuah perantara medium tertentu. Pengibaratan cara kerja pulsa seperti sebuah ikat ujung tali ke tiang, dan pegang ujung satu lagi. Tarik tali dengan berjalan mundur sampai lurus, kemudian hentakan tali dan kembali ke posisi awal, denyutan dari hentakan tali tadi akan bergerak menuju tiang dan akan kembali lagi ke arah kita yang menghentakan tali tersebut, bahwa suatu pulsa gelombang yang diberikan dan akan kembali kepada kita.

37. pulsa terbuat dari apa...


Pulsa berasal dari berbagai macam bentuk dari sinyal, seperti sinyal listrik, sinyal optik, hingga sampai gelombang radio, relatif sesuai tiang sinyal listrik, dan operator tranmisinya yang akan digunakan. Pulsa juga bisa dianggap sebagai bagian dari gelombang sinusoidal, yang bergerak atau merambat melewati sebuah perantara medium tertentu. Pengibaratan cara kerja pulsa seperti sebuah ikat ujung tali ke tiang, dan pegang ujung satu lagi. Tarik tali dengan berjalan mundur sampai lurus, kemudian hentakan tali dan kembali ke posisi awal, denyutan dari hentakan tali tadi akan bergerak menuju tiang dan akan kembali lagi ke arah kita yang menghentakan tali tersebut, bahwa suatu pulsa gelombang yang diberikan dan akan kembali kepada kita.

Pembahasan

Pulsa bisa dianggap sama seperti halnya dengan saldo, yang dibuat oleh jasa telekomunikasi tertentu. Pulsa juga seperti sarana pembayaran virtual untuk mendapatkan jasa dari telekomunikasi tersebut, jumlah angka pulsa sebenarnya hanya formalitas untuk memberikan jumlah layanan, harga yang bisa digunakan oleh konsumen. Semisal perusahaan telkomunikasi pulsa seperti timi, membuat saldo sendiri yang akan dibeli oleh agen, dan akan berjualan pulsa elektrik. Sehingga dari situlah pembeli membeli pulsa langsung, ditukar dengan uang.
