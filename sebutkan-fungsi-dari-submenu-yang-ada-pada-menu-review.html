4. Sebutkan submenu-submenu yang ada pada menu edit beserta fungsinya


Submenu yang ada pada menu edit Ms. Word 2003:

Undo / Undo Typing : Perintah untuk membatalkan perintah terakhir.

Redo : Perintah untuk membatalkan perintah undo.

Repeat Typing : Perintah untuk mengulangi perintah terakhir.

Cut : Perintah untuk mengopi teks dengan menghapus teks yang dicopy.

Copy : Perintah untuk mengopi teks tanpa menghapus teks yang dicopy.

Office Clipboard : Perintah untuk menampilkan task pane clipboard.

Paste : Perintah untuk menyalin teks yang sudah dicopy.

Paste Spesial : Perintah untuk menyalin teks dengan berbagai macam tipe teks.

Paste as Hyperlink : Perintah untuk menyalin teks dengan bentuk teks hyperlink.

Clear : Perintah untuk menghapus teks atau gambar yang salah atau yang tidak diinginkan.

Select All : Perintah untuk memilih semua teks (yang sudah di tulis) yang terdapat di dokumen.

Find : Perintah mencari teks atau kata yang terdapat di dokumen.

Replace : Perintah untuk mengganti teks.

Go To : Perintah untuk berpindah halaman, baris, kolom, objek, footnote, endnote, bookmark, comment, table, dan lainnya.

Links : Perintah untuk membuat hubungan dengan file / dokumen lain.

Object : Perintah untuk mengedit objek.

17. sebutkan submenu yang ada pada menu insert Sebut kan sub menu dari insert


Sub menu dari menu Insert pada MS Word 2007:

PagesTablesIllustrationsLinksHeader & FooterTextSymbols

Simak penjelasan berikut.

Pembahasan

Menu yang terdapat pada Microsoft office Word 2007 adalah Home, Insert, Page Layout, References, Mailings, Review, dan View. Menu-menu tersebut masing-masing memiliki sub menu yang memiliki fungsi lebih spesifik.

Menu Insert memiliki fungsi umum untuk memasukkan/menyisipkan objek tertentu ke dalam dokumen. Objek-objek tersebut dapat berupa gambar, bentuk, halaman, link, rumus, tabel, simbol, dan lainnya. 22. sebutkan submenu dalam menu layout






1. New.

New adalah submenu dalam menu file yang di dalam submenu tersebut terdapat beberapa jenis pemilihan file mana yang akan digunakan. Misalnya saja, blank document, blog post, recent templates, sample templates, my templates dan new from existing. Dari semua hal tersebut, terdapat jenis atau bentuk yang berbeda-beda untuk menulis dokumen. Jadi, tinggal pilih jenis mana yang akan digunakan. Fungsi dari submenu new adalah untuk membuat dokumen baru.

2. Open.

Open adalah salah satu submenu dalam menu file yang pengguna mengklik submenu tersebut maka akan muncul beberapa dokumen atau beberapa file yang sudah disimpan sebelumnya. Selanjutnya, tinggal memilih saja file atau dokumen mana yang akan dibuka. Fungsi dari submenu open dalam menu file ini adalah untuk membuka dokumen atau file yang sudah disimpan sebelumnya.

3. Save.

Save adalah salah satu submenu dalam menu file yang di jika pengguna mengklik submenu tersebut maka pengguna berarti akan menyimpan file atau dokumen mereka. Kemudian, pengguna harus memilih di mana file atau dokumen tersebut akan diletakkan. Pengguna juga dapat memilih dalam bentuk atau tipe apa file atau dokumen mereka akan disimpan. Fungsi dari submenu save dalam menu file ini adalah untuk menyimpan dokumen atau file.

4. Save as.

Save as adalah submenu dalam menu file yang fungsinya hampir sama dengan submenu save. Pengguna harus memilih mau diletakkan dimana dokumen atau file mereka. Pengguna juga akan bisa memilih dalam bentuk atau tipe apa dokumen atau file mereka yang akan disimpan. Fungsi dari submenu save as dalam menu file ini adalah untuk menyimpan dokumen atau file dengan format file yang berbeda dari penyimpanan yang pertama.

5. Print.

Print adalah submenu dalam menu file yang jika pengguna mengklik submenu tersebut berarti pengguna akan mencetak dokumen mereka. Dalam submenu print, terdapat beberapa hal untuk mengecek kembali apakah dokumen yang akan dicetak sudah benar atau belum. Fungsinya adalah untuk mencetak dokumen dengan printer.
