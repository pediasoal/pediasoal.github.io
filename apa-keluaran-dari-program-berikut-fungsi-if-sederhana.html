1. Buatlah Program Sederhanamenggunakan Fungsi If Bersarang(Nested If) dengan menggunakan DevC++!!​


Program dalam bahasa C++

// AUTHOR : LordRimuruTempest

#include <iostream>

void tahunKabisat(int Tahun){

   if(Tahun % 100 == 0){

      if(Tahun % 400 == 0){

          std::cout<<"Tahun "<<Tahun<<" merupakan tahun kabisat";

      }else{

          std::cout<<"Tahun "<<Tahun<<" bukan tahun kabisat";

      }

  }else{

      if(Tahun % 4 == 0){

          std::cout<<"Tahun "<<Tahun<<" merupakan tahun kabisat";

      }else{

          std::cout<<"Tahun "<<Tahun<<" bukan tahun kabisat";

      }

  }

}

int main() {

  int Tahun;

  std::cout<<"Masukkan tahun : ";

  std::cin>>Tahun;

  tahunKabisat(Tahun);

  return 0;

}

Pembahasan

Contoh program yang dapat dibuat menggunakan nested if adalah program menentukan angka tahun kabisat. Sebelum mencoba menerapkan algoritma ke dalam program C++, maka harus diketahui algoritmanya sebagai berikut

1. Program dimulai

2. Masukkan angka tahunnya

3. Jika angka tahun habis dibagi 100 (sisa baginya 0), maka

Jika angka tahun habis dibagi 400 (sisa baginya 0), cetak "tahun kabisat".Jika angka tahun tidak habis dibagi 400 (sisa baginya bukan 0), cetak "bukan tahun kabisat".

4. Jika angka tahun tidak habis dibagi 100 (sisa baginya bukan 0), maka

Jika angka tahun habis dibagi 4(sisa baginya 0), cetak "tahun kabisat".Jika angka tahun tidak habis dibagi 4(sisa baginya bukan 0), cetak"bukan tahun kabisat".

5. Program selesai.

Algoritma diatas dalam bentuk pseudocode dapat ditulis

Algoritma tahunKabisat

program start

var tahun : integer;

if(tahun % 100 == 0) then

   if(tahun % 400 == 0) then

       write "Tahun kabisat"

   else then

       write "Bukan tahun kabisat"

   end if

else then

   if(tahun % 4 == 0) then

       write "Tahun kabisat"

   else then

       write "Bukan tahun kabisat"

   end if

end if

program end

Setelah algoritma diketahui, maka sisanya adalah menerapkan algoritma kedalam kode program. pertama-tama import library iostream(input output stream) ke dalam program dengan perintah #include <iostream>. Fungsi library ini adalah untuk menerima input dan mencetak output. Kemudian buatlah procedure/function bernama tahunKabisat, function tersebut bertipe void (tidak mengembalikan nilai apapun) dan memiliki parameter Tahun(bertipe integer). Dalam function tahunKabisat, dibuat nested if sesuai petunjuk/alur dari algoritma. Kemudian untuk main function, deklarasikan variabel Tahun bertipe integer dan minta user untuk memasukkan angka tahunnya dengan menggunakan cin(c input). Langkah terakhir adalah memanggil function tahunKabisat dan memasukkan nilai variabel Tahun tadi kedalam function tahunKabisat. Kode program beserta output-nya dapat dilihat pada lampiran gambar.

8. Fungsi IF bercabang dalam penerapannya dapat disederhanakan menggunakan fungsi .....


Fungsi IF bercabang dalam penerapannya dapat disederhanakan menggunakan fungsi switch. Switch menggunakan case pada percabangannya.

Pembahasan

Fungsi SWITCH adalah fungsi yang digunakan untuk memilih satu dari beberapa opsi yang mungkin berdasarkan nilai atau ekspresi yang diberikan. Fungsi SWITCH bekerja dengan cara mengevaluasi sebuah ekspresi atau nilai, kemudian memeriksa nilai tersebut terhadap beberapa opsi yang telah ditentukan sebelumnya. Jika nilai tersebut cocok dengan salah satu opsi, maka fungsi akan mengembalikan nilai yang sesuai dengan opsi tersebut.

Fungsi SWITCH dapat digunakan untuk menggantikan pernyataan IF bercabang yang lebih kompleks, terutama ketika ada banyak kondisi yang harus dipenuhi. Dengan menggunakan fungsi SWITCH, kita dapat membuat kode yang lebih mudah dibaca dan dipahami, serta lebih efisien dan singkat.

14. perbedaan fungsi if dengan switch dalam program


Perbedaan utama dari IF statement dengan switch-case adalah kondisi yang digunakan oleh IF dapat berupa range dan object sedangkan switch-case memiliki kondisi yang tetap. Perbedaan lainnya adalah IF dapat memiliki beberapa kondisi dari variable atau obyek yang berbeda sedangkan switch-case hanya dapat berisi 1 variable. Contoh: Jika nilai a sama dengan 0 atau 1 atau 2 maka print 'A', jika 3 maka print 'B', selain itu print 'C'.


Tampilan dalam IF dapat berupa seperti ini


IF(a>=0 && a <3)

   print 'A'

ELSE IF(a==3)

  print 'B'

ELSE

  print 'C'

ENDIF


Sedangkan switch-case akan terlihat seperti ini


SWITCH(a)

    case '0':

    case '1':

    case '2':

            print 'A'

            break;

    case '3':

            print 'B'

            break;

     default:

          print 'C'

          break;
