3. Apakah fungsi utama tanda sama dengan ganda (==) pada bahasa pemrograman seperti php,java,dan python ?

== di dalam bahasa pemrograman bisa di artikan sebagai sama dengan contoh

variable = "jawaban"

if variable == "jawaban" {
     print("variable adalah jawaban")
} else {

Fungsi utama tanda sama dengan ganda (==) pada bahasa pemrograman seperti PHP, Java, dan Python adalah sebagai operator logika kesamaan nilai dua variabel, yang mengembalikan nilai:

true (atau 1) jika nilai kedua variabel sama, atau false (atau 0) jika nilai kedua variabel berbeda.

Walaupun fungsi utamanya sama, terdapat perbedaan dalam penggunaannya, tergantung dari implementasi dan aturan yang ditetapkan pada setiap bahasa pemrograman tersebut.
 

Pembahasan

Untuk menentukan apakah dua variabel a dan b sama nilainya, bahasa pemrograman seperti PHP, Java, dan Python (bahkan C, C++, dsb.) menggunakan operator berupa tanda sama dengan ganda (==).
Alasan utamanya adalah untuk membedakan dengan operator atau tanda yang digunakan untuk memberi nilai (assignment) pada variabel, yaitu tanda sama dengan tunggal (=).

Agar lebih jelas, kita gunakan contoh ilustrasi.

10. Tuliskan program sederhana bahasa java


Program dalam bahasa java

// AUTHOR : LordRimuruTempest

import java.util.Scanner;

public class menghitungMean {

   public static void main(String[] args) {

       Scanner input = new Scanner(System.in);

       double data, rataRata, jumlahData = 0;

       System.out.print("Masukkan banyak data : ");

       int banyakData = input.nextInt();

       for (int i = 1; i <= banyakData; i++) {

           System.out.print("Masukkan data ke " + i + " : ");

           data = input.nextDouble();

           jumlahData += data;

       }

       rataRata = jumlahData / banyakData;

       System.out.println("Nilai rata-ratanya adalah : " + rataRata);

   }

}

Pembahasan

Bahasa java dapat digunakan untuk membuat program-program sederhana yang dapat membantu pekerjaan manusia. Salah satu contohnya adalah program menghitung rata-rata dari n data. Perhatikan bahwa rumus dari rata-rata(mean) adalah sebagai berikut :

[tex]\boxed{\bold{rata-rata = \frac{jumlah \hspace{1 mm} data}{banyak \hspace{1 mm} data} }}[/tex]

Maka algoritma program dalam bentuk pseudocode dapat ditulis sebagai berikut:

Algoritma menghitungMean

program start

var : data, rataRata, jumlahData : double

var : banyakData : integer

jumlahData <-- 0

read banyakData

for i = 1 to i = banyakData do

     read data

     jumlahData <-- jumlahData + data

end for

rataRata <-- jumlahData / banyakData

print rataRata

program end

Setelah alur program dibuat, maka langkah selanjutnya adalah menerapkan algoritma ke dalam bahasa pemrograman java. Pertama-tama import library java.util.Scanner untuk menggunakan objek scanner(untuk meminta input user) dan buat kelas bernama menghitungMean beserta main method-nya. Buat satu objek scanner java dengan nama input untuk memasukkan input dari user. Deklarasikan variabel data, jumlahData, dan rataRata bertipe double serta banyakData bertipe integer. Isi jumlahData dengan nilai awal 0 dan minta user memasukkan banyak data. Lakukan perulangan for untuk melakukan input data serta menghitung jumlah datanya. Terakhir tinggal menghitung rata-ratanya sesuai rumus yang berlaku kemudian nilai rata-ratanya dicetak. Program beserta output-nya dapat diamati pada lampiran foto.

2. JDK (Java Development Kit)
JDK adalah Perangkat lunak yang digunakan untuk melakukan proses kompilasi dari kode java ke bytecode yang dapat dimengerti dan dapat dijalankan oleh JRE (Java Runtime Envirotment). JDK wajib terinstall pada komputer yang akan melakukan proses pembuatan aplikasi berbasis java, namun tidak wajib terinstall di komputer yang akan menjalankan aplikasi yang dibangun dengan java.

JRE (Java Runtime Environment)
JRE adalah perangkat lunak yang digunakan untuk menjalankan aplikasi yang dibangun menggunakan java. Versi JRE harus sama atau lebih tinggi dengan JDK yang dugunakan untuk membangun aplikasi java.

3.JVM :

JVM (Java VIrtual Machine) adalah mechine abstract dengan spesifikasi tertentu yang menyediakan lingkungan untuk runtime untuk menjalankan bytecode dari Java (Apliksi yang dibuat dengan java). 
JVM melakukan tugas-tugas utama sbb:
Membuka kode (Loads code)
Menferifikasi kode
Mengeksekusi kode
Menyediakan runtime environment
JRE
 digunakan untuk menyediakan lingkuran agar dapat mengeksukusi program yang terbuat dari Java. JRE menyediakan seperangkat libraries serta file lain yang digunakan JVM saat runtime atau dengan kata lain JRE merupakan implementasi dari JVM itu sendiri. Jika kita akan menggunakan program yang terbuat dari Java, maka JRE wajib terinstal terlebih dulu pada OS yang akan digunakan.

33. tentang bahasa pemrogram java?​


Penjelasan:

Jawaban:1.Pengertian Bahasa Pemrograman Java

Java dikenal memiliki moto “Write Once, Run Anywhere”. Hal ini menunjukkan bahwa setiap kode program yang ditulis menggunakan Java bisa langsung dijalankan pada platform berbeda-beda tanpa perlu menyusun ulang.

Mengapa bisa demikian?

Penulisan kode program Java biasanya disimpan dalam file berekstensi java. Dengan menggunakan javac compiler, kamu bisa menyusun file berekstensi java menjadi file berekstensi class.

File berekstensi class inilah yang disebut sebagai bytecode dan bisa dijalankan di seluruh Java Virtual Machine (JVM).

Dilihat dari penggunaannya, sebagai bahasa pemrograman umum kamu bisa memanfaatkan Java untuk membuat berbagai bentuk aplikasi.
