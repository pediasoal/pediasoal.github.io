10. Sebutkan Macam-macam tool grup pada tab Ribbon Home dan jelaskan fungsinya


Lima tool group yang ada di ribbon tab Home Microsoft Word antara lain: (berurutan dari sisi kiri ke kanan)

Clipboard

Font

Paragraph

Styles

Editing  

Pembahasan

Tool group pada ribbon tab Home yang berjumlah lima dalam kelompok Microsoft Office hanya terdapat pada Microsoft Word. Kelima tool groups tersebut memiliki komponennya masing-masing yang berguna untuk pengguna dalam melakukan pengolahan dokumen. Kegunaan dan isi dari tool groups masing-masing antara lain:

Clipboard, tool group ini berisi empat icon untuk melakukan operasi cut, copy, paste, dan format painter. Letaknya berada pada sisi paling kiri dari kelima group tersebut.

Font, tool group ini berguna untuk melakukan styling pada font seperti pengaturan jenis font, ukuran font, mengatur jenis case, pembatalan semua formatting, warna font, highlighter, pemberian efek pada font, subscript, superscript, dan pengaturan tebal/miring/garis bawah pada font.

Paragraph, tool group ini berguna untuk melakukan styling pada paragraph di dalam dokumen seperti bullets and numbering, alignment pada text, jarak antar baris dan paragraph, indentasi, shading, sorting, dan borders.  

Styles, tool group ini secara khusus untuk mengatur jenis style teks yang akan diterapkan di dalam dokumen. Terdapat bermacam jenis dan pengguna juga dapat membuat style sendiri dengan menggunakan “Create a style”.

Editing, tool group ini terdiri dari tiga icon yang terdiri dari find, replace, dan select dimana group ini terkait untuk keperluan editing dokumen secara keseluruhan seperti melakukan pencarian kata/frase/kalimat, mereplace suatu kata/frase/kalimat yang sudah ditentukan maupun selection pada dokumen.

21. mengidentifikasi menu dan icon ribbon home



Ribbon Tab Home adalah menu yang paling pertama yang didalamnya terdapat 4 (empat) grup tool dengan fungsinya masing-masing yaitu: Clipboard, Slides, Font, Paragraph, Drawing dan Editing. Grup tool ini berfungsi;

Clipboard, memungkinkan Anda menggunakan tombol copy, paste, cut, dan format painter.
Slide, didalamnya terdapat tombol add slide, layout, resetdan delete.
Paragraph, didalamnya terdapat tombol untuk mengatur posisi rataan (alignment), bullet and numbering, line spacing, dan beberapa tombol untuk mengatur paragraph.
Drawing, didalamnya terdapat tombol Text box, Austoshape, Arrange, Quick Styles, Shape Fill, Shape Outline, dan Shape Effects.
Editing, didalamnya terdiri dari tombol Find, Replace, dan Select.

38. Sebutkan 5 toolgroup yang ada pada ribbon tab home?


Fungsi Toolgroup pada Ribbon Tabs PowerPoint
1. Tab Home

Saat Anda membuka lembar Kerja Ms. PowerPoint pertama kali, maka Anda akan dibawa ke halaman Tab Home.

2.Tab Insert
Pada tab insert, terdapat 5 toolgroup, yaitu: tables, illustrations, links, text dan media clips.
a. Tables

Pada toolgroup tables terdapat table, yang berifungsi untuk menyisipkan atau membuat tabel pada dokumen.

b. Illustrations

Pada toolgroup illustrations terdapat 6 tombol, yaitu:

Insert picture from file, berfungsi menyisipkan objek dari file tertentu, baik dari hard disk, CD, atau yang lainnya.ClipArt, berfungsi menyisipkan objek yang telah tersedia di dalam komputerNew Photo Album, berfungsi membuat album fotoShapes, berfungsi menyisipkan bentuk-bentuk shapeInsert SmartArt Graphic, berfungsi membuat baganInsert Chart, berfungsi menyisipkan objek dengan bentuk grafik

c. Link

Toolgroup link berisikan 2 buah tombol yan dapat digunakan untuk membuat link pada slide, yaitu:

Insert Hyperlink, berfungsi membuat link untuk web page, alamat email, gambar, atau program.Action, berfungsi membangun action sebuah objek

d. Text

Pada toolgroup text, terdapat 7 tombol, yaitu:

Text box, berfungsi menambahkan kotak tulisanHeader dan footer, berfungsi menambahkan catatan kepala dan catatan kakiWordArt, berfungsi menyisipkan teks dekoratif ke dalam dokumenDate and Time, berfungsi menyisipkan tanggal dan waktu sekarang ke dalam dokumen yang sedang dibuat.Insert Slide Number, berfungsi menyisipkan nomor pada slideSymbol, berfungsi menyisipkan karakter yang tidak terdapat pada keyboard.Insert Objek, berfungsi menyisipkan objek dari file

e. Media Clips

Pada toolgroup media clips, terdapat tombol-tombol untuk memperkaya tampilan slide, dimana Anda dapat menambahkan file sound/suara maupun movie/film. Ada 2 tombol pada toolgroup ini, yaitu:

Movie from file, berfungsi menyisipkan file video ke dalam presentasiSound from file, berfungsi menyisipkan file musik atau suara ke dalam suara.
