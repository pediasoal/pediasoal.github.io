1. Diketahui:

Fungsi produksi: Q = 150 + 10x² − (2 /3 )x³

Q adalah output dan X adalah input, maka Q=f(X) atau fungsi dari X.

Ditanya:

Nilai X jika nilai fungsi Q bernilai maksimum?

Nilai X jika nilai fungsi Q sebesar dua kali nilai X (Q=2X).

Jawab:

a. Fungsi produksi diubah ke bentuk turunannya yaitu Q' atau dQ/dx.

Q = 150 + 10x² − 2 /3 x³

Q' = 2*10x − 2*(2/3 )x²

Q' = 20x − 2 x²

Output Q bernilai maksimum jika Q'=0

0 = 20x − 2 x²

0 = x(20 − 2x)

0 = 20 − 2x

2x = 20

x = 10

Fungsi Q bernilai maksimum jika X=10.

Q = 150 + 10(10)² − 2 /3 (10)³

Q = 150 + 1000 - 666,66

Q = 483,33

Fungsi Q yang maksimum bernilai Q=483,33.

b. Keuntungan maksimum didapat bila produk marjinal (MP) sama dengan rasio antara harga input (PX) dibagi harga output (PQ), sedangkan MP merupakan turunan pertama dari fungsi produksi yaitu Q' = 20x − 2 x².

MP = PX : PQ

Q' = PX : PQ

Fungsi produksi untuk turunan pertama atau Q'

Q = 150 + 10x² − 2 /3 x³

Q' = 20x − 2 x²

Fungsi produksi untuk turunan kedua atau Q''

Q' = 20x − 2 x²

Q'' = 20 − 4 x

Output Q' bernilai maksimum jika Q''=0

0 = 20 − 4x

4x = 20

x = 5

Fungsi Q' bernilai maksimum jika X=5.

Q' = 20(5) − 2 (5)²

Q' = 100 − 50

Q' = 50

Fungsi Q' yang maksimum bernilai Q'=50 atau MP=50.

Q = 150 + 10(5)² − 2 /3 (5)³

Q = 150 + 250 − 41,66

Q = 358,33

Fungsi Q untuk X=5 bernilai Q=358,33 atau PQ=358,33.

Nilai PX dicari dengan persamaan Q' = PX : PQ

Q' = PX : PQ

50 = PX : 358,33

PX = 358,33*50

PX = 17916,66

# Pertama, subtitusikan titik (0,16),(1,9) dan (2,4) pada persamaan y

ax² + bx + c = y

(0, 16) ⇒ a (0)² + b (0) + c = 16

              c = 16

(1, 9) ⇒ a (1)² + b (1) + c = 9

             a + b + c = 9

(2,4) ⇒ a (2)² + b (2) + c = 4

             4a + 2b + c = 4


# Menentukan nilai a dan b

a + b + c = 9

a + b + 16 = 9

a + b =  9 - 16

a + b = -7 ... persamaan (1)


4a + 2b + c = 4

4a + 2b + 16 = 4

4a + 2b = 4 - 16

4a + 2b = -12

2a + b = -6 ... persamaan (2)


Eliminasikan persamaan (1) dan (2)

a + b = -7

2a + b = -6

________  _

-a = -1

a = 1


Subtitusikan nilai a pada persamaan (1)

a + b = -7

1 + b = -7

b = -7 - 1

b = -8


# Masukkan nilai a dan b ke dalam persamaan y:

y = 1x² + -8x + 16

Soal 2

Rumus untuk fungsi kuadrat yang memotong dua titik disumbu x adalah

y = a(x - x1)(x - x2)

Titik potong sumbu x terjadi pada (2,0) dan (3,0)

x1 = 2

x2 = 3

Selanjutnya masukkan nilai x pada fungsi y

y = a(x - x1) (x - x2)

y = a(x - 2) (x - 3) ... persamaan (1)


Inilah saatnya kita menggunakan titik yang lagi satu, yaitu (0, 6).

Titik (0,6) bisa diartikan bahwa :

x = 0

y = 6


Masukkan nilai x dan y ke persamaan (1)

y = a(x - 2) (x - 3)

6 = a (0 - 2) (0 - 3)

6 = a (-2) (-3)

6 = a (-6)

-6a = 6

a = -1


Sekarang kita masukkan nilai a di atas ke persamaan (1)

y = a (x - 2) (x - 3)

y = -1 (x - 2) (x - 3)

y = -1 (x² -3x  - 2x + 6)

y = -x² + 5x - 6

17. Soal:1.)Apa fungsi dari benang sari dan putik?Note: Apa cuma saya yg gak bisa pelajaran mtk tp malah jawab soal mtk :v​


Jawaban:

Fungsi Benang Sari dan Putik adalah untuk memperoleh Individu baru pada tumbuhan.

Penjelasan:

Benang Sari adalah sel kelamin jantan, Sedangkan Putik adalah sel kelamin betina pada Tumbuhan. Benang Sari dan Putik berfungsi untuk memeroleh individu baru. Adapun nama proses perkembangbiakan ini, yaitu Proses Penyerbukan. Proses Penyerbukan adalah proses jatuhnya serbuk sari yang dimiliki oleh Benang Sari ke Kepala Putik. 18. Diketahui fungsi produksi : TP = Y = 27 X2 – 2 X3, X adalah penggunaan input Tentukan : a) TP maksimum b) MP maksimum c) AP pada saat X = 4


Jawaban:

Untuk menentukan TP maksimum, kita perlu mencari titik di mana turunan kedua fungsi produksi terhadap input (X) sama dengan nol. Dalam hal ini, kita akan mencari titik maksimum dengan memeriksa perubahan tanda dari turunan kedua.

a) TP Maksimum:

Untuk mencari titik maksimum TP, kita harus mencari turunan pertama dan kedua fungsi produksi terhadap X.

TP = 27X^2 - 2X^3

Turunan pertama terhadap X:

TP' = 54X - 6X^2

Turunan kedua terhadap X:

TP'' = 54 - 12X

Untuk menentukan TP maksimum, kita atur TP'' = 0:

54 - 12X = 0

12X = 54

X = 4.5

Menggunakan uji perubahan tanda, kita dapat menentukan bahwa TP mencapai maksimum pada X = 4.5.

b) MP Maksimum:

Untuk menentukan MP maksimum, kita perlu mencari turunan pertama fungsi produksi terhadap X.

TP' = 54X - 6X^2

MP adalah turunan pertama fungsi produksi terhadap X, jadi:

MP = 54X - 6X^2

Untuk menentukan MP maksimum, kita dapat mencari turunan kedua MP terhadap X:

MP' = 54 - 12X

Kita set MP' = 0 untuk mencari nilai X yang menghasilkan MP maksimum:

54 - 12X = 0

12X = 54

X = 4.5

Menggunakan uji perubahan tanda, kita dapat menentukan bahwa MP mencapai maksimum pada X = 4.5.

c) AP pada saat X = 4:

AP (Average Product) adalah TP dibagi oleh jumlah input yang digunakan (X):

AP = TP / X

Untuk mencari AP saat X = 4, kita substitusikan nilai X = 4 ke dalam fungsi produksi:

TP = 27X^2 - 2X^3

TP = 27(4)^2 - 2(4)^3

TP = 27(16) - 2(64)

TP = 432 - 128

TP = 304

Kemudian, kita bagi TP dengan X:

AP = TP / X

AP = 304 / 4

AP = 76

Jadi, AP pada saat X = 4 adalah 76.
