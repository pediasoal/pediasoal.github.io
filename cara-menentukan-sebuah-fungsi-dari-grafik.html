a>0 maka grafik terbuka keatas
a<0 maka grafik terbuka ke bawah

D>0 maka grafik memotong sumbu x di 2titik
D= 0 maka grafik memotong sumbu x di 1 titik(menyinggung)
D< 0 maka grafik tidak memotong sumbu x

1.)y=3x²-6x+5, a= 3,b=-6,c=5
a>0 maka grafik terbuka keatas
2.) D= b²-4ac
D= (-6)²-4.3.5=36-60=-24
D<0 maka grafik tidak memotong sumbu x
