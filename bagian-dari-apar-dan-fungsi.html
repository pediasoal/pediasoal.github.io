3. jelaskan bagian bagian dari apar dan cara penggunaannya.​


Jawaban:

1.hose/Selang

Selang atau bisa juga disebut hose di dalam APAR berfungsi untuk menyalurkan media pemadam api dari dalam tabung menuju ke ujung nozzle

2. Nozzle

Bagian nozzle adalah corong keluarnya media pemadam api menuju ke titik api. Sehingga, kebakaran dapat dipadamkan. Jadi, pastikan nozzle tidak tersumbat, agar media pemadam api bisa keluar dengan maksimal.

3. Handle / Tuas

Tuas APAR atau handle berfungsi untuk membuka spindle. Saat tuas APAR ditekan, maka gas bertekanan akan membawa media pemadam api keluar dari dalam tabung APAR.

4. Tabung

Tabung berfungsi untuk menyimpan media di dalam APAR.

5. Safety Pin

Komponen ini berfungsi untuk menahan tuas APAR saat APAR tidak digunakan. Sehingga, jika tuas APAR tidak sengaja tertekan, tidak akan mengakibatkan media pemadam api keluar. Jadi, jika kamu mau menggunakan APAR harus mencabut safety pin terlebih dahulu supaya tuas APAR bisa ditekan.

5. Pressure Gauge

Ini adalah komponen yang digunakan untuk mengecek tekanan di dalam tabung APAR

24. sebutkan 3 jenis bahan apar beserta fungsinya?​


Penjelasan:

1. Alat Pemadam Api (APAR) Air / Water

APAR Jenis Air (Water) adalah Jenis APAR yang disikan oleh Air dengan tekanan tinggi.

2. Alat Pemadam Api (APAR) Busa / Foam (AFFF)

APAR Jenis Busa ini adalah Jenis APAR yang terdiri dari bahan kimia yang dapat membentuk busa.

3. Alat Pemadam Api (APAR) Serbuk Kimia / Dry Chemical Powder

APAR Jenis Serbuk Kimia atau Dry Chemical Powder Fire Extinguisher terdiri dari serbuk kering kimia yang merupakan kombinasi dari Mono-amonium danammonium sulphate.

Jawaban:

Jenis-jenis APAR (Alat Pemadam Api Ringan):

1. Alat Pemadam Api (APAR) Air / Water,berfungsi untuk memadamkan api yang dikarenakan oleh bahan-bahan padat non-logam seperti Kertas, Kain, Karet, Plastik dan lain sebagainya

2. Alat Pemadam Api (APAR) Busa / Foam (AFFF, efektif untuk memadamkan api yang ditimbulkan oleh bahan-bahan padat non-logam seperti Kertas, Kain, Karet dan lain sebagainya (Kebakaran Kelas A) serta kebakaran yang dikarenakan oleh bahan-bahan cair yang mudah terbakar seperti Minyak, Alkohol, Solvent dan lain sebagainya (Kebakaran Jenis B).

3. Alat Pemadam Api (APAR) Serbuk Kimia / Dry Chemical Powder,  efektif untuk memadamkan kebakaran di hampir semua kelas kebakaran seperti Kelas A, B dan C.

4. Alat Pemadam Api (APAR) Karbon Dioksida / Carbon Dioxide (CO2,  sangat cocok untuk Kebakaran Kelas B (bahan cair yang mudah terbakar) dan Kelas C (Instalasi Listrik yang bertegangan).

40. bagian ulasan yang memaparkan suatu pandangan seseorang tentang hal atau karya yang di ulas adalah


Telah terdapat bagian ulasan yang di mana telah memaparkan mengenai sebuah pandangan dari seseorang mengenai sebuah karya ataupun hal yang telah diulas yaitu ialah tafsiran.

Pembahasan

Teks ulasan yaitu ialah sebuah teks yang telah berisi dengan ulasan mengenai suatu karya tertentu. Umumnya, suatu teks ulasan akan  mengulas suatu karya seperti sebuah drama ataupun film.

Ulasan yang telah disajikan di dalam teks ini telah disajikan dengan sangat lengkap di dalam struktur teks ulasan yang telah disusun secara berurutan pula.

Tujuan yang dimiliki oleh sebuah teks ulasan yaitu ialah menyampaikan kepada para pembaca mengenai sebuah ulasan karya seperti drama atau film tersebut untuk dapat menarik minat dari para pembaca agar mereka dapat menjadi tertarik untuk menonton karya tersebut.

Telah terdapat pula struktur dari teks ulasan yaitu ialah sebagai berikut:

Orientasi Tafsiran Evaluasi Rangkuman

Umumnya, sebuah teks ulasan ini dapat pula disebut sebagai sebuah resensi. Teks ulasan ini dapat pula diciptakan apabila telah terdapat sebuah peristiwa tertentu seperti olahraga ataupun kegiatan sosial yang lainnya.

Selain itu, teks ulasan telah memiliki fungsi yaitu untuk dapat mengajukan, menilai, dan juga menimbang sebuah kritik kepada suatu peristiwa ataupun karya yang telah diulas.
