3. pengertian dari Hibernate Query Language


Hibernate Query Language (HQL) adalah bahasa query berorientasi objek, mirip dengan SQL, tapi bukannya beroperasi pada tabel dan kolom, HQL bekerja dengan objek persisten dan sifat mereka. query HQL diterjemahkan oleh Hibernate dalam query SQL konvensional yang pada gilirannya melakukan tindakan pada database.

4. sebutkan dan jelaskan fungsi masing masing dari tipe data dan variabel!​


Variabel sendiri dapat disebut sebagai simbol yang digunakan untuk menyimpan sebuah nilai. Sedangkan tipe data  yakni disebut dengan jenis nilai yang akan disimpan. Tipe-tipe data sangatlah banyak.

Pembahasan

Macam-macam tipe data antara lain :  

Integer : Tipe data ini dapat didefinisikan sebagai integer. Jadi program yang menggunakan tipe data integer tidak akan mendukung penggunaan huruf. Char : merupakan tipe data yang biasanya terdiri dari angka, tanda baca, huruf atau bahkan karakter khusus. Untuk menyimpan karakter, diperlukan 1 byte atau 8 bit ruang dalam memori. Array : tipe data ini termasuk dalam tipe data komposit karena dapat menyimpan. Data yang disimpan dalam tipe data array harus dari tipe yang sama atau homogen. Penggunaan tipe data array ini berbeda dari tipe data lain, yang tidak secara langsung dinyatakan, tetapi tipe data ini sering memainkan peran penting dalam penggunaan data agar lebih terstruktur. Date : tipe data yang digunakan untuk menyimpan data dari tipe tanggal, bulan dan tahun, misalnya tanggal lahir, tanggal transaksi. Numerik : tipe data yang dapat digunakan untuk menyimpan data dalam bentuk angka. Contoh 6.00 Float : tipe data yang sering disebut sebagai tipe data bilangan real. Small INT : tipe data integer yang jangkauannya lebih kecil dari integer. Decimal : tipe data yang dapat menampung data fraksional. String : Seringkali tipe data yang terdiri dari serangkaian karakter yang memiliki panjang tertentu, tipe data string ini sering dianggap sebagai tipe data dasar. Double : tipe data tipe float tetapi dengan contoh akurasi yang lebih tinggi 5.555555 Pelajari lebih lanjut

14. pal: . Ada empat tipe obyek yang ada dalam program aplikasi Microsoft Office, yaitu table, query, form, dan report. Jelaskan apa arti dari keempat tipe obyek tersebut?​


Jawaban:

Berikut adalah penjelasan mengenai keempat tipe obyek dalam program aplikasi Microsoft Office:

1. Table (Tabel): Tabel adalah obyek yang digunakan untuk menyimpan dan mengatur data dalam format yang terstruktur. Tabel terdiri dari kolom dan baris yang membentuk bidang data. Tabel digunakan untuk mengorganisir dan menyimpan informasi dalam database.

2. Query (Kueri): Query adalah obyek yang digunakan untuk mengambil, memfilter, dan memanipulasi data dari tabel atau sumber data lainnya. Dengan menggunakan query, Anda dapat mengambil data yang spesifik sesuai dengan kriteria yang ditentukan, melakukan perhitungan atau perubahan pada data, serta menggabungkan data dari beberapa tabel.

3. Form (Formulir): Formulir adalah obyek yang digunakan untuk memasukkan, mengedit, dan menampilkan data secara interaktif. Formulir menyediakan antarmuka pengguna yang lebih terstruktur dan terorganisir dibandingkan dengan tampilan tabel. Dengan menggunakan formulir, pengguna dapat dengan mudah memasukkan data ke dalam tabel dan melakukan operasi lainnya, seperti validasi data atau navigasi antarrekaman.

4. Report (Laporan): Laporan adalah obyek yang digunakan untuk menghasilkan tampilan cetak atau digital yang terstruktur dari data. Laporan biasanya digunakan untuk menganalisis, menyajikan, dan memvisualisasikan data dalam format yang lebih terstruktur dan informatif. Anda dapat mengatur tata letak, menambahkan grafik atau elemen visual lainnya, serta menambahkan ringkasan atau pemformatan khusus pada laporan.

Keempat tipe obyek tersebut saling berinteraksi dan digunakan dalam aplikasi Microsoft Office, seperti Microsoft Access atau Microsoft Excel, untuk mengelola dan menganalisis data dengan lebih efektif.

18. secara dasar ada dua tipe data manipulation language sebutkan dan jelaskan


Secara dasar, terdapat dua tipe data manipulation language (DML), yaitu:


1. Prosedural

DML Prosedural digunakan dalam bahasa pemrograman tingkat tinggi (contoh: Java, PHP, C) dengan cara pengguna memberikan spesifikasi kepada sistem tentang data apa yang dibutuhkan dan bagaimana cara memperoleh data tersebut.


2. Non prosedural (deklaratif)

DML non prosedural digunakan dalam bahasa pemrograman deklaratif (contoh: SQL, QBE) dengan cara pengguna memberikan spesifikasi data yang diperlukan tanpa perlu tahu bagaimana cara memperoleh data tersebut.


Perbedaan DML dan DDL

Selain DML, dalam konsep manajemen database juga dikenal adanya DDL (Data Definition Language). DDL adalah kumpulan perintah untuk membuat, mengubah, dan menghapus database maupun objek yang terdapat dalam database (tabel, indeks, prosedur, trigger). Contoh perintah DDL: CREATE, ALTER, DROP. Sedangkan DML sendiri merupakan kumpulan perintah untuk memanipulasi data yang terdapat dalam sebuah tabel. Contoh perintah DML: INSERT, UPDATE, DELETE.

19. Sebutkan yang termasuk kategori mengupdate data query


Select query

Sebuah select query adalah jenis query yang paling umum. Query ini mengambil data dari satu atau lebih tabel dan menampilkan hasilnya dalam datasheet dimana Anda dapat mengupdate record (dengan beberapa pembatasan). Anda juga dapat menggunakan select query untuk membuat mengelompokkan data dan menghitung jumlah dengan sum, menghitung dengan count, menghitung rata-rata, dan menghitung total.

Parameter query 

Sebuah query parameter adalah bahwa ketika query dijalankan akan menampilkan kotak dialog yang meminta Anda untuk mengisi parameter atau nilai variabel, seperti kriteria untuk mengambil record atau nilai yang ingin Anda masukkan dalam lapangan. Anda dapat merancang query yang meminta Anda untuk memesukkan lebih dari satu informasi, misalnya, Anda dapat merancang parameter query untuk dua tanggal. Akses kemudian dapat mengambil semua record yang berada diantara dua tanggal.

Parameter query juga berguna bila digunakan sebagai record source untuk form, report, dan access data page. Sebagai contoh, Anda dapat membuat laporan penghasilan bulanan berdasarkan parameter query. Ketika Anda mencetak laporan, Access akan menampilkan kotak dialog yang meminta untuk mengisi bulan yang akan ditampilkan pada laporan tersebut. Anda memasukkan bulan dan Akses mencetak laporan yang sesuai dengan kriteria bulan.

Crosstab query

Anda menggunakan crosstab query untuk menghitung dan restrukturisasi data agar analisis data lebih mudah. Crosstab query menghitung jumlah sum, rata-rata, jumlah count, atau jenis lain dari total data yang dikelompokan oleh dua jenis informasi - satu di sisi kiri dari datasheet dan yang lain di bagian atas.

Action query

Sebuah action query adalah query yang membuat update pada banyak record hanya dalam satu operasi. Ada empat jenis action query:

Delete Queries Sebuah delete query menghapus sekelompok record dari satu atau lebih tabel. Sebagai contoh, Anda bisa menggunakan delete query untuk menghapus produk yang dihentikan atau yang tidak ada dalam order. Dengan delete query, Anda selalu menghapus keseluruhan record, bukan hanya fileds yang dipilih dalam record.Update Query Sebuah update query membuat perubahan menyeluruh untuk sekelompok record dalam satu atau lebih tabel. Sebagai contoh, Anda dapat menaikkan harga sebesar 10 persen untuk semua produk susu, atau Anda dapat menaikkan gaji sebesar 5 persen untuk orang dalam kategori pekerjaan tertentu. Dengan update query, anda dapat mengubah data dalam tabel yang ada.Append Query Append query menambahkan sekelompok record dari satu tabel atau lebih pada akhir satu atau lebih tabel. Sebagai contoh, misalkan Anda mendapatkan beberapa pelanggan baru dan database berisi tabel informasi tentang pelanggan tersebut. Untuk menghindari mengetik semua informasi ini ke dalam database Anda sendiri, Anda dapat menambahkan pada tabel Pelanggan Anda.Make-Table Query Make-table query dapat membuat tabel baru dari seluruh atau sebagian dari data dalam satu atau lebih tabel. Membuat-tabel query sangat membantu untuk menciptakan tabel untuk ekspor ke Microsoft Access database lain atau tabel history yang berisi catatan lama.
SQL query

Sebuah query SQL adalah query yang anda buat dengan menggunakan pernyataan SQL. Anda dapat menggunakan Structured Query Language (SQL) untuk query, memperbarui, dan mengelola database relasional seperti Access.

Bila Anda membuat sebuah query dalam query Design view, Akses akan membuat SQL yang setara di belakang layar untuk Anda. Jika Anda ingin, Anda dapat melihat atau mengedit sintak SQL pada SQL view. Namun, setelah Anda membuat perubahan untuk query dalam tampilan SQL, query mungkin tidak ditampilkan dengan cara sebelumnya dalam tampilan Design.

27. jelaskan fungsi dari tipe data pada M.s Acces


Jawaban:

 

Fungsi dari tipe data pada MS Acces adalah:

 

1.    Text: tipe data ini berfungsi untuk menyimpan teks, angka atau kombinasinya. Ukuran yang dapat ditampung dalam tipe data ini maksimum adalah 255 karakter.         

2.    Memo: tipe data ini berfungsi untuk menyimpan teks, angka atau kombinasinya dan mirip seperti tipe data Text, namun tipe data Memo memiliki kapasitas lebih besar hingga 65,536 karakter. Meski lebih besar kapasitasnya, kelemahan tipe data Memo ini adalah tidak dapat disortir, namun masih dapat di-search (dicari isinya).

3.    Byte: untuk menyimpan angka dengan nilai antara 0 sampai 255.

4.    Integer: untuk menyimpan angka dengan nilai antara -32,768 dan 32,767.

5.    Long: untuk menyimpan angka dengan nilai antara -2,147,483,648 and 2,147,483,647.

6.    Single: untuk menyimpan data dengan jenis Single precision floating-point dengan ukuran 32 bit (4 byte)

7.    Double: untuk menyimpan data dengan jenis Double precision floating-point dengan ukuran 64 bit (8 byte)

8.    Currency: untuk menyimpan data, yang terdiri dari 15 digit nilai mata uang, umumnya dollar, dan 4 digit nilai desimal.

9.    AutoNumber: untuk menyimpan nilai yang diisi secara otomatis oleh database, dimulai dari angka 1.

10. Date/Time: untuk menyimpan yang terdiri dari tahun, bulan dan tanggal

11. Yes/No: untuk menyimpan nilai logika seperti ya/tidak, benar/salah, on/off. Tipe data ini tidak boleh kosong.

12. Ole Object: untuk menyimpan file berukuran besar seperti gambar, video dan musik, atau data BLOB (binary large object) lainya

13. Hyperlink: untuk menyimpan link ke halaman web.

31. Tipe data untuk fungsi mempunyai beberapa macam. Sebutkan macam-macam tipe data untuk dungsi


Jawaban:

Integer: tipe data yang digunakan untuk menyimpan bilangan bulat seperti 1, 2, 3, dan seterusnya. Contohnya pada bahasa pemrograman C++, Java, Python, dan sebagainya.

Float/double: tipe data yang digunakan untuk menyimpan bilangan pecahan seperti 1.23, 3.14, dan seterusnya. Contohnya pada bahasa pemrograman C++, Java, Python, dan sebagainya.

String: tipe data yang digunakan untuk menyimpan teks atau karakter seperti "Halo", "Nama Saya", dan seterusnya. Contohnya pada bahasa pemrograman C++, Java, Python, dan sebagainya.

Boolean: tipe data yang digunakan untuk menyimpan nilai kebenaran (true/false). Contohnya pada bahasa pemrograman C++, Java, Python, dan sebagainya.

Array/List: tipe data yang digunakan untuk menyimpan kumpulan nilai yang serupa. Contohnya pada bahasa pemrograman C++, Java, Python, dan sebagainya.

Object/Class: tipe data yang digunakan untuk mengelompokkan data dan fungsi ke dalam satu kesatuan. Contohnya pada bahasa pemrograman Java, Python, dan sebagainya.

Pointer/Reference: tipe data yang digunakan untuk menyimpan alamat memori dari suatu variabel atau fungsi. Contohnya pada bahasa pemrograman C++, dan sebagainya.

Update data melalui query

a. Membuat query


    Sebelum memulai kegiatn ini,sebaiknya kamu tahu terlebih dahulu pengertian dari query.query bisa    diartikan sebagai pernyataan atu permintaan untuk memproses tabel yang ada.untuk mengupdate data melalui query,kamu harus memiliki query terlebih dahulu.namun jika kamu belum memiliki query,maka kamu harus membuatnya terlebih dahulu.terdapat dua cara membuat query

yaitu:   1.      Create query in design view

           2.      Create query by using wizard


Kita gunakan metode create query by using wizard.berikut adalah langkah-langkahnya. Pada jendela objects pilih queries,kemudian klik ganda pada create query by using wizard. Pilih nama pada tabel daftar.jika ingin ditampilkan seluruh field dalam query yang kamu buat,klik tombol >> namun jika hanya ingin dipilih beberapa field saja,klik tombol  > sekarang pilih seluruh field dan klik next selanjutnya,dalam pilihan yang diberikan,pilihlah detail dan klik next. Selanjtnya,kamu akan diberi pilihan untuk memberi nama query yang kamu buat.secara otomatis,ms access memberi nama sesuai nama tabel yang dirujuk,yaitu daftar query kemudian klik finish.


Kita dapat memilih untuk menampilkan field tertentu setelah query dibuat.bagaimana caranya?lakukanlah langkah-langkah berikut :

=> simpan hasil query yang kamu buat dan tutup query

=> klik kanan daftar query dan pilih design view

=> selanjutnya kamu akan mendapatkan jendela design view

=> jika kamu hanya ingin menampilkan field nama siswa,prog.pilihan,math,bhs inggris,maka berikan tanda   

     checklist pada bagian show hanya pada field ini saja

=> simpan dengan nama daftar query baru.selanjutnya,buka daftar query baru.


Bagaimana menampilkan jumlah nilai dan nilai rata-rata pada query?perhatikan langkah-langkah berikut :


buka query daftar query.pada kolom jumlah nilai,ketikkan rumus berikut:


Jumlah nilai:[ppkn]+[bhs indo]+[math]+[bhs inggris]+[nilai ipa]+[nilai ips]


Simpan hasil perubahan yang kamu buat di design view


selanjutnya,kamu dapat melihat query daftar query telah menunjukkan jumlah nilai siswa.


jika kamu menginginkan   juga nilai rata-rata ikut diperlihatkan dalam query.beri tanda checklist pada kolom nilai rata-rata di design view,kemudian ketikkan persamaan berikut:


Nilai rata-rata: [jumlah nilai]/6


simpan perubahan yang kamu lakukan dan keluarlah dari design view.selanjutnya buka kembali daftar query


kamu dapat mengubah jumlah desimal dengan mengklik kanan kolom.nilai rata-rata pada jendela design view,kemudian klik properties

ubah format pada filed properties menjadi  standard

simpan hasil perubahan query dengan menekan ctrl+s dan buka daftar query.
