13. Line touches circle P. if AP = 39 cm and PB = 15 cm, the length of the tangent line AB is...cm


Bab Segitiga
Matematika SMP Kelas VII

Triple Pitagoras
AB = √(AP² - PB²)
AB = √(39² - 15²)
AB = √(1.521 - 225)
AB = √(1.296)
AB = 36 cm

20. Apa fungsi Polygon tool di corel draw


Fungsi Polygon tool  yaitu untuk menggambar dengan bentuk persegi secara banyak, sekedar dengan melakukan atau mengaktifkan drag dan klik mouse. Memiliki turunan 4 macam tools, yaitu :

Star tool : berfungsi untuk menggambar berbagai macam bentuk bintang.

Complex Star tool : berfungsi untuk menggambar bentuk bintang yang mempunyai banyak potongan sisi.

Graph Paper tool : berfungsi  untuk menggambar bentuk rangkaian  susunan kotak-kotak seperti jala-jala.

Spiral tool : berfungsi  untuk menggambar bentuk lingkaran spiral secara simetris atau logaritmis

Fungsi polygon tool
