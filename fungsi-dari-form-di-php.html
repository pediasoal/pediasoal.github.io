3. Cara Membuat Form Login Dan Register Dengan HTML, PHP


Untuk HTML, silahkan buat tag <form>.
Di dalam tag <form> tersebut, nanti ada tag lain yang berfungsi mendapat data dari user, yaitu <input>. <input> ini punya attribute type="" yang bisa menentukan apa yang diisi oleh user. Misal, <input type="password"> akan membuat apa yanh diketik user berubah jadi bulatan² seperti password pada umumnya. <input type="date"> akan mengisi tanggal. Jika ingin lebih tahu lagi, ada banyak attribute <input> yang lainnya silahkan lihat W3Schools.

Untuk PHPnya, nanti kamu harus mengintegrasikan HTML tag <form> dengan PHPmu. Yaitu dengan attribute action="namafilephp.php". File php ini berisi fungsi yang bisa membuat inputan user disimpan ke database, yaitu MySQL. Silahkan cari sumber lain lagi untuk lebih lengkapnya, karena kode PHP tiap orang beda².

25. Fungsi dari Php fungsi dari sql


PHP dapat disebut dengan bahasa pemrograman umum yang berarti php dapat disematkan ke dalam kode HTML, atau dapat digunakan dalam kombinasi dengan berbagai sistem templat web, sistem manajemen konten web, dan kerangka kerja web.  

Sedangkan SQL sendiri merupakan singkatan dari Structured Query Language. Secara umum SQL dapat diartikan sebagai bahasa yang digunakan dalam mengakses data.  

Pembahasan

Fungsi dari PHP yakni membuat atau mengembangkan situs web statis atau situs web dinamis atau aplikasi Web. PHP digunakan karena untuk membuat website dinamis bisa dipakai untuk menyimpan data ke dalam database, membuat halaman yang dapat berubah-ubah sesuai dengan input  user, memproses form, dan yang lain-lain. Dalam membuat file PHP dapat digabungkan dengan menggunakan tag html, kemudian ketika tanpa menggunakan tag html apa pun disebut sebagai file PHP Murni. Server menginterpretasikan kode PHP lalu mengeluarkan hasilnya sebagai kode HTML ke browser web. Agar server mengidentifikasi kode PHP dari kode HTML, yang dilakukan adalah harus selalu menyertakan kode PHP dalam tag PHP.

Fungsi dari SQL yakni memungkinkan anda mengakses dan memanipulasi database, dapat mengeksekusi query terhadap database, dapat mengambil data dari database, dapat menyisipkan data dalam database, dapat memperbarui data dalam database, dapat menghapus data dari database, dapat membuat database baru, dapat membuat tabel baru dalam database, dapat membuat prosedur yang tersimpan dalam database, dapat menggabungkan beberapa tabel didalam database, dapat mengatur hak akses pada tabel, prosedur, dan yang lain-lain.
