Fungsi f(x) = x³ + 3x² - 9x + 5 naik pada interval x < -3 atau x > 1

Pembahasan

Nilai Stasioner Fungsi

adalah nilai optimum fungsi yang diperoleh dengan cara mensubstitusikan nilai x pada f'(x) sehingga hasilnya nol. Adakalanya nilai maksimum atau minimum.

Interval naik diperoleh jika f'(x) > 0

Interval turun diperoleh jika f'(x) < 0

Diketahui:

Fungsi f(x) = x³ + 3x² - 9x + 5

Ditanya:

Interval naik fungsi f(x)

Jawab:

f(x) = x³ + 3x² - 9x + 5

f'(x) = 3x² + 6x - 9 = 3(x² + 2x - 3) = 3(x + 3)(x - 1)

f(x) naik jika f'(x) > 0

3x² + 6x - 9 > 0

3(x + 3)(x - 1) > 0

x + 3 < 0 atau x - 1 > 0

x < -3 atau x > 1

f(x) naik pada interval x < -3 atau x > 1
