2. • tugas OJK, contoh kasus• tugas BI, contoh kasus​


Kesimpulan:

Fungsi perlindungan dan pengawasan bank telah beralih dari BI ke OJK. Walaupun begitu, pengawasan di perbankan tidak hanya akan dijalankan oleh OJK saja melainkan lebih ke pembagian tugas. OJK bertugas dalam melakukan perlindungan dan pengawasan sektor perbankan secara mikroprudensial. Sedangkan BI bertugas melakukan pengawasan secara makroprudensial.

Mikroprudensial dalam aspek ini adalah bertujuan untuk melindungi konsumen di sektor perbankan dan mengawasi pelaku di sektor perbankan dalam hal yang lebih rinci.

Makroprudensial lebih berfokus pada dampak yang bisa saja timbul dari suatu kasus perbankan pada perekonomian secara agregat.

Contoh Kasus:

OJK : Pengawasan terhadap Lippo dan Bank Nobu terkait Kasus Meikarta

OJK dalam kasus ini bertujuan mengamati dan menyelidiki aliran dana dan keputusan kredit dari Lippo dan Bank Nobu pada kasus Meikarta yang macet.

Contoh Kasus BI:

BI: Pengawasan terhadap kasus pembobolan uang elektronik

BI dalam konteks ini lebih berfokus dalam mengawasi dampaknya pada sistem keuangan dan keamanan sistem pembayaran di Indonesia sedangkan OJK berfokus pada perlindungan data/rekening dari nasabah.

4. contoh program yang menampilkan DEBUGGING, kasus bebas. ​


Jawaban:

1. Mereproduksi Bug

Hal pertama yang harus Anda lakukan adalah mereproduksi bug. Caranya, mendokumentasikan semua issue bug yang ada dengan benar.  

Ada beberapa tujuan mengapa tahap ini sangat penting, yaitu:

Menentukan skala prioritas penyelesaian bug;

Bug dapat diselesaikan dengan tim yang tepat;

Memetakan mana bug yang belum diselesaikan, siap di uji coba, dan lulus quality control;

Uji coba di berbagai perangkat dan browser untuk melihat apakah ada bug lainnya.

Ketika Anda sudah mereproduksi bug artinya Anda sudah mengantongi bug apa saja yang harus dituntaskan. Nah, Anda dapat menunjuk tim-tim terkait untuk melakukan tahap berikutnya.

2. Mengidentifikasi Error

Selanjutnya, lakukan identifikasi error, yaitu dengan mengumpulkan laporan error atau kesalahan pada program. Contohnya ketika perintah software gagal dieksekusi, maka kemungkinan ada syntax bug pada skrip.

Daftar error ini biasanya didapatkan saat melakukan uji coba program, baik pengujian internal hingga uji coba oleh klien.

Pada tahap ini, pastikan benar-benar mengidentifikasi program secara menyeluruh. Sehingga proses debugging lebih efisien dan nantinya tidak ada lagi bug yang masih tersisa.

3. Menemukan Lokasi Bug

Setelah mengidentifikasi, saatnya menemukan di mana letak error di code tersebut berada. Misalnya, di code navigasi, tombol perintah, dll.

Itu mengapa identifikasi bug pada saat uji coba penting dilaksanakan. Karena lokasi bug bisa jadi akan terdeteksi pada saat itu juga.

4. Menganalisis Error

Oke, sudah tahu ada bug dan di mana lokasinya, maka mari analisis error tersebut, agar nantinya dapat dilakukan perbaikan yang tepat.

Tips jitu: Gunakan teknik bottom-up atau analisis dari kode paling kecil di lokasi error hingga ke kesatuan kode disekitarnya.  

Tujuannya, agar Anda mengetahui bug-bug lain yang ada di sekitar kode error. Serta meminimalisir resiko error tambahan pada saat perbaikan.  

Seperti sebuah radar, Anda dapat memperbaiki pada pusat error sekaligus menemukan error-error lainnya. Siapa tahu, kode yang error tersebut memiliki efek untuk kode-kode lainnya, kan.

5. Membuktikan Analisis Error

Nah, hasil analisis bug Anda mungkin menemukan adanya bug di lokasi tertentu. Sebelum memperbaikinya, Anda perlu membuktikan bahwa tidak ada error lainnya yang mungkin saja akan muncul pada software.

Caranya, Anda dapat menulis tes otomatis pada lokasi bug dengan bantuan framework test.

9. contoh soal program linear​


Jawaban:

Gambarlah garis ax + by = c pada bidang kartesius, cara lebih lengkapnya dapat dilihat di sini.

Garis Lurus

Ambil sembarang titik (x1, y1) di luar garis ax + by = c kemudian hitung nilai ax1 + by1 dan bandingkan dengan nilai pada ruas kanan pertidaksamaan (nilai c).

Jika nilai ax1 + by1 ≤ c maka daerah yang memuat titik (x1, y1) adalah daerah penyelesaian berada di bawah garis ax + by = c.

Daerah layak kurang dari

Jika ax1 + by1 ≥ c maka daerah yang memuat titik (x1, y1) adalah daerah penyelesaian berada di atas pertidaksamaan ax + by = c.

Daerah Layak Penyelesaian Sistem Persamaan Linear

Contoh cara menentukan daerah himpunan penyelesaian dari sistem pertidaksamaan linear dua peubah. Diberikan sebuah sistem pertidaksamaan linear yang terdiri dari empat pertidaksamaan. Perhatikan sistem pertidaksamaan berikut.

x ≥ 0

y ≥ 0

x + y ≤ 7

x + 3y ≤ 15

Cari tahu daerah yang memenuhi sistem pertidaksamaan linear di atas.

Daerah yang memenuhi pertidaksamaan x + y ≤ 7.

Penyelesaian Daerah Layak Kurang Dari

Daerah yang memenuhi pertidaksamaan x + 3y ≤ 15.

Penyelesaian daerah layak kurang dari

Menentukan daerah yang memenuhi gabungan dari empat sistem pertidaksamaan linear: x ≥ 0, y ≥ 0, x + y ≤ 7, dan x + 3y ≤ 15.

25. Berikan contoh kasus Coorporate Social Responsibility (CSR) beserta bentuk program nya?​


Contoh Kasus: PT ABC, sebuah perusahaan manufaktur yang beroperasi di wilayah pedesaan.

Bentuk Program CSR:

1. Pendidikan dan Pelatihan: PT ABC dapat memberikan beasiswa kepada anak-anak desa agar mereka dapat melanjutkan pendidikan ke jenjang yang lebih tinggi. Selain itu, perusahaan juga dapat menyelenggarakan pelatihan keterampilan untuk membantu meningkatkan kemampuan kerja masyarakat setempat.

2. Infrastruktur Desa: Perusahaan bisa mendukung pembangunan infrastruktur seperti jalan raya, saluran air bersih, atau fasilitas kesehatan di desa sekitar tempat operasionalnya. Hal ini akan meningkatkan aksesibilitas dan kualitas hidup masyarakat setempat.

3. Lingkungan Hidup: PT ABC dapat melakukan program penghijauan dengan menanam pohon-pohon di sekitar area produksi atau mengadakan kampanye pengurangan limbah plastik bagi komunitas lokal.

4. Pengembangan Usaha Mikro: Perusahaan bisa memberdayakan warga desa dengan mendirikan kelompok usaha mikro seperti home industry produk-produk daur ulang atau kerajinan tangan tradisional. Ini akan membantu meningkatkan ekonomi lokal dan menciptakan lapangan pekerjaan baru.

5. Kesehatan Masyarakat: Perusahaan bisa memberikan dukungan dalam bidang kesehatan seperti penyuluhan tentang pola hidup sehat, vaksinasi massal, atau memberikan bantuan medis kepada masyarakat yang membutuhkan.

6. Program Kemitraan: PT ABC dapat menjalin kemitraan dengan lembaga sosial atau organisasi nirlaba setempat untuk mengimplementasikan program-program CSR secara berkelanjutan dan efektif.

Penting untuk dicatat bahwa program CSR harus disesuaikan dengan kebutuhan dan karakteristik masyarakat setempat serta sesuai dengan visi misi perusahaan. Selain itu, melibatkan komunitas lokal dalam perencanaan dan pelaksanaan program akan meningkatkan dampak positif yang dihasilkan


26. buatlah 2 contoh kasus hukum politik yang ada di Indonesia beserta:kasus, sanksi,denda (kalo ada),dan akhir dikasus tersebut!​mohon di bantu ya kakak" soalnya besok mau dikumpul:)


**Contoh 1: Kasus Penggelapan Dana E-KTP**

**Kasus:**

Pada tahun 2011, terjadi kasus dugaan korupsi pengadaan E-KTP yang melibatkan sejumlah pejabat negara, termasuk mantan Ketua Umum Partai Golkar Setya Novanto. Kasus ini menyeret nama-nama besar di partai politik dan pemerintahan, sehingga menjadi salah satu kasus hukum politik terbesar di Indonesia.

**Sanksi:**

Setya Novanto divonis 15 tahun penjara dan denda Rp 1 miliar dalam kasus ini. Selain itu, ia juga dihukum membayar uang pengganti sebesar Rp 7,6 triliun.

**Akhir Kasus:**

Setya Novanto mengajukan banding atas vonis tersebut. Namun, pada tahun 2019, Mahkamah Agung (MA) menolak banding tersebut dan tetap memvonis Setya Novanto dengan hukuman 15 tahun penjara.

**Contoh 2: Kasus Kriminalisasi KPK**

**Kasus:**

Pada tahun 2015, terjadi upaya kriminalisasi terhadap Komisi Pemberantasan Korupsi (KPK). Upaya ini dilakukan dengan cara menerbitkan surat perintah penyelidikan (sprindik) terhadap pimpinan KPK. Sprindik ini dianggap sebagai upaya untuk melemahkan KPK dan menghambat upaya pemberantasan korupsi di Indonesia.

**Sanksi:**

Upaya kriminalisasi KPK ini tidak membuahkan hasil. KPK tetap melanjutkan upaya pemberantasan korupsi, dan berhasil mengungkap sejumlah kasus korupsi besar.

**Akhir Kasus:**

Upaya kriminalisasi KPK ini menjadi salah satu skandal politik terbesar di Indonesia. Skandal ini menunjukkan adanya upaya untuk menghalangi upaya pemberantasan korupsi di Indonesia.

Pembahasan : 

Misalkan Umur Pak Andi=x, umur Amira=y dan umur Ibu Andi=z

x = 28 + y …(1)

z = x – 6; atau x=z+6 …(2)

x + y + z = 119 …(3)

dengan melakukan operasi penjumlahan (1) pada (2) didapatkan

2x = y + z + 34 atau 2x – y – z = 34 …(4)

Lakukan operasi penambahan (3) pada (4) atau

x + y + z = 119

2x – y – z = 34

3x =153

Atau

x = 51

Dengan melakukan substitusi x pada (1) dan (2) didapatkan

Y = 23; z = 45

Sehingga

jumlah umur Amira (y) dan bu Andi (z) adalah y + z = 23 + 45 = 68

Setelah menelpon gurunya, pak Anto mengetahui bahwa range nilainya adalah 0 - 100. Jika anaknya mendapat nilai minimal 90, ia akan memberi hadiah berupa sebuah laptop baru. Untuk nilai 80 hingga 89, hadiahnya berupa sebuah smartphone baru. Untuk nilai 70 hingga 79, ia akan membelikan sebuah tas ransel baru. Untuk nilai di bawah 70, ia tidak akan memberi hadiah apapun.

INPUT nilai

IF nilai >= 90 THEN

   OUTPUT "Laptop baru"

ELSE

   IF nilai >= 80 THEN

       OUTPUT "Smartphone baru"

   ELSE

       IF nilai >= 70 THEN

           OUTPUT "Tas ransel baru"

       ELSE

           OUTPUT "Belajar lagi!"

       ENDIF

   ENDIF

ENDIF
