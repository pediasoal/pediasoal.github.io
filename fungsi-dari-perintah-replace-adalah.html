1. Apa fungsi perintah Find dan Replace pada tab home


Fungsi find itu untuk menemukan/mencari sesuatu pada tab home.Sedangkan fungsi replace itu untuk mengembalikan/menggeser menu untuk kembali pada tab home.fungsi find, adalah untuk mencari suku kata yg ingin dicari oleh user. kita bisa menggunakan shortcut keys CTRL + F (biasanya)

sedangkan fungsi replace, adalah mengganti suku kata yg sudah tertera pada lembaran (misalnya) dan mengganti dengan yg baru. contoh:
anda menulis yang dengan disingkat menjadi "yg", anda tinggal mencari kata "yg" dan di ganti dengan yg baru, yakni bisa "yang". sekian


2. apa fungsi perintah find and Replace pada tub home?


Find and Replace berfungsi untuk mencari kata juga sekaligus mengganti kata yang dicari tersebut.
Find Digunakan untuk mencari suatu kata, angka atau pun kalimat (asal sesuai) Ingat berdasarkan pada apa yang ada di dalam file word yang anda ingin cari.
Replace Berfungsi untuk mencari sekaligus juga mengubah file yang anda cari, jadi anda bisa langsung mengubah kata di replace tanpa harus lagi ke document yang di temukan.

18. fungsi REPLACE pada Foxpro adalah​


Fungsi REPLACE pada Foxpro adalah fungsi untuk mengganti karakter-karakter tertentu dalam string dengan karakter lain. Fungsi ini memiliki sintaks sebagai berikut:

Copy code

REPLACE(string, string_to_replace, replacement_string)

Contoh:

Copy code

REPLACE("Hello world", "world", "Foxpro")

Fungsi di atas akan mengganti string "world" dalam string "Hello world" dengan string "Foxpro", sehingga hasilnya adalah "Hello Foxpro".

Jawaban:
