17. tentukan amplitudo y = 0,5 sin π (0,5 t-2x)


Jawaban:

amplitudo gelombang tersebut adalah 0,5.

Penjelasan:

Rumus umum gelombang yang terdiri dari sin atau cos adalah:

y = A sin (ωt - kx + φ)

dengan:

A = amplitudo (m)

ω = frekuensi sudut (rad/s)

t = waktu (s)

k = bilangan gelombang (m⁻¹)

x = posisi (m)

φ = pergeseran fasa (rad)

Dalam soal diberikan persamaan:

y = 0,5 sin π (0,5 t-2x)

Dengan membandingkan persamaan tersebut dengan rumus umum, maka dapat diketahui bahwa:

A = 0,5

ω = π × 0,5 = 1,57 rad/s

k = 2

φ = 0

Sehingga amplitudo gelombang tersebut adalah 0,5.

Jawab:

Membuat grafik sinus

1. menentukan titik potong pada sumbu x, y = 0

  3 sin (2x-90) = 0

     sin (2x-90) = 0

                       = sin (k.360 +0)

     untuk k = 0

                   (45°, 0)                                (135°,0)

      untuk k = 1

      sin (2x-90) =  sin 360             sin (2x-90) = sin 540

            2x - 90 = 360                        2x - 90 = 540                

                     x  = 225                                   x = 315

                   (225°,0)                                  (315°, 0)

      sehingga titik pd sumbu x : (45°, 0) ; (135°,0) ;  (225°,0)  ; (315°, 0)

2. menentukan titik potong pada sumbu y, x = 0

     y = 3 sin (2.0 -90)

        = 3 sin (-90)

        = 3 (-1)

        = -3

      sehingga titik pd sumbu y : (0,-3)

3. titik puncak maksimum atau minimum saat y' = 0

    y = sin (2x-90)

    y' = 2 cos (2x-90)

    2 cos (2x-90) = 0

       cos (2x-90) = 0

      cos (2x-90) =  cos (k.360 + 90)

                          = cos (k.360 + 270)

     untuk k = -1

     cos (2x-90) =  cos (-270)            cos (2x-90) =  cos (-90)

             2x-90  = -270                             2x - 90 = -90

                       x = -90                                       x = 0

      y = 3 sin (2x-90)                         y = 3 sin (2x-90)

         = 3 sin (2.-90-90)                         = 3 sin (2.0-90)

          = 3 sin (-270)                               = 3 sin (-90)

          = 3 (1)                                           = 3 (-1)

          = 3                                               = -3

       ( -90°, 3)                                          (0°, -3)  

     

     untuk k = 0

     cos (2x-90) =  cos 90          cos (2x-90) =  cos 270

             2x - 90 = 90                        2x - 90 = 270

                       x = 90                                  x = 180

      y = 3 sin (2x-90)                         y = 3 sin (2x-90)

         = 3 sin (2.90-90)                         = 3 sin (2.180-90)

          = 3 sin 90                                   = 3 sin 270

          = 3 (1)                                          = 3 (-1)

          = 3                                               = -3

       ( 90°, 3)                                           (180°, -3)

       untuk k = 1

      cos (2x-90) =  cos 450          cos (2x-90) =  cos 630

             2x - 90 = 450                        2x - 90 = 630

                       x = 270                                  x = 360

37. Fungsi y = 3 sin 3x memiliki amplitudo sebesar


Fungsi y = 3 sin 3x memiliki amplitudo sebesar 3. Amplitudo fungsi trigonometri merupakan setengah dari selisih nilai terbesar dengan nilai terkecil fungsi tersebut. Bentuk fungsi trigonometri

y = A sin k (x ± b) ± c y = A cos k (x ± b) ± c y = A tan k (x ± b) ± c

dengan

A = amplitudo

Untuk fungsi trigonometri pada sinus dan kosinus

Besar periodenya: p = [tex]\frac{360^{o}}{k} = \frac{2 \pi}{k}[/tex]

Untuk fungsi trigonometri pada tangen

Besar periodenya: p = [tex]\frac{180^{o}}{k} = \frac{\pi}{k}[/tex]

Pembahasan

y = 3 sin 3x

A = 3 k = 3

Berdasarkan persamaan fungsi trigonometri tersebut

Besar amplitudonya: A = 2

Jika akan mencari besar periodenya:

p = [tex]\frac{360^{o}}{k} = \frac{360^{o}}{3}[/tex] = 120ᵒ (dalam satuan derajat)

p = [tex]\frac{2 \pi}{k} = \frac{2 \pi}{3} = \frac{2}{3} \pi[/tex] (dalam satuan π radian)
