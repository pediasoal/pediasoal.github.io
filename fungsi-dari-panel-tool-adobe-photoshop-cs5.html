13. fungsi custom shape tool pada Adobe Photoshop​


Shape Tool pada Photoshop bisa kita pilih beberapa macam antara lain sebagai berikut :

Rectangle Tool : Merupakan Tool untuk membuat objek shape dengan bentuk Persegi atau persegi panjang.

Rounded Rectangle Tool : Merupakan tool untuk membuat objek shape berbentu persegi dan persegi panjang, sama dengan Rectangle Tool, tetapi dengan sudut-sudut yang Melengkung atau Melingkar, yang Lengkungannya (radius) bisa kita sesuaikan dengan keinginan kita.

Ellipse Tool : Merupakan tool untuk membuat objek shape dengan bentuk Lingkaran atau pun Ellipse (lonjong).  

Polygon : Merupakan tool untuk membuat objek shape yang bersudut lebih dari 4 atau Polygon. Dengan Polygon tool kita juga bisa membuat bentuk Bintang menggunakan Photoshop.

Line Tool : Merupakan tool untuk membuat objek shape berupa Garis Lurus.

Custom Shape Tool : Merupakan tool untuk membuat objek shape dengan bentuk-bentuk lain yang bebas yang sudah disediakan oleh Photoshop, dan juga bisa kita tambahkan dengan mencari free download Custom Shape Tool untuk Photoshop di Google.

16. langkah-langkah mengaktifkan Adobe Photoshop CS5


Jawaban:

Cara Install dan Aktivasi Adobe Photoshop CS5

1. Setelah anda selesai mendownload file Adobe Photoshop CS5, selanjutnya anda perlu mengekstraknya. Dengan cara klik kanan > pilih Extract to.

2. Kemudian klik 2X file Set-up.

3. Pilih Accept.

4. Selanjutnya anda akan disuruh memasukkan serial number. Kita bisa skip tahap ini dengan cara memilih Install this product as a trial kemudian pada kolom bahasa pilih English (International). Kemudian pilih Next.

5. Jika laptop anda menggunakan Windows 32 Bit, maka disarankan memilih Adobe Photoshop CS5 (tanpa embel-embel 64 Bit). Jika anda tidak tahu Windows anda 32 atau 64-bit. Selanjutnya pilih Install.

6. Jika muncul notifikasi seperti ini, tinggal pilih Yes.

7. Tunggu proses instalasi Adobe Photoshop CS5 hingga selesai.

8. Adobe Photoshop CS5 sudah berhasil diinstall, selanjutnya pilih Done.

Tahapan aktivasi yang perlu kita lakukan

Download file dibawah ini untuk keperluan aktivasi Adobe Photoshop CS5. Selanjutnya ekstrak dengan cara klik kanan > pilih Extract to.

Selanjutnya buka folder Aktivasi.

Pilih 32 atau 64 bit

Kemudian copy file yang bernama amtlib.dll.

Buka Local Disk (C:) kemudian masuk ke folder Program Files.

Masuk ke folder Adobe > Adobe Photoshop CS5 (tergantung bit)

Pastekan file tadi kesini. Jika muncul notifikasi seperti ini, tinggal pilih Replace the file in the destination.

Jika muncul warning, pilih saja Continoe.

Sekarang anda sudah bisa menggunakan Adobe Photoshop CS5 secara maksimal tanpa perlu takut muncul popup yang menyuruh anda untuk melakukan aktivasi.

Beberapa Fungsi-fungsi Tools di Photoshop:

1. Move Tool, Fungsinya untuk memindah objek baik itu berupa teks maupun gambar.

2. Marque Tool

- Rectangular Marquee Tool : untuk membuat area seleksi berupa segi empat dengan sudut 90 derajat.

- Elliptical Marquee Tool : untuk membuat area seleksi berbentuk lingkaran.

- Single Row Marquee Tool : untuk membuat area seleksi berbentuk garis horizontal.

- Single Coloumn Marquee Tool : membuat area seleksi berbentuk garis vertikal.

3. Lasso Tool

Lasso Tool : untuk membuat area seleksi secara bebas sesuai dengan gerakan mouse.

Polygonal Lasso Tool : digunakan untuk membuat area seleksi berbentuk persegi/garis lurus.

Magnetic Lasso Tool : digunakan untuk membuat area seleksi namun akan secara otomatis dilelengketkan pada batas objek berdasarkan perbedaan warnanya.

4. Quick Selection & Magic Wand Tool

Quick Selection Tool : untuk membuat seleksi dengan cepat..

Magic Wand Tool : untuk membuat seleksi pada daerah yang memiliki kesamaan warna namun dapat diatur toleransinya pada options bar.

5. Croping & Slice Tool

Crop Tool : untuk memotong secara persegi dan sejajar

Perspektif Crop Tool : bisa disesuaikan sudutnya, tidak mesti 90 derajat.

Slice Tool : untuk membuat potongan-potongan gambar.

Slice Select Tool : untuk memilih potongan gambar jika kita masih ingin menggesernya.

6. Screen Mode : Untuk merubah tampilan jendela dari tampilan standar, full screen dengan menu, atau full screen.

7. Quick Mask Mode : Untuk mengolah file gambar atau foto ke dalam mode normal dan Quick Mask.

8. Set Foreground & Background : Untuk mengganti warna foreground dan background.

9. Zoom Tool : Untuk meperbesar dan Memperkecil tampilan gambar dalam dokumen kerja.

10. Hand and Rotate View Tool

Hand Tool : Untuk menggeser tampilan gambar dalam dokumen saja.

Rotate View Tool : Untuk memutar tampilan gambar dalam dokumen kerja.

(Membutuhkan komputer 64-bit-mampu dengan versi 64-bit Mac OS, Microsoft ® Windows ® 7, atau Windows Vista ®. Aktual kinerja bervariasi tergantung pada jumlah RAM, tipe driver, dan faktor lainnya.) jadi yang ngedit gambar kemaren lama maka dengan yang seri CS5 ini akan lebih mudah dan cepat. gak perlu ditinggal ngopy dan jalan jala

2. untuk edit teks maupun paragraf lebih mudah dan fleksibel sedangkankan pendahulunya lebih rumit dan sulit. jadi untuk sekarang cuma butuh waktu sedikit untuk editing text

3. Puppet Warp. Fitur ini digunakan untuk memperbaiki gambar/photo dengan cara menggerakkan bagian gambar/photo sesuai keinginan anda. Seperti misalnya; untuk mengecilkan bagian perut, memposisikan bentuk badan yang kurang pas, meluruskan tangkai bunga yang bengkok dan lain-lain. Namun, jika anda suka membuat animasi sederhana menggunakan Photoshop, fitur puppet warp ini tentu akan sangat membantu karena anda dapat dengan mudah menggerakkan bagian-bagian objek kearah yang anda inginkan.

4.Refine Edge. Fitur ini sebenarnya sudah ada pada versi Photoshop cs4, namun teknik refine edge pada Photoshop cs4 tidak secanggih pada Photoshop cs5. Fitur refine edge pada Photoshop cs5 ini telah disempurnakan sehingga mampu menyeleksi dengan detail bagian sulit dijangkau sekalipun. Fitur ini akan terlihat sangat powerfull jika digunakan untuk menyeleksi bagian-bagian yang kecil dan tipis, seperti seleksi pada rambut misalnya. Dengan teknik ini, menyeleksi rambut yang acak-acakan pun dapat dilakukan dengan mudah dan cepat. dan masih banyak lagi mungkin.
