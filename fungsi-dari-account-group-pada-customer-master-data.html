25. Fungsi in group dan out group


In-Group adalah kelompok social yang individu-individunya mengidenti-fikasikan dirinya dengan kelompoknya. Dalam menunjukkan In-Group-nya dalam kehidupan sehari-hari diungkapkan dengan kalimat : kelom-pok saya, group saya, dsb.


 Out-Group adalah kelompok social yang oleh individu-individu diartikan sebagai musuh kelompoknya atau lawan In-Group. Out-Group sering sering diungkapkan dengan istilah : kelompok mereka, group mereka, kelas mereka, dsb.

1) Cara membuat Account User Windows 2000 Server

Klik Start > Programs > Administrative Tools > Active Directory Users and Computers.Buka folder Users.Lalu klik ikon Create New User. Isi First Name dengan nama belakang pengguna baru, Last Name isi dengan nama belakang pengguna baru, Initials diisi dengan inisial pengguna baru, untuk User Logo Name diisi dengan nama pengguna atau username yang nanti digunakan untuk login. Setelah diisi lalu klik Next.Isi password atau confirm password jika ingin memberikan password pada user yang akan kuat. Serta di bawahnya ada beberapa pengaturan, centang yang perlu lalu klik Next.Selanjutnya jika sudah yakin dengan pengaturan user baru tadi klik Finish.

2) Cara mengubah Properti Account User

Pertama-tama buka control panel dengan view small, lalu pilih User Accounts. Setelah itu klik Manage another account.Kemudian setelah muncul Window, pilih salah satu dari user yang ingin kamu ubah tipenya.Setelah itu pilih Change the account type.Pilih tipe yang ingin kamu ubah, menjadi Standard user atau Administrator, kemudian klik Change Account Type.Buka Start Menu dan ketikkan netpliwz dan tekan Enter atau buka Windows Run (keyboard shortcut Win+R) dan ketikkan netpliwz, tekan Enter.Setelah itu akan muncul Window User Accounts, pilih salah satu user yang ingin kamu ubah tipenya dan klik Properties.Lalu pilih tipe user yang kamu inginkan dalam klik OK. Buka Local Users atau Groups dan klik User yang berada di sidebar kiri.Setelah itu klik kanan pada user yang kamu ingin ubah tipe usernya dan pilih Properties. Klik tab Member Of dan klik tombol Add.Kemudian klik Advanced untuk mengambah tipe user.Window Select Groups akan terbuka dan kamu bisa mengklik Find Now untuk mencari tipe user.Tipe user akan muncul, untuk memilihnya kamu tinggal mengklik tipe user tersebut dan klik OK.Setelah tipe user berhasil ditambahkan maka akan muncul di kolom bawah sendiri dan klik tombol OK.Sekarang kamu hanya perlu mengklil Apply dan OK jika tipe user yang kamu tambahkan ada di Member Of.

3) Cara menjadikan anggota group

Buka Outlook untuk Windows.Grup di panel folder kiri, pilih group Anda.Pada pita grup, pilih Tambahkan anggota.Dalam kotak Tambahkan anggota, cari orang di dalam organisasi Anda baik dengan nama lengkap atau alamat email mereka, lalu pilih nama untuk ditambahkan.Klik OK.

4) Menghapus Keanggotaan User dari Group

Grup di panel folder kiri, pilih grup Anda.Pada pita, pilih Edit Grup.Dalam kotak Grup Edit, arahkan ke nama anggota yang ingin Anda hapus, dan klik X.

10) Cara membuat Group Local membuat Sharing

Kemudian klik Advanced sharing.Akan terbuka jendela Advanced sharing, lalu beri tanda cek-box pada Share this folder. Kemudian klik tombol permissions untuk menambahkan user dan mengatur permission.Klik tombol Add jika kamu ingin menambahkan user atau grup, kalau saya cukup Everyone saja karena sudah mencakup semua. Kemudian atur permission-nya. Setelah itu klik OK.Kemudian kembali lagi ke jendela properties, beralih ke tab security. Lalu klik tombol edit untuk menambahkan user dan mengatur permissions.Akan muncul jendela permissions.
