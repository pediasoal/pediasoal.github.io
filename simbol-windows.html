1. Apa kah nama simbol A yang ada di windows


Jawaban:

Simbol "A" yang ada di Windows sering disebut sebagai "Drive A" atau "Floppy Drive A." Ini adalah simbol yang digunakan untuk mengidentifikasi drive floppy disk pada sistem operasi Windows yang lebih lama. Namun, seiring perkembangan teknologi, penggunaan drive floppy disk menjadi semakin jarang, dan banyak komputer modern bahkan tidak lagi dilengkapi dengan drive floppy disk.

Drive A biasanya digunakan untuk membaca dan menulis data ke floppy disk, yang merupakan media penyimpanan fisik berukuran kecil yang digunakan sebelum USB drive dan media penyimpanan modern lainnya menjadi umum. Seiring dengan penurunan penggunaan floppy disk, drive A pun menjadi kurang umum dalam penggunaan sehari-hari.


2. Jelaskan Simbol simbol / lambang - lambang perintah berupa gambar Windows dinamakan​


Jawaban:

Ikon, juga disebut simbol, ialah antarmuka grafik di sebuah data yang digambarkan oleh gambar kecil yang menggambarkan program komputer ataupun berkas komputer dalam pengelola berkas sebuah sistem operasi. Dibuat melalui manipulasi langsung atas simbol, sering ditunjuk melalui tetikus, sehingga pengguna dapat menjalankan fungsinya dengan menggerakkan tetikus tersebut, melihat informasi atau menghapus berkas itu.

Ikon juga digunakan di perangkat lunak. Ikon sering ditempatkan di sebuah bilah alat (toolbar) yang bisa dijalankan fungsinya oleh pengguna dalam program itu dengan mengeklik ikon itu.

Penjelasan:

Cara menyusun atau memindahkan ikon

Untuk menyusun ikon berdasarkan nama, tipe, tanggal, atau ukuran, klik kanan area kosong di desktop, lalu klik susun ikon. Klik perintah yang menunjukkan bagaimana Anda ingin menyusun ikon (menurut nama, menurut tipe, dan seterusnya). Jika Anda ingin agar ikon diatur secara otomatis, klik susun otomatis. Jika Anda ingin menyusun ikon sendiri, klik susun otomatis untuk menghapus tanda centang.

Cara menghapus ikon

Beberapa ikon adalah pintasan ke program di komputer Anda. Ikon pintasan biasanya memiliki panah di sudut kiri bawah. Jika Anda tidak menginginkan pintasan di desktop, klik ikon tersebut, lalu seret ke keranjang sampah. Tindakan ini hanya menghapus pintasan, bukan program yang diarahkan. Anda juga bisa mengklik kanan ikon, lalu klik Hapus untuk menghapus pintasan dari desktop Anda. Beberapa ikon seperti tempat jaringan saya, keranjang sampah, dan komputer saya tidak dapat dihapus.

Cara mengubah gambar ikon

Beberapa ikon memiliki opsi untuk mengubah gambar yang mewakili ikon. Tidak semua ikon memiliki opsi ini. Untuk mengubah gambar ikon:

1. Klik kanan ikon dan klik properti.

2. Klik tab pintasan (jika ada), lalu klik ubah ikon.

3. Klik ikon yang ingin Anda gunakan dari daftar, klik OK, lalu klik OK.

Jika tidak ada ikon yang tersedia dalam daftar, pabrikan mungkin tidak menyediakan ikon tambahan. Untuk menemukan ikon lainnya, ikuti prosedur yang sama pada ikon yang berbeda, temukan file sumbernya (biasanya file. ico), lalu kembali ke ikon asli yang ingin Anda ubah. Ikuti prosedur yang sama, tapi saat Anda mengklik tombol ubah ikon , telusuri ke lokasi file sumber dari ikon berbeda. Catatan: alat pihak ketiga tersedia yang memungkinkan Anda untuk mengubah ikon. Untuk informasi tambahan tentang mengubah gambar ikon, klik nomor artikel di bawah ini untuk menampilkan artikel di Basis Pengetahuan Microsoft:

253689 Tidak dapat mengubah ikon folder default di Windows Explorer

TweakUI

Catatan: TweakUI adalah alat yang tidak didukung. Informasi berikut ini disediakan untuk tujuan informasi saja. Layanan dukungan produk Microsoft tidak dapat menjawab pertanyaan tentang TweakUI. Jika Anda adalah pengguna Windows Power-, Anda bisa menggunakan TweakUI 1,33 untuk memanipulasi ikon. Namun, ini adalah alat yang tidak didukung. Baca dokumen yang menyertai TweakUI dengan seksama sebelum Anda mengunduh dan menggunakan alat tersebut.

34. sebutkan kelemahan dan kelebijan sistem operasi pada : windows 95 windows 98 windows xp windows vista windows 7 windows 8


kelemahan:
Sistem Operasi Windows 3.1.1
Kelebihan dari sistem operasi ini adalah :
a. Memudahkan komunikasi antar anggota workgroup lainnya
b. Menggunakan sharing printer
c. Memudahkan mengakses file dan program aplikasi yang berada di komputer lain.
d. Administrasi sistem jaringan (network) relative lebih mudah

Sistem Operasi Windows 95
Keunggulan dari sistem operasi ini adalah :
a. Sangat mudah dan cepat :
· Setiap saat kita dapat menggunakan seluruh kemampuan Windows 95, yaitu dengan mengklik tombol start untuk menjalankan program aplikasi, membuka dokumen, mencari data dan menggunakan fasilitas lainnya.
· Pada taskbar Windows 95 ditampilkan seluruh program aplikasi yang sedang anda gunakan, hal ini akan memudahkan untuk berpindah program aplikasi.
b. Berbagai fasilitas baru
1. Dengan sistem 32-bit, Kemampuan pengkodeaan 32-bit dari prosesor seri 80386, 80486, dan 80586 dapat dimanfaatkan secara optimal.
2. Dilengkapi fasilitas multithreading dan preemptive multitasking : multitasking lebih responsive dan dengan fasilitas ini dapat menjalankan beberapa program aplikasi secara bersama-sama.
3. Kemampuan memberi nama yang panjang pada file dan folder (maksimal 250 karakter).
4. Lalu lintas input/output yang lebih cepat yang membuat operasi baca tulis, dan mencetak ke printer menjadi lebih ditingkatkan kecepatannya.
5. Visual device driver 32-bit yang mengelola sumber daya sistem sehingga memungkinkan lebih dari satu aplikasi dapat mengunakan daya pada saat yang bersamaan.
Securiti secara terpusat : Sistem pengamanan dapat dilakukan secara terpusat pada server.
c. Fasilitas Bantuan (Help) yang terintegrasi Fasilitas Help apabila pemakai sedang menghadapi kesulitan dalam pengoprasian, program ini lebih luas.

Sistem Operasi Windows 98
Keunggulan dari sistem operasi ini dalah :
a. Lebih mudah digunakan membuka file, serta banyak tambahan tombol jelajah pada tiap jendela. meningkatkan ukuran bidang kerja. Dan instalasi perangkat keras baru akan lebih mudah karena Windows 98 mendukung Standar Universal serial Bus (USB). Standar ini memungkinkan kita menancapkan perangkat keras baru dan langsung menggunakannya tanpa perlu melakukan restart computer.
b. Lebih terjamin : memiliki akses ke Windows Update, ekstensi berbasis web untuk Windows 98. Fasilitas pemecahan masalah yang baru dilengkapi perangkat
c. Lebih cepat : windows dan program yang ada didalamnya dapat dijalankan lebih cepat daripada sebelumnya. Memanfaatkan Windows Tune-Up Wizard, kita dapat secara mudah meningkatkan kecepatan dan efisiensi komputer. Begitu juga dengan fasilitas Power Management dan kita pun dapat menggunakan FAT 32 untuk meningkatkan efisiensi penyimpanan file dan menghemat pemakaian ruang hard disk.
d. Lebih menghibur : windows 98 dapat mendukung DVD dan digital audio, jadi kita dapat memutar film dan audio digital berkualitas sangat tinggi pada komputer kita dan didukung oleh paket wallpaper, screen saver, icon 3-D, pointer mouse yang unik
