34. Tuliskan cara menyimpan (save) dokumen di Word yang kamu ketahui!​


Jawaban:

*Klik tab File.

*Klik Simpan Sebagai.

*Pilih lokasi file, seperti OneDrive atau PC Ini untuk menyimpan file Anda.

*Dalam kotak Nama file, masukkan nama baru untuk file tersebut.

*Di dalam daftar Simpan sebagai tipe, klik format file yang Anda inginkan untuk menyimpan file itu.

*Klik Simpan.

Untuk menyimpan (save) dokumen di Word, Anda dapat mengikuti langkah-langkah berikut:

1. Setelah selesai mengedit dokumen di Word, klik pada tab "File" di bagian atas layar.

2. Kemudian, akan muncul menu File, pilih "Save" atau "Save As" (tergantung pada versi Word yang Anda gunakan).

3. Jika Anda menggunakan "Save", maka dokumen akan disimpan dengan nama dan format yang sama di lokasi penyimpanan terakhir Anda.

4. Jika Anda menggunakan "Save As", maka akan muncul jendela dialog yang memungkinkan Anda untuk mengatur nama dokumen dan memilih lokasi penyimpanan yang baru. Pilih lokasi dan tentukan nama file yang Anda inginkan.

5. Selanjutnya, pilih format file yang diinginkan. Word biasanya menyimpan dokumen dalam format .docx, tetapi Anda juga dapat memilih format lainnya seperti .doc (format Word yang lebih lama) atau .pdf (format Portable Document Format).

6. Setelah itu, klik tombol "Save" untuk menyimpan dokumen dengan pengaturan yang Anda tentukan.

Dengan mengikuti langkah-langkah di atas, Anda dapat menyimpan dokumen Anda dengan aman di Word dan dapat mengaksesnya kembali di masa mendatang.
