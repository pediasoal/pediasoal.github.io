12. Perintah yang digunakan untuk menampilkan angka dengan format tampilan menggunakan lambang uang adalah.


Jawaban:

Penjelasan:

Untuk menampilkan angka dengan format tampilan menggunakan lambang uang, kita dapat menggunakan perintah format() dengan parameter tambahan ${:,.2f}. Perintah tersebut akan menampilkan angka dengan format dua angka di belakang koma, dan menggunakan tanda koma sebagai pemisah ribuan.

Sebagai contoh, jika kita ingin menampilkan angka 1000 menggunakan format lambang uang, maka Anda dapat menggunakan perintah berikut:

print(format(${:,.2f}, 1000))

Perintah tersebut akan menampilkan angka 1000 dengan format lambang uang, yaitu $1,000.00. Kita dapat mengganti angka 1000 dengan angka lain sesuai kebutuhan kita.

23. pengaturan format yang sering digunakan untuk mengatur format Ada berapa macam pengaturan​


Jawaban:

Beberapa pengaturan yang dapat dilakukan dalam format cells, yaitu:

klik tab home

pilih kategori cells

klik tombol format

pilih format cells (atau dapat juga menekan tombol ctrl1+) sehingga akan ditampilkan kotak dialog format cells.

Dalam kotak dialog format cell, terdapat 6 buah tab pilihan format, yaitu :

1. Numbering

Digunakan untuk mengatur format angka dari cell. Terdapat 12 pilihan kategori jenis angka antara lain :

Tabel 5 2. Alignment

Digunakan untuk mengatur perataan dari cell.

Terdapat 4 buah pengaturan :

1) Text alignment

Digunakan untuk mengatur perataan teks,terdapat 2 buah opsi pengatura yaitu:

Horizontal : Perataan secara horizontal

Vertical perataan secara vertical

2) Text control

Wrap text : Membuat teks mengikuti ukuran lebar cell

Shrink to fit : Membuat teks mengecil mengikuti lebar cell

Merge cells : Menggabungkan cell

3) Right-to-left

Text Direction : Mengatur arah penulisan teks

4) Orientation

Mengatur derajat kemiringan dari teks.

3. Font

Digunakan untuk mengatur karakter huruf dari cell

Terdapat 6 buah pengaturan :

1) Font

Digunakan untuk memilih jenis huruf yang diinginkan.

2) Font Style

Digunakan untuk memilih gaya penulisan (tergantung font),terdiri dari:

Regular

Italic (cetak miring) (Ctrl+I)

Bold (cetak tebal) (Ctrl+B)

Bold Italic (cetak tebal dan miring)

3) Size

Digunakan untuk mengatur ukuran huruf.

4) Underline

Digunakan untuk mengatur garis bawah huruf.

5) Color

Digunakan untuk mengatur warna huruf dan garis bawah.

6) Effects

Digunakan untuk memberi efek huruf, terdiri dari:

Strikethrough : efek coretan

Superscript : huruf berukuran kecil diatas karakter umum (seperti pangkat)

Subscript : huruf berukuran kecil dibawah karakter umum.

4. Border

Digunakan untuk mengatur bingkai/garis luar dari cell

Terdapat 4 buah pengaturan :

Line

Digunakan untuk mengatur jenis garis

Color

Digunakan untuk mengatur warna garis

Presets

None : tanpa bingkai

Outline : memberi garis luar cell/range.

Inside : memberi garis dalam cell/range

Border

Digunakan untuk memberi garis sesuai arah yang diinginkan.

5. Fill

Digunakan untuk memberi warna cell.

Terdapat 3 buah pengaturan :

Backgroud Color

Digunakan untuk memilih warna latar belakang cell. Tombol pilihan Fill Effect digunakan untuk memberi warna dengan efek tertentu.

Pattern Color

Digunakan untuk memilih warna arsiran dari cell

Pattern Style

Digunakan untuk memilih jenis arsiran untuk cell

6. Protection

Digunakan untuk melindugi isi cell dengan mengunci dan/atau menyembunyikan rumus didalam cell.

38. Jelaskan langkah langkah yang dilakukan untuk memformat teks menggunakan kotak dialog font


Jawaban:

Langkah pertama, ialah mengklik kanan teks yang ingin diformat.

Langkah berikutnya, klik pilihan Font dari menu yang tampil.

Saat muncul kotak dialog font, lakukan pemformatan sesuai selera:

- Font: untuk memformat jenis teks

- Font style: untuk memformat gaya teks

- Size: untuk memformat ukuran teks

- Font color: untuk memformat warna teks

- Underline style: untuk memformat gaya garis bawah

- Underline color: untuk memformat warna garis bawah

- Effects: untuk menambahkan format efek pada teks

Bila semuanya telah diformat sesuai selera, selanjutnya klik tombol OK

Penjelasan:

Untuk lebih jelasnya, silakan perhatikan gambar berikut ini

40. Lambang ikon Clipboard untuk format printer yg benar adalah.......​


Jawaban:

Ikon-ikon yang terdapat pada group Clipboard

- Cut: untuk memindahkan suatu teks atau objek dari satu dokumen/program ke dokumen lainnya

- Copy: untuk menyalin/mengkopi suatu teks atau objek lainnya

- Paste: untuk menempelkan atau memunculkan hasil dari Cut atau Copy

- Format Painter: untuk menyalin atau menyamakan format penulisan
