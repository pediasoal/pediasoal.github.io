Untuk fungsi f: x → 4 - x dengan domain {-2, -1, 0, 1, 2}, kita dapat menentukan range fungsi tersebut dengan menggantikan setiap nilai x dalam domain ke dalam persamaan f(x) = 4 - x.

Jadi, jika kita menggantikan setiap nilai x dalam domain:

f(-2) = 4 - (-2) = 6

f(-1) = 4 - (-1) = 5

f(0) = 4 - 0 = 4

f(1) = 4 - 1 = 3

f(2) = 4 - 2 = 2

Maka, range fungsi f adalah {2, 3, 4, 5, 6}.

Untuk fungsi h(x) = x² + 2 dengan range (18, 27, 38, 51), kita dapat menentukan domain fungsi tersebut dengan mencari nilai-nilai x yang menghasilkan nilai dalam range tersebut.

Jadi, jika kita mencari akar-akar persamaan kuadrat:

x² + 2 = 18

x² = 16

x = ±4

x² + 2 = 27

x² = 25

x = ±5

x² + 2 = 38

x² = 36

x = ±6

x² + 2 = 51

x² = 49

x = ±7

Maka, domain fungsi h adalah {-7, -6, -5, -4, 4, 5, 6, 7}.

19. h(x)=2x-2 dan g(x)=√x~ Domain dari? h(x) / g(x) adalah​


Penjelasan dengan langkah-langkah:

Untuk dapat menentukan domain dari h(x) / g(x), perlu mengecek apakah ada pembagian dengan nol pada akar dari g(x).

Fungsi g(x) mengandung akar kuadrat (√x), oleh karena itu x harus lebih besar atau sama dengan nol untuk menghindari bilangan imajiner dalam fungsi. Oleh karena itu, domain dari g(x) adalah [0, ∞).

Sementara itu, fungsi h(x) adalah sebuah fungsi linier, sehingga domainnya tidak terbatas.

Karena kita ingin membagi h(x) dengan g(x), maka domain dari h(x) / g(x) akan dipengaruhi oleh domain dari g(x). Oleh karena itu, domain dari h(x) / g(x) adalah [0, ∞).
