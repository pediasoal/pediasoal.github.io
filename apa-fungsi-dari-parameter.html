4.  - Pendeklarasian Fungsi:

void sapa(nama){

   cout << "Hello, " << nama << endl;

}

void = sebuah tipe dari fungsi, lebih tepatnya fungsi tanpa kembalian.

sapa = sebagai nama fungsi yang telah kita buat dan akan digunakan untuk memanggil fungsi itu nantinya di main program.

{} = menandakan kode yang ada di dalam tanda kurung kurawal ini sebagi bagian dari milik fungsi sapa.

nama = sebagai paramater.

cout = berarti console out, atau sederhannya menampilkan hasil dari program ke console/terminal.

<< = tanda yang berati akan mengeluarkan/menampilkan ke terminal

"Hello" = teks/string biasa.

nama = akan menampilkan nama(Parameter) ke console/terminal.

endl = bisa diartikan sebagia "\n" atau disebut juga dengan new line/baris baru.

; = titik dua/semicolom berarti sebagai penanda jika kode pada baris ini akan berhenti.

- Pendefinisian(pemanggilan?) :

#include <iostream>

using namespace std;

int main(){

string nama;

cout << "masukan nama kamu : " ; cin >> nama;

sapa(nama);

}

#include <iostream> = prosesor pengarah yang mengatakan kepada kompiler untuk meletakan kode dari header file iostream.

using namespace std; =  membuat kita tidak perlu menggunakan std:: lagi di dalam program.

int main() = sebagi main program.

{} = menandakan kode yang ada di dalam tanda kurung kurawal ini sebagi bagian dari milik main program.

str nama; = sebagai variabel yang menyimpan data/value yang dimasukan oleh user.

cout << =  berarti console out, atua sederhannya menampilkan hasil dari programnya ke console/terminal.

; = berarti sebagai penanda jika kode pada baris ini akan berhenti.

cin >> = console input, yang berarti user akan memasukan/input data ke variabel nama;.

sapa(nama); = memnaggil fungsi. kita akan mengoper data yang telah di-input-kan oleh user di variabel nama; ke dalam fungsi untuk natinya ditampilkan ke dalam console/terminal.

; = berarti sebagai penanda jika kode pada baris ini akan berhenti.

13. Sebutkan alasan parameter COD sering digunakan untuk menggantikan fungsi BOD


Pertanyaan yang telah diajukan adalah sebagai berikut : Sebutkan alasan parameter COD sering digunakan untuk menggantikan fungsi BOD

Apa yang dimaksud dengan COD ?

COD adalah singkatan dari Chemical Oxygen Demand yang mempunyai pengertian jumlah oksigen yang terlarut dalam air yang dibutuhkan untuk mengoksidasi bahan-bahan organik yang terdapat dalam air. Parameter COD menunjukan jumlah oksigen( mg O2 ) yang terkandung di dalam senyawa oksida yang dibutuhkan untuk menguraikan senyawa – senyawa organik yang terkandung dalam 1 liter limbah cair. Untuk lebih jelasnya mari kita simak ilustrasi berikut jika suatu limbah memiliki nilai COD = 230 mg/l, hal ini menunjukkan bahwa dalam 1 liter limbah cair terdapat senyawa organik sejumlah 230 mg O2.

Apa yang dimaksud dengan BOD ?

BOD adalah singkatan dari Biochemical Oxygen Demand yang mempunyai pengertian sebagai jumlah oksigen terlarut yang dibutuhkan oleh oleh mikroorganisme termasuk bakteri untuk mendekomposisi atau mengurai bahan organik.  

Melalui kedua cara tersebut dapat ditentukan tingkat pencemaran air lingkungan.

Parameter COD lebih mudah digunkan dikarenakan oleh beberapa factor seperti :

Membutuhkan waktu yang lebih lama dikarenakan Oksigen yang dikonsumsi dalam uji BOD ini dapat diketahui dengan menginkubasikan contoh air pada suhu 20 0C selama lima hari uji BOD ini tidak dapat digunakan untuk mengukur jumlah bahan-bahan organik yang sebenarnya terdapat di dalam air, tetapi hanya mengukur secara relatif jumlah konsumsi oksigen yang digunakan untuk mengoksidasi bahan organik Terdapat pembatasan BOD, apabila ion logam yang beracun terdapat dalam sampel maka aktivitas bakteri akan terhambat sehingga nilai BOD menjadi lebih rendah dari yang semestinya

function rangeWithStep(startNum, stopNum, step) {

   if (typeof stopNum == 'undefined') {

       // one param defined

       stopNum = startNum;

       startNum = 0;

   }

   if (typeof step == 'undefined') {

       step = 1;

   }

   if ((step > 0 && startNum >= stopNum) || (step < 0 && startNum <= stopNum)) {

       return [];

   }

   var result = [];

   for (var i = startNum; step > 0 ? i < stopNum : i > stopNum; i += step) {

       result.push(i);

   }

   return result;

};

23. • Apakah fungsi dari parameter ALT pada tag Image? • Apakah fungsi dari parameter WIDTH dan HEIGHT pada tag Image? • Kapan parameter tersebut digunakan? Apakah perlu digunakan keduanya? • Bagaimana cara membuat hyperlink dari suatu gambar yang rujukannya merupakan URL (misalnya http://www.google.co.id atau ftp://sun.uio.org)?


Sederhananya fungsi dari atribut "alt" pada elemen atau tag image <img> adalah sebagai teks alternatif; artibut "width" dah "height" berfungsi untuk mendeklarasikan ukuran dari elemen image <img> (gambar); kedua atribut (width dan height) tersebut harus selalu digunakan secara bersamaan pada sebuah gambar; Cara membuat hyperlink atau tautan dari sebuah gambar bisa dilakukan dengan mengapit tag <img> dengan tag <a> dimana gambar menjadi jangkarnya (anchor)

Pembahasan:

ALT

Atribut alt pada image adalah singkatan dari alternative text yang pada dasarnya digunakan untuk membuat alternatif teks dari sebuah gambar yang berasal dari elemen <img>. Bagi pengguna pada umumnya fungsi alt ini tidak terlalu signifikan karena pada dasarnya ditujukan untuk:

27. Uraikanlah parameter-parameter yang ada dalam getaran dan gelombang​


Jawaban:

Periode: T

Frekuensi: f

Amplitudo: A

Cepat rambat: v

Panjang gelombang: lambda

Penjelasan:

Periode:

Periode adalah waktu yang dibutuhkan gelombang untuk melakukan sebuah siklus. Periode dihitung dalam satuan detik dan dilambangkan sebagai huruf T (Time) yang berarti waktu. Periode diukur dengan satuan detik/siklus.

Frekuensi:

Frekuensi gelombang adalah jumlah siklus yang diselesaikan dalam waktu tertentu. Frekuensi dilambangkan dengan huruf f kecil dan kita ukur dalam satuan siklus/detik atau sama saja dengan Hertz (Hz).

Amplitudo:

Amplitudo adalah simpangan terjauh sebuah gelombang. Amplitudo bisa juga dibilang sebagai jarak antara garis puncak dan puncak atau palung dari suatu gelombang. Semakin besar amplitudo, semakin banyak energi yang dibawa.

Cepat rambat:

Cepat rambat gelombang adalah kecepatan gelombang. Kecepatan gelombang dapat diukur dalam meter/detik (m/s). Cepat rambat gelombang diukur berdasarkan jarak gelombang itu bergerak dalam kurun waktu tertentu. Kecepatan gelombang memberitahu kita berapa banyak siklus yang diselesaikan suatu gelombang per detiknya.

Panjang gelombang (lambda):

Panjang gelombang adalah jarak untuk setiap siklus gelombang (wavelength). Panjang gelombang bisa didapat dengan mengukur jarak antara dua puncak berurutan.

Jawaban dan Penjelasan:

Overloaded Functions

Saya buat 2 versi. Yang pertama, argumen bertipe string menggunakan tipe "string" dari namespace std. Karena ada baris "use namespace std;", maka nama tipenya "string" saja, tidak perlu "std::string".

Versi kedua, argumen bertipe string menggunakan "char*". Pada saat passing argumen ke fungsi, harus menggunakan casting berupa "(char*)", agar tidak ada warning pada saat kompilasi.

Versi 1: tipe data string menggunakan string dari namespace std (std::string)

#include <iostream>

using namespace std;

void overloadingFunc(int i) {

   cout << "Tipe data int, hasilnya " << (100*i) << endl;

}

void overloadingFunc(double d) {

   cout << "Tipe data double, hasilnya " << (1000*d) << endl;

}

void overloadingFunc(string s) {

   cout << "Tipe data string, nilainya Hello " << s << endl;

}

int main() {

   overloadingFunc(10);

   overloadingFunc(10.0);

   overloadingFunc("World");

   return 0;

}

Versi 2: tipe data string menggunakan char*

Bagian di atas ini persis sama seperti di atas.

void overloadingFunc(char* s) {

   cout << "Tipe data string, nilainya Hello " << s << endl;

}

int main() {

   overloadingFunc(10);

   overloadingFunc(10.0);

   overloadingFunc((char*)"World");

   return 0;

}

40. Diketahui persamaan parameter suatu fungsi vector y='2t, x=6t+2 dan -∞​


JAWABAN DAN PENJELASAN :

Karena y='2t dan x=6t+2, maka bentuk parameter dari fungsi vector tersebut adalah v(t) = <6(t+1), 2t>

Dengan menggunakan bentuk ini, kita dapat menentukan nilai vector pada suatu nilai t tertentu. Misalnya, jika t=0, maka v(0) = <6(0+1), 2(0)> = <6,0>. Jadi pada nilai t=0, vector tersebut memiliki nilai (6,0).

Jika nilai t berubah, maka nilai vector juga akan berubah sesuai dengan persamaan parameter v(t). Sebagai contoh, jika t=1, maka v(1) = <6(1+1), 2(1)> = <12,2>. Jadi pada nilai t=1, vector tersebut memiliki nilai (12,2).

Dengan begitu, kita dapat mengetahui bentuk dan nilai vector pada suatu nilai t tertentu menggunakan persamaan parameter v(t) = <6(t+1), 2t>.
