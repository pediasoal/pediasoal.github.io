3. Fungsi kotak teks pada kotak dialog page setup dalam pengaturan margin


Perintah Page Setup adalah salah satu perintah yang paling penting dalam Microsoft Word. Dalam menu page setup anda dapat mengatur Margins seluruh dokumen misalnya, Ukuran Kertas, Jarak/ukuran header dan footer, Paper Source serta Layout (tata letak kertas/halaman).

Pilih menu File (Alt + F) > Page Setup (Alt + U) atau klik 2 x ruler sebelah kiri Akan muncul kotak dialog Page Setup

5. fungsi kotak dialog page setup muncul ketika kita memberi perintah pada komputer untuk


Fungsi kotak dialog page setup muncul ketika kita memberi perintah pada komputer untuk melakukan pengaturan terhadap batas atau margin lembar kerja kita. Selain itu menu page setup juga terdapat pada saat kita akan mencetak lembar kerja kita.

Pembahasan

Menu Page setup sendiri dapat kita akses melalui menu page layout, atau bisa pada saat akan mencetak lembar kerja kita kemudian pilih print review. Adapun batas margin atau batas lembar kerja kita yang bisa diatur melalui menu page setup adalah sebagai berikut

Batas AtasBatas BawahBatas KananBatas Kiri

Selain itu pada page setup kita juga bisa melakukan beberapa pengaturan selain batas lembar kerja, pemilihan jenis kertas yang kita gunakan juga bisa melalui menu Page Setup.

30. Fungsi kotak teks pada kotak dialog page setup dalam pengaturan margin


Perintah Page Setup adalah salah satu perintah yang paling penting dalam Microsoft Word. Dalam menu page setup anda dapat mengatur Margins seluruh dokumen misalnya, Ukuran Kertas, Jarak/ukuran header dan footer, Paper Source serta Layout (tata letak kertas/halaman).

Pilih menu File (Alt + F) > Page Setup (Alt + U) atau klik 2 x ruler sebelah kiri Akan muncul kotak dialog Page Setup

31. Jelaskan 5 macam margin yg terdapat pada jendela page setup ?


- Normal: Pengaturan margin yg default/biasanya digunakan di Word. Ukuran/jarak marginnya (margin atas, bawah, kiri dan kanan) sama-sama 1 inci
- Narrow: Pengaturan argin dengan ukuran margin yg kecil daripada normal sehingga area dokumen lebih luas daripada margin Normal. Ukuran/jarak marginnya sama-sama 0,5 inci. Biasanya dipakai untuk dokumen yang banyak isinya atau dokumen berisi gambar dengan ukuran besar
- Moderate: Hampir sama seperti Normal, namun ukuran margin kiri dan kanannya beda dengan ukuran margin atas dan bawah. Ukuran/jarak margin atas dan bawah: 1 inci, margin kiri dan kanan: 0,75 inci
- Wide: Hampir sama dengan Moderate, cuma beda ukuran marginnya. Ukuran margin atas dan bawah: 1 inci, margin kiri dan kanan: 2 inci. Biasanya dipakai untuk dokumen yang akan dijilid atau dokumen yang memuat gambar dengan ukuran wide/lebar kesamping
- Mirrored: Pengaturan margin untuk membuat 2 kolom dalam 1 halaman atau seperti majalah. Ukuran/jarak margin atas dan bawah: 1 inci, margin diantara 2 kolom: 1,25 inci dan margin luar/ditepi luar kolom: 1 inci

NB: Margin: jarak antara batas tepi tulisan dengan ujung tepi kertas
