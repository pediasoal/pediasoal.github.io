29. Tuliskan menu dan ikon pada menu bar pada Microsoft PowerPoint​


MENU BAR

Macam-macam menu bar pada Ms. Power Point:

•File

berfungsi untuk melakukan perintah dasar, membuat, membuka, dan menyimpan file power point, serta keluar dari power point.

•Home

digunakan untuk mengatur font dan paragraf, menggambar bentuk, dan melakukan copy, cut, paste, dll.

•Insert

digunakan untuk menyisipkan objek, seperti gambar, tabel, suara, animasi, dll.

•Design

digunakan untuk mengatur tampilan slide, untuk mengubah tampilan slide sesuai dengan keinginan.

•Transitions

digunakan untuk memberikan efek transisi/perpindahan antar slide, dan menambahkan efek suara, mengatur waktu efek transisinya.

•Animations

digunakan untuk mengatur animasi yang ditambahkan ke slide.

•Slide show

digunakan untuk menampilkan presentasi, pengaturan penampilan file presentasi, dll.

•Review

digunakan untuk mengecek tata bahasa, pengejaan, dll.

•View

digunakan untuk mengatur tampilan lembar kerja.
