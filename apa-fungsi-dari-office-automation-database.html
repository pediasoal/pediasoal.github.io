fungsi office button adalah:

1. New, di gunakan untuk membuat halaman kerja baru. jadi untuk membuat halaman kerja baru sobat tidak perlu membuka ulang aplikasi microsoft office, cukup dengan klik office button dan klik new.

6. Prepare, digunakan untuk pengaturan dokumen contohnya ingin membuat pasword, dan masih banyak lagi, karena ada 7 di dalam menu prepare.

7. Send, digunakan untuk mengirim dokumen.

8. Publish, digunakan untuk membagikan file ke Workspace.

9. Close, digunakan untuk menutup aplikasi microsoft office.

16. kata automation berasal dari bahasa tolong bantuin yan


Automation berasal dari bahasa Inggris.

Pembahasan:

Istilah “Automatitation” (Otomatisi) pertama digunakan pada industri otomotif, sekitar tahun 1946, untuk menggambarkan peningkatan penggunaan perangkat otomatis dan kontrol di jalur produksi kendaraan bermotor. Istilah ini dicetuskan oleh D.S. Harder, seorang manajer teknik pada saat itu di Ford Motor Company .

Otomatisasi adalah proses untuk meningkatkan produktivitas, penggunaan bahan yang lebih efisien, dan kualitas produk yang lebih baik, dengan menggunakan peralatan yang bekerja dengan sedikit kebutuhan kendali manusia. Sehingga, otomatisasi juga mengurangi kebutuhan pekerja.

Otomatisasi dimulai dari proses Revolusi Industri, yaitu proses penggunaan mesin dan teknologi untuk menghasilkan produk dengan jumlah besar secara massal. Awalnya, teknologi ini sederhana seperti alat pemintal dan mesin uap. Namun kemudian berkembang menjadi mesin yang kompleks seperti perangkai kendaraan bermotor dan elektronik.

17. hubungan business process automation dengan bpm


Dari pengertiannya
●Business process management ("manajamen proses bisnis"), disingkat BPM, adalah suatu metode penyelarasan secara efisien suatu organisasi dengan keinginan dan kebutuhan organisasi tersebut. BPM merupakan suatu pendekatan manajemen holistik untuk meningkatkan efektivitas dan efisiensi bisnis seiring upaya untuk mencapai inovasi, fleksibilitas dan integrasi dengan teknologi. BPM berupaya untuk melakukan perbaikan proses secara berkelanjutan atau bisa juga disebut sebagai suatu proses 'optimalisasi proses'.

●Business process automation (BPA) ,
Otomatisasi proses bisnis , ialah strategi bisnis menggunakan untuk mengotomatisasi proses untuk mengendalikan biaya. Ini terdiri dari mengintegrasikan aplikasi, restrukturisasi sumber daya tenaga kerja dan menggunakan aplikasi perangkat lunak di seluruh organisasi.

21. sebutkan fungsi-fungsi pada office botton


New Digunakan untuk membuat file dokumen kosong baru. Open Icon ini digunakan untuk membuka file dokumen yang tersimpan. Save Sebuah perintah untuk menyimpan file dokumen aktif. Save As Seperti fungsinya sama seperti icon "Save" diatas digunakan untuk menyimpan file, akan tetapi save as digunakan lebih mempunyai fungsi khusus yaitu menyimpan dokumen ke dalam format tertentu. Print Digunakan untuk mencetak file dokumen yang sedang aktif. Prepare Digunakan untuk melakukan properti khusus terhadap file dokumen aktif. Send Digunakan untuk mengirimkan file dokumen aktif sebagai email / faksimile. Publish Perintah untuk menerbitkan dokumen aktif ke website / blog / dll. Close Digunakan untuk menutup file dokumen aktif. Word Option Menutup aplikasi word . Exit Word Tombol untuk merekayasa fitur-fitur penting didalam ms word seperti mengubah fitur tampilan lingkungan word, kustomisasi quick access toolbar, kustomisasi saving file, kustomisasi setting file dll. 29. Fungsi office button pada Microsoft Office Word 2007 mirip dengan fungsi salah satu menu pada Microsoft Office Word 2003 yaitu menu

Tidak hanya pada microsoft word 2007 saja, untuk aplikasi Excel 2007 dan Power Point 2007 juga sudah dilengkapi dengan adanya Office Button.
