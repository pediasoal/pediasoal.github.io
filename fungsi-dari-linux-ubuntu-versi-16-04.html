Ubuntu adalah sistem operasi lengkap berbasis Linux, tersedia secara bebas dan mempunyai dukungan baik yang berasal dari komunitas maupun tenaga ahli profesional.Ubuntu pertama kali dirilis pada 20 Oktober 2004.

Ubuntu dijadwalkan dirilis setiap 6 bulan sehingga sistem Ubuntu dapat terus diperbarui.Setiap rilis didukung selama 18 bulan untuk pembaruan sistem, keamanan, dan kesalahan (bug).
Setiap 2 tahun sekali (versi xx.04 dengan x angka genap) akan mendapatkan Long Term Support (LTS) selama 3 tahun untuk desktop dan 5 tahun untuk edisi server.Ubuntu sekarang dibiayai oleh Canonocal Ltd. Tujuan dari pendirian yayasan ini adalah untuk memastikan pengembangan dan dukungan semua versi Ubuntu dapat terus berjalan.Pada 31 Oktober 2011, Mark Shuttleworth mengumumkan bahwa Ubuntu 14.04 akan mendukung smartphone, tablet, dan smart TV. Dan desain-desainnya pun sudah mulai dirancang.

20. cara setting IP Address linux dan ubuntu


ada di menu settings. kemudian pilih network

1. Melalui Command Line Interface

Pada cara pertama ini, kita akan melakukan konfigurasi langsung melalui terminal linux atau Command Line Interface (CLI). Misal kita akan berikan konfigurasi pada interface eth1 berupa ip address 192.168.10.20 dan netmask 255.255.255.0. Maka perintah yang dapat kita jalankan di terminal sebagai berikut.

root@ubuntu:~# ifconfig eth1 192.168.10.20 netmask 255.255.255.0

Setelah menjalankan perintah tersebut, cek dengan perintah ifconfig eth1. Jika sudah benar, pasti akan muncul informasi IP address yang terkonfigurasi pada interfaces eth1.


2. Dengan mengedit file interfaces

Untuk membuat konfigurasinya permanen, kita bisa edit file interfaces yang lokasinya ada di “/etc/network/interfaces“. Caranya, jalankan perintah nano untuk membuka file tersebut, lalu tambahkan onfigurasi yang kita inginkan. Coba perhatikan yang di bawah ini.

root@ubuntu:~# nano /etc/network/interfaces

Setelah file interfaces terbuka, berikan konfigurasi yang kita inginkan. Misalnya seperti dibawah ini.

# The loopback network interface auto lo iface lo inet loopback auto eth1 iface eth1 inet static         address 192.168.10.20         netmask 255.255.255.0         network 192.168.10.0         broadcast 192.168.10.255         gateway 192.168.10.10         dns-nameservers 8.8.8.8

Setelah selesai memberikan konfigurasi, simpan perubahan dengan menekan tombol Ctrl+O (save). Lalu keluar dengan menekan tombol Ctrl+X (Exit).

Kemudian restart service nya agar konfigurasi tadi bisa beroperasi. Perintahnya lihat di bawah ini.

root@ubuntu:~# /etc/init.d/networking restart  * Running /etc/init.d/networking restart is deprecated because it may not enable again some interfaces  * Reconfiguring network interfaces...   Ignoring unknown interface eth1=eth1.                                                          [ OK ] root@ubuntu:~#

Status OK menandakan settingan telah siap. Setelah itu kita coba cek dengan menjalankan perintah ifconfig eth1.

25. kelebihan dan kekurangan Linux Ubuntu?


Kelebihan jika menggunakan Ubuntu
1.      Gratis dan bisa digunakan untuk banyak komputer
2.      Stabil, karena turunan dari Debian dan aman dari virus, worm, malware dan sejenisnya, walaupun tak memakai anti virus
3.      Bisa digunakan pada komputer yang memiliki spesifikasi hardware rendah, dikarenakan Ubuntu sangat ringan sehingga tak membebani kinerja komputer
4.      Kita bisa mencoba menggunakan Ubuntu tanpa perlu menginstalnya kedalam harddisk komputer, dengan menggunakan fitur Live CD pada Ubuntu melalui proses boot pada CD atau flashdisk saja.
5.      Tersedia banyak aplikasi mulai dari aplikasi Office ( libreOffice, OpenOffice), browsing (Firefox, chromunium), multimedia (Rhythmbox, VLC player), grafik (GIMP, shotwell), game (linecity, hedgewar), edukasi/pendidikan (educational suite gcomprize, quran) dan berbagai Aplikasi lainnya yang sebagian besar diantaranya adalah gratis (free)
6.      Driver kebanyakan telah ada di dalam CD, seperti LAN, Wifi, Audio, dan lain sebagainya sehingga tidak sulit bagi kita untuk melakukan instalasi
7.      Terdapatnya Ubuntu Software Center, dimana anda dapat mendownload berbagai aplikasi dan game dengan mudah
8.      Anda dibebaskan dan diperbolehkan untuk menggunakan, memodifikasi dan mendistribusikan sesuka anda
9.      Terdapat lebih dari 55 bahasa, termasuk bahasa Indonesia. Sehingga memudahkan anda dalam menggunakan Ubuntu, jika anda tak mengerti bahasa Inggris.
10.  Tampilan desktop yang keren dan menarik dengan Compiz fusion
11.  Aplikasi Windows tetap bisa dijalankan di Ubuntu menggunakan aplikasi yang bernama Wine (Windows Emulator)
12.  Versi terbaru dari Ubuntu diliris tiap 6 bulan sekali
13.  File ISO Ubuntu bisa didapat dari banyak cara, melalui download, pesan, minta dari teman dan sebagai tanpa melanggar hukum pembajakan software
14.  Bisa di instal bersamaan dengan Windows
15.  Bisa di instal melalui Windows menggunakan bantuan aplikasi bernama pada Windows Wubi
Kelemahan jika menggunakan Ubuntu
1.      Belum user friendly, dikarena sebagian besar pengguna Ubuntu berasal dari migrasi Windows dan lainnya
2.      Tak semua aplikasi windows anda kompatibel dengan wine sehingga aplikasi kegemaran anda mungkin tidak bisa digunakan di Ubuntu.
3.      Sedikit dikucilkan dari teman-teman dikarenakan sebagian besar dari mereka menggunakan Windows

33. merupakan jenis perangkat lunak DNS pada linux ubuntu server.


Jawaban:

Tata Cara Setting DNS Ubuntu Server dengan 5 Langkah

By Rianda

Tata Cara Setting DNS Ubuntu Server dengan 5 Langkah

SHARE

Pada dasarnya yang dinamakan dengan DNS merupakan bagian dari infrastruktur suatu web. Biasanya dalam DNS tersebut akan menyediakan tata cara penerjemahan domain hingga menjadi alamat IP. Nantinya untuk melakukan setting DNS ubuntu server akan memerlukan beberapa langkah mudah.

Dalam hal ini terdapat istilah yang kerap disebut dengan DNS nameserver yang tidak sedikit dikenal dengan nama resolver. Istilah yang dapat dipahami sebagai server yang dipergunakan oleh perangkat lunak lain guna mencari DNS untuk domain yang diminta: berikut langkah setting DNS ubuntu:

Daftar Isi sembunyikan

1 1. Membuka File

2 2. Menyimpan di Directory

3 3. Mengubah IP Menjadi DNS Server Pilihan

4 4. Mengedit File

5 5. Menghasilkan Konfigurasi

6 Rekomendasi

1. Membuka File

Langkah pertama untuk melakukan setting DNS ubuntu server yaitu dengan membuka file yang diperlukan. Perlu diketahui bahwa file yang diperlukan ini masih ada, namun mungkin bentuknya akan menjadi symlink. Yang diketahui dilakukan pengendalian oleh layanan lainnya.

Untuk itulah tak diperbolehkan untuk dilakukan pengeditan dengan cara manual. Istilah yang namanya systemd-resolved merupakan suatu layanan yang diketahui memberikan resolusi DNS Name. Biasanya diperuntukan bagi aplikasi lokal maupun layanan yang mampu dikonfigurasi memakai Netplan.

2. Menyimpan di Directory

Tahap berikutnya dalam melakukan pengaturan pada DNS Ubuntu server yakni melakukan penyimpanan atas file konfigurasi dari Netplan. Kemungkinan pengguna nantinya akan menjumpai satu ataupun dua jenis file YAML pada bagian directory ini.

Apa Itu Link Building, Ketahui Pengertian Dan Jenis-Jenisnya

Kemudian bisa saja terjadi perbedaan atas nama file dari satu pengaturan dengan pengaturan lainnya. Umumnya file ini mempunyai nama tersendiri yang kemungkinan pada sistem yang dimiliki oleh pengguna akan berbeda.

Dengan file ini pengguna berkemungkinan untuk melakukan konfigurasi interface jaringan. Hal ini juga mencakup tentang gateway kemudian alamat IP, lalu DNS nameserver dan lainnya. Nantinya untuk melakukan pembukaan terhadap konfigurasi ini bisa menggunakan teks editor.

3. Mengubah IP Menjadi DNS Server Pilihan

Perlu diketahui bahwa untuk melakukan konfigurasi DNS ini pengguna perlu melakukan perubahan terhadap alamat IP sekarang ini. Hingga alamat IP tersebut menjadi DNS server yang menjadi pilihan. Contohnya ketika pengguna ingin mengubah DNS server cloudflare.

Jika demikian maka pengguna perlu melakukan perubahan pada bagian baris address yang ada. Dalam penulisan yang dipakai, DNS server perlu dipisah dengan koma. Nantinya dalam hal ini pengguna dapat melakukan penambahan dua name server lebih.

4. Mengedit File

Ketika nantinya tidak didapati entri, maka dapat dilakukan penambahan pada bagian bawah dari blok nama interface. Di waktu melakukan pengeditan file YAML, pengguna perlu memastikan bahwasanya sudah memenuhi standar indentasi dari kode YAML itu sendiri.

Apabila kemudian didapati kesalahan dalam melakukan konfigurasi, maka Netplan tak akan melakukan parsing file. Jika sudah selesai tinggal simpan saja file kemudian terapkan perubahan yang dilakukan.

5. Menghasilkan Konfigurasi

Jika sejumlah langkah sebelumnya sudah dijalankan dengan baik. Nantinya Netplan akan menghadirkan yang namanya file konfigurasi bagi layanan yang dilakukan penyelesaian dengan memakai systemd.

Guna melakukan tahapan verifikasi terhadap DNS resolver bahwasanya sudah dalam keadaan diatur, maka perlu untuk menjalankan sejumlah perintah tertentu.

Dimana perintah tersebut berkemungkinan untuk mencetak banyak informasi. Ketika nantinya pengguna sudah memakai filter string, maka selanjutnya bagian outputnya akan tampak terlihat. Untuk itulah pengguna dituntut untuk memperhatikan hal yang ada dengan jeli dan teliti.
