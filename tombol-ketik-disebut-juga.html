22. Kim Hee Sun menggunakan 'shortcut' "ALT + F4" untuk mematikan komputernya. Jika kita melihat pengelompokan pada tombol-tombol 'keyboard', maka tombol "alt" dan tombol "F4" secara berturut-turut masuk ke dalam kelompok? * A. Alt (tombol fungsi); F4 (tombol ketik) B. Alt dan F4 sama-sama di kelompok tombol fungsiOpsi C. Alt (tombol ketik); F4 (tombol fungsi) D. Alt dan F4 sama-sama di kelompok tombol ketik


Jawaban:

A. Alt (tombol fungsi); F4 (tombol ketik)

Jika Alt menjadi tombol fungsi dan F4 menjadi tombol ketik akan susah mengoperasikan atau melakukannya dan tidak diperbolehkan atau diperkenankan.

B. Alt dan F4 sama-sama di kelompok tombol fungsi Opsi

Jika bersama-sama menjadi tombol opsi yang ada akan susah menggunakan ke2nya karena Alt di tombol opsi, sedangkan F4 tombol fungsi dan diperbolehkan atau diperkenankan juga memakai atau menggunakan seperti ini.

C. Alt (tombol ketik); F4 (tombol fungsi)

Ini baru diperbolehkan atau diperkenankan karena ini yang umu dipakai orang untuk memudahkan menutup item, aplikasi, dan program yang sedang aktif.

D. Alt dan F4 sama-sama di kelompok tombol ketik

Jika bersama-sama menjadi tombol ketik yang ada akan susah menggunakan ke2nya karena Alt dan F4 tidak ada di tombol ketik. Alt di tombol opsi, sedangkan F4 di tombol fungsi, jika ke2 itu pindah bersama-sama akan susah mengoperasikan atau melakukannya dan tidak diperbolehkan atau diperkenankan.

Penjelasan:

A. Alt (tombol fungsi); F4 (tombol ketik)

Jika Alt menjadi tombol fungsi dan F4 menjadi tombol ketik akan susah mengoperasikan atau melakukannya dan tidak diperbolehkan atau diperkenankan (Karena Alt di tombol fungsi dan F4 di tombol ketik itu akan berlawanan dengan yang umum atau yang sering dipakai kebanyakan orang).

B. Alt dan F4 sama-sama di kelompok tombol fungsi opsi

Jika bersama-sama menjadi tombol opsi yang ada akan susah menggunakan ke2nya karena Alt di tombol opsi, sedangkan F4 tombol fungsi dan diperbolehkan atau diperkenankan juga memakai atau menggunakan seperti ini. (Jika ke2nya berada pada satu tombol yang ada akan rusak keyboardnya karena ke2 itu lain tombolnya).

C. Alt (tombol ketik); F4 (tombol fungsi)

Ini baru diperbolehkan atau diperkenankan karena ini yang umu dipakai orang untuk memudahkan menutup item, aplikasi, dan program yang sedang aktif (Ini baru diperkenankan atau diperbolehkan karena yang banyak digunakan atau dipakai).

D. Alt dan F4 sama-sama di kelompok tombol ketik

Jika bersama-sama menjadi tombol ketik yang ada akan susah menggunakan ke2nya karena Alt dan F4 tidak ada di tombol ketik. Alt di tombol opsi, sedangkan F4 di tombol fungsi, jika ke2 itu pindah bersama-sama akan susah mengoperasikan atau melakukannya dan tidak diperbolehkan atau diperkenankan (Jika ke2nya berada pada satu tombol yang ada akan rusak keyboardnya karena ke2 itu lain tombolnya).

37. 1. Sebutkan tombol-tombol pada tombol ketik (typing keys)2.Sebutkan tombol tombol pada numeric keypad 3.Sebutkan tombol tombol pada tombol fungsi (function keys)4.Sebutkan tombol tombol pada tombol kontrol (control keys)​


Jawaban:

1.Tombol tombol pada typing keys yaitu tombol abjad A-Z

2. Tombol tombol pada numeric keys yaitu tombol angka 0-9

3. Tombol tombol pada function keys yaitu tombol F1 sampai F12

4. Tombol tombol pada control keys contohnya tombol Ctrl, Alt, Esc.

Penjelasan:

1. Selain tombol abjad, ada juga tombol alfanumeric. Yaitu tombol yang menjadi gabungan antara angka dan simbol-simbol tertentu.

2. Selain tombol angka, yang termasuk dalam numeric keys ialah tombol /*-+.

3. Tombol tombol fungsi umumnya terletak di bagian paling atas di keyboard (berderet dari F1 sampai F12)

4. Selain contoh yang sudah disebutkan di atas, yang termasuk tombol control key dapat dilihat pada gambar berikut ini (peta keyboard).
