19. FungsiTentukan fungsi invers dari :• f(x) = 9x + 3• f(x) = (x- 3)⁵• f(x) = 4x + 5/3x - 2​


Penjelasan dengan langkah-langkah:

fungsi invers

Jika :

f(x) = (ax + b)/(cx + d)

maka,

f^-1(x) = (-dx + b)/(cx - a)

sehingga, tentukan fungsi invers dari :

• f(x) = 9x + 3

misalkan f(x) = y

f(x) = 9x + 3

y = 9x + 3

9x + 3 = y

9x = y - 3

x = (y - 3)/9

maka, fungsi invers nya f^-1(x) = (x - 3)/9

........

• f(x) = (x - 3)⁵

misalkan f(x) = y

(x - 3)⁵ = f(x)

(x - 3)⁵ = y

x - 3 = ⁵√y

x = ⁵√y + 3

jadi, fungsi invers nya f^-1(x) = (⁵√x + 3)

.........

• f(x) = (4x + 5)/(3x - 2)

misalkan f(x) = y

f(x) = (4x + 5)/(3x - 2)

y = (4x + 5)/(3x - 2)

y(3x - 2) = (4x + 5)

3xy - 2y = 4x + 5

3xy - 4x = 2y + 5

x(3y - 4) = 2y + 5

x = (2y + 5)/(3y - 4)

jadi, fungsi invers nya f^-1(x) = (2x + 5)/(3x - 4)

24. Diketahui fungsi f(x) = 3 cos(4x+60°) -2.tentukan amplitudo, periode, serta nilai maksimum dan minimum grafik fungsinya!​


Fungsi f(x) = 3 cos (4x + 60°) - 2 memiliki

A. Amplitudo sebesar 3

B. Periode sebesar 90°

C. Nilai Maksimum adalah 1

D. Nilai Minimum adalah -5

Dalam soal ini, kita membahas fungsi trigonometri. Sebelumnya, kita mengenal macam bentuk fungsi trigonometri. Bentuk fungsi trigonometri adalah :

a. y = A sin (kx ± b) ± c

b. y = A cos (kx ± b) ± c

c. y = A tan (kx ± b) ± c

Nilai maksimum dan minimum bisa dicari dengan rumus:

Nilai maksimum = |A| + c

Nilai minimum = - |A| + c

Maka, untuk mencari besar amplitudo, menggunakan rumus:

Amplitudo = (Nilai maksimum - Nilai minimum) / 2

Sementara periodenya menggunakan rumus:

Besar periode = 360°/k = 2π/k

Pembahasan

Fungsi trigonometri y = 3 cos (4x + 60°) - 2

A = 3; k = 4; c = -2

maka,

A. Menghitung nilai maksimum dan minimum

Nilai maksimum = |A| + c = |3| - 2 = 3 - 2 = 1

Nilai maksimum = - |A| + c = - |3| - 2 = -3 - 2 = -5

B. Menghitung nilai amplitudo

Amplitudo = (Nilai maksimum - Nilai minimum)/2 = (1 - (-5)) / 2

Amplitudo = 6 / 2 = 3 (sama dengan nilai A, maka A itu bisa dikatakan adalah besar amplitudonya)

C. Menghitung nilai periode

Periode = 360°/k = 360°/4 = 90°

29. 5. Tentukan periode dan amplitudo dari fungsi f(x) = √5 cos 8(x - 90°) 12​


Jawaban:

Untuk mencari periode dan amplitudo dari fungsi f(x) = √5 cos 8(x - 90°), kita perlu mengenali bentuk umum dari persamaan cosinus:

A cos (Bx + C)

dalam persamaan tersebut, A adalah amplitudo, B adalah frekuensi, dan periode fungsi adalah 2π/B. C digunakan untuk menyesuaikan fase awal dari fungsi, tetapi tidak mempengaruhi amplitudo, frekuensi, atau periode.

Dalam kasus fungsi f(x) = √5 cos 8(x - 90°), kita dapat mengamati bahwa:

A = √5

B = 8

C = 90°

Sehingga, amplitudo fungsi adalah √5, frekuensi adalah 8, dan fase awal diatur sebesar 90° ke kiri. Periode fungsi dapat dihitung sebagai 2π/B = 2π/8 = π/4.

Jadi, periode dari fungsi f(x) = √5 cos 8(x - 90°) adalah π/4 dan amplitudo fungsi adalah √5.
