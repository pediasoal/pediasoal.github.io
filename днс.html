Untuk menghitung entalpi (AH) dari setiap reaksi, kita perlu melihat koefisien stoikiometri dari setiap zat dalam reaksi dan nilai entalpi molar yang diberikan.

A. S + 2O2 + H2 → H2SO4, ΔH = -10 kJ

B. 2SO2 + O2 → 2SO3, ΔH = -77 kJ

C. 3H2 + 1.5O2 → 3H2O, ΔH = -286 kJ

D. CS2 + 3O2 → CO2 + 2SO2, ΔH = -1076 kJ

Mari kita hitung entalpi total dari reaksi-reaksi tersebut dengan menggunakan hukum Hess.

Reaksi yang diberikan:

S + 2O2 + H2 → H2SO4, ΔH = -10 kJ

Reaksi balikan dari reaksi B:

2SO3 → 2SO2 + O2, ΔH = +77 kJ

Reaksi balikan dari reaksi C:

3H2O → 3H2 + 1.5O2, ΔH = +286 kJ

Kombinasi dari reaksi B dan C:

2SO3 + 3H2O → 2SO2 + 3H2 + 1.5O2, ΔH = - 77 kJ + 286 kJ = +209 kJ

Reaksi balikan dari reaksi A:

H2SO4 → S + 2O2 + H2, ΔH = +10 kJ

Kombinasi dari reaksi A dan reaksi balikan dari reaksi B dan C:

H2SO4 + 2SO3 + 3H2O → S + 2O2 + H2 + 2SO2 + 3H2 + 1.5O2, ΔH = -10 kJ + 209 kJ + 10 kJ = +209 kJ

Reaksi yang diberikan:

SO2 + O2 → SO3, ΔH = -15 kJ

Kombinasi dari reaksi yang diberikan dan reaksi balikan dari reaksi B:

2SO2 + O2 - 2SO2 - O2 → SO3 + 2SO2 - SO3, ΔH = -15 kJ + (-77 kJ) = -92 kJ

Reaksi yang diberikan:

3H2 + 0.5O2 → H2O, ΔH = -90 kJ

Kombinasi dari reaksi yang diberikan dan reaksi balikan dari reaksi C:

3H2 + 0.5O2 - 3H2O + 1.5O2 → H2O + 3H2O - H2, ΔH = -90 kJ + (-286 kJ) = -376 kJ

Reaksi yang diberikan:

CS2 + 3O2 → CO2 + 2SO2, ΔH = - 1076 kJ

Kombinasi dari reaksi yang diberikan dan reaksi balikan dari reaksi A dan reaksi balikan dari reaksi B:

CS2 + 3O2 - S - 2O2 - H2 + 2SO2 + 2SO3 → CO2 + 2SO2 + S + 2O2 + H2 + 2SO2 + 2O2 - CO2 + 2SO3, ΔH = -1076 kJ + (-10 kJ) + (-209 kJ) = -1295 kJ

Dalam pendekatan ini, total entalpi terhitung sebagai jumlah dari entalpi reaksi-reaksi yang diberikan. Sebagai hasilnya, kita mendapatkan ΔH = +209 kJ + (-92 kJ) + (-376 kJ) + (-1295 kJ) = -1554 kJ.
