1. pengertian dan fungsi tag html, elemen html, atribut html?


Tag Adalah teks khusus (markup) berupa dua karakter "<" dan ">" (tanpa tanda kutip) yang merupakan cara untuk memberitahu web browser tentang apa fungsi dari suatu teks yang kita ketik di dokumen HTML, apakah text itu ditulis sebagai sebuah paragraf, list, atau sebagai link. 


Hampir semua tag dalam HTML ditulis secara berpasangan (beberapa berupa tag tunggal), tag pembuka dan tag penutup, dimana objek yang dikenai perintah tag berada di antara tag pembuka dan tag penutup.

Element adalah isi dari tag yang berada diantara tag pembuka dan tag penutup (termasuk tag itu sendiri). Sebagai contoh perhatikan kode HTML berikut:

<i> Ini adalah sebuah paragraf </i>

Pada contoh diatas, “<i> Ini adalah sebuah paragraf <i>” merupakan element i. Element tidak hanya berisi text, namun juga bisa tag lain.



Atribut adalah informasi tambahan yang diberikan kepada tag. Informasi ini bisa berupa instruksi untuk warna dari text, besar huruf dari text, dll. Setiap atribut memiliki pasangan nama dan nilai (value), dan ditulis dengan name=”value”. Value diapit tanda kutip, boleh menggunakan tanda kutip satu (‘) atau dua (“).

16. jelaskan pengertian HTML dan CSS!​


HTML adalah, (HyperText Markup Language) sebuah bahasa standar yang digunakan oleh browser Internet untuk membuat halaman dan dokumen pada sebuah Web yang kemudian dapat diakses dan dibaca layaknya sebuah artikel. Cascading Style Sheet (CSS) merupakan salah satu bahasa pemrograman web untuk mengendalikan beberapa komponen dalam sebuah web sehingga akan lebih terstruktur dan seragam.- HTML (Hypertext Markup Language), yaitu bahasa markup standar untuk membuat dan menyusun halaman dan aplikasi web. Kita bisa menggunakan bahasa markup ini untuk bagian paragraf, heading, maupun link pada suatu web page.

- CSS (Cascading Style Sheet) dan biasanya digunakan untuk mengatur tampilan elemen yang tertulis dalam bahasa markup, seperti HTML. CSS berfungsi untuk memisahkan konten dari tampilan visualnya di situs.

Perbedaan utama antara HTML vs. CSS adalah: HTML menentukan konten dan struktur halaman web, dan CSS menentukan desain dan tampilan elemen HTML. File CSS dapat digunakan dalam file HTML, tetapi file CSS bukanlah halaman web yang akan ditampilkan sendiri.

38. jelaskan fungsi kontak bantu kontaktor​


Pengertian :

Kontak bantu kontaktor adalah kontak yang bekerja jika kontaktor magnet aktif dan mati, juga jika kontaktor tidak aktif.

Mayortitasnya Kontak bantu kontakto digunakan sebagai pengunci, kontak ini akan menjaga kontaktor supaya tetap aktif. Karena dalam sebuah instalasi panel menggunakan sebuah saklar pengendali jenis Push button, saklar ini hanya bekerja pada saat ditekan dan mati ketika dilepas.

Fungsi :

sebagai pengunci : kontak ini akan menjaga kontaktor agar tetap aktif.sebagai pengendali lampu indikator : lampu indikator akan menandakan juga jika kontaktor masih dalam kondisi bekerja.sebagai pengendali perangkat elektronik : kontak ini akan berfungsi juga untuk perangkat-perangkat elektronik yang lain.
