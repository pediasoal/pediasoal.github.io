Objek kontrol adalah elemen visual yang digunakan dalam aplikasi Visual Basic .NET untuk membuat tampilan dan fungsi aplikasi. Berikut ini adalah beberapa objek kontrol yang tersedia dalam Visual Basic .NET, beserta penjelasannya:

6. RadioButton: Objek ini berfungsi sebagai tombol radio yang dapat digunakan untuk memilih satu opsi dari beberapa opsi yang tersedia.

7. Label: Objek ini merupakan teks statis yang digunakan untuk menampilkan informasi atau petunjuk kepada pengguna.


4. Bagaimana deklarasi variable panjang, lebar dan luas pada Visual Basic. Net?​.


Deklarasi variabel panjang, lebar dan luas pada Visual Basic .Net dapat dilakukan dengan:

Cara pertama:
[tex]\begin{aligned}&\texttt{Dim panjang As Single}\\&\texttt{Dim lebar As Single}\\&\texttt{Dim luas As Single}\end{aligned}[/tex]Cara kedua:
[tex]\begin{aligned}&\texttt{Dim panjang, lebar, luas As Single}\end{aligned}[/tex]

 

Pembahasan

Pada Visual Basic .NET, cara mendeklarasikan variabel sama seperti pada Visual Basic, yaitu dengan sintaks:

[tex]\boxed{\,\texttt{Dim namaVariabel As tipeData}\,}[/tex]

Hal pertama yang harus dilakukan adalah menentukan tipe data yang digunakan untuk variabel panjang, lebar, dan luas. Agar dapat menampung nilai bilangan bulat dan bilangan pecahan/rasional, kita gunakan tipe data floating point, yaitu:

[tex]\tt Single[/tex] (single-precision floating point, 32-bit), atau [tex]\tt Double[/tex] (double-precicion floating point, 64-bit).

Kita gunakan [tex]\tt Single[/tex] saja.

Deklarasi ketiga variabel tersebut dapat dilakukan dengan dua cara.

Cara pertama:
[tex]\begin{aligned}&\texttt{Dim panjang As Single}\\&\texttt{Dim lebar As Single}\\&\texttt{Dim luas As Single}\end{aligned}[/tex]Cara kedua: gabungkan tiga baris deklarasi tersebut menjadi 1 baris
[tex]\begin{aligned}&\texttt{Dim panjang, lebar, luas As Single}\end{aligned}[/tex]Contoh Penggunaan

Berikut ini diberikan contoh sederhana sebuah program VB.Net untuk menghitung luas persegipanjang.

' luas-persegipanjang.vb
Option Explicit
Imports System
Module luasPersegiPanjang
   Sub Main()
       ' Deklarasi variabel
       Dim panjang, lebar, luas As Single

       ' Input panjang dan lebar
       Console.Write("Panjang = ")
       panjang = Console.ReadLine()
       Console.Write("Lebar = ")
       lebar = Console.ReadLine()
       ' Menghitung luas persegipanjang
       luas = panjang * lebar
       ' Cetak luas persegipanjang
       Console.WriteLine("Luas persegipanjang = {0}", luas)
   End Sub
End Module

Karena saya menggunakan .Net di Debian Linux, maka saya kompilasi dan jalankan pada terminal/shell Debian dengan perintah:
vbc /nologo luas-persegipanjang.vb && ./luas-persegipanjang.exe

Hasil eksekusi menginputkan panjang dan lebar adalah sebagai berikut:
Panjang = 16
Lebar = 33.3
Luas persegipanjang = 532.8

9. Apa perbedaan struktur kontrol yg digunakan free basic dengan visual basic? ​


Penjelasan:

Struktur pemrograman dalam Visual Basic adalah perintah dengan struktur yang telah ditentukan untuk mengolah jalannya suatu program.

Struktur kontrol pemrograman biasa dibagi menjadi 3 bagian, yaitu :

Struktur kontrol keputusan, digunakan untuk memutuskan kode program mana yang akan dikerjakan bedasarkan suatu kondisi.

Struktur Kontrol Pengulangan, digunakan untuk melakukan pengulangan kode program.

Penanganan kesalahan, digunakan untuk menanggulangi kesalahan pada program.

A. Struktur Kontrol KeputusanStruktur

1. IF…THEN

Bentuk penulisan (syntax) struktur IF…THEN :

IF <kondisi> THEN <kode program>

Bila <kondisi> bernilai True maka <kode program>akan di kerjakan

IF <kondisi> THEN

<blok kode program 1>

ELSE

<blok kode program 2>

END IF

Bila <kondisi> bernilai True maka <blok kode program 1> akan dikerjakan, tetapi

bila<kondisi> bernilai False maka <blok kode program 2> yang akan dikerjakan.

Contoh :

IF umur <= 17 THEN

Pesan = “kamu tidak boleh masuk”

ELSE

Pesan = “silahkan masuk”

END IF

Jika umur nilainya kurang dari 17 atau sama dengan 17, maka pesan yang muncul “kamu tidak boleh masuk”. Sedangkan, jika umurnya lebih dari 17,maka pesan yang muncul “silahkan masuk”.

2. Select … Case

Bentuk penulisan (syntax) struktur SELECT…CASE :

SELECT CASE <pilihan>

CASE <pilihan 1>

<blok kode program 1>

CASE <pilihan 2>

<blok kode program 2

CASE <pilihan n>

<blok kode program n>

[CASE ELSE

<blok kode program x>]

END SELECT

Bila <pilihan> sesuai dengan <pilihan 1> maka <blok kode program 1>

akan dikerjakan, dan seterusnya. Tetapi bila <pilihan> tidak ada yang

sesuai dengan <pilihan 1> s/d <pilhan n> maka <blok kode program x>

yang akan dikerjakan.

Contoh :

SELECT CASE Nilai

CASE “A” = keterangan = “sangat baik”

CASE “B” = keterangan = “baik”

CASE “C” = keterangan = “cukup”

CASE “D” = keterangan = “kurang”

CASE “E” = keterangan = “gagal”

CASE ELSE = keterangan = “belum ujian”

END SELECT

24. Berikan contoh kode program visual Basic yang menggunakan funtion procedure ???​.


Jawaban:

1. Pengertian

Procedure yaitu sub program yang digunakan untuk melakukan proses tertentu dan tidak mengembalikan nilai, bisa disimpan dalam database sebagai object skema, sehingga suatu procedure bisa digunakan berulangkali tanpa harus melakukan parsing dan compile ulang.

Function yaitu Sebuah kumpulan Statement yang akan mengembalikan sebuah nilai balik pada pemanggilnya. Nilai yang dihasilkan Function harus ditampung kedalam sebuah variabel.

2. Perbedaan Function dan Procedure

Function akan mengembalikan suatu nilai pada pemanggilnya, sedangkan.

Procedure tidak akan mengembalikan nilai apapun pada fungsi pemanggilnya.

3. Bentuk Penulisan (Syntax)

Bentuk penulisan (syntax) procedure sub,

[public|private]Sub(||)

      …..

     

      …..

End Sub

Bentuk penulisan (syntax) procedure function,

[public|private]Function(||) As

     …..

   

     …..

End Function

Pernyataan [public|private] menyatakan ruang lingkup (scope) kerja prosedur. Jika prosedur tersebut berada pada scope public berarti digunakan dalam lingkup project. Dan jika pada scope private berarti digunakan dalam lingkup form.


25. sebutkan beberapa project yang bisa digunakan oleh banyak pengguna visual basic !


Microsoft Visual Basic adalah sebuah aplikasi yang digunakan untuk melakukan pengembangan berbagai macam aplikasi. Selain itu, environment yang ditawarkan dalam pembangunannya adalah dengan menggunakan Integrated Development Environment atau lebih sering disebut dengan IDE. Selain itu, aplikasi ini memiliki bentuk visual yang akan membantu kita dalam proses pembuatan program.

Visual Basic sendiri adalah sebuah aplikasi yang pada awalnya menggunakan Bahasa pemrogramman BASIC dimana Bahasa BASIC tersebut digunakan untuk melakukan pengembangan aplikasi perangkat lunak berbasis user interface dengan sangat cepat. Kemudian, aplikasi ini menyediakan komponen-komponen yang akan mendukung berbagai jenis project yang ada di Visual Basic. Berikut adalah jenis project-project yang ada pada Visual Basic

 Jenis project pada Visual Basic

- Standard EXE

Merupakan jenis aplikasi yang digunakan untuk membangun kepentingan bisnis yang biasanya ada dan tidak harus menggunakan aplikasi-aplikasi lainnya. Dengan menggunakan Standard EXE apa yang digunakan juga sangat sederhana dan terdapat keunggulan dimana semua komponen yang terdapat dapat pada Standard EXE diakui oleh setiap unit komputer yang tersedia meskipun user tersebut bukanlah seorang administrator.

- Active EXE

Active EXE merupakan salah satu project dengan fungsi yang difokuskan untuk aplikasi lain untuk menjadi sebuah layanan service pada aplikasi tersebut. Kemudian, service yang dibuat ini dapat digunakan didalam komputer yang sama maupun komputer yang dioperasikan secara online dengan menggunakan remote automation. Untuk service ini, biasanya tidak terdapat desain user interface dan project dari ActiveX memiliki komponen-komponen yang dapat melakukan komunikasi dengan semua aplikasi yang terdapat dan serta dapat digunakan oleh sistem operasi windows.

- ActiveX DLL

Active DLL adalah jenis aplikasi yang digunakan untuk aplikasi lainnya yang kemudian menjadi bagian dari service. Berbeda dengan Active EXE, hanya terdapat pada satu komputer yang digunakan pada ActiveX DLL saja. Serta aplikasi ini juga tidak menyediakan desain dari user interface. Tetapi, aplikasi ini dapat dijalankan melalui aplikasi Microsoft Transaction Server apabila ingin dijalan pada komputer lainnya.

- ActiveX Control

Merupakan salah satu komponen yang digunakan pada device yang memakai dan mendukung aplikasi dari ActiveX dan berupa sebuah aplikasi.

- VB Application Wizard

Project dari aplikasi ini akan membantu pengguna aplikasi untuk melakukan pembuatan aplikasi dengan sangat mudah.

- Addin

Project yang seperti Standard EXE yang memiliki lebih banyak kebebasan

- Data project

Data project didesain untuk pembuatan aplikasi berbentuk database

- DHTML

DHTML digunakan untuk melakukan pembuatan aplikasi internet dengan sistem client side sehingga dapat digunakan pada client dengan metode serta fungsi-fungsi yang terdapat pada DHTML

- IIS application

Seperti DHTML, tetapi sistem yang digunakan adalah server side. Untuk komponen yang dimiliki IIS juga menggunakan komponen dari CGI atau lebih dikenal dengan Common Gateway Interface.

- VB Wizard Manager

38. tipe data betikut yang merupakan default dari visual basic .Net adalah​


Mengenal Tipe Data pada Visual Basic 6.0 (VB6)

1. Boolean

    Boolean merupaka tipe data yang memiliki dua(2) buah nilai yaitu nilai Benar (True) dan nilai Salah (False).  

2. Byte

    Byte merupakan tipe data yang berupa nilai bulat positif yang mempunyai kisaran antara 0 s/d 225.  

3. Currency

    Currency merupakan tipe data yang digunakan untuk menyimpan nilai uang baik dalam bentuk nilai dollar maupun sejenisnya, kisaran tipe data ini antara -922.337.203.685.477,5808 s/d 922.337.203.685.477,5807.  

4. Date

    Date merupakan tipe data yang digunakan untuk meyimpan nilai tanggal, bulan, dan tahun. Kisaran tipe data ini antara 1 januari 100 s/d 31 Desember 9999.  

5. Decimal

    Decimal merupakan tipe data yang digunakan untuk menyimpan data desimal dan mempunyai ketepatan hingga 28 angka desimal.  

6. Double

    Double merupakan tipe data yang memiliki kisaran antara -1.7976931348622E+308 s/d 1.7976931348622E+308.  

7. Integer

     Integer merupakan tipe data numerik yang berupa bilangan bulat yang mempunyai kisaran antara -32.768 s/d 32.767.  

8. Long

    Long merupakan tipe data yang memiliki kisaran antara -2.147.483.648 s/d 2.147.483.648. tipe data ini sebenarnya tidak jauh berbeda dengan tipe data Integer tetapi perbedaannya terletak pada nilai kisarannya.  

9. Object

    Object merupakan tipe data yang mampu menyimpan data object seperti object form, kontrol dan sebagainya.  

10. Singgel

    Singgel merupakan tipe data yang memiliki kisaran antara -3.402823e+308 s/d 3.402823e+38.  

11. String

    String merupakan tipe data yang memiliki nilai yang termasuk dalam kategoti alfanumerik. biasanya karakter dari tipe data ini ditulis dalam tanda kutip ("").  

12. Variant

    Variant merupakan tipe data yang akan menyesuaikan dengan kebutuhan nilainya. Dan jika sebuah variable tidak dideklarasikan tipe datanya, maka secara otomatis visual basic akan mendeklarasiakan variable tersebut menjadi tipe data Variant.

Microsoft Visual Basic .NET adalah sebuah alat untuk mengembangkan dan membangun aplikasi yang bergerak di atas sistem .NET Framework, dengan menggunakan bahasa BASIC. Dengan menggunakan alat ini, para programmer dapat membangun aplikasi Windows Forms, Aplikasi web berbasis ASP.NET, dan juga aplikasi command-line. Alat ini dapat diperoleh secara terpisah dari beberapa produk lainnya (seperti Microsoft Visual C++, Visual C#, atau Visual J#), atau juga dapat diperoleh secara terpadu dalam Microsoft Visual Studio .NET. Bahasa Visual Basic .NET sendiri menganut paradigma bahasa pemrograman berorientasi objek yang dapat dilihat sebagai evolusi dari Microsoft Visual Basic versi sebelumnya yang diimplementasikan di atas .NET Framework. Peluncurannya mengundang kontroversi, mengingat banyak sekali perubahan yang dilakukan oleh Microsoft, dan versi baru ini tidak kompatibel dengan versi terdahulu.

Apakah Visual Basic .NET dianggap sebagai sebuah versi Visual Basic atau benar-benar bahasa yang berbeda merupakan sebuah topik perdebatan yang hangat. Hal ini dikarenakan sintaksis bahasa Visual Basic .NET tidak mengalami perubahan yang sangat drastis, dan hanya menambahkan beberapa dukungan fitur baru seperti penanganan eksepsi secara terstruktur dan ekspresi yang bisa di-short-circuit-kan. Dua perubahan tipe data pun terjadi saat berpindah ke Visual Basic .NET. Dibandingkan dengan Visual Basic 6.0, tipe data Integer yang dimiliki oleh Visual Basic .NET memiliki panjang dua kali lebih panjang, dari 16 bit menjadi 32 bit. Selain itu, tipe data Long juga sama-sama berubah menjadi dua kali lipat lebih panjang, dari 32 bit menjadi 64 bit. Bilangan bulat 16-bit dalam Visual Basic .NET dinamakan dengan Short. Lagi pula, desainer GUI Windows Forms yang terdapat di dalam Visual Studio .NET atau Visual Basic .NET memiliki gaya yang sangat mirip dengan editor form Visual Basic klasik.

Jika sintaksis tidak banyak yang berubah, lain halnya dengan semantik, yang berubah secara signifikan. Visual Basic .NET merupakan sebuah bahasa pemrograman yang mendukung fitur "Bahasa Pemrograman Berorientasi Objek" secara penuh, karena memang didukung oleh arsitektur Microsoft .NET Framework, yang mengandung kombinasi dari Common Language Runtime dan Base Class Library. Visual Basic klasik, hanya merupakan sebuah bahasa pemrogaman berbasis objek, yang berjalan di atas arsitektur Component Object Model (COM).

Perubahan ini telah mengubah banyak asumsi tentang hal yang benar yang harus dilakukan dengan mempertimbangkan performa dan kemudahan untuk dipelihara. Beberapa fungsi dan pustaka perangkat lunak, yang ada di dalam Visual Basic klasik, kini tidak terdapat di dalam Visual Basic .NET; mungkin masih banyak yang masih terdapat di dalam Visual Basic .NET, tetapi tidak seefisien apa yang ditawarkan oleh .NET Framework. Bahkan jika program Visual Basic klasik bisa dikompilasi dengan benar, sebagian besar program Visual Basic klasik harus melalui beberapa proses refactoring untuk mengadopsi fitur bahasa baru secara keseluruhan.
