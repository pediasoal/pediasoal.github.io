13. pengertian extend dan include dalam use case


Extend (Perluasan): Extend digunakan ketika suatu use case memiliki aksi tambahan yang dapat terjadi tergantung pada kondisi tertentu. Use case yang diperluas adalah use case utama, sedangkan use case yang memperluas adalah ekstensi. Use case ekstensi hanya terjadi jika kondisi tertentu dipenuhi. Dengan kata lain, use case ekstensi "meng-extend" fungsionalitas use case utama.

Include (Penggabungan): Include digunakan ketika suatu use case membutuhkan fungsionalitas dari use case lain untuk menyelesaikan tugasnya. Use case yang membutuhkan fungsionalitas tambahan adalah use case inklusi, sedangkan use case yang menyediakan fungsionalitas tambahan adalah use case yang diikutsertakan. Use case inklusi selalu terjadi dan dipanggil oleh use case utama.

Pembahasan

Contoh penggunaan extend:

Misalkan kita memiliki use case "Pesan Tiket" sebagai use case utama. Namun, terdapat ekstensi yang disebut "Tawarkan Upgrade Kelas" yang hanya terjadi jika pengguna memilih opsi upgrade kelas saat memesan tiket. Use case "Tawarkan Upgrade Kelas" adalah contoh dari extend yang memperluas fungsionalitas use case utama "Pesan Tiket".

Contoh penggunaan include:

Misalkan kita memiliki use case "Bayar Tagihan" sebagai use case utama. Di dalam use case ini, terdapat fungsionalitas untuk "Verifikasi Pembayaran". Namun, "Verifikasi Pembayaran" juga dapat digunakan oleh use case lain, seperti "Lihat Riwayat Pembayaran". Dalam hal ini, use case "Bayar Tagihan" akan mencakup atau mengikutsertakan use case "Verifikasi Pembayaran".

33. Kapan sebaiknya menggunakan directive: –#include atau –#include ”header.h”


Jawaban:

saat kita mau menggunakan fungsi atau konstanta atau kelas(hanya c++) yang terdapat dalam header tersebut.

Penjelasan:

preprocessor directive "#include <contohheaderstandar.h>" atau "#include <contohheaderc++standar>" atau "#include <contohheaderbuatan.h>" akan memasukkan semua kode yang ada di file header tersebut kedalam file utama. contoh:

header.h :

int tambah (int a, int b) {

return a + b;

}

main.c :

#include <stdio.h> // untuk header standar

#include "header.h" //untuk header buatan

int main () {

printf(tambah(1,2));

}

kode main diatas sebelum dikompilasi akan diubah menjadi berikut:

//kode kode yang terdapat pada stdio.h

//....

//kode pada header.h

int tambah (int a, int b) {

return a + b;

}

//kode utama

int main () {

printf(tambah(1,2));

}

37. Jelaskan kegunaan atau fungsi dari code program berikut ini #include


Include
Include adalah salah satu pengarah prepocessor directive yang tersedia pada C++. Preprocessor selalu dijalankan terlebih dahulu pada saat proses kompilasi terjadi. Bentuk umumnya :
                                   # include <nama file>
Tidak diakhiri dengan tanda semicolon, karena bentuk tersebut bukanlah suatu bentuk pernyataan, tetapi merupakan prepocessor directive. Baris tersebut menginstrusikan kepada kompiler untuk menyisipkan file lain dalam hal ini file yang berakhiran .h (file header) yaitu file yang berisi C++ standard library.

contohnya: - # include <iostream.h>   : diperlukan pada program yang                                                                          melibatkanobjek cout dan cin
                 - # include <conio.h>   : diperlukan bila melibatkan clrscr( ), yaitu                                                        perintah untuk membersihkan layar dan                                                          fungsi getch( ) untuk menerima sembarang                                                      input keyboard dari user.
                 - # include <iomanip.h>   : diperlukan bila melibatkan setw( ) yang                                                            bermanfaat untuk mengatur lebar dari                                                              suatu tampilan data.
                 - # include <math.h>   : diperlukan pada program yang                                                                    menggunkan  operasi sqrt( ) yang                                                                   bermanfaat untuk operasi matematika kuadrat.
