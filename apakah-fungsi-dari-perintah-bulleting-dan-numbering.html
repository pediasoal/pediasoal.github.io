15. untuk mengubah nomor tertentu pada bullets and numbering digunakan perintah


Untuk mengubah nomor tertentu pada bullets and numbering di Microsoft Word, Anda dapat menggunakan fitur "Renumber" atau "Change List Level". Berikut cara menggunakannya:

Pilih item yang ingin Anda ubah nomornya.Klik kanan pada item tersebut dan pilih "Renumber" atau "Change List Level".Masukkan nomor yang diinginkan pada kolom "Enter new numbering value".Klik "OK" untuk mengubah nomor item tersebut.

Anda juga dapat menggunakan shortcut untuk mengubah nomor item. Caranya:

Pilih item yang ingin Anda ubah nomornya.Tekan tombol "Alt" + "Shift" + "Left Arrow" atau "Right Arrow" untuk mengubah level item tersebut.

Perlu diingat bahwa fitur ini hanya bisa digunakan jika item tersebut sudah tergabung dalam daftar bullets atau numbering. Jika item tersebut belum tergabung dalam daftar, Anda perlu memformat item tersebut terlebih dahulu menjadi bullets atau numbering.

Word processor adalah evolusi dari mesin ketik konvensional yang dibuat setelah ditemukannya komputer.

Word Processor atau aplikasi pengolah kata adalah software / perangkat lunak yang berfungsi untuk mengolah kata dalam membuat suatu dokumen, hal yang bisa dikerjakan pada aplikasi word processor atara lain membuat, mengedit, memodifikasi, hingga mencetak dokumen tersebut. Aplikasi word processor banyak digunakan untuk membuat dokumen laporan, artikel, brosur, karya tulis, surat, dan lain sebagainnya.

Berikut ini beberapa contoh aplikasi pengolah kata yang sering digunakan:

OpenOfficeNotepadWordpadMicrosoft WordGoogle Docs

Dalam aplikasi word processor, khususnya Microsoft Word, terdapat fitur untuk merapikan sebuat lis informasi. Fitur tersebut adalah bullets dan numbering.

Bullets adalah fitur yang memungkinkan kita memberi tanda atau lingkaran pada setiap paragraf sebelum teks. Bullets pada microsoft word tidak hanya berbentuk lingkaran saja ada beberapa macam bentuk yang sudah disediakan.

Numbering adalah memberikan nomor atau angka pada setian paragraf sehingga tulisan akan tampak urut sesuai dengan penomoran yang diinginkan. Numbering juga tidak hanya angka desimal tetapi juga bisa berupa angka romawi dan lain-lain.

Cara menggunakan Numbering :  

Pilih tab Home di Microsoft WordKlik Numbering pada kolom paragrafPilih Numbering yang di inginkanPembahasan

Pada menu numbering, terdapat pilihan Define New Number Format. Setelah di klik, akan muncul jendela Define New Number Format ada di lampiran.

Kesimpulan

Ada beberapa pilihan perintah pada pilihan Define New Number Format yaitu:

Number StyleNumber FormatAlignment
