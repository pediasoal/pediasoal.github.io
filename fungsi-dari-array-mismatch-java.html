Untuk menampilkan semua hari dari sebuah array hari dalam bahasa Java, kita dapat menggunakan perulangan for seperti berikut:

String[] hari = {"Senin", "Selasa", "Rabu", "Kamis", "Jumat", "Sabtu", "Minggu"};

for (int i = 0; i < hari.length; i++) {

System.out.println(hari[i]);

}

Untuk menampilkan hari Selasa sampai Jumat saja, kita dapat mengubah perulangan for menjadi seperti berikut:

for (int i = 1; i <= 3; i++) {

for (int i = 1; i <= 3; i++) {

25. Mohon bantuannya Buatlah program Java membalik kalimat dan menentukan sebuah kalimat termasuk palindrom atau tidak menggunakan stack Array dan Array list


Jawaban:

import java.util.Stack;

import java.util.ArrayList;

public class PengecekanPalindrome{

   public static void main(String[] args) {

       String kalimat = "katak";

       // Membalik kalimat menggunakan stack Array

       char[] kalimatArray = kalimat.toCharArray();

       Stack<Character> stack = new Stack<>();

       for (char c : kalimatArray) {

           stack.push(c);

       }

       StringBuilder kalimatBalik= new StringBuilder();

       while (!stack.isEmpty()) {

           kalimatBalik.append(stack.pop());

       }

       // Menentukan apakah kalimat merupakan palindrom atau tidak

       boolean Palindrome = kalimat.equalsIgnoreCase(kalimatBalik.toString());

       // Menampilkan hasil

       System.out.println("Kalimat asli: " + kalimat);

       System.out.println("Kalimat terbalik: " + kalimatBalik.toString());

       if (Palindrome) {

           System.out.println("Kalimat merupakan palindrom.");

       } else {

           System.out.println("Kalimat bukan palindrom.");

       }

   }

}

36. Haii gays mohon bantuan. Buatlah program java menkoversikan nilai desimal ke biner mengunakan stack dengan array dan array list desimal= 25 biner = 11001


Jawaban:

Berikut adalah contoh program Java untuk mengkonversi nilai desimal ke biner menggunakan stack dengan implementasi array dan ArrayList:

Menggunakan stack dengan array:

import java.util.Scanner;

public class DecimalToBinary {

   public static void main(String[] args) {

       Scanner input = new Scanner(System.in);

       System.out.print("Masukkan nilai desimal: ");

       int decimal = input.nextInt();

       int[] binaryArray = new int[32]; // Array untuk menyimpan nilai biner

       int index = 0;

       // Konversi desimal ke biner menggunakan stack dengan array

       while (decimal > 0) {

           binaryArray[index] = decimal % 2;

           decimal = decimal / 2;

           index++;

       }

       System.out.print("Nilai biner: ");

       for (int i = index - 1; i >= 0; i--) {

           System.out.print(binaryArray[i]);

       }

   }

}

Menggunakan stack dengan ArrayList:

import java.util.ArrayList;

import java.util.Scanner;

public class DecimalToBinary {

   public static void main(String[] args) {

       Scanner input = new Scanner(System.in);

       System.out.print("Masukkan nilai desimal: ");

       int decimal = input.nextInt();

       ArrayList<Integer> binaryList = new ArrayList<>();

       // Konversi desimal ke biner menggunakan stack dengan ArrayList

       while (decimal > 0) {

           binaryList.add(decimal % 2);

           decimal = decimal / 2;

       }

       System.out.print("Nilai biner: ");

       for (int i = binaryList.size() - 1; i >= 0; i--) {

           System.out.print(binaryList.get(i));

       }

   }

}

Dalam kedua contoh di atas, program akan meminta pengguna memasukkan nilai desimal yang akan dikonversi menjadi biner. Kemudian, proses konversi dilakukan menggunakan stack dengan array atau ArrayList, dengan membagi desimal secara berulang dengan 2 dan menyimpan sisa hasil bagi (0 atau 1) ke dalam stack. Akhirnya, nilai biner akan dicetak ke layar.

Catatan: Program ini hanya mengkonversi nilai desimal positif ke biner. Jika ingin mengatasi nilai desimal negatif atau nilai desimal yang lebih besar, Anda dapat melakukan penyesuaian pada program tersebut.

           {1, 2, 3},

           {4, 5, 6},

           {7, 8, 9}

       };

       // Deklarasi dan Inisialisasi variable arrayA

       int arrayB[][] = {

           {9, 8, 7},

           {6, 5, 4},

           {4, 2, 1}

       };

       // Perulangan FOR untuk baris Array

       for(int z = 0; z < arrayA.length; z++) {

           // Perulangan FOR untuk data pada baris Z

           for(int x = 0; x < arrayA[z].length; x++) {

               // Operasi penjumlahan array A dan B

               System.out.print(arrayA[z][x] + arrayB[z][x] + " ");

           }

           System.out.println("");

       }

   }

}

Penjelasan:

Array yang digunakan adalah array2D yang menyerupai sebuah table. Untuk mengambil nilai dari Array2D menggunakan Nested Loop atau perulangan bersarang. Perulangan(DOR) pertama digunakan untuk baris pada array. Perulangan (FOR) kedua untuk  kolom pada array.

Pada gambar langkah 1, memilih baris pada array. Langkah 2,3,4 milih kolom pada array. Karena Array2D menyerupai table untuk memanggil nilai array dengan cara "nama_array[baris][kolom]".
