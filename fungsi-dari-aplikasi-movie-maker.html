21. Sebutkan 10 jenis tools di software windows movie maker dan fungsinya​


Video Effects : Berfungsi memberikan efek efek pada video yang ingin dibuat (efek blur, miror, sepia, watercolor, dan lain lain) Video Transition: Berfungsi memberikan efek pada saat video dimainkan (gambarnya bergilir kesamping, ke kanan, horizontal, vertical, dll.) Collections: Berfungsi menampung file file yang akan dijadikan video. Import Video : Berfungsi membuka file video yang akan ditambahkan ke collections. Import Picture: Berfungsi membuka file gambar yang akan ditambahkan ke collections. Import Audio or music: Berfungsi membuka file suara yang akan ditambahkan ke collections. Show collections : Berfungsi menampilkan semua file yang dipilih untuk dijadikan video. View video effects: Berfungsi menampilkan gambaran effect yang telah dipilih untuk video tersebut. View video transition: Berfungsi menampilkan gaya perubahan dari satu video ke video selanjutnya yang telah dipilih Save to computer : untuk menyimpan hasil video ke driver komputer
