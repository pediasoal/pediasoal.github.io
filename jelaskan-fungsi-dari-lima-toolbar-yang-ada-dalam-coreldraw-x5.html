1. Jelaskan fungsi dari lima toolbar yang ada dalam coreldraw X5


Pick tool. Tool ini berfungsi untuk merubah ukuran sesuai yang kita inginkan dan juga untuk menyeleksi gambar pada Corel draw.

Shape tool. Mempunyai fungsi yaitu digunakan untuk mengubah suatu bentuk objek yang terdapat di halaman kerja Corel draw. Selain itu, shape tool juga mempunya beberapa sup tool lagi yang berjumlah 4 tool. Yang mana 4 sub tool ini saling berhubangan satu dengan yang lainnya. 4 tool itu adalah shape, Smudge Brush, Free transform dan Roughen Brush. Adapun jalan pintasnya atau sering di sebut dengan tombol shortcut, kita cukup menekan tombol F10, mudah kan.

Crop tool merupakan tool yang dapat kita gunakan untuk memotong objek yang tidak kita butuhkan. Tool ini juga sama seperti shape tool, sama – sama mempunyai sub tool berjumlah 4 yaitu Crop, Eraser, Knife dan Virtual Segment Delete. Yang mana sub tool ini saling berhubungan.

Zoom tool. Tool ini berfungsi untuk memperbesar dan memperkecil objek atau halaman kerja Corel draw. Tool i9ni juga mempunyai 2 sub tool yaitu Zoom atau dengan menekan Z dan Pan cukup menekan H.

Freehand tool merupakan tool yang berfungsi untuk membuat berbagai macam garis. Di dalam Freehand tool juga terdapat berbagai tool antara lain : Freehand, Bezier, 2-point line, Pen, Artistic Media, Polyne, B-spline dan 3-point curve. Dapat juga kita menekan F5 dan untuk Artistic Media kita menekan tombol I.

20. Macam macam ikon yang terdapat dalam toolbar standar pada coreldraw?


Ada banyak ikon menu – menu pada jendela CorelDraw X3 yang memiliki fungsi yang berbeda – beda. Adapun bagian – bagian ikon menu tersebut antara lain :

Menu Bar : berisikan menu utama yang ada diatas dan banyak digunakan untuk membantu saat bekerja dalam CorelDraw, dimana menu – menu tersebut dikelompokkan sesuai dengan kategorinya. Menu Bar ini terdiri dari File, Edit, View, Layout, Arrange dllProperty Bar : menu ini berfungsi untuk menunjukkan properti / nilai dari objek yang ada di tampilan lembar kerja CorelDraw sehingga Property Bar akan selalu berubah – ubah tergantung dari objek yang ditampilkan. Misal lembar kerja kertas secara default adalah letter, coba ubah menjadi A4 maka nilai Property Bar akan berubahToolbar : menu yang standar dimunculkan dalam ikon – ikon menu seperti undo, redo, copy, paste, save.Title Bar : menu ini berada paling atas dan sebagai penunjuk nama file yang sedang dibuka. Secara default saat membuka awal program CorelDraw X3 maka nama Title Bar’nya yaitu CorelDraw X3 -[Graphic 1]Ruler : sebagai penggaris yang sangat berguna sebagai alat ukur saat bekerja dengan CorelDrawToolbox : menu sangat penting dalam CorelDraw karena berisikan ikon – ikon menu yang berguna melakukan desain kotak, lingkaran, gradasi, dll. Menu ini biasanya ada disamping kiri dari lembar kerjaDrawing window : area diluar media kerja Corel Draw. Biasanya klik kanan di menu Drawing window dengan Drawing Page (media / lembar kerja) akan berbeda hasilnya.Serta objek gambar yang ada di area ini akan masuk dalam PrintDrawing page : area media / lembar kerja CorelDraw yang dipakai untuk menggambar / menulis. Jika objek yang digambar sampai keluar area Drawing page maka akan terjadi perintah trouble saat PrintColor Pallete : berisikan kumpulan warna yang bisa dipakai untuk mewarnai objek gambar / tulisan di media kerja CorelDraw.Secara default letak Color Pallete ada disamping kanan dari media kerja CorelDrawDocker : menu ini berisikan tampilan – tampilan menu dari Menu Bar. Coba saja klik Text pilih Insert Symbol Character, maka akan ada menu baru disamping kanan media kerja CorelDrawStatus Bar : menu yang berada di paling bawah dan berisikan informasi text dari objek yang aktif / diklikDocument navigator : berfungsi sebagai navigasi untuk menambah jumlah halaman dari media kerja, biasanya ditulis dalam Page 1, Page 2 dllNavigator : berfungsi untuk membantu mengarahkan tampilan di media kerja CorelDraw.

1. CorelDraw adalah sebuah program komputer editor grafik vektor

2.

Tool Box

Tool Box adalah suatu kotak atau alat yang memiliki fungsi untuk menggambar, modifikasi objek, memberi warna, menulis teks. Tool Box memiliki beberapa fungsi di satu kotak atau tempat yang sama, jika ingin membuka atau mempergunakan fungi lain maka silahkan klik tanda segitiga kecil di pojok kanan bawah salah satu tool box.

Tool Bar

Tool Bar merupakan sekumpulan tombol shortcut yang memiliki fungsi untuk menjalankan perintah dengan cepat. ShortCut Ini adalah bagian yang paling sering digunakan oleh desainer grafis dalam melakukan sesuatu. CorelDraw memberikan fitur dalam toolbar yang powerfull namun mudah sekali digunakan.

Menu Bar

Menu Bar berisi tombol-tombol text yang memiliki banyak sekali fungsi yang berbeda sesuai dengan namanya masing-masing. Desainer pun sering menggunakan fasilitas ini biasanya dalam memberikan suatu effect berhubungan dengan vector dan bitmap.

Property Bar

Property Bar berisi sekumpulan tombol yang isinya itu menyesuaikan dengan tools yang sedang kita pilih di dalam tool box.

Docker

Docker merupakan kotak dialog yang di dalamnya berisi berbagai tombol perintah untuk mengatur, memberi effect kepada sebuah objek maupun teks.

Rulers

Rulers merupakan alat ukur berupa penggaris dibagian atas dari drawing page, dan di bagian kanan drawing page, rullers digunakan sebagai acuan yang mampu menunjukkan ukuran suatu objek atau dokumen, serta tata letak suatu desain.

Status Bar

Status Bar terletak dibagian bagian bawah area program sebelah kanan. Status Bar berisi informasi mengenai objek (teks) seperti ukuran, bentuk, warna, outline, format gambar, jenis huruf beserta  ukurannya.

Drawing Page

Drawing Page adalah adalah sebuah area kerja utama di dalam CorelDraw di mana Anda dapat mendesain suatu objek di dalamnya dan merupakan area cetak suatu desain.
