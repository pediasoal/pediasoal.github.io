11. Tania is a philatelist. She is very fond ofcollecting.....a. Envelopes b. tetters c. stampsd. coins​


Jawaban:

C. Stamps

Penjelasan:

Karena Philatelist adalah orang yang mengumpulkan perangko.

Istilah ‘philately’ berasal dari bahasa Yunani, yaitu kata ‘philllo’ dan ‘ateleia’.

Kata ‘philllo’ berarti ketertarikan pada sesuatu dan ‘ateleia’ artinya bebas pajak.

Maksud dari ‘bebas pajak’ adalah prangko sebagai surat pengiriman yang tidak perlu dibayar saat pengiriman.

Konon, kata itu awalnya dibuat menjadi istilah ‘philatelie’ oleh filateli asal Prancis, Georges Herpin. Kemudian dalam bahasa Inggris, istilah itu diartikan menjadi ‘philately’.

29. Terletak di sebelah kanan menu References yang memuat icon Envelopes, Labels, Start Mail Merge, dan lain-lain pada program Microsoft Word merupakan .....​


Mailings

Pembahasan:

Menu Referensi (References) pada program Microsoft Word berfungsi untuk membantu pengguna dalam membuat referensi pada sebuah dokumen. Terdapat beberapa ikon dan fitur di dalam menu ini, seperti Envelopes, Labels, Table of Contents, Footnotes, dan lain-lain. Namun, untuk membahas lebih lanjut, pertanyaan kali ini akan difokuskan pada fitur Start Mail Merge yang ada pada menu References.

Start Mail Merge adalah salah satu fitur pada menu Mailings di Microsoft Word yang sangat berguna bagi pengguna. Fitur ini digunakan untuk mengirimkan dokumen yang sama ke banyak penerima dengan mudah. Caranya, pengguna hanya perlu membuat dokumen sebagai template atau contoh, kemudian mengisi informasi tentang penerima di database atau daftar kontak. Setelah itu, pengguna dapat menggabungkan konten dari dokumen dan data dari database untuk membentuk surat baru yang unik, dan mengirimkannya ke penerima.

Fitur ini sangat efektif dan efisien bagi pengguna yang perlu mengirimkan jumlah surat yang besar dalam waktu yang singkat. Misalnya, dalam membuat undangan acara, pengguna hanya perlu membuat satu dokumen undangan saja, dan menggabungkannya dengan data daftar tamu. Dengan menggunakan fitur Start Mail Merge, pengguna dapat membuat surat undangan yang unik dan spesifik untuk masing-masing tamu dengan cepat dan mudah.

Dalam penggunaannya, start mail merge terdiri dari tiga tahap. Pertama, pengguna harus memilih jenis dokumen yang ingin dibuat, misalnya surat atau label. Kedua, pengguna harus memilih database atau daftar kontak yang akan digunakan sebagai sumber data. Terakhir, pengguna dapat memasukkan field dari database ke dalam dokumen untuk mencocokkan isinya.

Dalam mengisi field dari database ke dalam dokumen, pengguna dapat menggunakan fitur Mail Merge Helper untuk memudahkan pekerjaannya. Fitur ini akan membantu pengguna untuk memilih field yang ingin digunakan dan menunjukkan preview dari hasil akhir untuk setiap penerima.

Dalam menyimpan file hasil merge, pengguna dapat memilih format file asli, seperti Docx atau PDF, atau format lain yang berfungsi sebagai file master seperti Excel. File dalam format master ini dapat digunakan kembali untuk proses start mail merge selanjutnya atau untuk keperluan lainnya.

Dalam kesimpulannya, start mail merge pada menu Mailings di Microsoft Word sangat membantu bagi pengguna yang perlu mengirimkan surat atau dokumen yang sama ke banyak penerima. Dengan menggunakan fitur ini, pengguna dapat membuat banyak surat yang unik dan spesifik dengan cepat dan mudah. Selain itu, pengguna juga dapat menyimpan file hasil merge untuk penggunaan selanjutnya.
