18. Pengertian remove page break ms excel


Page Break digunakan untuk membuat halaman secara terpisah pada suatu sheet. Jadi walaupun jarak antara range satu dengan range lainnya ataupun cell satu dengan cell lainnya begitu dekat, ia dapat dipisah halamannya saat di print preview/print. Marilah kita belajar membuat page break di dalam Microsoft Excel.

Membuat Page Break

1. Bukalah Microsoft Excel, lalu buatlah file Excel baru.
2. Ketikan pada cell A1: "Halaman 1".
3. Ketikan pada cell A2: "Halaman 2".
4. Misal, yang kita inginkan adalah saat di print preview, cell A1 akan berada pada halaman satu, dan cell A2 akan berada pada halaman dua.
5. Tempatkan kursor di cell A2.
6. Klik menu Insert à Page Break.


7. Silahkan klik menu File à Print Preview.

8. Maka sekarang kedua cell ini tampil secara terpisah di dalam dua halaman, padahal jarak mereka sangat berdekatan.


Mengedit Jangkauan Page Break

1. Klik menu View à Page Break Preview. Jika muncul dialog box, klik OK.

2. Perbesar tampilan Excel dengan cara mengubah ukuran zooming menjadi 200%.

3. Untuk memperbesar/memperkecil area page break, klik and drag kotak yang berwarna biru. Misal seperti pada gambar sebagai berikut:



4. Kita dapat pula menambah data tambahan di halaman 1 dan di halaman 2 tersebut. Keluarlah dari mode print preview, kemudian ketikkan pada cell A3: "TEST". Maka secara otomatis ia akan terlingkup oleh page break sebagai object pada halaman 2.

5. Jika sudah selesai, klik menu File à Print Preview.

6. Maka walaupun cell-cell ini letaknya saling berdekatan, halamannya menjadi terpisah di dalam 2 halaman.


Menghapus Page Break

1. Klik menu View à Page Break Preview. Jika muncul dialog box, klik OK.

2. Perbesar tampilan Excel dengan cara mengubah ukuran zooming menjadi 200%.

3. Klik area page break yang hendak dihapus

4. Pilih menu Insert à Remove Page Break.


Kesimpulan

Page break akan dibuat berdasarkan lokasi cell yang sedang kita aktifkan/pilih. Jadi jika cell C5 yang sedang kita pilih, maka ia akan membelah area dari sebelah kirinya dan sebelah atasnya. Perhatikan gambar berikut:

19. apakah perbedaan cover page blank page & page break​


Fungsi Cover Page, Blank Page & Page Break di Ms. Word - Suatu laporan atau makalah yang dicetak dalam bentuk kliping; Cover pasti akan selalu ada di halaman paling depan.

Bahasa lainnya yakni sampul depan; Cover adalah suatu halaman yang berisi informasi utama dari isi makalah tersebut; misalnya judul makalah, nama pengarang, daftar anggota, dan masih banyak lagi.

Fungsi Cover Page, Blank Page & Page Break di Ms. Word

Berkaitan dengan Cover; kali ini Office Praktis bakal jelasin apa yang dimaksud dengan Cover Page, Blank Page, dan Page Break serta fungsinya masing2.

Pengertian Cover Page, Blank Page, dan Page Break serta Fungsinya

Cover Page adalah suatu cara untuk membuat halaman "sampul depan" dengan template yang sudah disajikan secara default (bawaan) oleh program aplikasi Microsoft Word.

Nah ada lagi yang dinamakan Blank Page. Blank Page adalah suatu cara untuk menampilkan dan/atau membuat halaman terbaru pada dokumen yang sama.

Tindakan dari Blank Page pun tidak akan mengganggu halaman lainnya; dan bahkan, penomoran dari tiap2 halaman pada dokumen tersebut akan berubah secara otomatis.

Mau bukti dari Blank Page? Coba ketikan sesuatu pada halaman pertama di Ms. Word. Lalu tekan Blank Page, secara otomatis kalian akan diarahkan ke halaman selanjutnya tanpa peduli ruang spasi pada halaman pertama tadi.

Bisa dibilang, dengan Blank Page; kita gak perlu lagi tekan tombol enter-enter dan enter untuk menuju ke halaman selanjutnya.

Terakhir ada Page Break. Tahu gak sih apa itu Page Break?

Pengertian dan fungsi dari Page Break itu hampir mirip dengan Blank Page; hanya saja, Blank Page bisa dilakukan secara bebas, sedangan Page Break harus mengasarkan kursor pada paragraf tertentu untuk membuat halaman baru selanjutnya.

Page Break bisa dibilang membuat halaman baru. Caranya? Buat 5 baris paragraf. Pada paragraf ketiga, arahkan kursor dibagian terakhir, dan klik tombol Page Break.

Maka baris paragraf pertama dan kedua akan dihalaman pertama, sedangkan baris paragraf ketiga, keempat, dan kelima akan diarahkan ke halaman selanjutnya (pada dokumen yang sama).

28. Cari tahu perbedaan blank page dan page break.


Blank page dan page break merupakan konsep yang berbeda dalam dokumen Microsoft Word atau aplikasi pengolah kata lainnya.

Blank page atau halaman kosong adalah halaman yang tidak memiliki konten apapun. Halaman kosong ini dapat dibuat dengan menekan tombol "Enter" sebanyak mungkin di akhir dokumen sebelumnya, atau dengan menambahkan "Section Break" di akhir dokumen.Page break atau jeda halaman adalah kontrol yang digunakan untuk memisahkan konten dari satu halaman dengan halaman berikutnya. Page break dapat ditambahkan dengan menekan tombol "Ctrl + Enter" atau dengan menggunakan menu "Page Layout" dan memilih "Page Break".

Secara umum, blank page digunakan untuk memberikan ruang kosong yang diinginkan pada dokumen, sedangkan page break digunakan untuk memisahkan konten dari satu halaman dengan halaman berikutnya.
