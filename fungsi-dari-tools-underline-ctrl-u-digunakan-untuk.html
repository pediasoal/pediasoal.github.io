1. Perbedaan save dan save as adalah pada nama dokumen yang akan disimpan. Bila save untuk menyimpan dokumen yang belum dinamai. Sedangkan save as untuk menyimpan dokumen yang telah disimpan sebelumnya dengan nama baru.

Save dan Save as adalah menu-menu yang digunakan dalam penyimpanan data.

Perbedaan save dan save as adalah:

Save digunakan untuk menyimpan dokumen yang belum memiliki nama.

Save as digunakan untuk menyimpan dokumen yang sebelumnya telah memiliki nama dengan nama dokumen yang baru.

Dengan kata lain, save digunakan untuk menyimpan dokumen baru, sedangkan save as digunakan untuk menyimpan dokumen lama dengan nama yang baru.

Ctrl C:Fungsi menyalin teks yang telah di blok (Copy)

4. bold adalah, ikon yang terdapat dalam MS.WORD yang berguna untuk menebalkan huruf.

25. a. fungsi dari Ctrl + fb. fungsi dari Ctrl + oc. fungsi dari Ctrl + dd. fungsi dari Ctrl + ve. fungsi dari Ctrl + uf. fungsi dari Ctrl + cnote:✨ mau habisin points ✨jawab dengan benar ✔️❌ ngasal auto lapor mod ❌​


Jawaban:

a. Ctrl + F - Temukan Kalimat/Huruf (Find). Ctrl + G - Pergi ke (go to). Ctrl + H - Temukan Kalimat dan Tindih (Find and Replace). Ctrl + I - Membuat cetak miring (Italic)

b. Fungsi ini menggantikan klik kanan > open pada mouse. Ctrl + N = fungsi shortcut untuk membuat berkas baru/New. Ctrl + O = fungsi shortcut untuk membuka berkas sebelumnya/Open yang sudah disimpan

c. Ctrl + T = fungsi shortcut untuk membuka tab baru pada browser. Ctrl + D = fungsi shortcut untuk melakukan bookmark pada halaman yang sedang di buka

d. Ctrl + X = fungsi untuk cut/memotong teks yang dipilih. Ctrl + C = fungsi untuk copy/menduplikat teks yang dipilih. Ctrl + V = fungsi untuk paste/meletakan teks yang dicopy/cut sebelumnya

e. Ctrl + B = fungsi untuk membuat bold/mencetak tebal teks yang dipilih. Ctrl + I = fungsi untuk membuat italic/garis miring teks yang dipilih. Ctrl + U = fungsi untuk membuat underline/garis bawah teks yang dipilih

f. Ctrl + C = fungsi untuk copy/menduplikat teks yang dipilih.

● Fungsi dari Ctrl + F = mencari File dan membuka menu/kotak dialog Find and Replace

● Fungsi dari CTRL + O = membuka menu/kotak dialog open

● Fungsi dari CTRL + V = mentempel/menampilkan hasil teks yang sudah di copy sebelumnya

● Fungsi dari CTRL + U = membuat garis bawah teks

● Fungsi dari CTRL + C = mengcopy/menyalin teks maupun File

Jawaban:

Pendahuluan :

Taukah kamu apa itu Microsoft Office word ? Ya, itu merupakan sebuah aplikasi yang paling populer dalam mengetik, memprint sesuatu baik berupa surat (dokumen) maupun berupa foto (gambar). Tentunya dalam mengetik sesuatu pada aplikasi ini membutuhkan yang namanya tombol untuk mengetik. Tombol komputer tersebut biasa kita kenal dengan sebutan keyboard. Keyboard adalah alat bantu kita untuk mengetik maupun memberi perintah sesuai yang kita inginkan. Di dalam tombol keyboard terdiri atas huruf, angka, maupun simbol - simbol lainnya.

Keyboard (Qwerty) diciptakan oleh Christopher Latham Sholes pada tahun 1868. Ia lahir di negara Amerika Serikat pada tanggal 14 Februari 1819. Setelah anda menyimak penjelasan di atas, kini saatnya membahas pertanyaan fungsi tombol dari keyboard.

Pertanyaan :

Tombol kombinasi yang digunakan untuk menebalkan, membuat garis bawah dan tulisan miring pada aplikasi pengolah kata adalah . . . ?

Pembahasan :

Tombol perintah (ctrl) yang berfungsi untuk menebalkan tulisan di aplikasi software microsoft office adalah tombol Ctrl + B.

Sesuai nama tombolnya, ctrl artinya perintah dan b (bold) artinya menebalkan huruf.

Tombol perintah (ctrl) yang berfungsi untuk membuat garis bawah tulisan di aplikasi software microsoft office adalah tombol Ctrl + U.

Sesuai nama tombolnya, huruf u merupakan awalan kata Underline yang artinya garis bawah.

Tombol kombinasi untuk membuat huruf menjadi miring adalah tombol Ctrl + I.

Sesuai nama tombolnya, huruf I merupakan awalan kata Italic yang artinya tulisan miring.

Kesimpulan :

Ctrl + B = Berfungsi untuk mempertebal huruf

Ctrl + U = Berfungsi untuk mengaris bawahi huruf

Ctrl + I = Berfungsi untuk membuat garis miring
