24. Hasil dari kode program berikut adalah... #include <iostream>using namespace std; int main() { int x = 157, y = ⁸; int hasil; hasil = x & y; cout<<hasil; return 0;} O 1 O 5 O 2 O 4 O 3 ​


Jawaban:

Kode program tersebut akan menghasilkan output berupa angka desimal, yaitu 8.

Penjelasan:

Baris pertama mendeklarasikan variabel x dengan nilai 157 dan variabel y dengan nilai ⁸. Namun, terdapat kesalahan penulisan pada variabel y yaitu menggunakan karakter superskrip yang salah, sehingga programnya akan mengalami error saat dijalankan.

Namun, bila kita asumsikan karakter superskrip itu sebenarnya adalah angka 8, maka program akan berjalan dan menghasilkan output 8.

Pada baris selanjutnya, nilai variabel hasil dihitung dengan operasi bitwise AND antara nilai variabel x dan y (dalam biner). Nilai biner 157 adalah 10011101, sedangkan nilai biner 8 adalah 00001000. Jika kedua nilai tersebut di-operasikan dengan AND, maka hasilnya adalah 00001000 (dalam biner) atau 8 (dalam desimal).
