Jawaban:

a) Tom opens the door/ The door opens by Tom

b) Tom is opening the door/ The door is opening by Tom.

c) Tom has opened the door/ The door has opened by Tom.

d) Tom opened the door/ The door opened by Tom.

e) Tom was opening the door/ The door was opening by Tom.

f) Tom had opened the door/ The door had opened by Tom.  

g) Tom will opens the door/ The door will opens by Tom.

h) Tom is going to open the door/ The door is going to open by Tom.

i) Tom will have opened the door/ The door will have opened by Tom.  

j) Is Tom opening the door?/ Is the door opening by Tom.

k) Did Tom opens the door?/ did the door opens by Tom  

l) Has Tom opened the door?/ has the door opened by Tom.

Explanation:

You can change a sentence from Active to Passive simply as exchange the position from Subject + tobe + Verb/ Nominal + Object, to Object + tobe + Verb/ Nomina + by/ use/ with + Subject

-

Kamu dapat mengubah sebuah paragraf dari Aktif menjadi pasif semudah menukar posisi Subjek (Pelaku) + Kata hubung + Kata Kerja/ Kata Sifat + Objek (sasaran) menjadi Objek + Kata hubung + Kata Kerja/ Kata Sifat + by/ use/with + Subjek

18. we closed the door atau we closing the door?


we closed the door artinya kami menutup pintu
we closing the door artinya kami menutup pintuClosed => kata dasar: close (+ed)
Imbuhan ed berarti menunjukkan kegiatan/sesuatu yang waktu pengerjaannya sudah berlalu. Tapi, tataan kata dalam kalimat 'We Closed the Door' yang benar adalah 'We HAVE Closed the door'. Harus ada kata HAVE nya, karena kegiatan yang dilakukan waktunya telah berlalu/selesai. Arti Have pada kalimat tersebut berarti 'telah/selesai/menyele- saikan'
Closing => kata dasar: close (+ing)
Imbuhan ing berarti menunjukkan bahwa sesuatu itu sedang dikerjakan (belum terselesaikan/menunjukkan waktu yang sekarang).

31. Change the active to the passive by supplying the correct form of beExample: Tom opens the door : The door is opened by Tom1. Tom is opening the door : The door ... opened by Tom2. Tom has opened the door : The door ... opened by Tom3. Tom opened the door : The door ... opened by Tom4. Tom was opening the door : The door ... opened by Tom5. Tom had open the door : The door ... opened by Tom6. Tom will open the door : The door ... opened by Tom7. Tom is going to open the door : The door ... opened by Tom8. Tom will have opened the door : The door ... opened by Tom​


Bentuk passive nya adalah..

Answer :

1). The door is being opened by Tom. (Present continuous)

2). The door has been opened by Tom. (Present perfect)

3). The door was opened by Tom. (Simple past)

4). The door was being opened by Tom. (Past continuous)

5). The door had been opened by Tom. (Past perfect)

6). The door will be opened by Tom. (Simple future)

7). The door is going to be opened by Tom. (Simple future "be going to")

8). The door will have been opened by Tom. (Future perfect)

Penjelasan :

Dalam mengubah kalimat aktif ke pasif, maka perhatikan :

Tenses yang digunakanPronounSubject & Object yang digunakan

Berikut ini rumus yang digunakan dari kalimat di atas

Present continuous

Active : S + To be (am/are/is) + V-ing + O

Passive : S + To be (am/are/is) + being + V3 + by..

Present perfect

Active : S + Have/has + V3 + O

Passive : S + Have/has + been + V3 + by..

Simple past

Active : S + V2 + O

Passive : S + To be (was/were) + V3 + by..

Past continuous

Active : S + To be (was/were) + V-ing + O

Passive : S + To be (was/were) + being + V3 + by..

Past perfect

Active : S + Had + V3 + O

Passive : S + Had + been + V3 + by..

Simple future

Active : S + will + V1 + O

Passive : S + will + be + V3 + by .

Simple future "be going to"

Active : S + To be (am/are/is) + going to + V1 + O

Passive : S + To be (am/are/is) + going to + be + V3 + by..

Future perfect

Active : S + will + have + V3 + O

Passive : S + will + have + been + V3 + by..

39. Tom opened the door = the door ...............by Tom


PASSIVE VOICE

The door was opened by Tom.

---------------------

EXPLANATION

Passive voice adalah grammatical voice di mana subject kalimat adalah yang dikenai aksi, bukan yang melakukan aksi seperti pada active voice.

Konstruksi kalimat dalam passive voice adalah:

S + be + past participle

Subjek kalimat pasif adalah objek di kalimat aktif. Subjek kalimat aktif dapat menjadi adverb by+agent bila diperlukan.

--------------------

EXPLICATION

Pada kalimat aktif Tom opened the door, kata kerjanya menggunakan bentuk past, opened.

Jadi, be yang digunakan dalam kalimat pasifnya pun dalam bentuk past yaitu was atau were. Karena the door tunggal, gunakan was.

Setelah be, gunakan bentuk past participle untuk base verb open. Open adalah regular verb dan bentuk past participlenya adalah opened.

Maka kata-kata yang tepat untuk melengkapi kalimat pasif dalam soal adalah was opened.
