8. apa fungsi ikon cut,copy,font,font size,bold,italic,underline,subseript,superscript


cut : memindahkan objek dari dokumen ke clipboard

copy : mengkopi data / menggandakan

font : memilih jenis huruf

font size : memilih ukuran huruf

bold : menebalkan huruf

italic : membuat tulisan miring

underline : membuat tulisan bergaris bawah

subsecript : membuat pangkat di bagian bawah

superscript : membuat pangkat di bagian atas


9. sebutkan fungsi ikon cut copy dan curve


1. Cut : Berfungsi untuk memotong/memindahkan suatu cell ke cell yang lain tanpa meninggalkan isinya.

2. Copy : Berfungsi untuk menggandakan isi cell pada lembar kerja ke cell yang lainnya.

3. Paste : Berfungsi untuk menampilkan hasil cut dan copy dari cell yang satu ke cell yang lainnya

cut = untuk memotong gambar atau media lain
copy = untuk menyalin gambar atau tulisan
