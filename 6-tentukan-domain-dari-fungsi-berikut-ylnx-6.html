12. 3. Tentukan Domain Fungsi berikut a). F(x) = 4 per x²+x-6 b). F(x) = √5x-6​


Jawaban:

a). F(x) = 4 per x²+x-6

F(x) = 4 dibagi dengan x²+x-6

Domain fungsi ini adalah semua bilangan real kecuali akar-akar dari ekspresi kuadrat x²+x-6. Akar-akar dari ekspresi ini adalah -3 dan 2, jadi domain fungsinya adalah x ∈ ℝ {-3, 2}.

b). F(x) = √5x-6

F(x) = akar kuadrat dari 5x-6

Domain fungsi ini adalah semua bilangan real x sedemikian rupa sehingga 5x-6 ≥ 0, atau x ≥ 6/5. Jadi, domain fungsinya adalah x ∈ ℝ {x ∈ ℝ | x < 6/5}.

jadi domain dua fungsinya adalah:

a. F(x) = 4 per x²+x-6: x ∈ ℝ {-3, 2}

b. F(x) = √5x-6: x ∈ ℝ {x ∈ ℝ | x ≥ 6/5}

20. 3. Tentukan Domain Fungsi berikut a) F(x) = y per x²+x-6b) F(x) = √5x-6​


Jawab:

Untuk menentukan domain fungsi, kita harus mencari nilai-nilai x yang memenuhi batasan atau pembatasan pada fungsi tersebut. Domain adalah himpunan semua nilai x yang dapat dimasukkan ke dalam fungsi sehingga menghasilkan nilai y yang terdefinisi.

Penjelasan dengan langkah-langkah:

a) F(x) = y per x² + x - 6

Pada fungsi ini, pembaginya adalah x² + x - 6. Untuk menjaga fungsi tetap terdefinisi, kita harus memastikan bahwa pembaginya tidak sama dengan nol (0), karena pembagian dengan nol tidak diperbolehkan. Oleh karena itu, kita perlu mencari nilai-nilai x yang membuat x² + x - 6 ≠ 0.

Untuk mencari nilai-nilai x yang membuat x² + x - 6 ≠ 0, kita bisa faktorkan persamaan ini:

x² + x - 6 = (x + 3)(x - 2)

Kemudian, kita mencari akar-akar persamaan dengan menyamakan faktor-faktornya dengan nol:

x + 3 = 0 --> x = -3

x - 2 = 0 --> x = 2

Jadi, domain fungsi F(x) adalah semua bilangan real kecuali -3 dan 2.

Domain F(x) = {x ∈ ℝ | x ≠ -3 dan x ≠ 2}

b) F(x) = √(5x - 6)

Pada fungsi ini, kita memiliki akar dari suatu ekspresi 5x - 6. Untuk menjaga fungsi tetap terdefinisi, kita harus memastikan bahwa ekspresi di bawah akar, yaitu 5x - 6, harus lebih besar atau sama dengan nol (5x - 6 ≥ 0). Ini karena kita tidak dapat mengambil akar dari angka negatif.

Untuk mencari nilai-nilai x yang memenuhi 5x - 6 ≥ 0, kita perlu menyelesaikan ketidaksetaraan:

5x - 6 ≥ 0

5x ≥ 6

x ≥ 6/5

Jadi, domain fungsi F(x) adalah semua bilangan real x yang lebih besar atau sama dengan 6/5.

Domain F(x) = {x ∈ ℝ | x ≥ 6/5}

24. Diketahui fungsi f : x →(x-6)(x 6). jika domain {0,1,2,3}. Tentukan jumlah daerah hasilnya adalah


Domain yang diberikan adalah {0, 1, 2, 3}. Mari kita substitusikan nilai-nilai tersebut ke dalam fungsi f:

1. Untuk x = 0:

f(0) = (0-6)(0+6) = (-6)(6) = -36.

2. Untuk x = 1:

f(1) = (1-6)(1+6) = (-5)(7) = -35.

3. Untuk x = 2:

f(2) = (2-6)(2+6) = (-4)(8) = -32.

4. Untuk x = 3:

f(3) = (3-6)(3+6) = (-3)(9) = -27.

Berdasarkan hasil substitusi di atas, kita dapat melihat bahwa fungsi f memiliki hasil negatif pada setiap elemen domain yang diberikan.

Jumlah daerah hasil fungsi adalah jumlah daerah di atas sumbu x di mana hasil fungsi berada di bawah sumbu x. Karena hasil fungsi f adalah negatif pada setiap elemen domain yang diberikan, maka jumlah daerah hasilnya adalah 4.

Jadi, jumlah daerah hasil fungsi f adalah 4.

40. tentukan domain untuk fungsi fungsi berikut​


Jika diketahui fungsi  [tex]f(x) = \frac{x^2}{x - 1}[/tex] , maka domain fungsi tersebut adalah x ≠ 1.

Pembahasan

Suatu fungsi dapat dinyatakan dalam bentuk: f : x ---> f(x). Nilai fungsi untuk setiap nilai x yang diberikan dapat dihitung dengan cara mensubstitusikan nilai x ke dalam rumus fungsi f(x).

Domain (daerah asal) fungsi adalah himpunan nilai-nilai agar suatu fungsi terdefinisi.

Daerah hasil fungsi (range) adalah himpunan nilai-nilai fungsi dari setiap anggota daerah asal (domain).

Penyelesaian

diket:

[tex]f(x) = \frac{x^2}{x - 1}[/tex]

ditanya:

domain....?

jawab:

karena fungsi [tex]f(x) = \frac{x^2}{x - 1}[/tex]  berbentuk pecahan a/b, maka fungsi f(x) agar terdefinisi adalah b ≠ 0, sehingga

x - 1 ≠ 0

x ≠ 1

Kesimpulan

Jadi, domain untuk fungsi [tex]f(x) = \frac{x^2}{x - 1}[/tex] adalah x ≠ 1.
