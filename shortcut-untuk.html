35. shortcut untuk memperbanyak data


Tombol shortcut untuk memperbanyak data dalam Windows adalah "Ctrl" + "C" untuk mengkopi data dan "Ctrl" + "V" untuk memperbanyak data yang sudah dikopi sebelumnya. Khusus untuk pengguna MacOS adalah "Command" + "C" untuk mengkopi dan "Command" + "V" untuk memperbanyak data.
