11. Apa yang dimaksud absolute value dan position value??


Maksud dari absolute value dan position value adalah salah nilai pada sistem bilangan yang terdapat komputer yang menggunakan bilangan desimal sebagai acuannya. Adapun rincian dari absolute value dan position value adalah

Absolute value adalah suatu bilangan pada komputer yang menunjukan nilai mutlak dari suatu bilanganPosition value adalah suatu nilai yang berfungsi sebagai penimbang atau pemberi bobot dari masing-masing bilangan sesuai dengan letak bilangan tersebut.Pembahasan

Setelah mengetahui apa itu absolute value dan position value, maka selanjutnya kita akan menyebutkan beberapa macam bilangan pada komputer, seperti kita ketahui bilangan-bilangan pada komputer terdiri dari beberapa jenis, jenis-jenis bilangan pada komputer adalah

Desimal atau dikenal dengan basis 10Biner atau dikenal dengan basis 2Oktal atau dikenal dengan basis 8Hexadesimal atau dikenal dengan basis 16

Berbagai macam bilangan pada komputer memiliki kegunaan yang berbeda satu dengan yang lainnya. Namun biasanya bilangan desimal dan biner yang sering digunakan.

19. sebutkan fungsi value dalam seni rupa​


Jawaban:

Pada pembuatan desain grafis, value memiliki beberapa fungsi, antara lain :

Untuk mengubah cahaya yang mengenai obyek/benda ke dalam bentuk tiga dimensi semu, misalnya untuk membentuk gelap terang pada gambar bentuk. Value terang (tint) untuk bagian yang kena cahaya langsung, value sedang (tone) untuk bagian yang kena cahaya normal, value gelap (shade) untuk bagian yang tidak kena cahaya atau bagian bayangan

Untuk menciptakan atau menilai karya seni, fungsi ini khususnya berkaitan dengan karakter value yang antara lain adalah sebagai berikut :

-Value terang mempunyai karakter positif, bergairah, meriah, ringan

-Value normal mempunyai karakter tegas, jujur, terbuka, galak

-Value gelap mempunyai karakter berat, dalam, muram, mengerikan, menakutkan

-Untuk mengubah cahaya yang mengenai obyek berwarna-warni menjadi bentuk hitam putih

Penjelasan:

Value adalah dimensi mengenai derajat terang gelap atau tua muda warna. Value merupakan perangkat untuk mengukur derajat terang atau gelapnya suatu warna jika dibandingkan dengan skala value.

23. Apa persamaan employee value dental stakeholder value? Employee value=Stakeholder Value


persamaan employee value dental stakeholder value adalah sama-sama berupa nilai / value yang dikirim kepada semua pemangku kepentingan kebijakan perusahaan baik itu customer, supplier, pekerja, pemilik perusahaan, dan komunitas.


Pembahasan

Employee Value Proposition (EVP) pengertiannya adalah reward/benefit yang diterima pekerja sebagai return pada performa mereka di tempat kerja.

Sedangkan Stakeholder Value Perspective adalah nilai-nilai / value yang terkirim untuk semua pemangku kepentingan (segenap pihak yang terkait dengan bisnis) seperti customers, suppliers, employees, shareholders. Stakeholder Value Perspective adalah tanggung jawab sosial yang penting untuk organisasi dimana peradaban adalah paling baik dilayani oleh organisasi yang mengejar keuntungan dan simbiosis ekonomi.

31. tipe data nilai pada fungsi function yang dikirim balik return value dapat dilihat pada​


Jawaban:

Tipe data nilai pada fungsi function yang dikirim balik return value dapat dilihat dari deklarasi fungsi tersebut. Dalam bahasa pemrograman, deklarasi fungsi biasanya menentukan tipe data yang dikembalikan oleh fungsi tersebut. Contohnya, dalam bahasa pemrograman C++, jika fungsi "foo" dideklarasikan sebagai berikut:

int foo() {

   // ...

}

Maka tipe data nilai yang dikembalikan oleh fungsi "foo" adalah integer (int). Tipe data yang dikembalikan oleh fungsi dapat digunakan untuk memastikan bahwa nilai yang dikembalikan oleh fungsi tersebut sesuai dengan yang diharapkan oleh program yang memanggil fungsi tersebut. Misalnya, jika fungsi "foo" diharapkan untuk mengembalikan integer, maka program yang memanggil fungsi tersebut harus menangani nilai yang dikembalikan sebagai integer.
