4. sebutkan fungsi - fungsi dari tombol keyboard berikut tombol caps lock, tombol tab, tombol back pace, tombol delete, tombol entar, tombol esc, tombol home, tombol end, tombol Page up, tombol Page Down​


Jawaban:

Misalkan dipakai pada program pengolah data teks (seperti Ms. Word):

Caps Lock: untuk mengunci huruf kapital.

Tab: untuk memindah posisi kursor ke sebelah kanan 1 tabulasi.

Back space: untuk menghapus 1 karakter di sebelah kiri

Delete : untuk menghapus 1 karakter di sebelah kanan

Enter: untuk mengakhiri 1 baris dan mengawali pengetikan teks di baris baru (di bawahnya).

Esc: untuk menghilangkan tampilan menu yg muncul.

Home: untuk memindah posisi kursor ke awal baris teks.

End: untuk memindah posisi kursor ke akhir baris.

Page Up: untuk memindahkan posisi kursor 1 layar ke atas.

Page Down: untuk memindahkan posisi kursor 1 layar ke bawah.

Penjelasan:

Contoh pemakaian tombol Esc: misalkan mengklik kanan teks (dg mouse) lalu muncul suatu menu, maka tombol ini bisa dipakai untuk menghilangkan menu yg muncul tadi.

17. jelaskan papan makey makey untuk tombol keyboard dan gerakan mouse​


Jawaban:

Papan Makey Makey adalah sebuah alat yang memungkinkan Anda mengubah objek-objek sehari-hari menjadi tombol keyboard atau gerakan mouse. Prinsip kerja papan Makey Makey didasarkan pada konsep sirkuit yang sederhana.

Papan Makey Makey terdiri dari beberapa konektor yang dapat dihubungkan dengan objek atau benda lainnya, seperti pisang, gelas air, atau bahkan manusia. Setiap konektor pada papan memiliki fungsi yang dapat diatur dan dihubungkan ke perangkat komputer melalui kabel USB.

Untuk membuat objek atau benda tertentu berfungsi sebagai tombol keyboard, Anda perlu menghubungkan salah satu konektor pada papan Makey Makey ke objek tersebut. Misalnya, Anda dapat menghubungkan salah satu konektor ke pisang dengan menggunakan kabel penghubung. Setelah itu, Anda harus menghubungkan satu ujung kabel penghubung yang lain ke ground (tanah) pada papan Makey Makey. Ground ini berfungsi sebagai referensi sirkuit.

Selanjutnya, saat Anda menyentuh pisang yang terhubung dengan papan Makey Makey, sirkuit akan tertutup dan papan akan mendeteksi sinyal sebagai input dari tombol keyboard yang telah ditentukan sebelumnya. Anda dapat mengatur koneksi ini menggunakan perangkat lunak atau library yang disediakan oleh Makey Makey.

Hal yang sama juga berlaku untuk mengubah gerakan mouse menggunakan papan Makey Makey. Anda dapat menghubungkan satu konektor pada papan Makey Makey ke objek yang dapat dideteksi gerakannya, misalnya sepotong aluminium foil yang digulung menjadi bola. Ketika Anda menggulung bola tersebut dan menghubungkannya ke papan Makey Makey, gerakan bola tersebut akan diinterpretasikan sebagai gerakan mouse pada perangkat komputer.

Papan Makey Makey mendeteksi perubahan dalam sirkuit yang terjadi saat objek terhubung dengan konektor. Dengan mengkombinasikan objek-objek dan konektor yang berbeda, Anda dapat menciptakan antarmuka unik dan kreatif untuk mengontrol perangkat komputer Anda. Papan Makey Makey dapat digunakan untuk berbagai macam aplikasi, mulai dari permainan hingga eksperimen ilmiah atau karya seni interaktif.

23. dalam tata papan keyboard terdapat beberapa tombol huruf dan lampu indikator yang menandakan keyboard berfungsi dengan baik dan jelaskan susunan tombol keyboard tersebut​


Jawaban:1. Main Typing keypad

Main typing keyopad adalah papan ketik utama yang berisi huruf A sampai Z, tanda baca dan angka 1 – 0 yang digunakan untuk memasukkan teks ke dalam sistem komputer misalnya saat mengetik pada Microsoft Office atau pada browser internet.

2. Tombol Penunjuk Arah

Dalam keyboard posisi direction keys ini biasanya di sudut kanan bawah. Tombol itu biasanya berisi petunjuk arah dengan simbol anak panah.

3. Numeric/ Directional Keypad

Numerik atau NUM, adalah tombol yang berisi angka dan tombol pemindah kursor yang biasanya terletak pada sisi kanan main typing board.

Tombol ini hanya akan berfungsi jika tombol fungsi Numeric key dinyalakan. Tombol ini akan memudahkan seseorang dalam mengetik dan memasukkan angka ke dalam sistem.

4. Function Keys

Tombol-tombol ini berada diatas tombol main typing keypad dan dapat dikenali dari tanda F1 hingga F12. Tombol-tombol ini berfungsi untuk mengatur beberapa perintah seperti menyimpan dokumen, memunculkan menu help saat mengetik di Microsoft word dan lain sebagainya.

5. Special Keys

Tombol ini ada dibagian atas tombol directional keys dan terdiri dari tombol Home, Delete, Page up, Page Down, Insert, Print screen, pause break dan End.

6. Modifier Keys

Mofofier keysYang termasuk dalam golongan keypad ini adalah tombol Ctrl, Alt dan Shift. Sesuai dengan namanya, ketiga tombol ini akan merubah atau memodifikasi fungsi suatu tombol lain dalam keypad dan bisa ditekan bersamaan tombol lain tersebut untuk memunculkan suatu perintah.

Misalnya tombol Ctrl yang ditekan bersamaan dengan huruf C akan secara otomatis menyalin data yang sudah dipilih pada monitor. Kombinasi tombol-tombol tersebut juga sering disebut sebagai shortcut keys.

8. Lock Keys

Lock keyas adalah indikator cahaya yang menandakan bahwa suatu tombol sedang digunakan untuk mengatur suatu fungsi misalnya tombol Caps lock yang memungkinkan data diketik dengan huruf kapital dan numeric lock yang digunakan untuk mengaktifkan tombol numeric key.

9. Windows keys

Biasanya pada keyboard pada komputer windows memiliki tombol yang bergambar logo Microsoft windows itu sendiri. Tombol ini berfungsi untuk memunculkan menu pada jendela windows pada komputer.

32. Tombol apa saja yang merupakan tombol tombol khusus pada keyboard...


Penjelasan:

Secara umum, struktur tombol pada keyboard

terbagi atas 4, yaitu:

1. Tombol Ketik (typing keys)

Tombol ketik adalah salah satu bagian dari

keyboard yang berisi huruf dan angka serta

tanda baca. Secara umum, ada 2 jenis susunan

huruf pada keyboard, yaitu tipe QWERTY dan

DVORAK. Namun, yang terbanyak digunakan

sampai saat ini adalah susunan QWERTY.

2. Numeric Keypad

Numeric keypad merupakan bagian khusus dari

keyboard yang berisi angka dan sangat

berfungsi untuk memasukkan data berupa

angka dan operasi perhitungan. Struktur

angkanya disusun menyerupai kalkulator dan

alat hitung lainnya.

3. Tombol Fungsi (Function Keys)

Tahun 1986, IBM menambahkan beberapa

tombol fungsi pada keyboard standard. Tombol

ini dapat dipergunakan sebagai perintah khusus

yang disertakan pada sistem operasi maupun

aplikasi.

4. Tombol kontrol (Control keys)

Tombol ini menyediakan kontrol terhadap

kursor dan layar. Tombol yang termasuk dalam

kategori ini adalah 4 tombol bersimbol panah di

antara tombol ketik dan numeric keypad, home,

end, insert, delete, page up, page down, control

(ctrl), alternate (alt) dan escape (esc).
