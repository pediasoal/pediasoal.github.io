1. 1. Ubahlah fungsi asynchronous callback-based getProvinces menjadi Promise-based Catatan: * - Anda boleh menggunakan util.promisify untuk mengubah fungsi callback-based menjadi Promise-based. * - Jika nama fungsinya berubah, sesuaikan nilai yang diekspor tanpa mengubah nama properti di akhir berkas ini


Berikut ini adalah implementasi fungsi asynchronous callback-based  `getProvinces` yang diubah menjadi Promise-based menggunakan `util.promisify`;

const util = require('util');

const fs = require('fs');

const readFile = util.promisify(fs.readFile);

function getProvinces() {

 return readFile('provinces.txt', 'utf8')

   .then(data => {

     const provinces = data.split('\n');

     return provinces;

   })

   .catch(err => {

     console.error(err);

   });

}

module.exports = {

 getProvinces

};

Pembahasan

Dalam implementasi di atas, kita menggunakan util.promisify untuk mengubah fungsi `fs.readFile` menjadi `Promise-based`. Kemudian, kita menggunakan metode `then` dan `catch` untuk mengakses hasil pembacaan file dan menangani kesalahan jika terjadi. Akhirnya, kita mengembalikan nilai `provinces` sebagai hasil dari fungsi `getProvinces`.

Penting digarisbawahi, fungsi tetap sama seperti sebelumnya, sehingga tak perlu mengubah nama properti yang diekspor di akhir berkas.

8. Jelaskan pengertian dari density function atau fungsi kerapatan


Dalam teori probabilitas, Fungsi kepadatan probabilitas atau fkp merupakan segolongan fungsi yang sering digunakan dalam teori statistika untuk menjelaskan perilaku suatu distribusi probabilitas teoretis.

Suatu fungsi memenuhi kriteria sebagai fkp apabila nilainya selalu positif untuk setiap titik absis dan fungsi primitifnya merupakan distribusi probabilitas.

Ini berarti bahwa suatu fkp berharga non-negatif untuk semua nilai absis dan hasil integral tertentunya yang merentang dari −∞ menuju +∞ sama dengan satu.

Selain disebut sebagai fungsi kepekatan probabilitas, pustaka-pustaka juga menyebutnya sebagai fungsi kepekatan peluang atau fungsi kerapatan probabilitas

19. sebutkan function yang ada di microsoft excel beserta fungsinya...​


Function adalah rumus yang telah ditetapkan untuk melakukan perhitungan dengan menggunakan nilai-nilai tertentu dalam urutan tertentu. Microsoft Excel mempunyai banyak fungsi umum yang dapat digunakan seperti menghitung jumlah, rata-rata, nilai maksimum, dan nilai minimum pada berbagai sel.

Function pada microsoft excel beserta fungsinya:

1. Function SUM

SUM: untuk menjumlahkan.AVERAGE: untuk mencari rata-rata.AND: untuk menghasilkan nilai TRUE jika semua argumen yang di uji bernilai benar dan FALSE jika semua atau salah satu argumen bernilai salah.NOT: kebalikan dari AND yaitu menghasilkan TRUE jika kondisi yang diuji salah dan FALSE jika yang diuji benar.OR: untuk menghasilkan TRUE jika beberapa argumen bernilai benar dan FALSE jika semua argumen salah.

2. Function IF

IF: mengembalikan nilai, jika kondisi yang diuji bernilai benar dan mengembalikan sebuah nilai lainnya jika kondisi yang diuji bernilai salah.COUNTIF: menghitung jumlah cell dalam suatu range dengan kriteria tertentu.COUNTA: menghitung jumlah cell yang telah terisi.FIND: menemukan suatu teks atau potongan teks (string) dalam sebuah cell yang berisi teks yang panjang.

3. Function LOOKUP

LOOKUP: mencari nilai data pada suatu kolom atau row. Namun LOOKUP sudah jarang digunakan karena beralih ke VLOOKUP serta HLOOKUP.VLOOKUP: menampilkan data dari sebuah tabel yang disusun tegak/vertikal, dengan syarat penyusunan tabel data pada baris pertama harus urut dari kecil ke besar.HLOOKUP: menampilkan data dari sebuah tabel yang disusun mendatar/horizontal, dengan syarat penyusunan tabel data pada baris pertama harus urut dari kecil ke besar.AREAS: menampilkan jumlah area (range atau cell) dalam suatu referensi (alamat cell atau range yang disebutkan). CHOOSE: memilih dan menunjukkan suatu nilai di sekumpulan nilai.MATCH: menampilkan posisi suatu nilai yang ditetapkan sebelumnya yang terdapat dalam sekelompok data.

4. Function INDEX, mengambil atau menghasilkan nilai range atau tabel

DAY: untuk mencari hari (dalam angka 1-31) dari tipe data DATE.MONTH: untuk mencari bulan (dalam angka 1-12) dari tipe data DATE.YEAR: untuk mencari tahun (dalam angka 1900-9999) dari tipe data DATE.DATE: untuk mendapatkan tanggal yang bertipe data DATE dengan memasukkan angka tahun, bulan dan hari.
