6. perangkat keras komputer yang berfungsi untuk mengeluarkan hasil pemrosesan dan pengolahan data disebutA. output deviceB. storage deviceC. input deviceD. peripheral device​


Perangkat keras komputer yang berfungsi untuk mengeluarkan hasil pemrosesan dan pengolahan data disebut

A. output device

Pembahasan:

Sistem komputer secara umum terdiri dari software (perangkat lunak) dan hardware (perangkat keras).

Perangkat keras atau hardware adalah perangkat fisik, termasuk bagian-bagian atau komponen dari komputer. Perangkat keras dapat dilihat kasat mata dan merupakan perangkat elektronik.

Perangkat keras dapat dibagi menjadi:

A. Output device (perangkat keluaran), yang menghasilkan keluaran proses perngolahan data, berupa tampilan, suara atau cetakan. Misalnya adalah speaker, monitor, atau printer.

B. Storage device (perangkat penyimpanan), yang digunakan untuk menyimpan data yang digunakan. Misalnya adalah flash disk, hard disk, disket dan compact disk (CD).

C. Input device (perangkat masukan) yang menerima input dari pengguna, yang akan digunakan dalam pengolahan data. Misalnya adalah keyboard, mouse dan mikrofon.

D. Processing device (perangkat pengolah) yang melakukan pengolahan data dari input menjadi input. Misalnya adalah mikroprosesor dan prosesor grafis.

10. Perangkat yang berfungsi untuk memasukkan data agar di proses pada perangkat pemrosesan, adalah pengertian dari ... A. Input Device B. Process Device C. Storage Device D. Output Device


Jawaban:

A.  input device

Penjelasan:

1. Input Device (perangkat input/masukan)

Jenis perangkat keras komputer ini berfungsi untuk memasukkan data atau informasi (dalam bentuk teks, gambar, video, dan audio) ke komputer.  

Beberapa contoh hardware yang termasuk input device seperti keyboard, mouse pen, microphone, webcam, joystick, dan lain-lain.

2. Process Device/Hardware (perangkat proses)

Sesuai namanya, perangkat keras jenis process drive berfungsi untuk mengolah data dan menjalankan proses pada sistem komputer. Contoh process device yaitu CPU, RAM, VGA, dan sebagainya.

3. Output Device (perangkat output/keluaran)

Yang ketiga, ada jenis hardware untuk mencetak dan menampilkan informasi dari hasil input data. Untuk data tersebut berasal dari hardware input setelah hardware process. Contoh: speaker, monitor, dan printer.

4. Storage Device (perangkat untuk menyimpan)

Storage device merupakan perangkat keras komputer yang berfungsi sebagai media penyimpanan, berguna untuk menyimpan data-data komputer. Contohnya, hardisk, floppy disk, CD/DVD, dan flashdisk.

20. Alat yang berfungsi untuk memasukkan data atau perintah ke dalam komputer adalah... A. Internal device B. Input device C. Eksternal device D. Output device


Alat yang erfungsi untuk memasukkan data atau perintah ke dalam komputer adalah input device (B). Alat yang termasuk input device adalah keyboard. Selain keyboard, mouse juga termasuk input device.

Pembahasan

Agar komputer dapat melakukan atau bekerja sesuai peritah penggunanya, maka komputer akan dilengkapi dengan komponen-komponennya, yaitu:

Input devide atau alat input, yaitu alat yang berfungsi untuk memasukkan data atau perintah ke dalam komputer. Yang termasuk input device adalah keyboard, mouse, scanner, micropone, dan lainnya.Alat pemprosesan (Process) yaitu alat yang berfungsi memproses data yang dimasukkan ke dalam komputer. Yang termasuk alat pemprosesan adalah CPU.Output device atau alat pengeluaran, yaitu alat yang berfungsi menampilkan data yang telah diproses oleh CPU. Yang temasuk alat output adalah monitor, spiaker dan printer.Storage device, yaitu alat yang berfungsi menyimpan data dalam komputer, dibedakan menjaid dua yaitu:Primary storage, yaitu RAM, ROM DAN EPROM.Secondary storage, yaitu DVD, hardisk, USB Flash Drive.
