1. fungsi DIR yang ada pada internal command


Jika digunakan tanpa parameter, perintah ini dapat menampilkan daftar berkas-berkas dan subdirektori yang terdapat di dalam direktori aktif. Berkas ini memiliki satu parameter, yakni lokasi direktori di mana hendak menampilkan daftar isi direktori.

2. Jelaskan fungsi CHDIR dengan DIR


CHDIR dan DIR merupakan perintah dasar yang bisa kita gunakan pada command prompt yang terdapat di sistem operasi Windows. CHDIR merupakan perintah untuk berpindah direktori, perintah CHDIR hampir mirip dengan perintah CD, namun pada CHDIR kita bisa menggunakan nama lokasi yang kita tuju tanpa menggunakan tanda petik, apabila lokasi yang kita tuju memiliki nama yang mengandung spasi didalamnya.

Setelah mengetahui apa itu CHDIR, maka berikutnya adalah fungsi dari DIR, DIR pada command prompt sendiri berfungsi untuk menampilkan isi dari suatu direktori, baik file atau folder yang ada di dalam suatu direktori. Dengan menggunakan DIR, kita juga bisa menampilkan file-file yang tersembunyi.

17. apakah kegunaan dari perintah dir


Kegunaan Perintah Dir Pada DOS
Saat ini perintah Dir pada DOS masih digunakan meskipun terdapat fasilitas GUI pada PC. Berikut ini perintah-perintah dir pada windows, antara lain :
1. dir c:
Ini adalah perintah dasar yang digunakan untuk menampilkan file dan directory pada drive C.
2. dir c: d:
Digunakan untuk menampilkan file dan directory pada drive C dan D.
3. dir /b /s
Digunakan untuk menampilkan directory dan subdirectory dengan tampilan full path name.
4. dir /a
Digunakan untuk menampilkan semua file termasuk file system dan file yang tersembunyi (hidden).
Untuk menampilkan semua file yang ada di drive D gunakan perintah sbb: dir d: /a
5. dir /os
Digunakan untuk menampilkan file dan directory dengan urutan mulai dari file yang berukuran paling kecil sampai dengan file yang berukuran paling besar. Untuk membalik urutan Anda bisa menggunakan perintah: dir /o-s
Anda bisa mengganti parameter s pada perintah di atas dengan:
- n, perintahnya dir /on : urut nama
- e, perintahnya dir /oe: urut tipe file
- d, perintahnya dir /od: urut tanggal
6. dir /p
Untuk menampilkan file dan directory per “halaman”.

33. tuliskan perbedaan antara fungsi CHDIR dan DIR


Tuliskan perbedaan antara fungsi CHDIR dan DIR, CHDIR dan DIR merupakan perintah dasar yang bisa kita gunakan pada command prompt yang terdapat di sistem operasi Windows. CHDIR merupakan perintah untuk berpindah direktori, perintah CHDIR hampir mirip dengan perintah CD, namun pada CHDIR kita bisa menggunakan nama lokasi yang kita tuju tanpa menggunakan tanda petik, apabila lokasi yang kita tuju memiliki nama yang mengandung spasi didalamnya.

Setelah mengetahui apa itu CHDIR, maka berikutnya adalah fungsi dari DIR, DIR pada command prompt sendiri berfungsi untuk menampilkan isi dari suatu direktori, baik file atau folder yang ada di dalam suatu direktori. Dengan menggunakan DIR, kita juga bisa menampilkan file-file yang tersembunyi.

Jadi perbedaan antara perintah DIR dan CHDIR adalah dari fungsi kedua perintah tersebut sudah berbeda, DIR untuk menampilkan isi suatu direktori, CHDIR berfungsi untuk berpindah direktori.
