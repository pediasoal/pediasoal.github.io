15. sebuah fotoelektron keluar dari permukaan logam yang fungsi kerjanya 7 eV ketika disinari oleh foton yang panjang gelombangnya 00 nm. Energi kinetik maksimum (dalam eV) fotoelektron tersebut (dalam eV) adalah


Misalkan panjang gelombang cahaya yang diberikan λ nm. Maka terlebih dahulu dihitung energi yang dimiliki cahaya tersebut, dengan persamaan:

E = hf = hc/λ,

dimana h = 6,6 x 10^(-34) J.s (alias konstanta Planck)

λ = panjang gelombang cahaya yang digunakan menembak logam (meter) dan c = kecepatan cahaya di dalam vakum (3.10^8 m/s)

Setelah itu dihitung dan diubah ke satuan eV (bagi Joule dengan muatan elektron, 1,6 x 10^-19), kemudian dicek, apakah energi foton ini melebih 7 eV. Kalau melebihi barulah efek foto listrik bisa terjadi. Kalau kurang, maka tidak bisa terjadi efek fotolistrik.

Apabila efek foto listrik terjadi, maka selisih antara energi foton dan fungsi kerja adalah Energi Kinetik nya.

19. Fungsi kerja wolfram ialah 4,58 eV carilah frekuensi dan panjang gelombang ambang untuk efek fotolistrik


Fungsi kerja wolfram ialah 4,58 eV, maka frekuensi dan panjang gelombang ambang untuk efek fotolistrik adalah 1,1 x 10¹⁵ Hz dan 273 nm.


[tex]\boxed{\boxed{PEMBAHASAN}}[/tex]

[tex]\boxed{\boxed{BAB\:8.\:FISIKA\:\:KUANTUM}}[/tex]


[tex]\boxed{Efek\:Fotolistrik}[/tex]

Efek fololistrik merupakan pengeluaran elektron dari suatu permukaan (biasanya logam) ketika dikenai, dan menyerap, radiasi elektromagnetik (seperti cahaya tampak dan radiasi ultraungu) yang berada di atas frekuensi ambang tergantung pada jenis permukaan.

Energi Kineti Maksimum

Persamaan Energi kinetik maksimum adalah :

[tex]\boxed{Ek_{maks}=hf-W_{0}}[/tex]


Energi ambang

Persamaan energi ambang adalah :

[tex]\boxed{W_{0}=hf_{0}}[/tex]


Fungsi Kerja

Persamaan energi sinar adalah :

[tex]\boxed{E=hf}[/tex]


[tex]\boxed{Diketahui}[/tex]

Fungsi kerja (E) = 4,58 eV


[tex]\boxed{Ditanya}[/tex]

FrekuensiPanjang gelombang

[tex]\boxed{Penyelesaian}[/tex]

Besar frekuensi dan panjang gelombang ambag untuk efekfotolistrik bisa dicari dengan menggunakan cara sebagai berikut :

Konversi satuan

Konversi satuan fungsi energi ke dalam satuan Joule :

E = 4,58 eV = 4,58 x 1,6 x 10⁻¹⁹ J = 7,328 x 10⁻¹⁹ J


Frekuensi

Frekuensi gelombang tersebut bisa dicari dengan menggunakan persamaan :

E = h f

atau

f = E / h

dimana konstanta plang (h) = 6,63 x 10⁻³⁴ J.s, maka :

f = 7,328 x 10⁻¹⁹ J / 6,63 x 10⁻³⁴ Js

f = 1,1 x 10¹⁵ Hz

Jadi frekuensinya sebesar 1,1 x 10¹⁵ Hz.


Panjang gelombang

Besar panjang gelombang bisa dicari menggunakan persamaan :

c = λ f

atau

λ = c / f

diaman c = 3 x 10⁸ m/s, maka :

λ = 3 x 10⁸ m/s / 1,1 x 10¹⁵ Hz

λ = 2,73 x 10⁻⁷ m

atau

λ = 273 nm

32. 1. Sebuah logam mempunyai fungsi kerja sebesar1,7 eV. Jika logam tersebut disinari oleh cahayayang memiliki panjang gelombang 660 nm, besarenergi kinetik maksimum elektron foto yang keluardari logam adalah .... (h = 6,6 x 10-34 J·s)A. 0,125 eVD. 0,200 eVB. 0,150 eVE. 0,225 eVC. 0,175 eV​


diket:

panjang gelombang (λ) = 660 nm = 660 x 10⁻⁹ m = 6,6 x 10⁻⁷ m

fungsi kerja (W₀) = 1,7 eV

konstanta Planck (h) = 6,6 x 10⁻³⁴ Js

cepat rambat cahaya (c) = 3 x 10⁸ m/s

ditanya:

energi kinetik maksimum (Ek maks)....?

jawab:

E = W₀ + Ek makshc / λ = W₀ + Ek maks

(6,6 x 10⁻³⁴ Js x 3 x 10⁸ m/s) / (6,6 x 10⁻⁷ m) = 1,7 eV + Ek maks

3 x 10⁻³⁴⁺⁸⁻⁽⁻⁷⁾ J= 1,7 eV + Ek maks

3 x 10⁻¹⁹ J = 1,7 eV + Ek maks

ubah satuan Joule ke eV

1 eV = 1,6 x 10⁻¹⁹J

(3 x 10⁻¹⁹ J) / (1,6 x 10⁻¹⁹ J) = 1,7 eV + Ek maks

1,875 eV = 1,7 eV + Ek maks

Ek maks = 1,875 eV - 1,7 eV

Ek maks = 0,175 eV

Jadi, besar energi kinetik maksimum adalah 0,175 eV.

40. Suatu logam tertentu mempunyai fungsi kerja 3,7 eV. Maka besar frekuensi ambang logam adalah

Frekuensi ambang (fo)

[tex] \boxed{fo = \frac{Wo}{h}} [/tex]

[tex]fo = \frac{3,7(1,6 \times {10}^{ - 19}) }{6.626 \times {10}^{ - 34} } [/tex]

[tex]fo = \frac{5,92 \times {10}^{ - 19} }{6,626 \times {10}^{ - 34} } [/tex]

[tex]fo = 8,9345 \times {10}^{14} \: Hz[/tex]
