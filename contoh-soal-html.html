1. Jelaskan cara membuat daftar/list di html?

cara untuk membuat daftar/list di html kamu bisa mengunakan <ol>, <ul> dan <dl> untuk menentukan tipe listnya dan list mengunakan <li>(untuk <ol> dan <ul>) atau <dt> dan <dd>(untuk <dl>) untuk menyatakan bahwa teks didalamnya adalah bagian dari daftar atau listnya.

<ol> adalah list yang mempunyai urutannya contohnya

1. daftar 1

2. daftar 2

3. daftar 3

sedangkan <ol> adalah list yang tidak mempunyai urutannya contohnya

- daftar 1

- daftar 2

- daftar 3

sedangkan <dl> adalah list yang mempunyai deskripsinya, <dt> adalah sesuatu yang akan di deskripsikan sedangkan <dd> adalah deskripsinya contohnya

Kopi

    -minuman panas berwarna hitam

Susu

    -minuman dingin berwarna putih

22. buatlah Contoh koding html pembuatan tabel​


Jawaban:

Berikut adalah contoh kode HTML untuk membuat tabel sederhana dengan 3 kolom:

```html

<!DOCTYPE html>

<html>

<head>

<title>Contoh Tabel HTML</title>

</head>

<body>

<table>

<tr>

<th>Nama</th>

<th>Umur</th>

<th>Pekerjaan</th>

</tr>

<tr>

<td>John</td>

<td>25</td>

<td>Engineer</td>

</tr>

<tr>

<td>Lisa</td>

<td>30</td>

<td>Teacher</td>

</tr>

<tr>

<td>David</td>

<td>40</td>

<td>Doctor</td>

</tr>

</table>

</body>

</html>

```

Kode di atas menggunakan elemen `<table>` untuk membuat tabel dan elemen `<tr>` untuk membuat baris dalam tabel. Kemudian, elemen `<th>` digunakan untuk membuat sel header (kolom header) dalam tabel, sedangkan elemen `<td>` digunakan untuk membuat sel biasa (kolom) dalam tabel.

Tabel ini memiliki tiga kolom yaitu "Nama", "Umur", dan "Pekerjaan". Setiap baris tabel (`<tr>`) berisi data untuk masing-masing kolom (nama, umur, dan pekerjaan).

Dengan menggunakan kode di atas, kita dapat membuat tabel HTML sederhana dengan beberapa baris dan kolom yang dapat ditampilkan pada halaman web.


23. pengertian dan fungsi tag html, elemen html, atribut html?


Tag Adalah teks khusus (markup) berupa dua karakter "<" dan ">" (tanpa tanda kutip) yang merupakan cara untuk memberitahu web browser tentang apa fungsi dari suatu teks yang kita ketik di dokumen HTML, apakah text itu ditulis sebagai sebuah paragraf, list, atau sebagai link. 


Hampir semua tag dalam HTML ditulis secara berpasangan (beberapa berupa tag tunggal), tag pembuka dan tag penutup, dimana objek yang dikenai perintah tag berada di antara tag pembuka dan tag penutup.

Element adalah isi dari tag yang berada diantara tag pembuka dan tag penutup (termasuk tag itu sendiri). Sebagai contoh perhatikan kode HTML berikut:

<i> Ini adalah sebuah paragraf </i>

Pada contoh diatas, “<i> Ini adalah sebuah paragraf <i>” merupakan element i. Element tidak hanya berisi text, namun juga bisa tag lain.



Atribut adalah informasi tambahan yang diberikan kepada tag. Informasi ini bisa berupa instruksi untuk warna dari text, besar huruf dari text, dll. Setiap atribut memiliki pasangan nama dan nilai (value), dan ditulis dengan name=”value”. Value diapit tanda kutip, boleh menggunakan tanda kutip satu (‘) atau dua (“).

28. istilah yang di gunakan untuk mendefinisikan kode HTML yang di gunakan contoh adalah?


HTML (Hypertext Mark Up Language) adalah beberapa barisan kode yang dapat dilampirkan semisal dokumen. HTML lebih didefinisikan kepada hypertext (kode bahasa program untuk hypertext) dan hyperlink (tautan link menggunakan kode berbasis HTML ; biasanya tautan warna biru dan tulisan "klik disini"). HTML juga merupakan standar format untuk suatu dokumen di internet.
