1. jelaskan tentang system requirement untuk menginstal software kodu !​


Jawaban:

Sistem persyaratan untuk menginstal perangkat lunak Kodu dapat bervariasi tergantung pada versi perangkat lunak dan sistem operasi yang digunakan. Berikut adalah beberapa persyaratan umum yang diperlukan untuk menginstal Kodu:

1. Sistem Operasi:

-Microsoft Windows: Kodu dapat diinstal pada Windows 10, Windows 8, Windows 7, dan Windows Vista. Namun, untuk performa terbaik, disarankan menggunakan Windows 10.

2. Spesifikasi Hardware Minimum:

-Processor: Prosesor dengan kecepatan 1 GHz atau lebih cepat.

-RAM: Setidaknya 1 GB RAM (2 GB lebih disarankan).

-Ruang Penyimpanan: Ruang kosong minimal 200 MB untuk instalasi Kodu.

3. Grafis:

-Kartu grafis: Direkomendasikan memiliki kartu grafis dengan dukungan Shader Model 3.0 atau lebih tinggi.

-Resolusi Layar: Disarankan menggunakan monitor dengan resolusi 1024x768 atau lebih tinggi.

4. Perangkat Lunak Tambahan:

-.NET Framework: Kodu membutuhkan Microsoft .NET Framework 4.0 atau versi yang lebih baru. Jika tidak terpasang, installer Kodu akan menginstal .NET Framework secara otomatis.

Penjelasan:

Untuk mendapatkan informasi persyaratan yang lebih terperinci dan akurat, disarankan untuk mengunjungi situs web resmi Kodu atau melihat dokumentasi resmi yang disediakan oleh Microsoft.


2. 7.Sebutkan Komponen Komponen apa saja yang harus ada dalam Software Requirement Spesification (SRS) ! jelaskan bagian bagiannya !​


Jawaban:

Komponen dalam SRS biasanya mencakup hal sbb :

1. Tujuan dari software yang akan dikembangkan

2. Deskripsi software secara keseluruhan

3. Berbagai fungsionalitas yang ada di dalam sistem

4. Performa software dalam tahap produksi (production stage)

5. Kebutuhan non-fungsional dalam sistem

6. External interfaces atau hubungan interaksi software dengan perangkat yang lain

7. Batasan dari desain atau gambaran yang akan dijalankan oleh sistem

Penjelasan:

Bagaimana Menyusun Dokumen SRS?

Cara terbaik untuk menyusun dokumen SRS adalah memulainya dengan membuat kerangka dan informasi umum tentang software yang akan Anda kembangkan dan menambahkan detail untuk menyempurnakan draf dokumennya. Berikut adalah enam langkah yang bisa dilakukan untuk menyusun dokumen SRS Anda:

Membuat sebuah outline dokumen

Langkah paling awal dalam membuat dokumen SRS adalah dengan menyusun kerangka atau outline dokumen. Anda dapat menyusunnya dengan kreasi Anda sendiri atau menggunakan template dari dokumen SRS sebagai pedoman awal. Berikut ini contoh mendasar dari sebuah kerangka dalam dokumen SRS:

PengantarTujuan PengembanganAudiensTujuan PenggunaanCakupan PengembanganDefinisi Berbagai FungsiDeskripsi KeseluruhanKebutuhan PenggunaAsumsi dan DependensiFitur dan Kebutuhan SistemSistem FungsionalSistem External InterfaceFitur dalam SistemSistem Non-FungsionalMendefinisikan Tujuan Anda

Setelah Anda menyusun outline, Anda harus menyempurnakannya dengan menentukan tujuan softwareyang akan Anda buat dalam awalan dokumen SRS. Bagian ini merupakan penjelasan mengenai audiens yang dituju dan cara mereka menggunakan softwareyang akan dibuat.

2. Memberikan Gambaran Umum

Bagian berikutnya setelah Anda menentukan tujuan produk yang akan Anda hasilkan adalah merangkum cara kerjanya. Bagian ini merupakan penjelasan mengenai deskripsi yang cukup general tentang fitur di dalam software dan cara fitur-fitur tersebut berjalan sesuai dengan kebutuhan penggunanya. Anda juga perlu menjelaskan tentang asumsi yang Anda buat tentang fungsi dari software dan apapun yang bergantung padanya dalam ekosistem teknologi saat ini.

3. Mendeskripsikan Spesifikasi Fungsional dan Non-Fungsional

Sekarang Anda telah sampai pada bagian yang lebih spesifik yakni mendeskripsikan sistem fungsional dan non-fungsional. Penyusunan tiga hal sebelum masuk ke dalam bagian ini membuat Anda memiliki referensi untuk memastikan bahwa segala hal yang Anda lakukan telah memenuhi kebutuhan dasar dari pengguna softwareAnda. Hal tersebut juga memudahkan Anda dalam mengisi hal yang lebih detail ke depannya, seperti bagian spesifikasi fungsional dan non-fungsional ini.

Bagian ini merupakan deskripsi detail tentang kebutuhan sistem softwareAnda. Hal ini bisa dikatakan sebagai komponen paling penting dalam dokumen SRS Anda. Jelaskan kebutuhan sistem fungsional dengan cukup detail sehingga software developerAnda dapat mulai bekerja seperti yang Anda inginkan dan juga jangan lupakan sistem non-fungsional seperti performance software dan sekuritasnya.

Dalam bagian ini juga Anda dapat menambahkan use cases untuk menjelaskan tentang cara pengguna berinteraksi dengan sistem Anda. Di sinilah Anda memerinci tujuan proyek dan mengukur indikator kemajuan proyek selama proses pengembangan software Anda berlangsung.

4. Menambahkan Detail Pelengkap

Terakhir dalam penyusunan draf dokumen SRS, Anda perlu menambahkan detail pelengkap sehingga pihak ketiga selaku pengembang perangkat lunak Anda dapat memulai dan menyelesaikan pekerjaan mereka dalam bentuk lampiran, daftar istilah, dan referensi yang juga Anda ketahui.

5. Mengajukan Persetujuan Terhadap Dokumen

Ketika Anda sudah menambahkan berbagai hal detail yang dibutuhkan, kini saatnya meminta persetujuan dari pihak-pihak yang dianggap berwenang (seperti atasan Anda atau jajaran manajerial) terhadap draf dokumen SRS yang telah Anda buat. Kadang dibutuhkan presentasi kepada orang-orang yang terlibat dalam proses software development nantinya untuk menjelaskan draf dokumen SRS ini. Tidak menutup kemungkinan akan ada permintaan perubahan atau revisi sehingga Anda harus memperbarui draf dokumen SRS Anda. Tetapi Anda jangan berkecil hati. Ini artinya pihak-pihak yang terlibat, baik dari pengembang software Anda dan para pemangku kepentingan di perusahaan Anda, membuat dokumen tersebut lebih tepat sehingga proyek software development yang direncanakan tidak akan keluar jalur.

Macam-Macam Software dan Fungsinya

Berikut merupakan penjelasan mengenai beberapa macam-macam software atau aplikasi yang biasa digunakan dalam kehidupan sehari-hari:

1. Software Antivirus

Macam-Macam Software Antivirus

Software antivirus merupakan software yang digunakan untuk melindungi perangkat dari virus. Dimana, tujuan dari software ini adalah untuk mengamankan, mendeteksi dan menghapus virus dari perangkat komputer yang kita miliki.

2. Software Perkantoran

Software Perkantoran

Software perkantoran sering kali dibutuhkan dalam menjalankan pekerjaan dikantor. Tidak hanya itu saja, software ini juga digunakan dalam berbagai bidang.

3. Software Video Editor

Macam-Macam Software Video Editor

Softwaer video editor digunakan untuk mengedit video. Tujuannya adalah agar video yang dihasilkan menarik dan sesuai dengan keinginan. Sehingga video editor ini cukup populer dibanyak kalangan masyarakat.

4. Software Browser

Software Browser

Software browser atau yang biasa disebut dengan mesin pencari (search engine). Software browser ini banyak digunakan dan dibutuhkan dalam mencari informasi menggunakan google salah satu contohnya. Dengan menggunakan browser, segala informasi yang dibutuhkan dapat ditemukan dengan mudah dan cepat. Hanya dengan menggunakan perangkat yang tersedia dan jaringan internet.

5. Software Desain Grafis

Software Desain Grafis

Desain grafis menjadi salah satu profesi yang cukup menantang, karena dengan ini anda dapat menyalurkan seni dan kreatifitas masing-masing melalui desain grafis yang akan dibuat.

6. Software Jaringan

Macam-Macam Software Jaringan

Software jaringan adalah software yang digunakan untuk memantau segala aktifitas yang berhubungan dengan komunikasi jaringan antar device yang terhubung. Sehingga dengan melalui software jaringan, segala aktifitas dan komunikasi dilakukan dapat dipantau dengan jelas.

7. Software Audio Editor

Software Audio Editor

Software aduio editor adalah jenis software yang digunakan untuk menghasilkan kualitas audio yang terbaik. Sehingga audio ini dapat digunakan untuk melengkapi video, lagu, musikalisasi, atau recorder lainnya.

8. Software Akuntansi

Software Akuntansi

Software akuntansi merupakan bagian dari software perkantoran yang banyak digunakan. Namun bedanya adalah lebih ke permasalahan akuntansi. Software akuntansi difokuskan untuk mengumpulkan semua transaksi yang dilakukan, dan akhirnya dibutuhkan untuk laporan keuangan perusahaan.

9. Software Downloader

Software Downloader

Software downloader digunakan untuk mempermudah kita sebagai pengguna dalam melakukan download pada suatu file atau data tertentu. Sehingga proses mendownload data atau file lebih cepat dan mudah dilakukan.

10. Software PDF Reader

Macam-Macam Software PDF Reader

Dokumen dalam bentuk PDF membutuhkan software khusus agar dokumen tersebut dapat dibaca. Sehingga penggunaan software PDF reader sangat dibutuhkan bagi setiap orang yang memiliki dokumen beresktensi PDF. Dengan adanya software PDF reader ini pengalaman dalam membaca, mengedit (menandai) dokumen, dan mengirim dokumen menjadi lebih mudah.

13. sebutkan dan jelaskan fungsi 5 software dari software commersial​


Jawaban:

Berikut adalah lima perangkat lunak komersial dan penjelasan singkat tentang fungsi masing-masing:

1. Microsoft Office: Microsoft Office adalah paket perangkat lunak yang terdiri dari berbagai aplikasi, termasuk Word, Excel, PowerPoint, Outlook, dan lainnya. Word digunakan untuk mengolah dokumen teks, Excel untuk mengelola data dan membuat spreadsheet, PowerPoint untuk membuat presentasi, dan Outlook untuk mengelola email dan kalender. Fungsi utama dari Microsoft Office adalah menyediakan alat produktivitas yang komprehensif untuk pekerjaan kantor dan pengolahan data.

2. Adobe Photoshop: Adobe Photoshop adalah perangkat lunak pengolahan gambar yang populer di kalangan desainer grafis dan fotografer. Photoshop memiliki berbagai fitur dan alat untuk mengedit, memanipulasi, dan meningkatkan gambar. Fungsi utama Photoshop adalah untuk mengedit gambar secara profesional, termasuk retouching, penghapusan objek, penyesuaian warna, dan lainnya.

3. QuickBooks: QuickBooks adalah perangkat lunak akuntansi yang dirancang khusus untuk pengelolaan keuangan bisnis kecil dan menengah. QuickBooks memungkinkan pengguna untuk mengelola pembayaran, menghasilkan laporan keuangan, mengelola inventaris, mengirim faktur, dan lainnya. Fungsi utama QuickBooks adalah membantu pengguna mengelola dan melacak keuangan bisnis mereka dengan mudah dan efisien.

4. Salesforce: Salesforce adalah perangkat lunak manajemen hubungan pelanggan (CRM) yang digunakan oleh berbagai perusahaan untuk mengelola proses penjualan, pelayanan pelanggan, dan pemasaran. Salesforce menyediakan alat untuk melacak prospek, mengelola hubungan pelanggan, mengatur kegiatan penjualan, dan menganalisis data pelanggan. Fungsi utama Salesforce adalah membantu perusahaan meningkatkan efisiensi operasional dan produktivitas tim penjualan.

5. Autodesk AutoCAD: Autodesk AutoCAD adalah perangkat lunak desain dan pemodelan berbasis komputer yang digunakan oleh para profesional di bidang arsitektur, rekayasa, dan desain. AutoCAD memungkinkan pengguna untuk membuat gambar 2D dan model 3D, menghasilkan rencana konstruksi, dan melakukan simulasi visual. Fungsi utama AutoCAD adalah membantu pengguna merancang, menggambar, dan memvisualisasikan proyek dengan presisi dan detail yang tinggi.

Perangkat lunak komersial seperti yang disebutkan di atas memiliki berbagai fungsi yang dirancang untuk membantu pengguna dalam berbagai aspek pekerjaan mereka, mulai dari pengolahan dokumen hingga desain grafis dan manajemen bisnis.

Fungsi dari system software adalah sebagai “jembatan“ atau interface antara perangkat keras komputer dengan aplikasi, yang mengendalikan dan mengelola pengoperasian perangkat keras komputer sesuai instruksi dari aplikasi yang digunakan.
 

Pembahasan

System software adalah software yang “menjembatani” atau sebagai penghubung/interface antara perangkat keras komputer dengan aplikasi. System software mengendalikan dan mengelola pengoperasian perangkat keras komputer, sesuai dengan instruksi dari perangkat lunak aplikasi (user application). Dengan demikian, dapat dikatakan pula bahwa system software berfungsi untuk menjamin program-program aplikasi berjalan dengan benar/semestinya.

Contoh dari system software adalah sistem operasi (operating system), seperti Microsoft Windows, OS X, Linux, Unix, dsb.

38. Berdasarkan fungsinya, software dikategorikan sebagai .... a. Application software dan System software b. System software dan Compilers c. System software dan Compilers d. System software dan assemblers


Berdasarkan fungsinya, software yang digunakan dalam komputer dikategorikan menjadi Application software dan System software

Pembahasan:

Dalam penggunaan komputer, kita membutuhkan software atau perangkat lunak agar bisa mengoperasikan komputer. Tanpa adanya perangkat lunak, maka suatu komputer tidak bisa digunakan secara maksimal. Dilihat dari fungsinya, software secara garis besar dibedakan menjadi dua macam yaitu application software dan system software.

Application software merupakan perangkat lunak yang memiliki fungsi dan tujuan khusus. Kekhususannya ini membuat perangkat lunak ini efektif dalam mengerjakan suatu tugas secara spesifik, namun secara general tidak bisa. Fungsi spesifik antara lain untuk pengolahan kata bisa menggunakan microsoft word, untuk pengolahan gambar bisa menggunaka Adobe photoshop, dan pengolah angka bisa menggunakan Microsoft exel.

System software merupakan perangkat lunak yang membuat suatu komputer bisa bekerja secara umum seperti menyimpan file, mengorganisit file, hingga menggandakan suatu file. System software juga menjadi tempat untuk instalasi dari application software. Contoh dari system software antara lain Microsoft Windows 10, Ubuntu 19.04, Machintos.
