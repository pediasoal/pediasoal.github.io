1. Bagaimana cara mencari nilai maksimal dan minimum dengan menggunakan fungsi aljabar​


Jawaban:

Suatu fungsi akan mencapai optimal (maksimum atau minimum) jika gradiennya sama dengan nol (m = 0). Karena gradien sama dengan turunan pertama dari fungsi tersebut maka turunan pertama dari fungsi sama dengan nol (f'(x) = 0).

Titik tersebut dinyatakan dengan titik stasioner. Beberapa sifat dari turunan pertama dan kedua suatu fungsi pada x1 dapat kita nyatakan sebagai berikut:

f'(x1) = 0, maka titik (x1, f(x1)) disebut titik stasioner (kritis).

f'(x1) = 0 dan f''(x1)>0, maka titik (x1, f(x1)) disebut titik minimum.

f'(x1) = 0 dan f''(x1)<0, maka titik (x1, f(x1)) disebut titik maksimum.

f''(x1) = 0, maka titik (x1, f(x1)) disebut titik belok.

4. jika diketahui fungsi f(x) = 5+ 6x-x², maka nilai maksimal fungsi tersebut adalah..tolong bantu jawab​


Untuk mencari nilai maksimal dari fungsi f(x) = 5 + 6x - x², kita perlu mencari titik ekstrim dari fungsi tersebut. Dalam hal ini, karena koefisien x² adalah negatif (-1), fungsi tersebut adalah fungsi parabola yang terbuka ke bawah.

Untuk menemukan nilai maksimal, kita perlu mencari titik puncak (vertex) dari parabola. Dalam hal ini, vertex dapat ditemukan dengan menggunakan rumus x = -b/2a, di mana a adalah koefisien x², dan b adalah koefisien x.

Dalam fungsi f(x) = 5 + 6x - x², kita memiliki a = -1 dan b = 6. Substitusikan nilai-nilai ini ke rumus vertex:

x = -b/2a

x = -6 / (2 * (-1))

x = -6 / (-2)

x = 3

Sekarang kita telah menemukan nilai x dari vertex, kita dapat mencari nilai maksimal dengan menggantikan nilai x ke dalam fungsi f(x):

f(x) = 5 + 6x - x²

f(3) = 5 + 6(3) - (3)²

f(3) = 5 + 18 - 9

f(3) = 14

Jadi, nilai maksimal dari fungsi f(x) = 5 + 6x - x² adalah 14.

21. cari kata asli pluralisasi dan maksimal


Pluralisme (bahasa Inggris:pluralism), terdiri dari dua kata plural(=beragam) dan isme (=paham) yang berarti paham atas keberagaman. Definisi dari pluralisme seringkali disalahartikan menjadi keberagaman paham yang pada akhirnya memicu ambiguitas.

Berdasarkan Webster's Revised Unabridged Dictionary (1913 + 1828)arti pluralisme adalah:

hasil atau keadaan menjadi plural.keadaan seorang pluralis; memiliki lebih dari satu tentang keyakinan.

Pluralisme juga dapat berarti kesediaan untuk menerima keberagaman (pluralitas), artinya, untuk hidup secara toleran pada tatanan masyarakat yang berbeda suku, gologan, agama,adat, hingga pandangan hidup. Pluralisme mengimplikasikan pada tindakan yang bermuara pada pengakuan kebebasan beragama, kebebasan berpikir, atau kebebasan mencari informasi, sehingga untuk mencapai pluralisme diperlukanadanya kematangan dari kepribadian seseorang dan/atau sekelompok orang.

39. suatu fungsi f(x) = 2x³ + 3x² + 3 Tentukan a. interval saat fungsi tersebut dikatakan naik b. nilai maksimal


SOAL A.

Tentukan interval saat fungsi tersebut dikatakan naik.

Jawab :

f(x) = 2x³ + 3x² + 3

f'(x) = 2•3x² + 3•2x + 0 > 0

f'(x) = 6x² + 6x > 0

6x² + 6x = 0

6x•(x + 1) = 0

x = 0 atau x = -1

Jadi, f'(x) naik pada interval x < -1 atau x > 0

SOAL B.

Tentukan nilai maksimal pada fungsi f(x) = 2x³ + 3x² + 3.

Jawab :

f(x) = 2x³ + 3x² + 3

f(-1) = 2•(-1)³ + 3•(-1)² + 3

f(-1) = 2•(-1) + 3•(1) + 3

f(-1) = (-2) + 3 + 3 = 4

f(x) = 2x³ + 3x² + 3

f(0) = 2•(0)³ + 3•(0)² + 3

f(0) = 2•(0) + 3•(0) + 3

f(0) = 0 + 0 + 3 = 3

Jadi, nilai maksimal tersebut adalah 4 dan nilai minimal tersebut adalah 3.
