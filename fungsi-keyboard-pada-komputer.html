Tombol Tab

Fungsi tombol tab adalah untuk membuat kursor bergeser dengan jarak 1 tabulasi ke arah kanan. Fungsi tombol ini mirip dengan tombol space yang digunakan untuk memberi jarak antara kata 1 dan lainnya.  Letak tombol tab tepat diatas tombol caps lock.

Tombol Caps Lock

Fungsi tombol caps lock adalah untuk membuat huruf atau tulisan menjadi BESAR (UPPERCASE). Biasanya ada indikator atau lampu yang menyala begitu tombol ini dalam keadaan on.

Tombol Windows (Home)

Fungsi tombol home adalah untuk mengarahkan tampilan pada halaman awal atau halaman start secara cepat. Untuk mengembalikan pada halaman yang terakhir dibuka, maka cukup klik lagi tombol ini. Tombol home biasanya berupa logo windows atau jendela dan terletak antara CTRL dan ALT.

Tombol Space (Spasi)

Tombol space berbentuk panjang dan terletak dibagian paling bawah. Fungsinya adalah untuk menambahkan spasi atau jarak antara 1 kata dengan kata lainnya.

Tombol Delete

Tombol delete berfungsi untuk menghapus 1 karakter di kanan/depan kursor. Tombol ini akan menghapus kata yang ditulis setelah tanda kursos atau kata yang sebelumnya telah ditandai.

Tombol Enter

Fungsi tombol enter adalah untuk berpindah baris. Ketika mengetikan tombol enter maka tulisan di baris kursor akan berpindah ke baris baru. Tombol enter ada disebelah kanan keyboard dengan bentuk paling besar diantara tombol lainnya.

Tombol Backspace

Tombol Backspace ditandai dengan simbol <—- di keyboard. Fungsi tombol backspace adalah untuk menghapus satu karakter yang ditulis. Jika kamu ingin menghapus lebih cepat gunakan perpaduan tombol backspace dengan CTRL untuk menghapus sebuah kata.

Tombol PrtScr

Tombol print screen atau pada keyboard disingkat PrtScr/PrtSc adalah tombol yang digunakan untuk menangkap atau meng-capture layar. Tekan tombol ini 1 kali kemudian buka ms word ataupun paint. Ketik CTRL + P, maka gambar hasil screeshoot pun akan muncul.

Cara lain adalah dengan memadukan tombol windows/home dengan printscreen. Ketika di klik berbarengan maka akan muncul kedipan di layar yang menandakan layar sudah ter-screenshoot. Kamu dapat melihat hasilnya di folder PICTURES di my computer anda.

Tombol End

Fungsi tombol end adalah untuk memindahkan kursor ke baris akhir tulisan. Ini biasanya berjalan ketika anda sedang bekerja menggunakan ms word.

Tombol Page Up & Page Down

Tombol page up digunakan untuk menarik halaman ke atas (scrolling ke atas), dan page down untuk menarik halaman ke bawah (scrolling ke bawah). Dengan menggunakan tombol ini anda tak perlu repot ketika akan menurunkan atau menaikan posisi file di layar komputer.

Tombol Navigasi (Panah Kanan, kiri, atas, bawah)

Tombol navigasi berfungsi untuk menggerakan kursor ke atas, bawah, kanan ataupun kiri. Kursor sendiri adalah bagian di layar komputer yang berfungsi sebagai indikator posisi. Bentuknya lurus kecil dan berkedip-kedip.

Tombol Lain

Tombol NumLock -> tombol numlock berfungsi untuk mengubah bagian keyboard utama bertindak sebagai keypad numerik.

Tombol Scrooll Lock -> Pada microsoft excel tombol ini berfungsi untuk mengunci kolom agar tidak menggulung saat  tombol atas kanan kiri bawah ditekan.

Tombol Fn -> Tombol fn biasanya ada pada keyboard laptop ia hanya berfungsi ketika dikombinasikan dengan tombol lain. Misalnya untuk mencerahkan meredupkan layar, menaikan atau mengecilkan volume suara dll.

15. kenapa keyboard QWERTY skrng menjadi standar keyboard komputer ??


Pada dasarnya desain keyboard ada berbagai model, pembuatan desain keyboard dengan berbagai pertimbangan antara lain untuk menghindari kesalahan-kesalahan mekanik yang sering terjadi sebelumnya, desain keyboard qwerty dibuat sekitar 1860-an.
Pada tahun 1882, semua orang pada zaman itu menggunakan metode “cari dan ketik” empat jari. Pada saat itu ada seorang wanita yang mengajukan pendapatnya bahwa dengan menggunakan model QWERTY dapat melakukan pengetikan secara cepat dengan delapan jari. Lalu, wanita tersebut menyewa seorang pengetik professional untuk menghafalkan model QWERTY ini. Setelah itu dia menantang para professional pengetik lainnya untuk membuktikan pendapat wanita tersebut. Hasilnya adalah wanita tersebut tidak tertandingi dalam mengetik dengan menggunakan model QWERTY tersebut. dengan pertimbangan tersebut Keyboard QWERTY dinobatkan menjadi model keyboard standar internasional 
Keyboard QWERTY dinobatkan menjadi model keyboard standar internasional di tahun 1966 dan pada tahun 1973 diresmikan menjadi keyboard standart ISO (International Standard Organization) menjadi standar yang dipakai diseluruh dunia hingga saat ini. sebagian besar orang karna telah terbiasa keyboard qwerty akan berpikir beberapa kali jika harus beralih model keyboard meskipun desain tersebut lebih effisien

22. (Soal Logika Proposisi) Pilihlah kesimpulan yang paling tepat dari pernyataan atau premis yang tersedia pada soal! Semua komputer dilengkapi dengan keyboard. Keyboard bertekhnologi canggih. Maka.. (A) Tidak ada yang dilengkapi keyboard kecuali komputer (B) Tidak semua komputer canggih dilengkapi dengan komputer (C) Keyboard bertekhnologi canggih hanya ada di computer (D) Tidak ada komputer yang keyboardnya bertekhmologi canggih (E) Ada komputer yang memiliki keyboard bertekhnologi canggih


PERTANYAAN

Semua komputer dilengkapi dengan keyboard. Keyboard berteknologi canggih. Maka..

(A) Tidak ada yang dilengkapi keyboard kecuali komputer

(B) Tidak semua komputer canggih dilengkapi dengan komputer

(C) Keyboard berteknologi canggih hanya ada di computer

(D) Tidak ada komputer yang keyboardnya berteknologi canggih

(E) Ada komputer yang memiliki keyboard berteknologi canggih

PENJELASAN

Sebelum kita mencari yang mana benar,Sebaiknya kita akan menentukannya dari pilihannya,Maka :

(A) Tidak ada yang dilengkapi keyboard kecuali komputer

Berdasarkan pilihan diA yang merupakan pilihan yang salah,Karena tidak hanya komputer saja yang memiliki keyboard tetapi bisa juga laptop,Hp,Dll

(B) Tidak semua komputer canggih dilengkapi dengan komputer

Pilihan diB kita ubah menjadi Tidak semua keyboard canggih dilengkapi dengan Komputer,Berdasarkan pilihan diB tepat karena keyboard dipasang pada Komputer yang dimana keyboardnya tidak semua ada diKomputer.

(C) Keyboard berteknologi canggih hanya ada di computer

Sama seperti pilihan A,Karena tidak hanya komputer saja yang memiliki keyboard tetapi bisa juga laptop,Handphone,Dll

(D) Tidak ada komputer yang keyboardnya berteknologi canggih

Berdasarkan Pilihan diD yang merupakan salah satu pilihan salah yang dimana ada Komputer yang memiliki teknologi yang canggih

(E) Ada komputer yang memiliki keyboard berteknologi canggih

Berdasarkan pilihan diE yang merupakan pilihan yang benar karena ada Komputer yang memiliki keyboard yang berteknologi canggih

KESIMPULAN

Jadi,Pilihan yang tepat adalah antara B dan E tetapi yang lebih tepat dipilihan E yang merupakan Keyboard memiliki teknologi canggih dipasang didalam Komputer


23. Fungsi tombol F1 sampai F12 pada keyboard komputer ?​


• F1 berfungsi Help Screen (tombol F1 digunakan untuk menampilkan fitur Help pada hampir setiap program)

• F2 berfungsi Rename (mengganti nama file secara cepat)

• F3 berfungsi Search (Untuk mengaktifkan fitur Search pada aplikasi yang sedang berjalan/aktif )

• F4 berfungsi Close Window (Untuk menutup Window )

• F5 berfungsi Refresh/Reload (Memperbarui Window/Browser )

• F6 berfungsi Address Bar (Cara pintas ke Address bar)

• F7 berfungsi Spell and grammar (check Untuk mengaktifkan fungsi spell dan grammar di Microsoft Word)

• F8 berfungsi Boot (Mengakses boot/bios Mengakses boot/bios menu)

• F9 berfungsi Refresh (Refresh dokumen di Microsoft word)

• F10 berfungsi Menu bar (Mengakses menu bar)

27. sebagian besar keyboard komputer terbuat dari kapasitor. jelaskan cara kerja keyboard sehingga dapat di gunakan sesui fungsinya


Sebagian besar keyboard komputer terbuat dari kapasitor. Jelaskan cara kerja keyboard sehingga dapat di gunakan sesui fungsinya!

Jawaban:

Keyboard kapasitif adalah jenis keyboard komputer yang menggunakan perubahan kapasitansi pada lapisan pad kapasitor di bawah tomnol, untuk mendeteksi tombol mana yang ditekan pada keyboard.  

Keyboard kapasitif menyediakan fungsionalitas yang mirip dengan keyboard kontak standar, tetapi struktur internalnya berbeda dan dengan reaksi akan sentuhan yang lebih cepat dan lebih dapat diandalkan.

Pembahasan:

Keyboard kapasitif bekerja dengan mendeteksi perubahan kapasitansi pada kapasitor yang terletak di bagian bawah dan tersebar di seluruh keyboard.  

Tidak seperti keyboard tradisional, di mana aliran listrik dimulai secara internal ketika tombol ditekan, listrik selalu mengalir di keyboard kapasitif. Di bawah setiap kunci, muatan statis disimpan dalam kapasitor.  

Ketika sebuah tombol ditekan, tombol ini akan terhubung dengan kapasitor dan mengubah kapasitansi pada titik tertentu pada pad kapasitor, yang mana keyboard mengidentifikasi dan mencatat sebagai penekanan tombol tersebut.  

Keyboard kapasitif lebih cepat daripada keyboard kontak karena hanya memerlukan tekanan yang lembut untuk bisa terbaca tombolnya.

Dilihat dari arti katanya hardware merupakan perangkat keras, yang secara umum mencakup perangkat semua, device atau komponen apapun yang bebentuk fisik yang dapat dilihat serta dapat diraba.

Dalam dunia computer Hardware (perangkat keras) merupakan komponen-komponen fisik nyata yang membentuk sebuah system computer yang fungsinya untuk menunjang  kinerja dari computer itu sendiri. Terdapat banyak komponen Hardware dalam dunia computer. Kali ini WaroungTKJ akan memberikan daftarnya!

 

Macam-macam Perangkat Keras Komputer

MotherBoard/MainBoard

Merupakan papan sirkuit atau papan induk yang berfungsi sebagai tempat terpasangnya komponen-komponen computer seperti  Processor, RAM, VGA,Slot VGA, AGP, Slot AGP  serta Slot-slot untuk kartu tambahan seperti TV Tunner atau Radio Tunner  yang terhubung ke mainboard, dibawah ini merupakan komponen-komponen Mainboard.

 

RAM (Random Access Memory)

RAM atau Random Acces Memory merupakan perangkat keras yang memiliki fungsi untuk menyimpan  data serta-intruksi yang dilakukan/diberikan oleh prosessor. ram berbeda dengan hard disk, penyimpanan pada RAM bersifat sementara.  Jadi ketika komputer dimatikan, data maupun intruksi yang sedang disimpan oleh RAM akan hilang kecuali komputer tersebut dalam keadaan sleep atau hibernate. singkatnya yaitu ram merupakan media penyimpan data sementara ketika komputer hidup atau dihidupkan.

 

Processor atau CPU (Central Processing Unit)

Merupakan sebuah IC yang memiliki fungsi untuk mengontrol semua proses atau jalannya sebuah sistem komputer.  processor bertugas melakukan perhitungan aritmatika dan logika pada sebuah komputer serta memberikan instruksi untuk menjalankan program yang akan dijalankan oleh komputer yang diinput oleh user (branware). Processor juga dapat diartikan sebagai otak dari sebuah system computer.

 

Soket

Berfungsi sebagai  tempat dari processor.

 

Fan & Heatsink

Memiliki fungsi untuk mengurangi panas yang dihasilkan oleh computer computer, Fan & Headsink juga memiliki fungsi sebagai pendingin processor agar tidak overhead.

 

VGA (Video Graphic Adapter)

Merupakan perangkat keras komputer yang memiliki fungsi untuk pengolahan data

grafik yang nantinya akan ditampilkan oleh monitor.

 

Sound Card (Kartu Suara)

Merupakan perangkat keras berfungsi untuk mengolah data audio atau suara pada sebuah computer yang nantinya diteruskan ke speaker agar keluar Output suara.

 

NIC (Network Interface Card)

Merupakan sebuah kartu yang memiliki fungsi untuk menghubungkan komputer ke sebuah jaringan.

 

HDD (Hard Disk Drive)

Merupakan perangkat keras komputer yang berfungsi sebagai media penyimpanan data utama pada sebuah komputer, serta tempat terinstallnya sebuah system operasi dan system aplikasi.

 

CDROOM atau Optical Drive

Merupakan perangkat keras yang berfungsi untuk membaca (read) maupun menulis (write) data dari kepingan CD/ DVD yang dilakukan oleh optic pada cdroom.

Jenis-jenis CDROOM :

ü  CD ROM

ü  CD R (Read)

ü  CD RW (Read-Write)

ü  DVD ROM

ü  DVD R (Read)

ü  DVD RW (Read-Write)

Printer

Speaker
