17. 2. Tentukan asimtot datar dan asimtot tegak 3x-4 sin x dari fungsi f(x) = X-4​


Penjelasan dengan langkah-langkah:

Untuk menentukan asimtot datar dan asimtot tegak dari fungsi ( f(x)={3x-4sin(x)}{x-4} ), kita perlu memeriksa perilaku fungsi ini saat ( x ) mendekati nilai tak hingga dan saat ( x ) mendekati nilai ( x=4 ).

1. Asimtot datar:

Ketika ( x ) mendekati nilai tak hingga positif atau negatif, kita perhatikan bahwa fungsi ( f(x) ) memiliki suku ( {3x}/{x} ), yang mendekati nilai 3. Oleh karena itu, terdapat asimtot datar di ( y=3 ).

2. Asimtot tegak:

Ketika ( x ) mendekati nilai ( x=4 ), kita harus memeriksa limit dari fungsi tersebut. Dengan memeriksa limit secara aljabar atau menggunakan aturan L'Hopital, kita dapat melihat bahwa limit tersebut memiliki nilai tak hingga. Oleh karena itu, terdapat asimtot tegak di ( x=4 ).

Jadi, asimtot datar dari fungsi ( f(x) ) adalah ( y=3 ) dan asimtot tegak adalah ( x=4 ).
