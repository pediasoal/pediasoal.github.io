2. JDK (Java Development Kit)
JDK adalah Perangkat lunak yang digunakan untuk melakukan proses kompilasi dari kode java ke bytecode yang dapat dimengerti dan dapat dijalankan oleh JRE (Java Runtime Envirotment). JDK wajib terinstall pada komputer yang akan melakukan proses pembuatan aplikasi berbasis java, namun tidak wajib terinstall di komputer yang akan menjalankan aplikasi yang dibangun dengan java.

JRE (Java Runtime Environment)
JRE adalah perangkat lunak yang digunakan untuk menjalankan aplikasi yang dibangun menggunakan java. Versi JRE harus sama atau lebih tinggi dengan JDK yang dugunakan untuk membangun aplikasi java.

3.JVM :

JVM (Java VIrtual Machine) adalah mechine abstract dengan spesifikasi tertentu yang menyediakan lingkungan untuk runtime untuk menjalankan bytecode dari Java (Apliksi yang dibuat dengan java). 
JVM melakukan tugas-tugas utama sbb:
Membuka kode (Loads code)
Menferifikasi kode
Mengeksekusi kode
Menyediakan runtime environment
JRE
 digunakan untuk menyediakan lingkuran agar dapat mengeksukusi program yang terbuat dari Java. JRE menyediakan seperangkat libraries serta file lain yang digunakan JVM saat runtime atau dengan kata lain JRE merupakan implementasi dari JVM itu sendiri. Jika kita akan menggunakan program yang terbuat dari Java, maka JRE wajib terinstal terlebih dulu pada OS yang akan digunakan.

23. Cara mindahin tugas word jadi pdf???


Jawaban:

Cara mengubah Word ke PDF kini bisa dilakukan dengan sangat mudah, cepat, dan praktis. Anda bisa melakukannya dengan cara offline maupun online. Cara mengubah Word ke PDF via online misalkan saja menggunakan website seperti ilovepdf atau Google Drive. Jika secara offline anda bisa mengubahnya menggunakan Microsoft Word.

Penjelasan:

Salah satu cara mengubah Word ke PDF secara offline tanpa aplikasi tambahan yakni melalui Microsoft Word. Setelah anda selesai menulis atau mengedit dokumen, anda bisa langsung menyimpannya dengan format PDF. Namun, fitur ini hanya bisa dilakukan dengan Microsoft Word versi 2007 ke atas.

Berikut cara mengubah Word ke PDF dengan Microsoft Word:

1. Buka dokumen yang ingin Anda ubah, lalu klik tab "File".

2. Pilih opsi "Save As" dari daftar di sebelah kiri.

3. Pada opsi Sava As, pilih tempat Anda ingin menyimpan PDF (OneDrive, PC, folder tertentu, atau di mana pun).

4. Selanjutnya, klik panah dropdown di sisi kanan kotak "Save as type", dan pilih "PDF (* .pdf)" dari menu dropdown.

5. Jika suda selesai, klik tombol "Save".

Setelah menyimpan PDF, Anda akan dikembalikan ke dokumen Word, dan PDF baru akan terbuka secara otomatis di penampil PDF defaultmu.

Salah satu kekurangan mengubah Word ke PDF melalui Microsoft Word yakni anda tidak bisa mengonversi beberapa file sekaligus. Jadi hanya satu file saja, padahal sering kali kita membutuhkan beberapa file Word untuk diringkas menjadi satu file PDF supaya lebih mudah dibaca.

Tapi tenang saja, karena selain melalui Microsoft Word, anda juga bisa mengubah Word ke PDF melalui Google Drive. Cara ini juga cukup praktis karena bisa dilakukan secara online kapan saja dan dimana saja.

Berikut cara mengubah Word ke PDF dengan Google Drive:

1. Masuk ke drive.google.com menggunakan akun Google.

2. Upload file Word yang ingin Anda ubah menjadi PDF ke Google Drive.

3. Buka file yang sudah terupload dengan melakukan double click pada file yang ingin Anda ubah formatnya. Klik Open With, kemudian pilih Google Docs. Maka file akan terbuka di tab baru dalam halaman editor Google Docs.

4. Setelah halaman dokumen terbuka, klik pada File > Download As > pilih PDF Document (.pdf).

5. Download dan simpan file PDF di folder yang Anda inginkan.

Cara mengubah Word ke PDF melalui pdf2doc.com

Situs pdf2doc.com menyediakan fitur untuk convert file Word ke PDF secara gratis. Berikut ini langkah-langkahnya:
