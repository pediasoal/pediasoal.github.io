7. apa fungsi dari subject verb object?​


Jawaban & Penjelasannya

Subject Verb Object (SVO) adalah struktur kalimat yang paling umum digunakan dalam bahasa Inggris. Subjek (S) adalah entitas yang melakukan tindakan, kata kerja (V) adalah tindakan yang dilakukan, dan objek (O) adalah hasil dari tindakan tersebut.

Rumus: Subjek + Kata Kerja + Objek

Contoh:

John eats an apple.

John (subjek) eats (kata kerja) an apple (objek).

10. Jelaskan apa maksud pilihan object in pada saat menempatkan grafik


Jawaban singkat:

Maksudnya adalah pemilihan tempat dalam menampilkan grafik.

Penjelasan panjang:

Misalkan pada program MS. Excel 2013, terdapat 2 sheet.

Data utama terdapat dalam sheet 1.

Bila pengguna program ingin menyisipkan grafik, maka pilihan Object in dapat dipakai untuk memilih tempat dalam menampilkan grafik:

- grafik dapat disisipkan di sheet 1 (sheet yg sama dengan data utama)

- grafik dapat disisipkan di sheet 2 (sheet yg beda dengan data utama)

Bila diterapkan di program Ms. Excel, bisa dilihat pada lampiran (Object in ditunjukkan tanda panah merah).

26. Apa fungsi dari toolbox,Ellipse tool dan Object Tool?


A) toolbox

Toolbox berisi kumpulan tool untuk mengedit dan menggambar objek. Beberapa tool memiliki tanda segitiga kecil berwarna hitam pada sudut kanan bawah, dengan memilih tool tersebut menu flyout berisi perintah akan tampil

B) Ellipse tool

Ellipse tool digunakan untuk membuat objek gambar lingkaran, termasuk ellips, pie, dan arc. Pada tools ini terdapat 2 (dua) flyout, yaitu Ellipse Tool dan 3D Point Ellipse Tool

C) Object tool

Object tool digunakan untuk membentuk objek gambar poligon atau bintang, spiral, dan kerangka tabel. Pada tools ini terdapat 3 (tiga) flyout, yaitu Polygon/Star, Graph Paper, dan Spiral.
