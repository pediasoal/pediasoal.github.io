Baris penggulung scroll bar berfungsi untuk menggulung tampilan dokumen aktif.
scrollbar vertical = untuk menggulung tampilan dokumen secara vertikal (dari atas ke bawah/ sebaliknya)

scrollbar horizontal= untuk menggulung tampilan dokumen secara horizontal (mendatar kiri ke kanan/ sebaliknya)untuk menggulung layar ke arah vertical dan horizonta

25. je;askan dan tuiskan fungsi penggulung layar


Baris penggulung pada Ms. Word juga ada 2, yaitu :
1. Baris penggulung mendatar (horizontal scrollbar )
Baris penggulung mendatar digunakan untuk menggeser layar dari kiri ke kanan
atau sebaliknya. Baris penggulung ini terletak di atas toolbar drawing.



2. Baris penggulung tegak (Vertical scrollbar)
Baris penggulung tegak diganakan untuk menggeser layar ke atas atau ke bawah.
baris penggulung ini biasanya terdapat di sebelah kanan pada lembar kerja.

40. Tuliskan Langkah langkah mengatur tampilan baris penggulung microsoft Excel


Penjelasan:

Menghilangkan tampilan baris penggulung, lakukan langkah berikut :

Pilih Office Button > Excel Option

Pada kotak dialog Excel Option sebelah kiri pilih Advanced

Selanjutnya cari opsi Display option for this workbook

Hilangkah cawang pada pilihan Show horizontal scroll bar atau pada pilihan Show vertical scroll bar

OK
