2. Mengapa dalam pemrograman Java tidak memiliki keyword procedure dan function


Salah satu alasan kuat kenapa dalam pemrograman Java tidak memiliki fungsi/perintah procedure dan function adalah dikarenakan metode atau pendekatan yang digunakan oleh java itu sendiri. Secara teknis Java termasuk dalam kategori object oriented programming, sedangkan procedure dan function pada umumnya digunakan oleh procedural programming.

Pembahasan:

Object Oriented Programming (OOP)

Sesuai dengan namanya, Object Oriented Programming adalah bahasan pemrograman yang  memiliki orientasi pada atribut dan perilaku setiap objek. Sebagai iustrasinya bayangkan kamu sedang memilah saluran televisi dengan bantuan remote kontrol. Dalam hal ini remot kontrol tersebut adalah objek yang memiliki atribut dan perilaku. Dalam ilustrasi tersebut OOP diibaratkan sekumpulan kode yang fokus pada "bagaimana" objek berprilaku bukan untuk memberi tahu bagaimana objek harus berprilaku. Berikut ini adalah beberapa bahasa pemrograman yang termasuk dalam kategori OOP, diantaranya adalah:

JavaPythonRubyC++Visual Basic .NETObjective-C

Procedural Programming

Bahasa pemrograman seperti Pascal adalah salah satu yang termasuk dalam Prcoedural Programming, yaitu bahasa pemrograman yang pada dasarnya memiliki pendekatan berupa rangkaian perintah yang berurutan. Pada umumnya bahasa pemroograman yang termasuk dalam kategori ini terdiri dari 3 paradigma yaitu algoritma yang ditambah dengan struktur data lalu menghasilkan program.

3. Jelaskan function ( operator ) yang digunakan untuk menghitung nilai perpangkatan dalam C++ dan java.


Jawaban:

1) Fungsi Perpangkatan bahasa Java

Fungsi yang digunakan untuk menghitung perpangkatan didalam java menggunakan fungsi Math.pow(nilai, pangkat) fungsi ini akan menggembalikan nilai yang bertipe data double.

Fungsi pow() berada pada library java yaitu java.lang.Math. Untuk menambahkan library Math secara manual kita dapat menuliskan :

import java.lang.Math;

2) Fungsi perpangkatan pada bahasa CPP.

Untuk menghitung perpangkatan pada bahasa CPP menggunakan fungsi pow(nilai, pangkat) fungsi ini mirip dengan fungsi yang ada di bahasa Java. Untuk menggunakan fungsi ini diperlukan library Math yang dapat dituliskan seperti ini :

#include <math.h>

31. Lari cepat diawali dengan start


Lari cepat diawali dengan start jongkok.

Pembahasan

Lari jarak pendek adalah gerkaan berlari yang menggunkana kecepatan penuh mulai dari garis strat hingga garis finish. Dalam lari jarak pendek terdapat tiga nomor yang masuk kedelam kategorii lari jarak pendek, yakni 100 meter hingga 400 meter. Dalam lari jarak pendek, terdapat beberapa teknik dasar yang harus dikuasai oleh setiap pelari.  Teknik dasar dalam lari jarak pendek adalah.

1. Teknik start.

Dalam lari jarak pendek, start yang digunakan adalah start jongkok.

2. Teknik berlari

Teknik berlari pada lari jarak pendek dilakukan dengan posisi badan sedikit condong ke depan dan menggunakan langkah lebar dengan tolakan yang kuat.

3. Teknik memasuki garis finish.

Ketika akan memasuki garis finish,  gerakan kaki dipercepat dan salah satu bahu dijatuhkan kedepan agar terlebih dahulu memasuk garis finish dibandingkan anggota badan lainnya.
