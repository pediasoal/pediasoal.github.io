Abstract
Definisi : Abstract adalah kata kunci yang digunakan untuk menyatakan kelas atau metode untuk menjadi abstract.
Kegunaan : Sebuah metode tidak memiliki implementasi, semua kelas yang
berisi metode abstract harus dari mereka abstrak, walaupun tidak semua kelas abstrak
memiliki metode abstrak.
Syntax : [abstract] class [class_mahasiswa]
Contoh :
public abstract class Belajar {
public int jamBelajar;
public String tempat;
public abstract void belajar();
}
//Class Turunan
//JadwalBelajar.java
class Telkom extends Belajar{
public void setJam(int jam){
jamBelajar = jam;
}
public void setKelas(String sekolah){
tempat = sekolah;
}
public void belajar() {
System.out.println(“Kita belajar”
+ ” java di kelas “+tempat+” selama ”
+jamBelajar+” jam”);
}
}
class Primagama extends Belajar{
public void setJam(int jam){
jamBelajar = jam;
}
public void setCabang(String cabang){
tempat = cabang;
}
public void belajar() {
System.out.println(“Kita belajar”
+ ” java di primagama cabang ”
+tempat+” selama “+jamBelajar+” jam”);
}
}
public class JadwalBelajar{
public static void main(String[] args) {

6. fungsi syntax union digunakan untuk


Fungsi Syntax Union digunakan untuk melakukan penggabungan query dalam melakukan pemanggilan select. Syntax Union sendiri berkaitan dengan penggunaan dalam MySQL. Seperti halnya dalam matematika, syntax union akan menghilangkan duplikasi hasil dari pemanggilan select. Dalam pemanggilan, unsur dalam expression yang boleh berubah hanyalah nama tabel.

Pembahasan

Dalam penggunaan fungsi syntax union, ada beberapa hal yang harus diperhatikan antara lain:

Nama tabel dapat berbeda antar SELECT, namun, harus ada minimal satu nama tabel di salah satu expression SELECT.Jumlah field antara SELECT awal dan lainnya harus sama. Contoh: Pada select pertama terdapat 3 field, maka select lainnya harus juga 3 field.Syntax Union secara otomatis akan menghapus setiap duplikasi yang muncul pada hasil sehingga tidak diperlukan suatu pengubah distinctJika menggunakan pernyataan ORDER, maka field pada setiap SELECT yang diisikan berupa alias / nama lain bukan nama asliPenggunaan pernyataan ORDER yang tidak diikuti dengan pernyataan LIMIT harus diletakkan pada susunan terakhir agar dapat terbaca

15. apa fungsi distro linux


Secara umum fungsi Sistem Operasi yaitu:

1. MULTI TASKING: memungkinkan beberapa program sekaligus diproses pada saat bersamaan di sebuah komputer dengan single user.
2. MULTIPROGRAMMING: memungkinkan beberapa program menggunakan resources komputer pada saat waktu yang bersamaan bergantian (cth: proses memasukan data, melakukam cetak pada printer)
3. TIME SHARING: memungkinkan beberapa pemakai komputer menggunakan CPU bersamaan dan CPU akan memberikan waktunya bergantian kepada user untuk melayaninya.

Secara detail fungsi Sistem operasi yaitu:

1. Pengaturan Processor, yaitu mengatur processor untuk dapat dijalankan oleh sistem komputer.
2. Pengaturan Memori, yaitu mengatur pembagian dan mengirimkan instruksi dari memori utama dan tempat penyimpanan lain ke sistem komputer.
3. Pengaturan Input/Output, yaitu koordinasi dan penugasan dari berbagai perangkat input/output ketika satu atau lebih program sedang dieksekusi atau dijalankan.
4. Pengaturan File, yaitu mengatur penyimpanan file dari berbagai tempat ke perangkat penyimpanan lainnya. Juga memungkinkan semua file dapat dengan mudah diubah dan dimodifikasi dengan menggunakan text editor atau dengan lainnya.
5. Menjalankan sistem prioritas, yaitu menentukan urutan pekerjaan yang harus dilaksanakan dalam sistem komputer, mulai dari yang paling utama, lalu yang kedua, dan seterusnya.
6. Interpretasi atau penerjemahan perintah-perintah dan instruksi-instruksi.
7. Sebagai fasilitas komunikasi yang mudah antara sistem komputer dan komputer operator (manusia).
8. Bertanggung jawab atas keamanan data dan integritas.

21. 2. Apakah Syntax yang digunakan dalam fungsi AVERAGEIF! Jawab: ...​


Syntax yang digunakan dalam fungsi AVERAGEIF adalah sebagai berikut:

AVERAGEIF(range, criteria, [average_range])

- range: Rentang sel yang akan dievaluasi berdasarkan kriteria tertentu.

- criteria: Kriteria yang digunakan untuk menentukan data mana yang akan dihitung rata-ratanya.

- average_range (opsional): Rentang sel yang berisi data yang ingin dihitung rata-ratanya. Jika tidak ditentukan, maka range akan digunakan sebagai average_range.

Contoh penggunaan fungsi AVERAGEIF:

Misalnya, kita memiliki rentang sel A1:A5 yang berisi nilai-nilai, dan kita ingin menghitung rata-rata hanya dari nilai-nilai yang lebih besar dari 5.

AVERAGEIF(A1:A5, ">5")

Dalam contoh di atas, range adalah A1:A5, dan kriteria adalah ">5". Fungsi ini akan menghitung rata-rata dari nilai-nilai yang memenuhi kriteria tersebut.

Catatan: Syntax dan cara penggunaan fungsi AVERAGEIF dapat bervariasi tergantung pada program spreadsheet atau bahasa pemrograman yang digunakan. Contoh di atas mengacu pada penggunaan fungsi AVERAGEIF dalam aplikasi spreadsheet seperti Microsoft Excel atau Google Sheets.

31. pengertian dan fungsi linux


Pengertian dan fungsi linux adalah salah satu sistem operasi yang bermodel open source dan berbasis UNIX. Fungsi linux sendiri tidak jauh berbeda dengan sistem operasi lain yaitu melakukan multitasking, multiprogramming, multiprocessing dan time sharing.

Pembahasan

Sebagai salah satu sistem operasi yang memiliki model open source, sistem operasi memberikan akses bagi penggunanya untuk melakukan modifikasi bagi sistem operasi ini sendiri.  Tentunya dengan kebebasan akses linux tetap mendapat lisensi dari GNU untuk General Public License.

Fungsi LINUX sendiri tidak jauh dengan sistem operasi lain, antara lain :

Multi-tasking

Sebagai sistem operasi yang dikendalikan oleh penggunanya, sistem operasi dapat menjalankan program - program secara bersamaan tanpa menimbulkan masalah.

Multi-programming

Seperti yang kita tahu, fungsi sistem operasi sendiri adalah mengatur manajemen resources perangkat berserta operasinya sehingga fungsi ini membuat sistem operasi dapat mengakses dan melakukan tindakan - tindakan atas resources ini dalam waktu yang sama.

Multi-Processing

Sistem operasi dapat menjalankan processor pada beberapa cpu dalam waktu yang sama

Time-Sharing

Dengan adanya sistem operasi ini, dapat berjalan model multiuser yang menggunakan cpu secara serentak dan secara otomatis akan diatur pembagian waktu antar usernya untuk dapat menjalankan layanan.

33. Jelaskan fungsi-fungsi dari perintah dasar linux


Perintah-perintah dasar Linux Debian
cd (change directory),untuk berpindah direktori
cp,untuk mengcopy suatu file
rm (remove files),menghapus suatu data/file
rm -fr (resurcive remove),menghapus file,directory,atau sub directory
mkdir (make directory),membuat directory baru
cat,melihat isi file
more,melihat isi suatu file yg dapat ditampilkan layar per layar
clear,memebersihakan layar directory yg aktif
who,mengetahui daftar pemakai yg sedang aktif (login)
exit,keluar dari sistem
cal,menampilkan kalender
date,menunjukkan/mengatur tanggal
ls,menampilkan directory yg aktif
arch,menampilkan arsitektur prosessor
nano,editor text standar debian untuk mengedit file
eject,perintah mengeluarkan cd-rom
eject-t, perintah memasukkan cd-rom
apt-get update, printah mengupdate sistem
apt-get upgrade,menginstall seluruh aplikasi baru scr otomatis dari catatan yg telah diterima
addusser,menambah user baru
psswd,menggnti password user
reboot,merestar pc
poweroff / halt,mematikan pc
ifconfig,menampilkan eth yg hidup dan networknya
ifconfig - a,menampilkan semua eth dan networknya
hostname,menampilkan hostname
apt-get-install,menginstal aplikasi untuk pc dari repository
apt-get-remove,menghapus aplikasi yg terinstal
apt-get install -f,menginstal semua kekurangan paket dalam instalasi aplikasi tertentu

34. fungsi partisi linux


a. / : sebuah direktori yang menempati posisi yang paling atas yang kemudian di ikuti oleh direktori " lainnya

b. /home : merupakan direktori user, di dalamnya terdapat konfigurasi – konfigurasi tentang user dan tempat meyimpan data untuk user, document & setting kalo di sistem operasi windows

c. /boot : merupakan direktori yang digunakan untuk menyimpan konfigurasi dan file-file yang berhubungan dengan proses booting

d. /usr : merupakan direktori yang digunakan untuk penyimpanan konfiguasi aplikasi yang akan digunakan oleh user

e. /var : merupakan direktori yang digunakan untuk penyimpanan Log system

f. /tmp : singkatan dari temporer adalah direktori yang disediakan ketika dibutuhkan ruang sementara dalam melakukan pekerjaan, contoh ketika melakukan proses burn cd maka image (file iso ) secara default dimasukkan ke direktori ini sebelum di burn ke cd

g. /opt : merupakan direktori yang digunakna untuk menyimpan aplikasi tambahan diluar aplikasi bawaan dari distro

l. swap : virtual memory dari sistem operasi linux. Biasanya pengguna linux memberi ukuran 2 kali jumlah memory RAM. digunakan untuk membantu kerja memory

37. tuliskan syntax fungsi logika IF


IF Tunggal  
 Fungsi logika IF yang hanya memiliki satu kondisi / syarat, sehingga dipastikan memiliki dua hasil yang akan ditampilkan yaitu hasil yang sesuai syarat (kondisi) atau hasil yang tidak sesuai syarat (kondisi) akibat dari satu kondisi / syarat tersebut, sehingga hanya membutuhkan satu IF (Tunggal) 

Rumus seperti di atas ....
Contoh :
Dalam pelajaran TIK, Jika Nilai TIK kalian kurang dari 75 maka harus remidi dan jika mendapatkan nilai 75, 76 ke atas berarti kalian lulus.
