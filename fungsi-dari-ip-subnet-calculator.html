maka IP Subnet adalah 192.168.18.0 dan IP Broadcast 192.168.18.255Cara menghitung IP Subnet (IP Network):
Ubah IP menjadi biner:
11000000.10101000.00010010.10110001
Tidak ada suffix (Nilai CIDR)? Gunakan default netmask: /24 yaitu 255.255.255.0.
Ubah Subnet mask tsb menjadi biner juga:
11111111.11111111.11111111.00000000

Perhitungan Network:
Subnet mask AND IP:
[tex]11111111.11111111.11111111.00000000[/tex]
[tex]11000000.10101000.00010010.10110001[/tex]
-------------------------------------------------------- AND
[tex]11000000.10101000.00010010.00000000[/tex] ATAU 192.168.18.0

Perhitungan Broadcast:
Invers Subnet Mask OR Network
[tex]00000000.00000000.00000000.11111111[/tex]
[tex]11000000.10101000.00010010.00000000[/tex]
------------------------------------------------------------------- OR
[tex]11000000.10101000.00010010.11111111[/tex] ATAU 192.168.18.255

Maka dengan IP 192.168.18.177, 
subnet masknya adalah 255.255.255.0

Networknya adalah 192.168.18.0 dengan broadcast 192.168.18.255

8. Jika diberikan IP Address 192.168.170.155/27 lakukan subnetting ! Berapa ? 1. IP BC (IP Awal) 2. IP Net (IP Akhir) 3. IP Host (rentang IP) 4.Subnet Mask 5.Jumlah Subnet 6. Blok Subnet 7. Host per subnet dan beserta caranya​


Jawaban:

IP Address 192.168.170.155/27 adalah alamat IP dengan subnet mask /27, yang menunjukkan bahwa 27 bit dari 32 bit alamat IP tersebut digunakan untuk menentukan subnet, sisanya 5 bit digunakan untuk menentukan host.

1. IP BC (IP Awal): 192.168.170.128

Untuk menentukan IP BC, pertama-tama kita harus menentukan subnet mask dalam bentuk desimal, yaitu 255.255.255.224. Kemudian, kita menentukan subnet dalam bentuk biner, yaitu 11000000.10101000.10101010.10000000. Setelah itu, kita membuat IP BC dengan mengubah hanya satu bit dari bit paling kanan dari subnet menjadi nol, sehingga memperoleh alamat IP dalam bentuk desimal sebagai berikut: 192.168.170.128.

2. IP Net (IP Akhir): 192.168.170.159

Untuk menentukan IP Net, kita melakukan hal yang sama dengan menentukan IP BC, tetapi dengan mengubah satu bit dari bit paling kanan dari subnet menjadi satu, sehingga memperoleh alamat IP dalam bentuk desimal sebagai berikut: 192.168.170.159.

3. IP Host (rentang IP): 192.168.170.129 - 192.168.170.154

IP Host adalah rentang IP yang dapat digunakan untuk menentukan host pada jaringan, dan dapat ditemukan dengan mengurangi IP BC dengan satu dan menambahkan IP Net dengan satu.

4. Subnet Mask: 255.255.255.224

Subnet Mask adalah alamat IP yang digunakan untuk menentukan bagian dari alamat IP yang digunakan untuk subnet.

5. Jumlah Subnet: 8

Jumlah subnet adalah jumlah subnet yang dapat dibuat dengan menggunakan alamat IP dan subnet mask tertentu. Dalam hal ini, karena kita memiliki 5 bit untuk menentukan host, kita memiliki 2^5 = 32 subnet yang dapat dibuat.

6. Blok Subnet: 32

Blok Subnet adalah jumlah alamat IP yang tersedia pada setiap subnet. Dalam hal ini, karena kita memiliki 2^5 = 32 subnet, maka blok subnet adalah 32.

7. Host per subnet: 30

Jumlah host per subnet adalah jumlah host yang dapat ditempatkan pada setiap subnet. Dalam hal ini, karena kita memiliki 2^5 - 2 = 30 host per subnet.

10. Pada IP 196.189.200.1/26 tentukanjumlah subnetjumlah host persubnetblok subnet​


Jawaban:

IP 196.189.200.1/26 merupakan alamat IP dengan subnet mask 255.255.255.192 atau dalam bentuk CIDR /26. Subnet mask tersebut menunjukkan bahwa terdapat 26 bit yang direservasi untuk network ID dan sisa bit (6 bit) digunakan untuk host ID.

Jumlah Subnet

Jumlah subnet dapat dihitung dengan menggunakan formula 2^n dimana n adalah jumlah bit yang direservasi untuk subnet. Pada kasus ini, terdapat 6 bit yang direservasi untuk subnet, sehingga jumlah subnet yang dapat dibentuk adalah 2^6 = 64 subnet.

Jumlah Host Persubnet

Jumlah host persubnet dapat dihitung dengan menggunakan formula 2^n-2 dimana n adalah jumlah bit yang direservasi untuk host. Karena pada kasus ini terdapat 6 bit yang direservasi untuk host ID, maka jumlah host persubnet yang dapat dihitung adalah 2^6-2 = 62 host.

Blok Subnet

Untuk menentukan blok subnet, kita perlu menghitung nilai dari setiap oktet pada alamat IP dan subnet mask. Berikut adalah nilai dari setiap oktet pada alamat IP dan subnet mask:

Alamat IP: 196.189.200.1 = 11000100.10111101.11001000.00000001

Subnet mask: 255.255.255.192 = 11111111.11111111.11111111.11000000

Untuk menentukan blok subnet, kita cukup melakukan operasi bitwise AND antara alamat IP dan subnet mask. Berikut adalah hasil operasi bitwise AND tersebut:

11000100.10111101.11001000.00000001 (IP Address)

11111111.11111111.11111111.11000000 (Subnet Mask)

11000100.10111101.11001000.00000000 (Network ID)

Berdasarkan hasil tersebut, blok subnet untuk alamat IP 196.189.200.1/26 adalah 196.189.200.0/26.

15. jumlah subnet dan host per subnet dari ip 192.168.1.1/19 berapa?


Jumlah subnet dan host per subnet dari IP 192.168.1.1/19 adalah sebagai berikut:

- Notasi CIDR /19 menandakan bahwa 19 bit pertama pada alamat IP tersebut digunakan untuk network ID, sehingga sisa 13 bit digunakan untuk host ID.
- Jumlah subnet dapat dihitung dengan rumus 2^k, di mana k adalah jumlah bit yang digunakan untuk network ID. Dalam hal ini, k = 19, sehingga jumlah subnet adalah 2^19 = 524,288 subnet.
- Jumlah host per subnet dapat dihitung dengan rumus 2^h - 2, di mana h adalah jumlah bit yang digunakan untuk host ID. Dalam hal ini, h = 13, sehingga jumlah host per subnet adalah 2^13 - 2 = 8,190 host per subnet.

Jadi, IP 192.168.1.1/19 memiliki 524,288 subnet dan 8,190 host per subnet.

19. diketahui suatu IP 10.10.0.0/16 hitunglah jumlah subnet, host per subnet, blok subnet dan buat tabelnya!​


IP: 10.10.0.0/16

Jumlah subnet = 256 subnetHost per subnet = 65.534 host per subnetBlok subnet = 1Tabel subnet terdapat pada bagian pembahasan.

 

Pembahasan

Diketahui IP: 10.10.0.0/16

Oktet pertama = 10, maka termasuk pada class A.CIDR subnet mask = /16Subnet mask (biner)
= 11111111.11111111.00000000.00000000Subnet mask = 255.255.0.0x: banyak bit 1 pada oktet ke-2 sampai oktet ke-4 dari subnet mask
⇒ x = 8y: banyak bit 0 pada oktet ke-2 sampai oktet ke-4 dari subnet mask
⇒ y = 16

JUMLAH SUBNET (S)
S = [tex]2^x = 2^8[/tex] = 256 subnet.

JUMLAH HOST PER SUBNET (H)
H = [tex]2^y - 2 = 2^{16} - 2[/tex]
⇒ H = 65.536 – 2
⇒ H = 65.534 host per subnet

BLOK SUBNET (B)
B = 256 – nilai oktet kedua
⇒ B = 256 – 255
⇒ B = 1

Jadi, pada oktet kedua network address dari subnet, angkanya adalah 0, 1, 2, 3, ..., 255.Untuk setiap blok subnet, range dari oktet ketiga adalah dari 0 sampai 255.Untuk setiap blok subnet, semua kemungkinan nilai oktet ketiga dan keempat dari IP host dapat digunakan, kecuali yang bernilai *.*.0.0 (untuk network address), dan *.*.255.255 (untuk broadcast address).

Jawaban:

Alamat IP 192.68.2.1 dengan subnet 3 bit akan menghasilkan 8 subnet baru dengan subnet mask 255.255.255.224 dan tiap subnet dapat menampung 254 alamat IP. Tiap subnet memiliki alamat IP dari network address hingga broadcast address.

Penjelasan:

Untuk membagi alamat IP 192.68.2.1 menjadi 8 network baru, kita perlu menggunakan proses subnetting. Dalam subnetting, kita membagi alamat IP menjadi beberapa subnet dengan mengubah bagian dari bit host menjadi bit subnet.

Langkah-langkah untuk menghitung subnet dan alamat IP tiap subnet adalah sebagai berikut:

Tentukan jumlah subnet yang dibutuhkan, dalam kasus ini adalah 8.

Tentukan jumlah bit subnet yang diperlukan dengan menggunakan rumus 2^b >= jumlah subnet, dimana b adalah jumlah bit subnet yang diperlukan. Dalam kasus ini, 2^3 >= 8, maka diperlukan 3 bit subnet.

Hitung nilai dari setiap bit subnet. Dalam kasus ini, bit subnet adalah 3 bit paling kiri dari octet ke-3. Maka nilai dari bit subnet adalah 128 + 64 + 32 = 224.

Hitung jumlah alamat IP dalam setiap subnet dengan menggunakan rumus 2^(32-b)-2, dimana b adalah jumlah bit subnet yang diperlukan. Dalam kasus ini, 2^(32-3)-2 = 254, artinya setiap subnet dapat menampung 254 alamat IP.

Bagi alamat IP menjadi 8 subnet dengan menambahkan nilai bit subnet ke setiap subnet. Untuk menghitung alamat IP tiap subnet, gunakan rumus subnet mask = 256 - nilai bit subnet.

34. Sebuah IP 192.168.80.∅/ 2+1A. Jumlah subnet B. Jumlah hasil persubnetC. Block subnetD. Range IP ​


A. Jumlah subnet:

IP 192.168.80.0/2+1 memiliki 2^(2+1) = 2^3 = 8 subnet.

B. Jumlah hasil per subnet:

Setiap subnet memiliki 2^(32-2-1) = 2^29 = 536,870,912 alamat IP.

C. Block subnet:

Block subnet dalam IP 192.168.80.0/2+1 adalah 192.168.80.0/2+1, dimana "/2+1" menunjukkan mask subnet /2+1, yang berarti 2 bit dari 32-bit alamat IP digunakan untuk subnetting.

D. Range IP:

Range IP setiap subnet akan bervariasi tergantung pada subnet mask yang digunakan. Misalnya, jika subnet mask adalah /25, maka setiap subnet akan memiliki 128 alamat IP (dari 64 hingga 191) dan range IP akan menjadi 192.168.80.0 hingga 192.168.80.127. Namun, jika subnet mask adalah /26, maka setiap subnet akan memiliki 64 alamat IP (dari 0 hingga 63 atau dari 64 hingga 127) dan range IP akan menjadi 192.168.80.0 hingga 192.168.80.63 atau 192.168.80.64 hingga 192.168.80.127.

2. **Jumlah Host per Subnet**: Dalam subnet /30, dua alamat digunakan untuk jaringan dan broadcast, sementara dua alamat tersedia untuk host. Oleh karena itu, setiap subnet memiliki 2 host yang dapat digunakan.

3. **Blok Subnet**: Subnet ini menggunakan prefix /30, yang berarti blok subnetnya adalah 4 alamat berurutan. Dalam hal ini, blok subnetnya adalah 170.238.102.176/30 hingga 170.238.102.179/30.

4. **Alamat IP Jaringan**: Alamat IP jaringan untuk subnet ini adalah 170.238.102.176.

5. **Host Pertama**: Host pertama dalam subnet ini adalah 170.238.102.177.

6. **Host Terakhir**: Host terakhir dalam subnet ini adalah 170.238.102.178.

7. **IP Broadcast**: IP broadcast dalam subnet ini adalah 170.238.102.179.

8. **Kelas**: Alamat IP 170.238.102.177/30 tidak termasuk dalam kelas A, B, atau C, karena ia adalah bagian dari prefix /30 yang lebih spesifik. Kelas khusus ini digunakan untuk subnetting.

9. **IP Address**: Alamat IP adalah 170.238.102.177.

10. **Subnetwork Ke Berapa**: Dengan prefix /30, ini adalah subnet ke-1 dalam jaringan 170.238.102.176/30.

C. Block subnet adalah jumlah alamat IP yang tersedia dalam setiap subnet. Dalam hal ini, block subnet adalah 14 alamat IP.

D. Range IP adalah jangkauan alamat IP yang tersedia dalam setiap subnet. Dalam hal ini, range IP dapat ditemukan dengan menggunakan block subnet dan alamat IP awal dari setiap subnet. Misalnya, range IP dari subnet pertama adalah 192.168.6.0 - 192.168.6.14.
