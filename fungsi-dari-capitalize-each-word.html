2. Apa fungsi dari "CAPITALIZE EACH WORD"


Fungsi dari "Capitalize Each Word" adalah untuk mengubah huruf-huruf pada setiap awal kata yang dipilih/diblok menjadi huruf kapital.

Pembahasan

Capitalize each word merupakan salah satu opsi perintah pada Microsot Word yang berada pada tab Home. Perintah ini tepatnya termasuk bagian dari menu di bagian sub-tab Font--Change case. Change case adalah perintah untuk melakukan perubahan format karakter atau huruf pada tampilan lembar kerja terkait kapitalisasi. Beberapa opsi pada perintah change case di antaranya:

Sentence caselower caseUPPERCASE CASECapitalize Each WordtOGGLE cASE

Fungsi dari Capitalize Each Word adalah mengubah seluruh huruf pertama setiap kata yang diblok/dipilih menjadi huruh kapital. Semisal adalah :

Ada kalimat :
Alat pemadam api ringan (APAR).Kemudian blok kalimat "Alat pemadam api ringan"Klik pilihan change case pada tab Home, sub-tab font.Kemudian, pilih Capitalize Each Word, maka tampilan kalimat tadi menjadi :
Alat Pemadam Api Ringan (APAR).

4. 10. Apa yang dimaksud dengan UPPERCASE, Sentence case, lowercase, Capitalize Each Word, TOGGLECASE​


Jawaban:

Semuanya itu dimaksudkan sebagai salah 1 fasilitas dalam program pengolah kata (contohnya Ms. Word) untuk melakukan pengaturan huruf yang aktif.

Penjelasan:

- Uppercase: mengatur supaya semua huruf/ teks menjadi huruf kapital.

- Sentence case: mengatur agar hanya huruf awal saja yang kapital (dalam suatu kalimat)

- lowercase: mengatur supaya semua huruf/ teks menjadi bukan huruf kapital.

- Capitalize Each Word: mengatur supaya huruf/ teks yang menjadi huruf depan saja yang kapital (dalam suatu kalimat)

- Toggle case: mengatur pembalikan huruf kapital (yg sebelumnya bukan huruf kapital menjadi huruf kapital dan yg sebelumnya huruf kapital menjadi bukan huruf kapital).

Fasilitas-fasilitas di atas disebut Change Case dan dalam tampilan program Ms. Word 2016, dapat diakses melalui tab ribbon Home (perhatikan gambar berikut ini/ ditunjukkan tanda panah merah).

14. 10.mengubah semua huruf menjadi huruf besar adalah fungsi dari .....A.toggle case b.lowercase c.capitalize Each word d.uppercase ​


Jawaban:

D. Uppercase

Penjelasan:

Change Case merupakan salah satu perintah pemformatan karakter di Microsoft Word yang berfungsi untuk menyesuaikan kapitalisasi huruf (capitalization) yaitu jenis besar dan kecil huruf berdasarkan posisi dan format awal dalam suatu teks atau kalimat. Perintah ini terletak di ribbon "Home" tepatnya pada grup perintah "Font".

Perintah Change Case terdiri dari 5 jenis kapitalisasi huruf, yaitu: Sentence case, lower case, UPPERCASE, Capitalize Each Word, dan tOGGLE cASE.

Adapun fungsi dari tiap fitur Change Case tersebut adalah sebagai berikut:

Sentence Case: Membuat semua huruf yang berada di awal kalimat menjadi huruf kapital.lower case: Mengubah semua huruf yang dipilih menjadi huruf kecil.UPPERCASE: Berbeda dengan "lower case", UPPERCASE berfungsi untuk mengubah semua huruf yang dipilih menjadi huruf kapital.Capitalize Each Word: Mengubah huruf depan di setiap kata menjadi huruf kapital.tOGGLE cASE: Mengubah tampilan huruf kapital menjadi huruf kecil dan huruf kecil menjadi huruf kapital.

Semoga membantu!
