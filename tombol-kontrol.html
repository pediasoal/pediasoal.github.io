1. tuliskan 10 tombol kontrol dan kategori tombolnya


1. Fungsi Tombol Esc

Tombol Escape untuk membatalkan perintah yang sudah dilakukkan. Jika kita menekan tombol ini maka perintah atau dialog yang kita masukkan akan dibatalkan.

2. Fungsi Tombol Tab

Tombol Tab berfungsi jika ditekan akan memindahkan cursor ke arah kanan dengan satu tabulansi.

3. Fungsi Tombol Caps Lock 

Jika menekan tombol ini maka lampu indikator caps lock pada keyboard menyala, hal ini menandakan bahwa anda menggunakan mode semua huruf besar atau semua huruf kapital, jika anda menekan lagi dan lampu indikator mati maka dalam mode seperti biasa atau semua huruf kecil.

4. Fungsi Tombol Shift

Fungsinya yaitu untuk membuat huruf besar secara manual (dengan menekan shift+a = A), dan untuk memasukkan input yang diatas angka 1,2,3 dan seterusnya (contoh : shift+1 = ! , shift+/ = ?).

5. Fungsi Tombol Ctrl

Khusus untuk tombol ini fungsinya itu untuk menyisipkan perintah-perintah pintas. Contohnya untuk mencopy  file tinggal tekan ctrl+c.

6. Fungsi Tombol Alt

Fungsinya sama seperti tombol Ctrl yaitu untuk menyisipkan perintah secara pintas. Contohnya alt+F4, itu untuk menutup program yang sedang dijalankan.

7. Fungsi Tombol Enter

Sebagai perintah "ok" yang diberikan oleh program pada komputer dan untuk membuat paragraf baru.

8. Fungsi Tombol Back Space 

Fungsinya untuk mendelete satu huruf yang berada disebelah kiri kursor. Jika kita kombinasikan dengan tombol ctrl, ctrl+backspace maka anda menghapus satu kata disebelah kiri kursor.

9. Tombol Delete

Tombol ini sama seperti tombol backspace, bedanya itu jika tombol delete menghapus satu huruf di sebelah kanan kursor.

10. Tombol Macro 

Macro ini jika ditekan akan muncul seperti ini "\".

7. Bagian keyboard yang terdiri atas tombol yang berisi angka dan tombol pemindah kursor yang biasanya terletak pada sisi kanan main typing board adalah…. A. function Keys B. typing Writer C. kontrol Keys D. numeric keypad


Jawaban:

Bagian-bagian keyboard komputer/ laptop :

1. Typewriter Key

Terbentang dari tombol ~ di ujung kiri atas hingga tombol ctrl di bagian kanan bawah. Contoh : tab, capslock, shift, ctrl, alt, enter, backspace.

2. Function Key

Dapat menampilkan help support, melakukan proses rename, dan lain sebagainya. Tombol fungsi ini terbentang dari Esc pada ujung kiri atas, dilanjutkan dengan kombinasi tombol F1 hingga F12.

3. Navigation Key

Terdiri dari 4 tombol panah (kiri, kanan, atas, bawah), tombol insert dan delete, tombol page up dan page down, serta tombol home dan end.

4. Numeric Keypad

Kombinasi tombol danmemudahkan kita untuk mengetikkan angka.

5. Extra Key (untuk komp/lap. tertentu saja)

Terdiri dari beberapa tombol yang biasanya memiliki fungsi sesuai dengan apa yang tertulis pada tombol tersebut. Fitur printscreen, Scroll Lock, dan Pause/Break sering dijumpai pada keyboard dengan jenis ini.

Sehingga, jawaban dari pertanyaan di atas adalah Numeric Keypad.

22. 4 macam tombol kontrol jendela beserta fungsinya


Jawaban ringkas:

Empat macam tombol kontrol jendela (control window) yaitu:

a. Minimize Button.

b. Maximize Button.

c. Restore Button.

d. Close Button.

Penjelasan panjang:

a. Minimize Button : tombol untuk memperkecil ukuran jendela menjadi ikon aktif di taskbar.

b. Maximize Button : tombol untuk memperbesar ukuran jendela menjadi layar penuh.

c. Restore Button : tombol untuk mengembalikan jendela ke ukuran semula. (sebelum diklik Maximize).

d. Close Button : tombol untuk menutup jendela dan mengakhiri program yg aktif.

Perlu diketahui bahwa: tombol Maximize hanya aktif, bila sudah mengklik tombol Restore.

Begitu juga sebaliknya: tombol Restore hanya aktif, bila sudah mengklik tombol Maximize.
