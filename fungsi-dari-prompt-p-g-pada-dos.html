Jawaban:

a. ECHO: Menampilkan pesan, atau mengubah command echoing on atau off.

b. COLOR: Mengubah warna teks dan latar pada command prompt

c. VER: Menampilkan versi sistem operasi yang digunakan

d. CLS: Membersihkan layar dan menempatkan kursor pada pojok kiri layar. Perintah ini tidak memiliki parameter.

e. PROMPT: Mengubah tampilan command prompt MS-DOS

f. DATE: Perintah ini akan menampilkan tanggal saat ini. Perintah ini juga dapat mengeset tanggal komputer.

g. LABEL: Berfungsi untuk menampilkan label atau merubah nama label pada disk.

h. COPY CON: untuk membuat file

i. TYPE: Menampilkan isi dari sebuah berkas (dalam bentuk teks) ke dalam standard output.

j. DIR: Jika digunakan tanpa parameter, perintah ini dapat menampilkan daftar berkas-berkas dan subdirektori yang terdapat di dalam direktori aktif. Berkas ini memiliki satu parameter, yakni lokasi direktori di mana hendak menampilkan daftar isi direktori.

k. COPY: Digunakan untuk meng-copy isi file yang satu ke file yang lain.

l. CHDIR/CD: Mengganti direktori aktif ke direktori lainnya yang ditentukan dalam parameter. Jika dijalankan tanpa parameter, maka perintah ini akan menampilkan lokasi di mana direktori aktif berada.

m. REN/RENAME: Mengubah nama sebuah atau beberapa berkas (dengan menggunakan karakter wildcard).

n. MOVE: Berfungsi untuk memindahkan suatu file ke lokasi lain yang berbeda.

o. DEL/ERASE: Menghapus berkas yang ditentukan dalam parameter. Parameter dapat berupa nama berkas atau beberapa nama berkas yang disusun menggunakan karakter wildcard.

p. VOL: Menampilkan nama label dari sebuah volume atau partisi.

q. ATTRIB: Menampilkan atau mengubah atribut suatu file.

a. XCOPY: Menyalin file serta pohon direktori.

b. FORMAT: Mem-Format sebuah hardisk/disket untuk digunakan di Windows.

c. CHKDSK: Berfungsi untuk melihat kondisi media penyimpanan yang kita pakai.

d. SCANDISK: untuk pemeriksaan dan perbaikan sistem berkas dan buruk Cluster pada Hard drive.

e. DEFRAG: untuk mengoptimalkan kinerja disk oleh reorganisasi file pada disk dengan mencari bad sector, dan mengompimalkan kinerja hardisk.

f. DISKCOPY: Menyalin seluruh isi disket sama persis ke disket lainnya.

g. FDISK: untuk mengkonfigurasikan partisi suatu hardisk untuk MS-DOS

7. Sebutkan tiga macam cara untuk masuk ke command DOS prompt


Langkah pertama untuk mengawali bekerja dengan sistem komputer adalah dengan memuatsistem operasi (MS-DOS) ke dalam memori komputer yaitu dengan menggunakan aplikasiCommand Promt.
Ada beberapa langkah untuk menjalankan aplikasi command prompt, yang pertama :
1.    Pertama Anda Klik Start
2.    Kemudian pilih All Program
3.    Selanjutnya pilih Accessories
4.    Dan Klik Command PromptCara lain untuk menjalankan command prompt yaitu bisa melalui run:
1.    Pada keyboard Tekan tombol yang bergambar windows dan tekan tombol r (run)
2.    Setelah itu anda ketikan “cmd” tanpa tanda petik pada tempat yang telah disediakan.
3.    Setelah itu Layar command prompt akan terbuka seperti pada gambar berikut:Pandangan pemakai terhadap MS-DOS

Command Interpreter / processor (shell) menginterpretasikan perintah. Command Intepreter disimpan di file command.com yang dimuatkan ke memori secara otomatis saat MS-DOS mem-boot komputer.

Macam-macam/ jenis perintah yang dimuat dalam MS-DOS

Terdapat 2 jenis perintah MS-DOS:
    Perintah internal
    Perintah eksternal

    Perintah internal
Perintah Internal merupakan bagian dari command.com yaitu perintah yang  sudah di memori saat command.com dimuatkan, 
    Perintah eksternal
Perintah eksternal dimuatkan ke memori dari disk jika diperlukan (yaitu saat dipanggil pemakai). Command interpreter mengeksekusi file program. File dieksekusi adalah file berekstensi .COM, .EXE, dan .BAT

Perintah-perintah di MS-DOS antara lain :
    Komunikasi dengan sistem
    Bekerja dengan disk
    Bekerja dengan direktori
    Pengelolaan file
    Pengelolaan perangkat masukan / keluaran
    Penyesuaian (setting) lingkungan kerja
    Pengelolaan  batch files

10. parameter pada perintah prompt yang di gunakan untuk menghapus karakter sebelumnya dan fungsinya sama dengan backspace adalah


Perintah prompt ( command prompt ) pada komputer adalah sebuah aplikasi penerjemah perintah yang terdapat pada setiap sistem operasi windows. Aplikasi ini digunakan untuk menjalankan perintah yng diinput. Kebanyakan dari perintah tersebut menjalankan tugas melalui scripts dan batch file.

Bentuk umumnya adalah PROMPT [prompt_text] [$parameter]

Beberapa parameter yang tersedia adalah

- $g :menampilkan karakter>

- $n :menampilkan default disk yang digunakan.

- $p :menampilkan posisi direktori yang sedang aktif dari drive default.

- $q :menampikan karakter “=”

- $t :menampilkan jam yang aktif.

- $v :menampilkan versi MS-DOS yang dipakai.

- $ :menampilkan karakter “$”.

- $_ :pindah baris (line feed).

- $1 :menampilkan karakter <.

- $b :menampilkan karakter !.

- $d :menampilkan tanggal yang aktif.

- $e :berfungsi sebagai tombol ESC.

Parameter pada perintah prompt yang di gunakan untuk menghapus karakter sebelumnya dan fungsinya sama dengan backspace adalah $h

14. berikut file dalam dos yang berfungsi sebagai boot loader adalah


file dalam dos yang berfungsi sebagai boot loader adalah mbldr.exe (CMIIW) File tersebut berfungsi sebagai boot loader membawa OS untuk memasuki boot yang ada di dalam sistem operasi.

MBLDR merupakan kependekan atau singkatan dari Master Boot Loader. Mbldr dibuat dengan base OS Boot Select (OS-BS) by Thomas wolfRAM


Pembahasan

Sebagai intermezzo, Boot loader atau dalam bahasa indonesia pemuat boot. Bootloader merupakan program komputer yang memuat sistem operasi (OS) untuk komputer secara sederhananya digunakan untuk memanggil boot sistem operasi dan memilih akan menggunakan Ssistem operasi yang mana.

Bootloader dapat digunakan untuk mengkonfigurasi  dan sarana untuk memilih darimana memulai kernel. Hal ini memungkinkan untuk melakukan aktivitas boot yang memiliki 2 Sistem operasi seperti contoh untuk mengunakan Linux alternatif, atau Windows. Bootloader adalah komponen penting dari proses booting yang harus dipastikan dalam keadaan baik, memori bootloader tersebut tersimpan dalam memori non-volatile contohnya seperti memori flash.
