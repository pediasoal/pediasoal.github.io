8. integral fungsi rasionalx+5/x^2-2x-3​


[tex]Hasil~dari~\int {\frac{x+5}{x^2-2x-3}} \, dx~adalah~-ln|x+1|+2ln|x-3|+C[/tex].

PEMBAHASAN

Integral merupakan operasi yang menjadi kebalikan dari operasi turunan/diferensial. Sehingga integral sering juga disebut sebagai antiturunan.

Sifat - sifat operasi pada integral adalah sebagai berikut

[tex]\int {ax^n} \, dx=\frac{a}{n+1}x^{n+1}+C~~~~~,dengan~C=konstanta\\\\\int {kf(x)} \, dx=k\int {f(x)} \, dx\\\\\int {[f(x)+g(x)]} \, dx=\int {f(x)} \, dx+\int {g(x)} \, dx\\\\\int {[f(x)-g(x)]} \, dx=\int {f(x)} \, dx-\int {g(x)} \, dx[/tex]

Untuk integral tentu dengan batas tepi a dan b berlaku :

[tex]\int\limits^b_a {f(x)} \, dx=F(b)-F(a)[/tex]

.

DIKETAHUI

[tex]\int\ {\frac{x+5}{x^2-2x-3}} \, dx\\[/tex]

.

DITANYA

Tentukan hasil integral tak tentu dari fungsi tersebut.

.

PENYELESAIAN

Kita sederhanakan dahulu fungsi rasionalnya.

[tex]\frac{x+5}{x^2-2x-3}=\frac{A}{x+1}+\frac{B}{x-3}\\\\\frac{x+5}{(x+1)(x-3)}=\frac{A(x-3)+B(x+1)}{(x+1)(x+3)}\\\\\frac{x+5}{(x+1)(x-3)}=\frac{(A+B)x+(-3A+B)}{(x+1)(x+3)}\\[/tex]

Dengan menyamakan kedua ruas diperoleh 2 buah persamaan, yaitu :

[tex]A+B=1\\\\B=1-A~~~~~~~~~...(i)\\\\\\-3A+B=5~~~~~~~~~...substitusi~pers.(i)\\\\-3A+(1-A)=5\\\\-4A=4\\\\A=-1\\\\\\B=1-A\\\\B=1-(-1)\\\\B=2\\[/tex]

Sehingga  [tex]\frac{x+5}{x^2-2x-3}=-\frac{1}{x+1}+\frac{2}{x-3}[/tex]

.

Kembali ke soal

[tex]\int {\frac{x+5}{x^2-2x-3}} \, dx\\\\=\int {[-\frac{1}{x+1}+\frac{2}{x-3}]} \, dx\\\\=-ln|x+1|+2ln|x-3|+C[/tex]

.

KESIMPULAN

[tex]Hasil~dari~\int {\frac{x+5}{x^2-2x-3}} \, dx~adalah~-ln|x+1|+2ln|x-3|+C[/tex].

.

9. Integral rasional 3x² / x² + x - 2


Penjelasan dengan langkah-langkah:

Untuk menyelesaikan integral rasional 3x² / (x² + x - 2), kita perlu membagi penyebut menjadi faktor-faktor. Kita dapat menulis x² + x - 2 sebagai (x + 2)(x - 1). Dengan demikian, integral dapat ditulis sebagai:

3x² / (x + 2)(x - 1)

Untuk menyelesaikan integral ini, kita perlu membagi pecahan menjadi dua pecahan parsial, yaitu:

3x² / (x + 2)(x - 1) = A / (x + 2) + B / (x - 1)

Kita sekarang perlu mencari nilai-nilai A dan B. Untuk melakukan ini, kita dapat mengalikan kedua sisi dengan (x + 2)(x - 1) dan menyelesaikan persamaan yang dihasilkan untuk A dan B. Setelah melakukan beberapa manipulasi aljabar, kita dapat menemukan bahwa A = 2 dan B = 1.

Dengan demikian, integral dapat ditulis sebagai:

3x² / (x + 2)(x - 1) = 2 / (x + 2) + 1 / (x - 1)

Jadi, integral dari 3x² / (x² + x - 2) adalah 2 ln|x + 2| + ln|x - 1| + C, di mana C adalah konstanta integrasi.

[tex]\tt \frac{3}{x^{2} - 1} = \frac{3}{(x + 1)(x - 1)} \\ \tt \frac{3}{x^{2} - 1} = \frac{A}{x + 1} + \frac{B}{x - 1}[/tex]

Sehingga :

[tex]\boxed{\tt \int \frac{3}{x^{2} - 1} \: dx = \int \frac{A}{x + 1} \: dx + \int \frac{B}{x - 1} \: dx}[/tex]

Dari bentuk tersebut juga diperoleh :

[tex]\tt 3 = Ax - A + Bx + B \\ \tt 3 = (A + B)x + (-A + B) \\ \\ \red{\tt Maka :} \\ \tt 0 = A + B \\ \tt 3 = B - A \\ ................. [-] \\ \tt B = \frac{3}{2} \to A = -\frac{3}{2}[/tex]

Hasil akhir diperoleh :

[tex]\tt \int \frac{3}{x^{2} - 1} \: dx = \int \frac{A}{x + 1} \: dx + \int \frac{B}{x - 1} \: dx \\ \tt \int \frac{3}{x^{2} - 1} \: dx = - \frac{3}{2}\int \frac{dx}{x + 1} + \frac{3}{2}\int \frac{dx}{x - 1} \\ \tt \int \frac{3}{x^{2} - 1} \: dx = \frac{3}{2} \ln (x - 1) - \frac{3}{2} \ln (x + 1) + C[/tex]

30. domain dan range fungsi dari fungsi rasional (pecahan)​


Penjelasan dengan langkah-langkah:

Dalam matematika, fungsi rasional adalah fungsi yang dinyatakan sebagai pecahan polinomial, di mana pembilang dan penyebutnya adalah polinomial. Untuk fungsi rasional f(x), domain dan range-nya dapat dijelaskan sebagai berikut:

1. Domain (daerah asal):

Domain dari fungsi rasional adalah himpunan semua nilai x yang dapat digunakan sebagai input dalam fungsi tersebut tanpa menyebabkan pembagian dengan nol dalam penyebut. Jadi, domain f(x) adalah semua nilai x yang membuat penyebutnya bukan nol.

2. Range (daerah hasil):

Range dari fungsi rasional tergantung pada bentuk fungsi dan batasan-batasan yang mungkin ada. Range adalah himpunan semua nilai y yang dapat dihasilkan oleh fungsi tersebut untuk nilai x dalam domainnya. Range bisa beragam tergantung pada karakteristik fungsi rasional tersebut.

Sebagai contoh, jika kita memiliki fungsi rasional f(x) = (3x + 2) / (x - 1), maka domain f(x) adalah semua nilai x kecuali x = 1 (karena itu akan membuat penyebutnya nol). Range f(x) akan bergantung pada cara grafik fungsi ini berperilaku dan apakah ada batasan lain pada variabel x. Range biasanya dapat ditentukan dengan mengamati grafik fungsi atau dengan menggunakan analisis matematis lebih lanjut.

36. Selesaikan dengan Metode Integral Fungsi Rasional


Selesaikan dengan Metode Integral Fungsi Rasional

[tex]\int {\frac{3x - 1}{x^{2} + 2x + 1}} \: dx [/tex] = 3 ln |x + 1| + [tex]\frac{4}{x + 1}[/tex] + C

Integral adalah lawan dari turunan (anti turunan).  

Rumus dasar:

∫ kxⁿ dx = [tex]\frac{k}{n + 1} x^{n + 1}[/tex] + C, dengan n ≠ –1

Jika n = –1, maka

∫ kx⁻¹ dx = [tex]\int {\frac{k}{x}} \: dx[/tex] = k ln |x| + C

Bentuk fungsi (ax + b)

∫ k(ax + b)ⁿ dx = [tex]\frac{k}{a(n + 1)} (ax + b)^{n + 1}[/tex] + C, dengan n ≠ –1

Jika n = –1, maka

∫ k(ax + b)⁻¹ dx = [tex]\int {\frac{k}{ax + b}} \: dx = \frac{k}{a} \: ln \: |ax + b| [/tex] + C

Pembahasan  

[tex]\int {\frac{3x - 1}{x^{2} + 2x + 1}} \: dx [/tex]

= [tex]\int {\frac{3x + 3 - 4}{(x + 1)^{2}}} \: dx [/tex]

= [tex]\int {\frac{3(x + 1) - 4}{(x + 1)^{2}}} \: dx [/tex]

= [tex]\int {\frac{3(x + 1)}{(x + 1)^{2}} - \frac{4}{(x + 1)^{2}}} \: dx [/tex]

= [tex]\int {\frac{3}{x + 1} - 4(x + 1)^{-2}} \: dx [/tex]

= [tex]\int {\frac{3}{x + 1}} \: dx - \int{4(x + 1)^{-2}} \: dx [/tex]

= 3 ln |x + 1| – [tex]\frac{4}{-1}[/tex](x + 1)⁻¹ + C

= 3 ln |x + 1| + [tex]\frac{4}{x + 1}[/tex] + C
