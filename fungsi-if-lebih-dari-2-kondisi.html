3. Fungsi yang digunakan untuk menentukan hasil jika kondisi bukan dari kondisi yang ditentukan adalah:

a. NOT

Penjelasan:

Berikut adalah penjelasan mengenai fungsi-fungsi tersebut:

a. NOT (Negasi): Fungsi NOT digunakan untuk membalik kondisi atau nilai yang diberikan. Jika kondisi awal bernilai TRUE, maka NOT akan menghasilkan nilai FALSE, dan sebaliknya. Dengan kata lain, NOT mengubah TRUE menjadi FALSE, dan FALSE menjadi TRUE.

Contoh penggunaan: Jika NOT (x > 5), maka ini berarti jika x tidak lebih besar dari 5.

b. OR (Alternatif): Fungsi OR digunakan untuk membandingkan dua atau lebih kondisi dan menghasilkan nilai TRUE jika salah satu atau lebih kondisi tersebut terpenuhi. Jika semua kondisi bernilai FALSE, maka hasilnya akan menjadi FALSE.

Contoh penggunaan: Jika x > 5 OR y < 10, maka ini berarti jika x lebih besar dari 5 atau y lebih kecil dari 10.

c. IF (Jika): Fungsi IF digunakan untuk mengevaluasi kondisi tertentu, dan jika kondisi tersebut terpenuhi (nilai TRUE), maka pernyataan pertama akan dijalankan, jika tidak (nilai FALSE), maka pernyataan kedua atau yang berikutnya akan dijalankan.

Contoh penggunaan: IF (x > 5, "Nilai x lebih besar dari 5", "Nilai x tidak lebih besar dari 5")

d. AND (Dan): Fungsi AND digunakan untuk membandingkan dua atau lebih kondisi dan menghasilkan nilai TRUE hanya jika semua kondisi tersebut terpenuhi. Jika salah satu atau lebih kondisi bernilai FALSE, maka hasilnya akan menjadi FALSE.

Contoh penggunaan: Jika x > 5 AND y < 10, maka ini berarti kondisi akan benar jika x lebih besar dari 5 dan y lebih kecil dari 10.

e. IF-OR (Jika-atau): Fungsi IF-OR adalah kombinasi dari fungsi IF dan fungsi OR. Fungsi ini digunakan untuk mengevaluasi beberapa kondisi dan menghasilkan nilai TRUE jika setidaknya satu kondisi terpenuhi, atau FALSE jika semua kondisi tidak terpenuhi.

Contoh penggunaan: IF (x > 5 OR y < 10, "Setidaknya salah satu kondisi terpenuhi", "Tidak ada kondisi yang terpenuhi")

Semoga penjelasan ini membantu dalam pemahaman mengenai fungsi-fungsi tersebut.

function detectTriangle(a,b,c,callback){

 try{

   if(typeof (a) !== 'number'){

     throw new Error("Argumen pertama harus number");

   }

   if(typeof (b) !== 'number'){

     throw new Error("Argumen kedua harus number");

   }

   if(typeof (c) !== 'number'){

     throw new Error("Argumen ketiga harus number");

   }

   if(a === b && b === c){

     callback(null,"Segitiga sama sisi");

   }else if(a === b || a === c || b === c){

     callback(null,"Segitiga sama kaki");

   }else{

     callback(null,"Segitiga sembarang");

   }

 }catch(err){

   callback(err,null);

 }

}

Pembahasan

Agar fungsi detectTriangle bisa berjalan dengan baik dan merespon error dengan tepat maka kita harus:

3. Menambahkan sebuah fungsi anonim sebagai argumen keempat saat memanggil fungsi detectTriangle, yang akan berfungsi sebagai parameter callback.
