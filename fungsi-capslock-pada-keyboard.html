17. Tombol yang berfungsi untuk membalikkan fungsi Capslock Adalah ...


TOMBOL CAPSLOCK Jawaban

Tombol yang berfungsi untuk membalikkan fungsi Caps lock adalah dengan menekan kembali tombol Caps lock.

Atau alternatif lain adalah dengan menekan tombol Shift.

Pembahasan

Caps Lock, atau singkatan dari Capitalis Lock, adalah sebuah tombol di keyboard yang memiliki fungsi untuk mengubah semua huruf yang diketik menjadi huruf besar. Tombol Caps Lock biasaya berada tepat di sebelah kiri tombol A (atau di sebelah kiri rangkaian tombol ASD), dan berada diantara tombol Tab dan Shift.

Kegunaan

Tombol Caps Lock digunakan saat kamu ingin mengetik dengan huruf besar semua. Kamu cukup menekan sekali tombol caps lock maka otomatis semua huruf yang kamu ketik akan menjadi huruf kapital. Untuk mengembalikan menjadi huruf kecil, kamu cukup menekan tombol caps lock kembali.  

Perbedaan tombol caps lock dan shift adalah penggunaannya. Untuk membuat huruf kapital dengan tombol shift, kamu perlu menekan dan menahannya. Sementara caps lock tidak perlu. Cukup tekan sekali saja. Menekannya kembali akan menonaktifkan fungsi caps lock.

Harap perhatikan penggunaan caps lock dalam pengiriman pesan. Menggunakan caps lock membuat semua huruf menjadi kapital dan biasanya penerima pesan akan merasa tersinggung. Membuat pesanmu dalam huruf kapital seolah – olah sama saja dengan kamu meninggikan nada suara kamu saat berbicara.

27. beberapa tombol keyboard seperti enter,shift,capslock,numlock,spacebar,tab dan print screen merupakan bagian dari


beberapa tombol keyboard yang sebutkan merupakan bagian dari tombol-tombol utama pada keyboard komputer. Berikut ini adalah penjelasan tentang masing-masing tombol tersebut:

Enter: Tombol Enter digunakan untuk memberi isyarat ke sistem bahwa Anda telah selesai mengetik sebuah perintah atau teks, sehingga sistem akan memproses perintah atau teks tersebut.Shift: Tombol Shift digunakan untuk mengaktifkan huruf kapital (uppercase) atau karakter khusus pada tombol-tombol lain yang terdapat di sebelah kiri tombol Shift.Caps Lock: Tombol Caps Lock digunakan untuk mengaktifkan huruf kapital (uppercase) secara permanen. Sehingga semua huruf yang diketik setelah tombol Caps Lock diaktifkan akan ditulis dengan huruf kapital.Num Lock: Tombol Num Lock digunakan untuk mengaktifkan atau menonaktifkan penggunaan tombol-tombol angka di bagian kanan keyboard. Jika tombol Num Lock diaktifkan, tombol-tombol angka tersebut akan berfungsi sebagai tombol angka, sedangkan jika tombol Num Lock dinonaktifkan, tombol-tombol tersebut akan berfungsi sebagai tombol kursor.Spacebar: Tombol Spacebar digunakan untuk memberikan jarak antar kata atau untuk memindahkan kursor ke kanan sejauh satu spasi.Tab: Tombol Tab digunakan untuk memindahkan kursor ke kanan sejauh satu tab atau untuk mengatur tingkat paragraf dalam dokumen teks.Print Screen: Tombol Print Screen digunakan untuk mengambil tangkapan layar (screenshot) dari layar komputer Anda. Hasil tangkapan layar tersebut dapat disimpan sebagai file gambar atau dicetak langsung.

Caps Lock : tombol pada keyboard untuk setting huruf yang diolah apakah kapital (huruf besar) atau nonkapital (huruf kecil). Caps Lock adalah tombol sakelar pada keyboard komputer yang menyebabkan semua huruf yang diketik dalam huruf besar hingga dinonaktifkan.

Misalnya, ketika Caps Lock diaktifkan, huruf yang diketik akan muncul “LIKE THIS.” Ketika dinonaktifkan, mereka muncul “like this.” Gambar penunjuk LED pada keyboard akan menunjukkan apakah tombol CAPs Lock diaktifkan atau tidak. Dalam contoh ini, LED biru menunjukkan bahwa Caps Lock diaktifkan.

huruf kapital

Penjelasan panjang:

Tombol Capsclok di keyboard sebenarnya berfungsi menyalakan / mematikan lampu indikator Capsclok di keyboard.

Bila lampu indikator Capsclok di keyboard berada dalam kondisi menyala, maka seluruh huruf yg diketik di keyboard jadi huruf kapital.

Selama lampu indikator Capsclok di keyboard berada dalam kondisi mati, maka seluruh huruf yg diketik di keyboard jadi huruf non kapital.

Letak tombol capslock di keyboard dapat dilihat contohnya di lampiran (dalam kotak kuning).
