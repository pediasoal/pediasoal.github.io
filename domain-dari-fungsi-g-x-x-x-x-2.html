8. Jika diketahui f(x) = 2/x²-9 dan g(x) = 2x, daerah asal (domain) fungsi (f+g)(x) adalah​


JAWABAN

Hasil dari [tex] \bold{hasil \: dari \: (f + g)(x)}[/tex] adalah [tex]\bold{ \frac{ {3x}^{3 } - 18x + 2 }{ {x}^{2} - 9} }[/tex]

PEMBAHASAN

Fungsi dari A ke B adalah relasi khusus yang memasangkan setiap anggota himpunan A dengan tepat satu anggota himpunan B.

Ada dua syarat yang harus dipenuhi agar suatu relasi menjadi suatu fungsi

Setiap anggota A mempunyai pasangan di B

Setiap anggota A dipasangkan dengan tepat satu anggota B

Daerah Asal, Daerah, dan Daerah Hasil

Daerah asal disebut juga domain yaitu himpunan yang akan dipasangkan ke himpunan lainnya Daerah kawan disebut juga kodomain yaitu himpun yang akan dipasangkan dari himpunan daerah asalDaerah hasil disebut juga range atau bayangan yaitu anggota daerah kawan yang memiliki pasangan dengan anggota daerah asal.

Nama, Notasi, dan Rumus Fungsi

Nama fungsi biasanya dinyatakan dalam huruf kecil, Misalnya f, g, h. Suatu fungsi yang memetakan x anggota himpunan ke A ke y anggota himpunan B dinotasikan

[tex] \boxed{f : x \rightarrow \: y \: \bold{ atau }\: f : x \rightarrow f(x)}[/tex]

Fungsi dapat juga dirumuskan sebagai berikut

[tex] \boxed{f(x) = ax + b} \\ \\ \bold{keterangan} \\ \bold{f = nama \: fungsi} \\ \bold{x = anggota \: daerah \: asal} \\ \bold{ax + b = bayangan \: dari \: x}[/tex]

Diketahui

[tex] \bold{f(x) = \frac{2}{ {x}^{2} - 9} } [/tex]

[tex] \bold{g(x) = 2x}[/tex]

Ditanya

[tex] \bold{hasil \: dari \: (f + g)(x)}[/tex]

Jawab

[tex] \bold{(f + g)(x) = \frac{2}{ {x}^{2} - 9 } + 2x} \\ \\ \: \: \: \: \: \: \: \: \: \: \: \: \: \: \: \: \: \: \: \: = \bold{\frac{2}{ {x}^{2} - 9} + \frac{2x( {x}^{2} - 9) }{ {x}^{2} - 9} } \\ \\ \: \: \: \: \: \: \: \: \: \: \: \: \: \: \: \: \: \: \: = \bold{ \frac{2}{ {x}^{2} - 9} + \frac{ {3x}^{3} - 18x}{ {x}^{2} - 9} } \\ \\ \: \: \: \: \: \: \: \: \: \: \: \: \: \: \: \: \: \: \: = \bold{ \frac{ {3x}^{3 } - 18x + 2 }{ {x}^{2} - 9} }[/tex]

jadi hasil dari [tex] \bold{hasil \: dari \: (f + g)(x)}[/tex] adalah [tex]\bold{ \frac{ {3x}^{3 } - 18x + 2 }{ {x}^{2} - 9} }[/tex]

19. domain fungsi bilangan asli kurang dari 100 dengan fungsi f(x)=2^x-3 dan g(x)=3^x-3


Domain fungsi adalah himpunan semua nilai input (biasanya disimbolkan dengan "x") yang dapat diterima atau digunakan dalam suatu fungsi matematika tertentu tanpa menyebabkan masalah atau pembagian oleh nol yang tidak terdefinisi. Dalam kata lain, domain adalah kumpulan semua nilai x yang membuat fungsi tersebut memiliki nilai yang terdefinisi.

Sebagai contoh, dalam fungsi matematika sederhana seperti f(x) = 2x, domainnya adalah semua bilangan real karena Anda dapat menggunakan nilai x apa pun, baik bilangan bulat, pecahan, atau bilangan desimal, dan fungsi tersebut akan memberikan nilai f(x) yang terdefinisi.

Penjelasan dengan langkah-langkah:

Diketahui:

fungsi f(x)=2^x-3fungsi g(x)=3^x-3

Ditanya:

domain fungsi bilangan asli kurang dari 100 dengan fungsi f(x)=2^x-3 dan g(x)=3^x-3?

Penyelesaian:

Untuk menentukan domain fungsi f(x) = 2^x - 3 dan g(x) = 3^x - 3 untuk bilangan asli kurang dari 100, Anda perlu memastikan bahwa nilai x memenuhi kondisi tersebut. Bilangan asli adalah bilangan bulat positif (termasuk nol), sehingga kita harus memeriksa kapan fungsi tersebut menghasilkan bilangan asli kurang dari 100.

Mari kita analisis fungsi-fungsi tersebut:

Fungsi f(x) = 2^x - 3

Untuk f(x) menghasilkan bilangan asli kurang dari 100, kita harus mencari nilai-nilai x yang memenuhi:

2^x - 3 < 100

Selanjutnya, kita bisa mencari nilai-nilai x yang memenuhi ketidaksetaraan di atas. Namun, perlu dicatat bahwa f(x) = 2^x - 3 akan menghasilkan nilai asli hanya untuk x yang cukup kecil, karena eksponensial tumbuh sangat cepat. Jadi, kita perlu menguji nilai-nilai x secara berurutan untuk menemukan domainnya.

Misalnya, kita bisa mencoba x = 0, x = 1, x = 2, dan seterusnya, hingga mendapatkan batas nilai x yang menghasilkan f(x) < 100.

Fungsi g(x) = 3^x - 3

Sama seperti sebelumnya, untuk g(x) menghasilkan bilangan asli kurang dari 100, kita harus mencari nilai-nilai x yang memenuhi:

3^x - 3 < 100

Seperti sebelumnya, kita perlu menguji nilai-nilai x secara berurutan untuk menentukan domainnya.

Penyelesaian eksak dari kedua fungsi ini akan memerlukan perhitungan yang cukup rumit, tetapi Anda dapat menggunakan metode uji nilai-nilai x secara berurutan untuk menentukan domainnya.

20. Diketahui fungsi f dengan domain bilangan real. Bila g(x)=f(x)+f(-x), tunjukkanbahwa g(x) adalah fungsi genap.​


Jawab:

Suatu fungsi dikatakan fungsi genap jika [tex]f(-x)=x[/tex] untuk setiap [tex]x[/tex] dan suatu fungsi dikatakan fungsi ganjil jika [tex]f(-x) = -f(x)[/tex] untuk setiap [tex]x[/tex].

Jika [tex]g(x)=f(x)+f(-x)[/tex],

[tex]\begin{aligned}g(-x)&=f(-x)+f(-(-x))\\&=f(-x)+f(x)\\&=g(x)\\\end{aligned}[/tex]

Karena [tex]g(-x)=g(x)[/tex], g(x) merupakan fungsi genap.


21. h(x)=2x-2 dan g(x)=√x~ Domain dari? h(x) / g(x) adalah​


Penjelasan dengan langkah-langkah:

Untuk dapat menentukan domain dari h(x) / g(x), perlu mengecek apakah ada pembagian dengan nol pada akar dari g(x).

Fungsi g(x) mengandung akar kuadrat (√x), oleh karena itu x harus lebih besar atau sama dengan nol untuk menghindari bilangan imajiner dalam fungsi. Oleh karena itu, domain dari g(x) adalah [0, ∞).

Sementara itu, fungsi h(x) adalah sebuah fungsi linier, sehingga domainnya tidak terbatas.

Karena kita ingin membagi h(x) dengan g(x), maka domain dari h(x) / g(x) akan dipengaruhi oleh domain dari g(x). Oleh karena itu, domain dari h(x) / g(x) adalah [0, ∞).

27. diketahui f(x)=√1-x dan g(x)=x². tentukan (a) domain dan range fungsi f(x). (b) domain dan range fungsi g. (c) domain FoG​


Jika diketahui fungsi  [tex]f(x) = \sqrt{1- x}[/tex] dan g(x) = x² , maka

a. domain dan range dari fungsi [tex]f(x) = \sqrt{1 - x}[/tex] adalah Df = {x| x ≤ 1, x ∈ R} dan Rf = {y| y ≥ 0, y ∈ R}.

b. domain dan range dari fungsi g(x) = x² adalah Dg = {x| x ∈ R} dan Rg = {y| y ≥ 0, y ∈ R }.

c. domain fungsi [tex]fog(x) = \sqrt{1 - x^2}[/tex] adalah [tex]D_{fog(x)}[/tex] = {x| -1 ≤ x ≤ 1, x ∈ R}.

Pembahasan

Suatu fungsi dapat dinyatakan dalam bentuk: f : x ---> f(x).

Nilai fungsi untuk setiap nilai x yang diberikan dapat dihitung dengan cara mensubstitusikan nilai x ke dalam rumus fungsi f(x).

Domain (daerah asal) fungsi adalah himpunan nilai-nilai agar suatu fungsi terdefinisi.

Daerah hasil fungsi (range) adalah himpunan nilai-nilai fungsi dari setiap anggota daerah asal (domain).

Fungsi komposisi adalah suatu penggabungan fungsi dari operasi dua jenis fungsi yaitu f(x) dan g(x) sehingga menghasilkan fungsi baru.

Operasi fungsi komposisi dinotasikan dengan "o" dibaca sebagai bundaran. Fungsi baru yang terbentuk dari fungsi f(x) dan fungsi g(x) sebagai berikut.

(fog(x)) = f(g(x)) ---> maksudnya adalah nilai x pada fungsi f(x) diganti dengan fungsi g(x) (gof(x)) = g(f(x)) ---> maksudnya adalah nilai x pada fungsi g(x) diganti dengan fungsi f(x)

Penyelesaian

diket:

[tex]f(x) = \sqrt{1 - x}[/tex]

g(x) = x²

ditanya:

a. domain dan range fungsi f(x)....?

b. domain dan range fungsi g(x)....?

c. domain fungsi fog(x)...?

jawab:

a. - mencari domain fungsi f(x)

Karena fungsi [tex]f(x) = \sqrt{1 - x}[/tex] berbentuk [tex]\sqrt{a}[/tex] , maka agar fungsi tersebut dapat terdefinisi adalah dengan syarat a ≥ 0, sehingga

1 - x ≥ 0

1 ≥ x

x ≤ 1

- mencari range fungsi f(x)

 Karena domain fungsi f(x) adalah x ≤ 1, maka range dapat ditentukan dengan memasukkan nilai x ≤ 1, yaitu 1, 0, -1, -2, -3 .... dan seterusnya.

misal

 x = 1, maka  [tex]f(1) = \sqrt{1 - 1} = 0[/tex]

 x = 0, maka  [tex]f(0) = \sqrt{1-0} = 1[/tex]

 dan seterusnya

 Karena nilai range paling kecil adalah 0, maka range dapat dituliskan sebagai berikut.

Rf = {x| x ≥ 0, x ∈ R}

b. - mencari domain fungsi g(x)

     Karena fungsi g(x) berbentuk fungsi kuadrat yaitu g(x) = x², maka domain untuk fungsi g(x) adalah semua anggota real.

  - mencari range fungsi g(x)

    Karena domain fungsi g(x) semua anggota real, maka daerah hasil (range) fungsi g(x) juga semua anggota real.

c. - mencari fungsi fog(x) lebih dulu

     fog(x) = f(g(x))

               = [tex]\sqrt{1 - x^2}[/tex]

   

    - mencari domain fungsi fog(x)

      Karena fungsi [tex]fog(x) = \sqrt{1 - x^2}[/tex] berbentuk [tex]\sqrt{a}[/tex] , maka agar fungsi tersebut dapat terdefinisi adalah dengan syarat a ≥ 0, sehingga

      1 - x² ≥ 0

      x² - 1 ≤ 0

      (x - 1)(x + 1) ≤ 0

       x - 1 = 0     atau x + 1 = 0

           x = 1                     x = -1

   Tuliskan nilai x pada garis bilangan

   ++++++++--------------+++++++++

                 -1    ≤ x ≤    1

   sehingga domain fungsi fog(x) adalah [tex]D_{fog(x)}[/tex] = {x| -1 ≤ x ≤ 1, x ∈ R}.

Kesimpulan

Jadi,

a. domain dan range dari fungsi [tex]f(x) = \sqrt{1 - x}[/tex] adalah Df = {x| x ≤ 1, x ∈ R} dan Rf = {y| y ≥ 0, y ∈ R}.

b. domain dan range dari fungsi g(x) = x² adalah Dg = {x| x ∈ R} dan Rg = {y| y ≥ 0, y ∈ R }.

c. domain fungsi [tex]fog(x) = \sqrt{1 - x^2}[/tex] adalah [tex]D_{fog(x)}[/tex] = {x| -1 ≤ x ≤ 1, x ∈ R}.
