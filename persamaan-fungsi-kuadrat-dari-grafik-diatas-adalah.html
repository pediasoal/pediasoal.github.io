Jika suatu grafik memotong sumbu-x di titik (p, 0) dan (q, 0) dan memotong sumbu-y di titik (0, r), maka

persamaan fungsi kuadrat:

y = a (x - p)(x - q)

==================================

Grafik tersebut memotong sumbu-x di titik (1, 0) dan (5, 0) dan memotong sumbu-y di titik (0, -5)

y = a (x - 1)(x - 5)

Tentukan nilai a.

y = a(x - 1)(x - 5)

-5 = a(0 - 1)(0 - 5)

-5 = a(-1)(-5)

-5 = 5a

a = -1

Persamaan Fungsi Kuadrat:

y = -1(x - 1)(x - 5)

y = -1(x² - 5x - x + 5)

y = -1(x² - 6x + 5)

y = -x² + 6x - 5

(0 , 4) , (-2 , 0) , (2 , 0)

→ untuk (0 , 4)
y = ax² + bx + c
4 = a(0)² + b(0) + c
4 = c ..... (1)

→ untuk (-2 , 0)
y = ax² + bx + c
0 = a(-2)² + b(-2) + c
0 = 4a - 2b + c .... (2)

→ untuk (2 , 0)
y = ax² + bx + c
0 = a(2)² + b(2) + c
0 = 4a + 2b + c ...... (3)

karena c = 4 , maka pers (2) & (3) menjadi :

(2). 0 = 4a - 2b + 4
2b = 4a + 4 , dibagi 2 semua
b = 2a + 2

(3). 0 = 4a + 2b + 4
0 = 4a + 2(2a + 2) + 4
0 = 4a + 4a + 4 + 4
0 = 8a + 8
-8 = 8a
-1 = a

→ b = 2a + 2
b = 2(-1) + 2
b = -2 + 2
b = 0

maka, telah diketahui :
a = -1
b = 0

31. persamaan grafik fungsi kuadrat dari gambar diatas adalahtolong besok dikumpul​


Jawaban:

-x² + 2x + 3

Penjelasan dengan langkah-langkah:

Titik Puncak (1,4)

x = -b/2a = (1,0)

y = D/-4a = (0,4)

Sumbu y = (0,3)

Maka,

Sumbu y :

f(x) = ax² + bx + c

y = ax² + bx + c

Maka x = 0

y = a(0)² + (0)x + 3

y = 0 + 0 + 3

y = 3

Didapatkan persamaan :

-x² + 2x + 3

Pembuktian :

Titik Puncak :

x = -b / 2a

= -2 / 2(-1)

= -2 / -2 = 1 (BENAR)

y = D / -4a

= b² - 4ac / -4a

= (2)² - 4 (-1)(3) / -4(-1)

= 4 + 12 / 4

= 4 (BENAR)

Sumbu Y :

y = -x² + 2x + 3

x = 0

y = -(0)² + 2(0) + 3

y = 3 (BENAR)

NOTE :

Grafik ke bawah maka ax² bertanda negatif (-)

﻿16a + 4b + c = 10 ... (ii)

﻿y = ax² + bx + c

﻿16a + 4b + c = 10

﻿4a + 2b + c = 6

﻿12a + 2b = 4

﻿6a + b = 2 ... (iv)

﻿64a + 8b + c = -6

﻿4a + 2b + c = 6

﻿_____________

﻿60a + 6b = -12

﻿10a + b = -2 ... (v)
