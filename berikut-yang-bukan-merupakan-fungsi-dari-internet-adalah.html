16. Sebutkan istilah-istilah yang ada di internetberikut dengan fungsinya?minimal 20​


Jawaban:

ISP (Internet Service Provider)

Adalah lembaga/instansi/perusahaan yang menyediakan jasa pelayanan penyedia jaringan internet.

Internet

Istilah internet adalah sejumlah besar network yang membentuk jaringan inter-koneksi yang terhubung melalui protokol TCP/IP.

Koneksi Internasional (IX)

Koneksi Internasional atau (IX) adalah koneksi keluar negeri atau server yang terletak diluar negeri seperti yahoo.com, facebook.com, dll.

Koneksi Lokal (IIX)

Koneksi lokal (IIX) adalah koneksi lokal indonesia ke server yang terletak di indonesia seperti detik.com, dll. Koneksi game online indonesia juga termasuk menggunakan bandwith lokal (IIX).

Bandwith

Bandwith merupakan suatu ukuran kemampuan sebuah jaringan dalam mentransfer data pada jangka waktu tertentu.

Upload / Unggah

Kegiatan pengiriman data (berupa file) dari komputer lokal ke sebuah situs web internet yang terhubung dalam sebuah network. Kebalikan dari kegiatan ini disebut download.

Download / unduh

adalah proses pengambilan atau transfer file dari sebuah situs web internet ke media penyimpanan yang terdapat di komputer yang dipakai oleh user.

bit

Binary digit. Satuan terkecil dalam komputasi, terdiri dari sebuah besaran yang memiliki nilai antara 0 atau 1.

Byte

Byte merupakan kumpulan beberapa bit (1 Byte = 8 bit ). Byte biasanya merepresentasikan sebuah karakter (Misalkan seperti A, ?, -, dll). Karakter ini bisa berupa huruf, angka ataupun simbol tertentu.

Network

Network adalah sekumpulan dua atau lebih sistem komputer yang digandeng dan membentuk sebuah jaringan. Internet sebenarnya adalah sebuah network dengan skala yang sangat besar.

TCP/IP

Transmission Control Protocol/Internet Protocol. Satu set protokol standar yang digunakan untuk menghubungkan jaringan komputer dan mengalamati lalu lintas dalam jaringan. Protokol ini mengatur format data yang diijinkan, penanganan kesalahan (error handling), lalu lintas pesan, dan standar komunikasi lainnya. TCP/IP harus dapat bekerja diatas segala jenis komputer, tanpa terpengaruh oleh perbedaan perangkat keras maupun sistem operasi yang digunakan.

IP Address

Alamat IP (Internet Protocol), yaitu sistem pengalamatan di network yang direpresentasikan dengan sederetan angka berupa kombinasi 4 deret bilangan antara 0 s/d 255 yang masing-masing dipisahkan oleh tanda titik (.), mulai dari 0.0.0.1 hingga 255.255.255.255.

IP Public

IP Public adalah IP address yang dapat di akses oleh pengguna internet diseluruh dunia.

IP Private (Pribadi)

Sebuah alamat IP dianggap pribadi jika nomor IP termasuk dalam salah satu rentang alamat IP untuk jaringan pribadi seperti Local Area Network (LAN).

IP Dinamik

IP address yang berubah-ubah. Untuk layanan Personal akan mendapatkan fasilitas IP Dinamik

IP Statik

IP yang tidak berubah/tetap. Untuk layanan Corporate akan mendapatkan IP Statik.
