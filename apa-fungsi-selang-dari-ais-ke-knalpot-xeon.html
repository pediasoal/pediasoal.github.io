16. 1) Sebuah kotak mengandungi 98 batang ais krim milo, 82 ais krim oren dan 76 ais krim merah. Apabila sebatang ais krim di pilih secara rawak, apakah kebarangkalian mendapat sebatang ais krim merah


Sebuah kotak mengandungi 98 batang ais krim milo, 82 ais krim oren dan 76 ais krim merah. Apabila sebatang ais krim di pilih secara rawak. Maka kebarangkalian mendapat sebatang ais krim merah adalah [tex]\frac{19}{64}[/tex].

Pembahasan

Peluang adalah harapan terjadinya suatu kejadian dengan dikuantitatifkan

rumus peluang ↓

P(A) = [tex]\frac{n(A)}{n(S)}[/tex]

Penyelesaian Soal

Diketahui:

98 batang ais krim milo,

82 batang ais krim oren dan

76 batang ais krim merah

Total batang ais krim 256

Ditanya:

Apabila sebatang ais krim di pilih secara rawak, apakah kebarangkalian mendapat sebatang ais krim merah

Jawab:

Gunakan rumus P(A) = [tex]\frac{n(A)}{n(S)}[/tex]

P(merah) = [tex]\frac{n(Merah)}{n(Total)}[/tex]

    = [tex]\frac{76}{256}[/tex]

    = [tex]\frac{76:4}{256:4}[/tex]

    = [tex]\frac{19}{64}[/tex]

17. periksalah Apakah fungsi-fungsi berikut pada selang merupakan fungsi naik atau fungsi turun​


aplikasi turunan

f(x) = 3x² - 12x + 9

fungsi naik → f'(x) > 0

fungsi turun → f'(x) < 0

f'(x) = 6x - 12

interval fungsi naik

6x - 12 > 0

6x > 12

x > 2

interval fungsi turun

6x - 12 < 0

x < 2

selang :

(0,2) → 0 < x < 2

fungsi turun

(-1,2) → -1 < x < 2

fungsi turun

(-2,0) → -2 < x < 0

fungsi turun
