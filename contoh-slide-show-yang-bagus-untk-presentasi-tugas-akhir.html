Pembahasan

Menampilkan dan menjalankan presentasi

Langkah untu menampilkan atau menjalankan file presentasi dapat dilakukan dengan mengakses ribbon Slide Show pada Microsoft Powerpoint, kemudian, pada start slide show, akan terdapat beberapa pilihan yang disesuaikan dengan kebutuhan diantaranya, From beginning, From current slide, Present online, dan Custom slide show. Sedangkan, shortcut untuk mejalankan presentasi dapat dilakukan dengan F5 atau pada beberapa laptop dapat juga dengan fn + F5.

Menjalankan file presentasi juga dapat dilakukan tanpa harus membuka file tersebut terlebih dahulu, hal ini dapat dilakukan dengan mengklik kanan file yang akan dirpesentasikan, lalu pilih Show.

Jika pengguna ingin menampilkan keseluruhan slide sekaligus, maka dapat digunakan slide sorter view yang dapat diakses di sisi bawah kanan window dimana iconnya berupa kotak sebanyak empat. Cara lain dapat dillakukan dengan mengakses ribbon view, lalu pada section presentation views, pilih Slide Sorter.

Memberikan halaman slide

Langkah dalam memberikan halaman slide dapat dilakukan dengan cara mengakses ribbon Insert, lalu pada section text, pilih Header & Footer. Lalu akan muncul small window untuk format Header & Footer, pilih tab Slide, lalu beri centang pada bagian checkbox Slide Number dan apply to all.

Read Text File

Read text file adalah file yang berupa text. Contoh dari read text file antara lain:

File dengan format .rtf (Rich text format)File dengan format .txt (pada umumnya berasal dari notepad/wordpad)
