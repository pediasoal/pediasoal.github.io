3. apa fungsi dari ESC?


Jawaban:

Tombol ESC dapat digunakan tersendiri sebagai perintah No (tidak), Stop (hentikan), Abort (membatalkan), Quit (kembali), atau Exit (keluar). Juga dapat digunakan sebagai shortcut dengan menekan tombol lainnya pada keyboard untuk menjalankan sebuah fungsi.

Penjelasan:

1. Membatalkan perintah/dialog

Tombol ESC digunakan untuk membatalkan atau menghentikan secara cepat sebuah perintah yang terlanjur diberikan. Fungsinya kira-kira sama seperti saat kita menutup sebuah kotak dialog dengan menggunakan mouse (tetikus) yakni mengklik tanda silang (x) pada pojok kanan atas di kotak dialog tersebut.

Ketika akan membuka sebuah halaman web di browser, menekan tombol ESC akan meng-cancel halaman yang hendak dibuka tersebut sehingga proses loading-nya dibatalkan. Sementara di beberapa jenis browser, menekan tombol ESC akan menghentikan proses pencarian web.

2. Membuka Start Menu

Tombol ESC bisa berfungsi sebagai shortcut untuk membuka Start Menu pada layar komputer Windows yakni dengan menekan tombol Ctrl lalu menekan ESC (Ctrl + ESC). Start Menu di pojok kiri bawah layar komputer akan terbuka dengan cepat dan pengguna dapat memilih menu-menu program yang tersedia.

3. Menjalankan Program Sebelumnya

Ada juga fungsi lain dari tombol ESC yang relatif jarang diketahui, khususnya oleh pengguna awam, yakni untuk menjalankan atau membuka kembali program yang terakhir digunakan. Maksudnya, ketika pengguna sedang membuka sebuah program dan ingin beralih ke program yang telah ditutup satu tahap sebelumnya, cukup dengan menekan tombol Alt dan ESC secara bersamaan.

4. Membuka Task Manager

Tombol ESC yang digunakan bersama dengan tombol Ctrl dan Shift (shortcut Ctrl + Shift + ESC) berfungsi untuk membuka Task Manager dengan lebih cepat dan mudah. Task Manager adalah tool yang terdapat pada sistem Windows yang fungsinya untuk mengontrol atau memberikan informasi kepada pengguna tentang aplikasi dan service apa yang sedang berjalan.

Dengan Task Manager ini seorang pengguna dapat melihat proses apa yang sedang berjalan dan seberapa besar penggunaan sumber dayanya. Pengguna bisa menghentikan suatu aplikasi yang sedang berjalan di latar belakang apabila tidak dibutuhkan. Sebab, kalaupun dibiarkan, aplikasi yang tidak dibutuhkan itu akan memberatkan kinerja komputer.

Terkadang seorang pengguna tidak sengaja menghapus tema yang menjadi penyebab layar laptop blank atau terjadi kesalahan pada file sistem sehingga tidak bisa membuka sebuah program secara normal. Maka, Task Manager bisa digunakan untuk membuka program yang dibutuhkan tersebut dengan cara menjalankan fungsi New Task (Run) melalui tab File. Ketika muncul kotak dialog, pengguna dapat mengetikkan nama program yang akan dijalankan lalu klik pilihan OK untuk membukanya.

Fungsi lain Task Manager adalah menampilkan informasi tentang hardware komputer, mengubah prioritas program, mencari lokasi file, dan melihat informasi pengguna.

5. Fungsi Pause pada Game Komputer

Hampir pada semua game komputer, tombol ESC berfungsi untuk menghentikan sementara sebuah game yang sedang berjalan atau sebagai fungsi Pause. Misalnya, ketika sedang bermain game komputer, pengguna ingin melakukan aktivitas lainnya maka pengguna tersebut menekan tombol ESC untuk menghentikan game sementara. Dengan begitu pengguna tidak perlu keluar dari game yang bisa menyebabkan dia kehilangan progres yang belum tersimpan.

6. Membuka Menu Game Komputer

Saat game komputer sedang berjalan, lalu pengguna ingin membuka pilihan menu game tersebut maka tombol ESC dapat digunakan untuk membukanya. Dengan demikian pengguna dapat mengubah pengaturan game secara lebih mudah dan ringkas. Jadi, pengguna tidak harus keluar dari game yang sedang berjalan untuk mengakses setting pada menu utama.

7. Keluar dari Game Komputer

Untuk sebagian jenis game di komputer, tombol ESC berfungsi sebagai pilihan Quit atau Exit. Misalnya saja digunakan apabila pengguna ingin keluar dari game yang berjalan untuk kembali ke menu utama.

36. apa arti dari caps lock, tab, dan esc...??


Caps Lock berfungsi untuk menjadikan seluruh huruf ketikan kapital (huruf besar)

Tab berfungsi untuk memberikan spasi atau jarak antar kata yang standar (untuk paragraf baru)

Esc adalah singkatan dari escape dalam bahasa indonesia artinya melarikan diri. Yaitu tombol pintas untuk keluar dari jendela laman ke jendela utama
