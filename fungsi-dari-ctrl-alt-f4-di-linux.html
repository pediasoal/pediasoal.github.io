Jawaban:

Tombol CTRL + C = Menyalin/menggandakan (Copy).

Tombol CTRL + X = Potong (Cut).

Tombol CTRL + V = Tempel (Paste).

Tombol CTRL + Z = Membatalkan perintah sebelumnya (Undo)

Tombol DELETE = Menghapus/mendelete  

F1 =  

- Hampir selalu digunakan sebagai bantuan tombol, hampir setiap program ini akan membantu membuka layar saat ini tombol yang ditekan.

- Windows Key + F1 akan membuka Microsoft Windows pusat bantuan dan dukungan.

- Buka Task Pane.  

F2 =  

- Dalam Windows umumnya digunakan untuk mengubah nama yang disorot atau file icon.

- Ctrl + Alt + F2 untuk membuka dokumen baru dalam Microsoft Word.

- Ctrl + F2 menampilkan pratinjau cetak jendela Microsoft Word.    

F3 =  

- Seringkali membuka fitur pencarian untuk berbagai program termasuk Microsoft Windows.

- Shift + F3 akan mengubah teks dalam Microsoft Word dari atas ke bawah kasus atau huruf

F4 =

- Terbuka menemukan jendela.

- Ulangi terakhir dilakukan tindakan (Word 2000 +)

- Alt + F4 akan menutup program sedang aktif di Microsoft Windows.

- Ctrl + F4 akan menutup jendela yang terbuka di jendela yang sedang aktif dalam Microsoft Windows.    

F5 =  

-Dalam semua modern browser Internet akan menekan F5 refresh atau reload halaman atau dokumen jendela.

- Buka menemukan, mengganti, dan pergi ke jendela Microsoft Word.

- Mulai tayangan slide dalam PowerPoint.  

 

Ctrl+Shift+Z = Redo (kebalikan dari Undo)  

Ctrl+Shift+N = Membuat folder baru pada windows explorer.    

Ctrl+A = Memilih semua teks  

Alt+Enter = Menampilkan properti dari objek yang dipilih

Alt+F4 = Menutup item aktif, atau keluar dari program aktif

Alt+Enter = Menampilkan properti dari objek yang dipilih  

Alt+Spacebar = Buka menu shortcut untuk jendela aktif  

Ctrl+F4 = menutup program sedang aktif di Microsoft Windows

Alt+Tab = berpindah antara satu aplikasi ke aplikasi lain  

Ctrl+Alt+Tab = berfungsi untuk Move forward melalui tab  

Alt+Esc = Cycle melalui item dalam urutan yang telah dibuka

Shift+F10 = menunjukan icon yang diseret, file / link internet.    

Ctrl+Esc = berfungsi untuk menampilkan menu Start

8. Apa fungsi tombol berikut ini A) Ctrl+Alt+Del b) F12 c) Alt+F4 d) F5 e) Capslock f) reset


Jawaban:

A) tombol kombinasi yg berfungsi untuk menampilkan jendela Task Manager

b) tombol yg berfungsi untuk menampilkan kotak dialog Save As

c) tombol kombinasi yg berfungsi untuk menutup aplikasi aktif.

d) tombol yg berfungsi untuk melakukan refresh tampilan desktop

e) tombol yg berfungsi untuk mengaktifkan/ menonaktifkan penulisan huruf kapital pada keyboard

f) tombol yg berfungsi untuk melakukan warm boot pada komputer.

Penjelasan:

A) tombol kombinasi ini lebih sering ditekan pada komputer dengan sistem operasi Windows.

b) tombol ini lebih tepat ditekan pada saat mengaktifkan program office, seperti: Ms. Office Word, Ms. Office Excel, Ms. Office Powerpoint, dll.

c) misalkan aplikasi Ms. Office Word sedang aktif, maka tombol kombinasi ini dapat ditekan untuk menutupnya.

d) tombol ini lebih tepat ditekan dalam sistem operasi Windows saat monitor sedang menampilkan desktop saja (belum ada aplikasi yg aktif)

e) selain fungsi di jawaban yg sudah disebutkan, tombol ini juga digunakan untuk menyalakan dan mematikan lampu indikator Capslock di keyboard.

f) tombol ini dapat ditekan saat komputer mengalami hang (macet).

Penjelasan tambahan:

Warm booting ialah proses booting komputer saat komputer sudah dalam keadaan menyala. Hal ini dilakukan apabila komputer mengalami hang namun masih mampu merespon tombol-tombol di keyboard.

Misalkan komputer mengalami hang namun tidak mampu merespon tombol-tombol di keyboard, maka warm booting bisa dilakukan dg menekan tombol Restart yg ada di bagian casing komputer.

18. Shortcut untuk menambahkan Sheet adalah .... a. Ctrl + B b. Shift + F11 c. d. Alt + F4 Ctrl + F2​


Jawaban:

b). Shift + F11

Penjelasan:

Shortcut Shift + F11 digunakan untuk menambahkan sheet baru dalam program spreadsheet seperti Microsoft Excel atau Google Sheets. Ketika Anda menggunakan shortcut ini, program akan membuat sebuah sheet baru di sebelah kiri sheet aktif Anda.

Shortcut ini sangat berguna ketika Anda ingin dengan cepat menambahkan sheet baru tanpa harus menggunakan menu atau ikon tambahan. Anda dapat langsung menekan kombinasi tombol Shift + F11 dan sheet baru akan langsung ditambahkan ke dalam workbook Anda.

Namun, penting untuk dicatat bahwa shortcut ini mungkin berbeda tergantung pada program spreadsheet yang Anda gunakan. Oleh karena itu, pastikan untuk memeriksa panduan atau dokumentasi program yang Anda gunakan jika shortcut ini tidak berfungsi untuk Anda.
