9. buatlah grafik fungsi x²+x-12=0 tolong dijawab kak bukan gambar nya aja tapi cara nya jugak ​


Grafik Fungsi Kuadrat Dapat Di Gambarkan Melalui Nilai Diskriminan.

Grafik Fungsi Kuadrat :

→ x² + x - 12 = 0

a = 1 , b = 1 , c = - 12

Karena a > 0 Maka Grafiknya akan Terbuka Keatas.

Nilai Diskriminan :

D = b² – 4.a.c

D = 1² – 4.- 12.1

D = 1 + 48

D = 49

Karena D > 0 Maka Grafiknya Memotong Sumbu X

Penjelasan dengan langkah-langkah:

Fungsi persamaan kuadrat y =x² + x - 12

Pertama, cari titik potong sumbu x, (nilai y = 0)

[tex]y = {x}^{2} + x - 12 \\ 0 = {x}^{2} + x - 12 \\ = (x + 4)(x - 3)[/tex]

[tex]x1 = x + 4 = 0 \\ x = - 4[/tex]

[tex]x2 = x - 3 \\ x = 3[/tex]

Maka titik potong terhadap sumbu x adalah, (-4,0) dan (3,0)

Kemudian cari titik potong sumbu y (nilai x = 0)

[tex]y = {x}^{2} + x - 12 \\ y = {0}^{2} + 0 - 12 \\ y = - 12[/tex]

Maka, titik potong terhadap sumbu y adalah (-12,0)

Kemudian cari titik puncak nya

[tex]xp = \frac{ - b}{2a} \\ = \frac{ - 1}{2 \times 1} \\ = \frac{ - 1}{2} \\ = - 0.5[/tex]

[tex]yp = \frac{ {b}^{2} - 4 ac}{4a} \\ = \frac{{1}^{2} - 4 \times 1 \times - 12}{4 \times 1} \\ = - \frac{1 + 48}{4} \\ [/tex]

[tex] = - \frac{49}{4} \\ = - 12.25[/tex]

Dan kemudian gambar grafiknya...

25. Buatlah grafik fungsi kuadrat berikut ini ! F(x)=x²+x-12 (buat caranya)


Grafik fungsi kuadrat berikut ini bisa di lihat pada gambar di lampiran

F(x)=x²+x-12

PEMBAHASAN

Sebelumnya marilah kita pelajari dulu prinsip-prinsip dari fungsi kuadrat sebagai berikut ini.

Fungsi kuadrat memiliki bentuk umum y = ax² + bx + c

dimana :

x = -b/2a → persamaan sumbu simetri

y = ( b² - 4ac ) / (-4a) → nilai maksimum fungsi

Okay dengan prinsip ini kita akan mencoba selesaikan soalnya.

Diketahui fungsi kuadrat f(x)=x²+x-12

Untuk menggambarkan fungsi kuadrat ini dapat dicari beberapa titik penting sebagai berikut :

1. Titik potong dengan sumbu x

f(x)=x²+x-12=0

(x+4)(x-3)=0

x = -4 atau x = 3

Titik potong dengan sumbu x adalah di ( -4,0 ) dan ( 3,0 )

2. Titik potong dengan sumbu y

f(x) = x²+x-12

f(0) =  0²+0-12

f(0) = -12

Titik potong dengan sumbu y adalah di (0,-12)

3. Titik balik

x = -b/2a = -1 / 2(1) = -1/2 = -0,5

y = ( b² - 4ac ) / (-4a) = ( 1² - 4(1)(-12) ) / (-4(1)) = -12,25

Titik Balik Minimum adalah di ( -0,5 , -12,25 )

Selanjutnya kita gambarkan titik-titik penting ini di koordinat kartesius dan akan di dapatkan gambar seperti pada lampiran.
