3. nilai maksimum dari fungsi y = x3+3x2-24x adalah


[tex] f(x) = x^3 + 3x^2 - 24x [/tex]

[tex] \textit{Absolute Extrema,} +\infty, -\infty [/tex]

[tex] \textit{Relative Extrema,} [/tex]
[tex] f'(x) = 3x^2 + 6x - 24 [/tex]
[tex] f'(x) = 0 [/tex]
[tex] 0 = 3(x^2+2x-8) [/tex]
[tex] 0 = 3(x-2)(x+4) [/tex]
[tex] x = 2 \vee x = -4 [/tex]

[tex]f''(x) = 6x+6 [/tex]
[tex]f''(2) = 12+6 = 18 [/tex]
[tex] \therefore x=2, \, \textit{Relative minima.} [/tex]

[tex]f''(-4) = -24+6 = -18 [/tex]
[tex] \therefore x=-4, \, \textit{Relative maxima.} [/tex]

[tex]f(-4) = (-4)^3+3\cdot 4^2 - 24 \cdot 4 = 80[/tex]
[tex] \textit{Relative maxima,} \, (-4,80) [/tex]

6. fungsi f yang dirumuskan dengan f(x) = x3 + 3x2 - 9x - 1, naik pada interval....


Fungsi f yang dirumuskan dengan f(x) = x³ + 3x² – 9x – 1, naik pada interval x < – 3 atau x > 1. Suatu fungsi f(x) dikatakan

naik jika f’(x) > 0 turun jika f’(x) < 0 memiliki titik stasioner jika f’(x) = 0

Notasi dari turunan fungsi y adalah y’ atau dy/dx. Jika y = kxⁿ maka y’ = kn xⁿ⁻¹

Pembahasan

f(x) = x³ + 3x² – 9x – 1

f’(x) = 3x² + 6x – 9

Grafik f(x) naik jika  

f’(x) > 0

3x² + 6x – 9 > 0

3(x² + 2x – 3) > 0

3(x + 3)(x – 1) > 0

x = –3 atau x = 1

Garis bilangan

+++++ (–3) – – – – (1) +++++

x < –3 atau x > 1

Jadi grafik f(x) = x³ + 3x² – 9x – 1 naik pada interval x < –3 atau x > 1

Keterangan:

Untuk menguji tanda pada garis bilangan, kita uji dengan x yang dipilih

Daerah sebelum x = –3, kita pilih x = –4

3(–4 + 3)(–4 – 1) = 3(–1)(–5) = 15 (positif)

Daerah antara x = –3 dan x = 1, kita pilih x = 0

3(0 + 3)(0 – 1) = 3(3)(–1) = –9 (negatif)

Daerah setelah x = 1, kita pilih x = 2

3(2 + 3)(2 – 1) = 3(5)(1) = 15 (positif)

11. Grafik fungsi f(x) = x3 – 3x2 - 9x + 7 naik pada interval.....​


Grafik fungsi f(x) = x³ - 3x² - 9x + 7 naik pada interval x < -1 dan x > 3.Pembahasan

Misalkan [tex]f(x) = ax^n[/tex], maka untuk menentukan turunan fungsi aljabar f(x) sebagai berikut.

[tex]f'(x) = n \times ax^{n -1}[/tex]

Aplikasi turunan juga dapat digunakan untuk menentukan naik dan turunnya suatu fungsi f(x) pada interval tertentu.

Interval naik dan turunnya suatu fungsi f(x) dapat ditentukan sebagai berikut.

Jika f'(x) > 0, untuk semua x yang berada di interval A, maka f(x) naik pada A.Jika f'(x) < 0, untuk semua x yang berada di interval A, maka f(x) turun pada A.

Penyelesaian

f(x) = x³ - 3x² - 9x + 7

f'(x) = 3x² - 6x - 9

f(x) akan naik, ketika f'(x) > 0, maka

3x² - 6x - 9 > 0

---------------------- bagi 3 (utk menyederhanakan)

x² - 2x - 3 > 0

(x - 3)(x + 1) > 0

x = 3 atau x = -1

gambar pada garis bilangan berikut.

++++++-------------++++++

 x <   -1              3 < x

Kesimpulan

Jadi, f(x) akan naik pada interval x < -1 dan x > 3.

[tex]y = {( - 4)}^{3} + 3 {( - 4)}^{2} - 24( - 4) + 12[/tex]
