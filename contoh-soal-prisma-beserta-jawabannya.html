4. contoh soal volume prisma beserta jawabannyatolong bantuan nya kakk​


Jawaban:

Sebuah prisma memiliki alas yang berbentuk belah ketupat yang panjang sisi nya adalah 10 cm dan panjang dari diagonal nya masing” iyalah 12 cm dan 16 cm. Dan jika tinggi dari prisma tersebut iyalah setinggi 20 cm, maka carilah berapa luas permukaan dari bangun prisma tersebut?

Jawaban :

Di ketahui :

Alas nya = 10 cm

Panjang diagonal = 12 cm dan 16 cm

Tinggi = 20 cm

Di tanya = luas permukaan dari prisma tersebut ?

Luas permukaan = 2 x ( Luas alas ) + ( Keliling alas x tinggi )

=2 x (½ x 12 cm x 16 cm) + (10 cm x 4 x 20 cm)

= 2 x ( 96 cm + 40 xm x 20 cm )

= 192 cm + 800 cm

= 922 cm2

Jadi, luas permukaan dari bangun prisma tersebut iyalah 922 cm2

Rumus Luas Permukaan Limas Segi Empat

Untuk mencari permukaan limas bisa memakai persamaan dibawah ini.

L = Luas alas + luas selimut

Jika alas limas berbentuk persegi, maka persamaan menjadi

L = ( s x s ) + ( 4 x 1/2 x s x t. selimut )

Jika alas memiliki bentuk persegi panjang maka persamaan yaitu

L = ( p x l ) + ( 2 x 1/2 x p x t.selimut ) + ( 2 x 1/2 x l x t.selimut )

= 4 x 10 x 25

= 40 x 25

= 1.000 cm²

Luas limas segiempat

soal:

sebuah prisma segitiga memiliki tinggi 15 cm, dengan alas 10 cm, dan tinggi 8 cm , tentukan volumenya?

jawaban:

Diketahui:

Tinggi prisma = 15 cm

Alas segitiga = 10 cm

Tinggi segitiga = 8 cm

Ditanyakan:

Volume prisma segitiga = ….?

Jawab

Luas alas = 1/2 x alas segitiga x tinggi segitiga

Luas alas = 1/2 x 10 x 8 = 40 cm²

Volume prisma segitiga = luas alas x tinggi prisma

Volume prisma segitiga = 40 x 15 = 600

Jadi volume prisma segitiga = 600 cm³
